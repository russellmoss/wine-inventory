# How to reprint a weigh tag?

You can reprint a weigh tag from the [Fruit Intake Console](#h_01EJBJNC1JNXC0WCVZH4NNMYQ2), or from the [Harvest Calendar](#h_01EJBJNJWA491S8WG577P15APH).

![Weighmaster_Certificate_20200916.png](https://support.vintrace.com/hc/article_attachments/32328914261012)

## From the Fruit Intake Console

To reprint a weigh tag from the Fruit Intake Console:

1. Click ![Fruit_Intake_Icon_20200403.png](https://support.vintrace.com/hc/article_attachments/32328932724884) Fruit Intake from the sidebar.
2. Search for the booking.
3. Click on the booking.
4. From the Print menu, select Docket (Original).

![Printing_Weigh_Tag_from_Fruit_Intake_Console_20200916.png](https://support.vintrace.com/hc/article_attachments/32328951532308)

The Delivery Receipt Print Options window displays.

5. If there are multiple deliveries, select the delivery.
6. Select the analysis you want to include on the weigh tag.

![Delivery_Receipt_Print_Options_20200916.png](https://support.vintrace.com/hc/article_attachments/32328923212692)

7. Click OK.

## From the Harvest Calendar

To reprint the weigh tag from the Harvest Calendar:

1. Click ![Calendar_Icon_20200423.png](https://support.vintrace.com/hc/article_attachments/32328932803988) Calendar from the sidebar.
2. Select Harvest.
3. Locate on the booking.
4. Click the booking.
5. From the Print menu, select Docket (Original).

![Printing_Weigh_Tag_from_Harvest_Calendar_20200916.png](https://support.vintrace.com/hc/article_attachments/32328914296340)

The Delivery Receipt Print Options window displays.

6. If there are multiple deliveries, select the delivery.
7. Select the analysis you want to include on the weigh tag.
8. Click OK.
