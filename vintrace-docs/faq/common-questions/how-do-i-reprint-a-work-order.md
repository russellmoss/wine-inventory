# How do I reprint a work order?

To reprint a [work order](https://support.vintrace.com/hc/en-us/articles/32303315610388) from the [Job Management console](https://support.vintrace.com/hc/en-us/articles/32303318317972):

1. Do one of the following to find the work order:

- If you know the work order number, you can enter the number in the Quick Search to find the work order.
- If you don't know the work order number, click ![Jobs_Menu_20200803.png](https://support.vintrace.com/hc/article_attachments/32329260079764) Jobs in the sidebar. Click the ![Filter_20200323.png](https://support.vintrace.com/hc/article_attachments/32329279062292) filter icon to find the work order.

![Finding_Work_Order_20220314.png](https://support.vintrace.com/hc/article_attachments/32329245842964)

2. Expand the Complete list that’s displayed to the right of the work order.
3. Select Print.

![Printing_Work_Order_20220314.png](https://support.vintrace.com/hc/article_attachments/32329279126548)
