# How do I change my password?

## Changing Your Own Password

To change your own password:

1. Click the ![Profile_20200402.png](https://support.vintrace.com/hc/article_attachments/32328967687700) user icon in the upper right.
2. Select Change Password. The Change Password window displays.

![Change_Password_20210312.png](https://support.vintrace.com/hc/article_attachments/32328954342932)

3. In the Current Password field, enter your current password.
4. In the New Password and Confirm Password fields, enter the new password that you’d like to use.

Your new password should be easy enough to remember, but not so easy that another person can guess it. Your new password should:

- Be 6-8 characters long.
- Include at least one number, lowercase character, uppercase character, or special character.

5. Click Change.

## Changing Another User’s Password

NOTE: Only local system administrators and the vintrace support team can change another user’s password.

To change another user’s password:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328935134100)More Options in the sidebar.
2. From the Address Book tile, click Manage System Users.
3. Click the user whose password you want to change. The Basic System User Widget displays the user’s information.

![Update_Basic_System_User_Widget_20210312.png](https://support.vintrace.com/hc/article_attachments/32328959286932)

4. In the Password and Confirm Password fields, enter a new password for the user.
5. Click Save.
