# Version 25.12.0

**Version roll-out dates**: Tue, 2 Dec, 2025\*

\* The December release consists of minor updates to the core system only, as well as some upgrades to the Vineyard management module. The major feature and enhancement updates will resume in January, once the roll-out of the latest platform upgrades has completed (as per 25.11.2 release notes).

## General availability

**The updates in this section are generally available to all customers as long as the module in question is within the customer package scope.**

### Vineyard management

#### Calendar: Grouped tasks

New grouped tasks design and improved management options.

We’ve upgraded grouped task management in the Calendar. You can now edit any number of selected tasks, not just one-by-one or all at once, but also, for example, 3 out of 5 tasks. In addition, the design of the task management options has been updated.

![7dbfe441-6d76-4181-967b-0a5ec4ea5d7c.png](https://support.vintrace.com/hc/article_attachments/43997336350868)

**Improved grouped tasks management.**

This new feature introduces several enhancements:

- Only the task properties you check out will be updated for the selected blocks.
- When selected blocks have different values for a task property, a warning icon will indicate that the information is not consistent across all blocks.

![335ef314-e9ee-4361-9948-76ecdf3d8f64.png](https://support.vintrace.com/hc/article_attachments/43997329208468)

**Updating task properties for selected vineyard blocks.**

#### Work orders: Board view

Introducing new feature: Grouped work orders.

A new **Grouped Work Order** option is now available in the Board view. This allows you to manage specific work orders as a single group. It is useful for machinery activities such as spraying or fertilisation, where you may want to create one work order for several blocks and record all blocks at once instead of individually.

This new feature introduces several enhancements:

- **In the Work Orders module:**

  - You can now add a Work Order in the Board view as a **Grouped Work Order**.
  - Grouped Work Orders work as a single task in the Board view, making them easier to manage collectively. ![6a083f11-95ab-4db4-a86e-889d3905dc60.png](https://support.vintrace.com/hc/article_attachments/43997329208596)**Adding a grouped work order.**

![6c7210bb-c22a-4f6c-a733-17dbd83c3f00.png](https://support.vintrace.com/hc/article_attachments/43997336351508)

**Grouped work order on Work orders Board view.**

- **Grouped Work Orders on the Calendar module:**

  - Once added to the Calendar, they are automatically grouped and marked with a special icon to differentiate them from individually created tasks.
  - On the Calendar, Grouped Work Orders can be managed just like any other task and benefit from the updated grouped-task design and improved management options.

![c6403139-010e-485a-a114-ccfef3ddc8ed.png](https://support.vintrace.com/hc/article_attachments/43997329210772)

**Grouped work order on Calendar with a "grouped icon".**

- **Grouped Work Orders on the Work Tracking module:**

  - Grouped Work Orders cannot be recorded as a group, but each component task can be recorded individually. However, they are marked with a special icon to distinguish them from other work orders.

![0dc38dec-c0ca-49f3-80d6-bffafe0aaa5d.png](https://support.vintrace.com/hc/article_attachments/43997336352020)

**Grouped work order on the Work Tracking module.**

- **Grouped Work Orders on the Documentation module:**

  - The full-page Work Orders report has been updated to support Grouped Work Orders. Those with defined timing can now be printed together on a single page.

#### Vineyard properties: General

Introducing vineyard “Block type” option.

We’ve added an option to mark each vineyard block as: Mature full production vineyard, Immature bearing vineyard, Newly planted vineyard, and a Block in replanting, making it possible to distinguish between blocks at different stages of vine development and manage them accordingly.
