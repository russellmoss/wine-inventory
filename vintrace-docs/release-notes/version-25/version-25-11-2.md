# Version 25.11.2

**Version roll-out dates**: late Nov 2025 - mid Jan 2026

We’re upgrading all vintrace production environments as part of our ongoing platform uplift program. This phase of deployments includes version upgrades across vintrace software components, with no material changes to the underlying infrastructure.

### **🚨 What’s Happening 🚨**

The roll-out will be implemented in waves between late November to the middle of January 2026. Client-specific maintenance date and time will be posted on the [vintrace status page](https://vintrace.statuspage.io/ "https://vintrace.statuspage.io/"). During the maintenance window, production environments will be temporarily unavailable - up to an hour downtime per client system.

### 👉 Do you need to do anything? 👉

Please avoid using vintrace or triggering integrations during your scheduled window. No configuration changes are required — all endpoints and credentials will remain unchanged.
