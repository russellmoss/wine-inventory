# Version 25.06.1

### Contract management module\*

**Updated Features**

- The Payment Details Report has been updated to allow a selection of whether it is based on contracted fruit or actual fruit received.
  - Only the 'Received fruit' option can be selected in the short term.
  - Validation for this Payment Details Report has been enhanced to show how many of the selected contracts can have the report processed.

**Additional Improvements**

- Fixed an issue in the Payment accordion where levy values for '% of total fruit cost' types did not take into account bonuses & penalties.
- Fixed an issue where some levies in the Levy accordion did not having the Actual Fruit Cost calculated/displayed where there were a mix of blocks with and without bonuses and penalties applied.

*\* This module is currently available to selected pilot clients only*
