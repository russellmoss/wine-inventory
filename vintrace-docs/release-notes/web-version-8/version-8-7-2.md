# Version 8.7.2

# vintrace app

## Major new features

- [Backdate jobs on the app](https://vinx2-old.zendesk.com/hc/en-us/articles/360001264115)

# vintrace

## Fixes and improvements

- [Sustainability flag and reporting](https://vinx2-old.zendesk.com/hc/en-us/articles/360001303636)
- Add **Name** column to **Dispatch reports**.
- Added Tirage group name, Batch name, Building, Storage area, Tirage date, Number of bins in tirage group search.

# New vintrace

## Fixes and improvements

- Faster loading of **Dashboard** insights.
- Fixed the error in the **Vessels** page when **Saved searches** are selected.
- Fixed the error on viewing a wine that has an addition in the jobs list where the additive rate unit is **Percentage**.
