# Version 8.5.1

# vintrace

## Major new features

- [Uploading attachments and photos to product notes](https://support.vintrace.com/hc/en-us/articles/32303291025940)
- [Tagging wines](https://support.vintrace.com/hc/en-us/articles/32303337168148)

## Fixes and improvements

- Fixed the issue where reports were showing the dates in UTC timezone instead of the local timezone.
- Fixed the error on printing extraction work orders due to daylight savings time shift.
- Fixed the error on saving a sales order that had a blank sales item code when Xero integration is turned on.
- Added **Tirage** group checkbox in vessel search in **Product Treatment**.
- Fixed the issue where syncing sale to Xero with Tons converts to KG.
- Fixed the error on clearing multi line service order charges.
- Allow grape and grape metrics to be available in harvest summary emails.
- Fixed the issue where adding metrics in **Configure metrics** for Laboratory displays an error when adding a duplicate metric name but has white space on it.
- Fixed the issue where **Stock item > Size** was incorrect for Glass/Container.
- Fixed the issue where self topping does not trigger appropriate billing charges.
- Fixed the issue where printing a Sampling work order where the sample set does not have a block set displays an error.
- Fixed the issue where a work order created from a **Bulk wine search** reverses the job order.
- Fixed the issue where the lab work sheet PDF export from the **Lab console** is missing the date from the main header and on each subsequent page.
- Fixed multi topping work order print template to only hide barrels list when **Hide barrel codes** is selected.
- Terminology changes such as "Stocktake" to "Physical inventory" and "Con. note" to "Shipping reference" for US locales, "Break Barrels" to "Split Barrel Group", "Amount" to "New amount" in Measurement screen, "Modify" to "Adjust template" and  "Set up Wizard" to “Search for wines” in Multi additions screen.
