# Version 8.5.2

# vintrace

## Major new feature

- [Bulk product allocation](https://support.winery-software.com/hc/en-us/articles/360000924455)

## Fixes and improvements

- Fixed the issue where having the same additive in a multi addition operation would double the charges for client billing.
- Fixed the issue where harvest chart in Block Overview shows out of order if you have blank booking weights for one of the harvests.
- Fixed the Organisation importer to be able to update the autocode via the importer.
- Fixed the issue where searching for batch/vessel from the new vintrace pages shows the search results multiple times while typing.
- Show Designated Variety and Designated Region/Sub AVA in batch create screen even if it they are not part of the autocode policy.
