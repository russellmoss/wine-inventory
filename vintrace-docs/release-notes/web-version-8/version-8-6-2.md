# Version 8.6.2

# vintrace

## Major new feature

- [Baker's lab integration](https://support.vintrace.com/hc/en-us/articles/32303350817044)

## Fixes and improvements

- Add sales report that shows where wine was shipped from.
- Export Sales Order/Invoices individually in .CSV format with Shipping Address.
- TTB Report under reporting on multiple spirit additions in the same reporting period.
- Fixed the issue where a full transfer into empty vessel with a stop ferment that changes the product to Bonded creates no tax movements.
- Fixed the issue where the rendered date and payment due date is not set when the billing invoice is created.
- Fixed the issue where the wine storage charges can be saved multiple times for the same period.
- Fixed the issue where the stock items with inactive lots are showing up in the inventory stock report.
- Fixed the issue related to scheduling multiple equipments on equipment treatment.
- Handling winery and its respective building and storage area in Stock Item Overview.
- Fixed the issue where the reverse charges are showing up in Invoice management console.
- Fixed the issue where the Allocation button in completed operation view is shown if there was no allocation movement.
- Analysis template is non-mandatory for ferment treatments.
