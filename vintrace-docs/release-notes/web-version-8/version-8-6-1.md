# vintrace

## Major new feature

- [Print QR codes for your vessels](https://support.vintrace.com/hc/en-us/articles/32303321067412)

## Fixes and improvements

- Bulk wine search by Vineyards.
- Transfer blends are not showing the correct service charges.
- Lab chart labels should allow for more decimals if the metrics have higher precision.
- The vessel search does not load when managing the allocations and adding new vessels.
- Updated the Stock Purchase Order PDF printout to show *Deliver by* instead of *Please ship by*.
- Added *Preliminary* result support for ETS.
- Reset your password from the 'Forgot your password' option on the login screen.
