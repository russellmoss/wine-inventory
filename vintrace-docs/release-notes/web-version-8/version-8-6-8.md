# Version 8.6.8

# vintrace app

## Major new features

- [Sign and email weigh tags](https://support.vintrace.com/hc/en-us/articles/32301385936148)
- [Adding attachments and notes to work orders](https://support.vintrace.com/hc/en-us/articles/32301332764948)

# vintrace

## Fixes and improvements

- Enabled ferment option for Product treatments to be able to customise Ferment activity for the mobile app.
- Fixed the typo in the barrel search console for work order.
- Fixed the issue in **Bulk Wine Intake Report** where the column header is showing **Cost per Litre** for Imperial metric scheme.
- Fixed the issue in **Crush/Extraction Report** where the **Pressings** column was not shown when Expanded fraction detail was ticked.
- **Current Barrel Exposure Report** showing litres instead of gallons.
- Fixed the issue where **Create work order** from **Operations menu** doesn't launch the second time in new vintrace > Vessels > Bulk wine.
- Fixed the issue where composition is not pulled from the fruit Parcel/Docket # while editing an extraction work order.
- Fixed the issue where **Packaged/Manufactured report** shows reversed costs.
- Fixed the issue where Crush/press load is not linked to fruit intake when fruit intake happens first before creating the crush/press load.
- Fixed the issue where **From Vessel Start Vol** for press cycle is incorrect in **Work Detail Report**, **Transaction Summary Report**, and **Transaction API**.
- Fixed the issue where **Ullaged** column in **Bulk wine search** shows barrels that have 0 volume.

# New vintrace (All plans)

## Fixes and improvements

- Fixed the issue where sorting by Variety and Region/Sub AVA filters out wines without one set on Vessels page.
- Fixed the issue where Archive vessel dialog shows the name of previously archived vessel.
