# Version 8.7.3 

# vintrace

## Major new features

- [TTB reporting for non-grape wine](https://support.vintrace.com/hc/en-us/articles/32303290103444)

## Fixes and improvements

- Fixed the issue where the**Bin count** column in **Bulk stock report** for tiraged wine is outputting blank
- Fixed the issue where the signature file is removed before the weight tag can be generated
- Fixed the issue where users without the **Can view product notes** permission can view the **Latest note** column of the **Bulk stock report**
- Warn if making a **Booking** for a **Block** that already has a **Booking** for the current vintage/winery that has the **Last booking** flag set
- When shipping from an AP bond, the shipped from details on the BOL are based off the AP bond

# vintrace app

## Major new features

- [Reset your password from the app](https://support.vintrace.com/hc/en-us/articles/32303267044884)
- [Support for metric weightings for additions on the app](https://support.vintrace.com/hc/en-us/articles/32303349581460)
