# Fixes and improvements

- New **Third Party Weigh Tag#** field in the **Fruit intake > QA/Cost/Analysis** tab and in the intake receipt.
- Fixed the issue where inactive lot numbers were displaying in the **Stock item overview > Distribution** tab.
- Show empty tank capacity in the Tank map.
- Allow the unit price to show four decimal places in Client billing invoice screens and printed invoices.
- Fixed the issue where billing charges are not showing the correct quantity for destination additions in extraction operation.
- Added an **Owner** filter in the wine batch search in **Billing > Adhoc charges.**
- Fixed the issue where the date field is cut off in the client billing printed invoice.
- Fixed the issue where x2 and x4 units are showing in **Production > Additive unit o**f addition.
- Fixed the issue where editing a work order and completing a job does not update the job to *Completed*.
- Fixed the issue in the T**ransaction Summary** report where press cycle with the same vessel in the source and destination was showing incorrect values.
- Fixed the issue in **Client Billing > vintrace (CSV)** where the tax rate shows the calculated rate instead of the rate set in the price list.
- Booking details are now available to print in the delivery intake receipt.
- Fixed the issue where the number of barrels is not counted in the quantity field in service charges on a treatment for multi transfer operation.
