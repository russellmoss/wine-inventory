# Version 8.6.7

# vintrace app

## Major new features

- [Process fruit from different blocks together](https://support.vintrace.com/hc/en-us/articles/32303309006612)

## Fixes and improvements

- Uploading attachments to extraction operation.
- Fixed the issue where parcels are not showing up in the extraction work order in the mobile app if fruit was received after scheduling the work order for the booking.

# vintrace

## Fixes and improvements

- Fixed the error on clicking **Reset** in the **Configure 3rd party lab interface** dialog.
- Fixed the issue where exporting a very large number of barrels is throwing an error.
- Fixed the issue where one-to-many and many-to-one transfers are not prompting for pre-operation losses.
- Fixed the issue where press cycle from app does not record a gain and will keep on showing the confirmation dialog.
