# Version 8.6.5

# vintrace app

## Major new feature

- [Press must operation](https://support.vintrace.com/hc/en-us/articles/32303320655892)

# vintrace

## Fixes and improvements

- Fixed the error on creating a new installment item.
- Updated trial blend to allow all system users to edit the name.
- Updated Product > Composition tab should show the vintage description and not the year for non-vintage.
- Fixed the issue of showing blank blend pie chart on a trial blend.
- Fixed the issue where packaging addition costs do not show up in the Packaged stock report.
- Added treatment tags within transfer list for Multi Topping and One to Many Transfer custom print templates and within pressings list for Press Cycle custom print template.
- Added dip measurement tags for Transfer/Rack/Blend, Bulk dispatch, and Bulk intake custom print templates.

# New vintrace

## Fixes and improvements

- Added Region/Sub AVA as a column in Vessels page
- Fixed the issue where clicking on drop down border does not expand drop down.
