# Version 8.6.6

# vintrace

## Fixes and improvements

- Submitting job from app sets the date and time on the operation in the web app.
- Fixed the issue where cost is not recalculated after Cost Allocation reversal.
- Fixed the auto login function.
- Fixed the issue where the barrel scanner shows the old wine colour and not the new beverage type.
- Fixed the issue where CSV of Bulk Stock Report doesn't show the Designated Variety, Region and Vintage without Program.

# New vintrace

## Fixes and improvements

- New filter for showing archived vessels in Vessels page.
- **Unarchive** vessels from Vessels page.
- Fixed the issue where duplicating Blocks in Vineyard View saves the **Add another** checkbox.
- Fixed the issue when creating a new block or editing a block with the same Vineyard that was used for a different Grower, the system will display a message that the Vineyard already exists and will load in the Grower for the existing Vineyard.
- Vessels page > Download > **Type** column is split to show **Vessel type** and **Wine type**.
