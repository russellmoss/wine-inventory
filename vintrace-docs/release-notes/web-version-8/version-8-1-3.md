# Additional fixes and improvements

- Fixed the overlapping icon in the **Tank Map**.
- Fixed the issue where SWNZ# is appended to Vineyard in the composition groupings for NZ.
- Fixed the **Contracted** and **Balance** values on **Grower contracts** > **Export as CSV**.
- Fixed the issue where photo and file upload was not working on a fruit intake correction.
- Fixed the issue where the **Edit work order** button in **View work order** has a white icon on a white background.
- Fixed the Vend sync error when there are positive and negative priced items on the same sale.
- Fixed the issue where the user is unable to select a custom template on creating a work order.
- Added the option to be able to pre-enter target start/end dips on scheduling work orders regardless if the tank is empty at the time of scheduling.
- Fixed the issue where clearing a tank leaves the batch code in place.
- Fixed discrepancy in Operations throughput when there is no owner selected.
- Fixed the issue where journal sync is sometimes picking up old costs.
- Fixed the “Product not found” error on the mobile app on selecting a wine from the app.

# vintrace Beta

## Major new features

- Added the **Operations** menu to the **Products** page.
- Added quick filters such as **In use/Empty** and **Tanks/Barrels/Bins** to the **Vessels** page.

## Additional fixes and improvements

- Added an indicator for demo databases.
- Show the filter panel by default for bigger screen resolutions in the **Job** and **Harvest** calendars, and in the **Job Management** page.
- Added an information dialog upon clicking the icons for pages that are coming soon.
- Changed the default columns in the **Vessels** page to **Type**, **Vessel**, **Batch**, **Volume**, **Year**, **Variety**, **Product state**, and **Location**.

---

# vintrace App fixes and improvements

- After adding a lab result for a product or a block the loading animation displays correctly while the lab results list is being refreshed.
- Some of the mandatory fields in a booking or a fruit receival didn’t clear the red highlight even after adding a value to them which is now fixed.
- Fixed an issue which caused the lock icon to incorrectly appear in some of the fields in work order jobs.
- Fixed a rare crash caused by selecting the checkmark in some of the dropdown fields when there were no options to select from.
