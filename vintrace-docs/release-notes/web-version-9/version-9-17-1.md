# Version 9.17.1

# Additional Fixes and Improvements

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

- Added the vessel forecasting functionality to the Bin Transfer operations
- Updated the allocations importer to accept all vessel/batch values regardless of if they are shown in the allocations matrix
- Fixed an issue with small and negative volumes on product allocations
- Fixed an issue that affected metric filters on the vessels page
- Fixed a rounding issue when moving additives
- Fixed an issue on the allocations matrix to filter out allocations updated to '0'
