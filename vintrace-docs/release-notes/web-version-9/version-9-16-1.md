# Version 9.16.1

# Additional Fixes and Improvements

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

- Added the ability to add a product to a trial blend without the "Can Add/Edit Allocation Products" permission
- Added the ability to run the Grower Contract Details report for a cost/area based contract
- Updated printed work orders for additive and measurement operations to include the entered pre and post dips and volumes
- Fixed an issue with auto-codes in Spanish
- Updated the upper limit for 'Max precision' values when creating an additive
- Fixed an issue with auto label printing not working when creating work orders from a template
- Fixed an issue that limited search results on the grower console
- Fixed an issue with vessel forecasting on transfer operation
- Fixed an issue with the layout of the Move Barrels printed work order template
- Fixed an issue causing discrepancies in the loss report after certain operations
