# Version 9.9.3

# Major New Features

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

## New Work Order Details Report

We added a new [work order details report](https://support.vintrace.com/hc/en-us/articles/32301280001812-Exporting-Jobs-Details) that groups the work orders selected on the Jobs page by category (turnover sheet).

# Additional Fixes and Improvements

- We fixed an issue where no products were exported from the Product importer when a specific winery was selected (i.e., it was only working in All Winery mode).
