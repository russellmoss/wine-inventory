# Version 9.21.1

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

# Additional Fixes and Improvements

- Added additional fields to the allocations report for description and reference
- Improved the vessel forecasting functionality on the Bulk dispatch (inter-winery) operation for full transfers
- Improved error messaging on the Bookings importer when the 'Block' field is not supplied
- Fixed an issue where the vessels listed in the Ferment tab did not match the selected analysis template
- Fixed an issue with incorrect ferment states on the Ferment Detail report
- Fixed an issue causing errors when running the System Audit report
- Removed the ability to receive fruit against an inactive booking
- Fixed an issue preventing the ability to complete or save a Receive stock operation
