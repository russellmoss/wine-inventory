# Version 9.0.1

# Major new features

- The new vintrace experience. The team has been working super hard shaping our current application and completely overhauling the look & feel for a more intuitive & user friendly experience. [More information](https://support.vintrace.com/hc/en-us/articles/32301370030868).

# Additional fixes and improvements

- Jobs tab is now available on the app in the bulk wine products page.
- Updated the **Add cost** in completed operation to show the operation's date and time by default.
- Fixed the issue where viewing a product after packaging in the new UI shows an error.
- Fixed the issue where the email report options displays the incorrect text.
- Fixed the issue when the Vessels page is still loading and selecting an action shows an error.
- Fixed the issue when saving a sales order as Approved without setting the invoice date shows an error.
