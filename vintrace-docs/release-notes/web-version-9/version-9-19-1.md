# Version 9.19.1

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

# Major New Features

## Allocations Enhancements

- Updated the Allocations Matrix and Products page to include the ability to use multiple filters and then 'search'
- Added an extra column to a product's history to link to the relevant job
- Updated the allocations export file with additional columns to reflect the matrix
- Updated the permission required for the allocations export file
- Added the ability to change the status of a product to 'archived'
- Added an additional field to a product for 'Reference'

# Additional Fixes and Improvements

- Fixed some inconsistencies with the heart icon
- Updated the heading on the stock allocation table from 'Sales sub AVA' to 'Sales Region'
- Fixed an issue in the trial blends console when populating batches using the 'From search'
- Fixed an issue affecting the displayed loss in the Operation Throughput report
