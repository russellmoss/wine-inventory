# Version 9.20.1

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

# Major New Features

## Allocations Enhancements

- Added separate filters for 'Product owner' and 'Wine owner'
- Added an additional row on the allocations matrix to show the 'Produced' value
- Added the ability to search within the 'Product allocations' filter on the allocations matrix by using a wildcard search
- Updated the product allocation importer to support importing using item codes

# Additional Fixes and Improvements

- Added an additional field to the Fruit Intake API for an external weigh tag
- Added the vessel forecasting functionality to the multi topping operation
- Improved the vessel forecasting functionality on the transfer/racking/blending operation for full transfers
- Addressed a performance issue when showing full history for historic wines
- Replaced 'Fruit Intake Number' with 'Booking Number' in multiple places
