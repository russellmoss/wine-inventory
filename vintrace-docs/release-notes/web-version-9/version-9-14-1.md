# Version 9.14.1

# Additional Fixes and Improvements

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

- We added a What's New link to the profile menu that enables you to view the vintrace release notes.
- We fixed an issue where the Fruit Arrival Date filters in the Grower Contract Console used the Expected Fruit Arrival Date instead of the Earliest Fruit Receival Date and Latest Fruit Receival Date found in the Harvest tab of Grower Contract.
- We fixed an issue where the [Work Order Job Details report (i.e., turnover sheet)](https://support.vintrace.com/hc/en-us/articles/32301280001812-Exporting-Jobs-Details) did not include the From Vessels and From Batches for multi-addition jobs.
- We fixed an issue that prevented the Configure Dashboard from opening in vintrace's legacy user interface.
- We fixed an issue where clearing the last line of some operations caused an error.
- We fixed an issue where setting the number of installments caused an error.
- We fixed an issue that caused the decimal precision in the 702 report to display incorrectly in Part IV.
- We fixed an issue that prevented the 702 report from displaying multiple materials in the same tax class in Part VI.
- We added support to receive the Juice tax glass for non-grape into a tank with water.
- We added bulk intake, packaging, and change batch operations to vessel forecasting.
- We added the Laboratory filter to the Analysis History window.
- We added the Bond filter to the [Wine Production Loss as CSV report](https://support.vintrace.com/hc/en-us/articles/5709688817423) for US locales. The Bond column is only available when the Show Each Loss Event checkbox is selected.
- We added sections for submitted and completed dip measurements to printed work orders.
- We updated the product importer to allow for different vintages for the same product name.
- We fixed typos in vintrace's windows and reports.
