# Version 9.0.2

# Additional fixes and improvements

- Added an expanded sidebar navigation view in the new vintrace
- Customer service number and contact options are available from the Help dialog in the new vintrace
- Classic mode and Help & Support options are available from the user profile menu in the new vintrace
- Updated the completed operation view to display the full date and time
- Vessels can now be viewed and edited on the Vessels page in the new vintrace
- Saved searches can now be added from the Vessels page in the new vintrace
- Fixed the issue where moving events are not saved when re-opening the Job and Harvest calendars
