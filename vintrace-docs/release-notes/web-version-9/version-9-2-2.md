# Version 9.2.2

# vintrace

## Major New Features

- Added the ability to [manage grape grower contributions on installments](https://support.vintrace.com/hc/en-us/articles/32301329022868) from the Grower Contract Console.

## Additional Fixes and Improvements

- Updated the Grower Contract Remittance Report from the Grower Contract Console to include the grape grower contributions.
- Fixed an issue where performing a double rollback and replacing the original vessel with an empty vessel lost the operation that it was rolling back to and impacted the costing.
- Fixed an issue where the order of the additives on the additive template was not remembered.
- Added a [Spanish work order print template](https://support.vintrace.com/hc/en-us/articles/32301319987988) for new and existing US customers.
- Added the ability to remember the "Print work order on save" setting for [work orders](https://support.vintrace.com/hc/en-us/articles/32303315610388).
- Added the Bill of Lading for the en\_GB locale.
- Fixed an issue where the Stock levels importer was creating a stock distribution without a location.
- Standardized the way that the per litre / gallon cost is calculated in the product overview.
- Fixed an issue where the Quick Search was disabled when the barcode scan button in the top bar was used.
- Fixed an issue where changing the town/suburb information from the Sales Order screen was updating the town/suburb on contacts that share the same town/suburb.
- Fixed an issue where a Refund could be approved multiple times.
