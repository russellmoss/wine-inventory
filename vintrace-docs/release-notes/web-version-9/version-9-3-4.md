# Version 9.3.4

# vintrace

## Major New Features

- We’ve added a Scale Certification placeholder text for US databases that can be checked from the Help & Support pop-up screen - accessed through the “?” in the bottom left corner of the screen.
- We've added the ability for users to have different time zones in each winery. The time zone will be displayed properly in the vintrace web application and reflected in the reports.

## Additional Fixes and Improvements

- Added an indicator on the Wine Product page for a Tirage Group if it can be combined with other groups. Users will need to have a special permission to access this.
- Added a Winery filter to the top filters of the Bulk wine list page for users of multi winery databases with the permission to switch between wineries.
- Fixed an issue in Product Allocation Matrix where users could interact with the screen while it was still loading.
