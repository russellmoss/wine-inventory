# Version 9.0.6

# vintrace

## Additional fixes and improvements

- Fixed an issue that prevented users from viewing the Ferment State, Malo State and Fraction Type for Bins on the Vessels page
- Fixed a conversion issue for non metric units on the Manufacture operation
- Fixed an issue that prevented users from viewing the lab results for Bins on the Vessels page
- Fixed an issue that prevented the import of Vend data into vintrace when the customer's email is invalid
- Fixed an issue that rounded up the Discount % for the Sales Order API when accounting integration is enabled
- Enabled excise duty for Great Britain customers

## iOS and Android app

- [Scan lab barcodes and record lab results in real time](https://support.vintrace.com/hc/en-us/articles/32301385573012)

## Additional fixes and improvements

- Fixed an issue that prevented users from scanning bins containing wine
- Fixed a rounding issue that prevented users from submitting fruit extractions
