# Version 9.30.1

# Major New Features

- A new GET Block API
  - This can be used to get the basic attributes about a block and optionally request the block's fruit placement information
- A new GET Batch API
  - This can be used to get the basic attributes about a batch and optionally request the batch's allocation information

# Additional Fixes and Improvements

- Fixed an issue on the TTB report affecting the reporting of inventory losses
- Fixed an issue with uncategorised costs and how they are populated on the bulk stock report
- Multiple typos/inconsistent terminology have been fixed
