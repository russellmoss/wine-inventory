# Version 9.11.1

# Additional Fixes and Improvements

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

- We made the following improvements to the vessel forecasting (previously referred to as future state) feature:
  - We added vessel forecasting for bulk dispatch and measurement operations.
  - We fixed an issue where the vessel volume forecasting created for the current job was included in the calculation when the job was edited.
  - We fixed an issue where jobs were not defaulting to Projected mode.
- We added new submitted dip tags for additive, multi-addition, bulk intake, and bulk dispatch operations.
- We fixed an issue where the Date field wasn't set when adding a new job to a work order.
