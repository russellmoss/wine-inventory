# Version 9.4.1

# vintrace

## Major New Features

- Added the ability to view the [days on skins](https://support.vintrace.com/hc/en-us/articles/32301304002964) for must.
- Added new columns to the Blocks page: Grading, Micro AVA/Sub Region, Intended Use, Area, Product Forecast, Earliest harvest date, Crop inspected, Default price.
- Added the ability to [edit and deactivate trial blends](https://support.vintrace.com/hc/en-us/articles/32303333476372).
- Added the ability to specify a [status for trial blends](https://support.vintrace.com/hc/en-us/articles/32303333476372) to better manage their approval process.
- Added the ability to set up an [auto-code policy](https://support.vintrace.com/hc/en-us/articles/32303292885908) for growers.
- Added the ability to search for [reversed dispatches](https://support.vintrace.com/hc/en-us/articles/32301313789460).
- Added the option to summarize the [Bulk Cost Summary report](https://support.vintrace.com/hc/en-us/articles/32301316497940) by product and group the output by stock item.
- Updated the [Wine Production Loss as CSV report’s](https://support.vintrace.com/hc/en-us/articles/32301303832084) output to include the $/gal, product code, and program code.
- Added two air compliance reports: the [Air Displacement Report](https://support.vintrace.com/hc/en-us/articles/32301303790996) and the [Ferment Detail Report](https://support.vintrace.com/hc/en-us/articles/32301313705748).

## Additional Fixes and Improvements

- Added [auto-code search in Address Book](https://support.vintrace.com/hc/en-us/articles/360001669836-Address-Book-Contacts#FilteringtheAddressBook).
- Added Product as an attribute to Batch auto-code.
- Added the ability to correct a Bulk Wine Intake (those without destination vessels) and Import Product to update the Metrics and Additions.
- Fixed the issue where sorting the Grower/Block/Vineyard on the Blocks page throws an error.
- Fixed the issue where Sales Orders are not saved because of number too low validation.
- Fixed the issue where multi-topping and self-topping results in incorrect product allocations.
- Added the ability to specify the construction material for [barrels](https://support.vintrace.com/hc/en-us/articles/32303355021588), [bins](https://support.vintrace.com/hc/en-us/articles/32301351818900), and [tanks](https://support.vintrace.com/hc/en-us/articles/32301359425428).
