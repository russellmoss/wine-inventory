# Version 9.18.1

# Additional Fixes and Improvements

Not all of the new features are enabled by default. If you would like to use any of these features, please contact our support team.

- Added the Tirage operation to the vessel forecasting functionality
- Fixed an issue with the filters when producing the Wine Production Loss report
- Fixed an issue when removing additional lines in the Bulk Dispatch operation
- Fixed an issue on the 'Crushed At' tab where locations were displaying as 'unspecified'
- Fixed an issue on the Air Displacement report where some metric results were incorrect
- Fixed an issue on the Block Assessment API when looking up brand allocations
- Added additional validations on the Fruit Intake API to ensure the booking and block fields match
- Updated the heading on the stock allocation table from 'Allocations' to 'Demand'
- Updated the vintrace logo on the login page
