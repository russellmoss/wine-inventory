# Version 9.2.3

# vintrace

## Major New Features

- [Improved the workflows for lab entry](https://support.vintrace.com/hc/en-us/articles/4404926419343).

## Additional Fixes and Improvements

- Fixed an issue where there was an error when an unattached analysis was saved while a wine was viewed in the Product Overview that longer exists.
- Fixed an issue where the Extraction and Press Cycle operations on the web still included the inactive Crushers and/or Presses in the drop down lists.
- Fixed an issue where the Sales order API ignored the override tax rate for the item, if set on the price list.
- Fixed an issue where there was an error when saving a Bulk Intake to the same vessel as the source and updating the destination vessel and there was a pending unattached analysis for destination.
- Improved the [mobile app so that it includes the equipment list for equipment treatment jobs](https://support.vintrace.com/hc/en-us/articles/32301315087892). Any discrepancies between the scheduled work order and the submitted work order display when you [complete the equipment treatment job](https://support.vintrace.com/hc/en-us/articles/32301315044500).
