# Version 9.3.6

# vintrace

## Major New Features

- Added [Okta](https://support.vintrace.com/hc/en-us/articles/32303289782676) as one of the sign-on options for vintrace.
- Added the option to [enable/disable different sign-on methods](https://support.vintrace.com/hc/en-us/articles/32303266949396) for vintrace.
- Added the [*Can Add/Edit Vessels* permission](https://support.vintrace.com/hc/en-us/articles/360000813755#Permissions) to restrict users from creating and editing vessels. This permission is added to new and existing users by default.
- Added the [*Can View Product Allocations* permission](https://support.vintrace.com/hc/en-us/articles/360000813755#Permissions) to manage who can view the Product Allocations page and Allocations report.
- Added the workflow to [return shipped juice back to the fermenter](https://support.vintrace.com/hc/en-us/articles/32301334239636).
- Added the [water addition percentage of the total resulting volume in vessel to the Adds tab of a wine](https://support.vintrace.com/hc/en-us/articles/32301314001940).

## Additional Fixes and Improvements

- Fixed the issue where multi work order templates did not schedule the work orders according to the the time setting.
- Fixed the issue where sorting of barrels in barrel groups was not consistent.
- Fixed the issue where the increase in the volume of wine also increased the recorded additive amounts.
- Fixed the issue where correcting costs on the bulk intake didn't show on the Bulk intake receipt report.
- Fixed the issue where the quick search was not working for some owner logins.
