# Version 9.3.1

# vintrace

## Additional Fixes and Improvements

- We improved the search and filter functionality on the [Product Allocation page](https://vinx2-old.zendesk.com/hc/en-us/articles/4413602205327) and [Allocations Table](https://vinx2-old.zendesk.com/hc/en-us/articles/4413615914639).
- We added tiles for Appellation/Geographical Indicator and Item Code to the [Product Allocation Details page](https://vinx2-old.zendesk.com/hc/en-us/articles/4413615829135).
- We updated the available [System User permissions](https://vinx2-old.zendesk.com/hc/en-us/articles/360000813755#Permissions).
- We have added an Additive Rate Unit for gal/1000gal.
- Added the ability [to filter the Xero Account Types from a list when creating or updating an Account](https://support.vintrace.com/hc/en-us/articles/360001790036#SetupaledgeraccountwithXeroconfigured).
- Added error handling when a currency is selected for a Sales Price List that is not configured in Xero.
- Added the connected Organization name to the Xero tile after successfully [connecting vintrace to Xero](https://support.vintrace.com/hc/en-us/articles/360000813655).
- [Added the ability to manage and create API tokens from Set up](https://vinx2-old.zendesk.com/hc/en-us/articles/4407443037711).
- Fixed an issue where the Lab tab for a Wine Product displayed the results for another Metric when the Sort Order was the same for both Metrics and/or all Metrics.
- Added the ability to import Programs from Set up.
- Resolved an issue that resulted in an unhandled error when making an Addition to a Trial Blend.
- Operation confirmations when saving operations have been removed for all existing and newly created [System Users](https://support.vintrace.com/hc/en-us/articles/360000825496-Managing-System-Users#AddingaSystemUser).
- Improved the error messages for invalid date formats when [importing data](https://support.vintrace.com/hc/en-us/articles/360000825396-Importing-and-Exporting-Data).
- Fixed an issue where [importing Fruit Sample Analysis results](https://support.vintrace.com/hc/en-us/articles/4404556203151-Recording-Fruit-Sample-Analysis#UsingtheCSVFile) did not validate the dates correctly.
- Improved the error messages when editing a Barrel that required the Time In and/or Time Out tracking details.
- Fixed an issue where the Time In and/or Time Out tracking details for Barrels were mandatory for scenarios where they did not apply.
- Added the following pack sizes for inventory [case stock items](https://support.vintrace.com/hc/en-us/articles/360001541376-Setting-Up-a-Case-Stock-Item):
  - Case x8
  - Case x 10
  - Case x14
  - Case x16
  - Case x48
- Fixed an issue where the Press Cycle operation did not empty Barrels correctly.
- Fixed an issue where the top filters were not remembered when viewing a Saved Search from the Dashboard.
- Fixed an issue where smaller rates of addition [imported from the eBOL file](https://support.vintrace.com/hc/en-us/articles/360000822536-Using-an-eBOL-Electronic-Bill-of-Lading-#ImportingtheeBOLFile) were shown as blank or "null".
- Fixed an issue where the Tirage Admin operation did not adjust costs correctly for fully emptied Tirage Group Batches.
- Fixed an issue where backdating a partial Transfer after a completed source vessel Analysis resulted in incorrect volumes on the Bulk Stock Report.
- Fixed an issue where [Liquid sugar (i) of PART IV](https://vinx2-old.zendesk.com/hc/en-us/articles/360000825636/#SugarintheTTBReport) of the [TTB Report](https://vinx2-old.zendesk.com/hc/en-us/articles/360000813955) did not group on stock items correctly.
- Fixed an issue where an unhandled error occurred on the [Allocations Table](https://support.vintrace.com/hc/en-us/articles/4413615914639-Allocations-Table) when loading vessels with no Owner filter set.

# iOS and Android app

## Additional fixes and improvements

- Added the ability to search for and view wines in Bins.
- Resolved an issue where the Press Cycle operation did not empty Barrels correctly.
