# Version 1.30.0 for Android

# Major New Features

The following features were added to the vintrace mobile app for Android.

## Change Language

We added the ability to [change the language used in the vintrace mobile app](https://support.vintrace.com/hc/en-us/articles/32301265677076).

![Mobile - Changing Language.gif](https://support.vintrace.com/hc/article_attachments/32329205683092)

## Filter Work Orders by Winery

We added the ability to [filter work orders on the mobile app](https://support.vintrace.com/hc/en-us/articles/32301291247508) by winery.

![Filter Work Orders by Winery 20240326.png](https://support.vintrace.com/hc/article_attachments/32329197149204)

## Delete Notes

We added the ability to delete notes for [products](https://support.vintrace.com/hc/en-us/articles/360001892115-Product-Notes-and-Attachments#DeletingaNote), [stock items](https://support.vintrace.com/hc/en-us/articles/9428482075791-Stock-Item-Notes#DeletingaNotefromaStockItem), and [blocks](https://support.vintrace.com/hc/en-us/articles/360001526295-Block-and-Vineyard-Assessments#DeletingaNote).

![Delete Note 20240328.png](https://support.vintrace.com/hc/article_attachments/32329220985364)

## Upload Existing Photos

We added the ability to upload existing photos from the phone.

![Adding Note - Photo Options 20240328.png](https://support.vintrace.com/hc/article_attachments/32329238295316)

Before this release, you could only upload a photo taken while using the vintrace mobile app.

## Add Notes to a Barrel

We added the ability to [add a note to a specific barrel in a barrel group](https://support.vintrace.com/hc/en-us/articles/9398984703887-Barrel-Notes-and-Attachments#AddingNotesandAttachmentstoaSpecificBarrel).

## Dip Calculator

We added a [dip calculator to vintrace mobile](https://support.vintrace.com/hc/en-us/articles/32301301350548) that lets you calculate from the dip or volume, or calculate the volume between two dips.

![Dip Calculator 20240424.png](https://support.vintrace.com/hc/article_attachments/32329197167636)

# Additional Fixes and Improvements

- We fixed an issue that could cause text in a vessel's tile to be truncated.
- We fixed an issue that caused the Yield Estimate and Available to Us calculators during a block assessment to incorrectly display "T" when it should have displayed "tn".
- We fixed an issue that bypassed the screen that displays a list of jobs in a work order when there was only one job in the work order.
