# What's new - monthly view

*An overview of our latest features and improvements on vintrace platform. Additional information can be found by clicking on the release specific link.*

## **June 2026**

### [26.06.1](https://support.vintrace.com/hc/en-us/articles/49730573917332-Version-26-06-1) [Rolling Out]This is your vintrace version

### General Availability

#### Bev AI Assistant

- Your helpful Support bot is now accessible within vintrace directly
- Simply click Bev's chat icon in the top right of vintrace to open and close the prompt window

#### Other Updates

- Saved searches can now be assigned by Winery
- The Vessel and Container Equipment search now automatically remembers your vessel type filter selections from your last session
- Enter lab results faster with Analysis Operations and Templates now accepting a Metric’s Code when adding data
- Dynamic MSO2 calculation is now available on standalone Analysis Operations
- The Inventory Stock Report now allows a decimal precision to be set, ensuring better reporting accuracy on both PDF and CSV outputs
- Exporting large lists of Grower Contracts to CSV is now optimised and includes a helpful loading screen

#### Mobile App

- You can now specify Barrel fill heights on the App by choosing 'specify' while filling a barrel
- The Dip Calculator is now available on empty tanks
- The App now returns details for empty vessels when their QR code or Barcode is scanned

Read more by clicking the release version in the heading.

### Features in pilot

#### Scalehouse

- Commodity Type records - i.e. the types of goods that can be weighed - can now be maintained via Set up > Classification > Commodity Types
- The Weighmaster fields are shown/hidden based on the Set up > Defaults setting ‘Require weighmaster’

Read more by clicking the release version in the heading.

## **May 2026**

### [26.05.1](https://support.vintrace.com/hc/en-us/articles/48434754107796-Version-26-05-1) [Rollout Complete]This is your vintrace version

### General Availability

#### MSO2 (Molecular Sulphur Dioxide) calculation

- vintrace now supports automatic calculation of MSO2 within the Lab Console
- This feature is currently in phased rollout but is available upon request
- General release is scheduled for June 2026

  - Contact vintrace Support to discuss access before this time

#### Other Updates

- The Classic Product Search is now available in the Products tile of More Options
- Users can now set Calendar, Jobs, and More Options as their default start page
- Stock Adjustment operations now restrict the ability to change the 'Keep unit costs @' setting and this can only be modified from the Adjustment type default with the 'can adjust costs' permission
- Press Cycles now stop the Days on Skins counter regardless of fraction type

Read more by clicking the release version in the heading above.

### Features in pilot

#### Scalehouse

- The pilot Scalehouse feature has been expanded on to include a ‘Weigh certificate report’ which lists all weigh certificates for a selected timeframe.
- The Commodity types available for selection in the Scalehouse weigh records can now be managed via a standard Set up page
- There is now a Winery search/filter option on the Scalehouse page

Read more by clicking the release version in the heading above.

## **April 2026**

### [26.04.1](https://support.vintrace.com/hc/en-us/articles/47327661464980-Version-26-04-1) [Rollout Complete]This is your vintrace version

### General Availability

#### Reporting

- The ‘Analysis Spreadsheet’ now downloads with its correct title
- The ‘Wine/Juice Costing Report’ correctly reflects measurement units in headers
- The ‘Bulk Inventory by Allocation’ report has an ‘Allocation precision’ field to control how many decimals values in the Allocation Quantity column are displayed
- The ‘Sample Day Sheet' has been updated to include the vineyard Sub AVA / Region information
- The ‘User Transaction Audit’ report can now be generated as a CSV file

#### Other Updates

- Enhanced expiration visibility, and new ‘Best Before’ dates for stock items
- Inclusive terminology updates, i.e. updated ‘Man Hours’ to ‘Person Hours’ within Client Billing and Service Charge areas
- You can now set a default vintage for fruit maturity sampling
- The ‘Vineyard Govt Reference’ in the block importer/exporter csv file has been renamed ‘Sustainable Cert#’ to match the on screen field
- Renamed the ‘Lab labels’ to ‘Lab request’ option under the ![Plus](https://enctech.atlassian.net/gateway/api/emoji/1c16f20a-11bd-4e2a-8735-5749a2657085/atlassian-plus/path?scale=MDPI) icon when viewing a vessel’s wine overview screen to better reflect functionality
- Added a paperclip icon to indicate spray diary attachments on Block Assessments

Read more by clicking the release version in the heading above.

### Features in pilot

#### Contract management

- Optimised the Levy Costs report, resulting in much faster generation times and fewer timeouts
- Updated the ‘Instalment’ column to displays the specific instalment number (e.g. "1 of 3")
- The ‘Process Next Payment’ bulk action now provides clear notifications if a contract is excluded from a run.
- The system now disables payment buttons while data is refreshing to ensure you only perform actions on the most up-to-date information.
- Corrected contract status displays to show clear, human-readable labels
- Enhanced the translations to ensure smoother performance and better reliability across international versions of the module

#### Scalehouse

- A new ‘Scalehouse’ feature has been introduced that provides an interface for managing deliveries and dispatches that go across the winery scales.

#### Claret Integration

- A new integration has been introduced to import wine product demand data from Claret.

Read more by clicking the release version in the heading above.

## **February 2026**

### [26.02.1](https://support.vintrace.com/hc/en-us/articles/45697447322772) [Rollout Complete]This is your vintrace version

### General Availability

#### Reporting

- An issue has been fixed in the Bulk Inventory By Allocation Report where occasionally a 0 gallon row was included.

#### Managing Tags

- A new Set Up tile has been introduced to allow the maintenance of Tags. This new tile can be found under Set Up > Classification.
- On clicking ‘Configure’ a list of existing Tags will display where these can be updated, including an option to make them inactive.
- Inactive Tags will no longer be available to attach to entities throughout vintrace.
- Any currently deployed Tags will remain in situ against bulk wines, blocks etc. after being made inactive. These tags can be manually removed against the individual entities.

#### Naming Consistency

We've made some label changes to helps provide better consistency throughout vintrace.

- **Cost console relabelled:** The link to the Cost console was previously labelled ‘Cost admin’ under More options > Accounts.
- **Tax Event console (US only):** The link to the Tax Event console was previously labelled ‘Tax Class Event history’ under More options > TTB. This label change helps provide better consistency throughout vintrace.
- **Change Ownership:** The link to the Change Ownership operation was previously labelled ‘Change Owner' under Vessels > Operations.
- **Bulk Intakes:** For the US, if a bulk intake of wine had the Product type ‘Neutral condensate’ then this caused issues with the TTB report. This Product type option has been removed for US based databases.

#### Mobile App UX update

- Work Order search: Users can now search for a Work Order from the App home screen.

#### APIs

- Performance of the v7 vintrace Report API has been improved to continue to bring you a better user experience

### Features in pilot

#### Enhanced Appellation Management (US databases)

- Appellation support: Appellation data can now be assigned to vineyards, which in turn is inherited by the blocks within those vineyards.
- Linear structure: Unlike the traditional tiered hierarchy of AVAs, Sub-AVAs and Micro-AVAs, Appellations are flat in that a vineyard can have multiple Appellation attributes assigned to it. This allows for instances where a vineyard falls within multiple, sometimes overlapping, Appellations.
- Historical assignment option: Existing bulk wines with defined block sources are eligible to have Appellation data assigned to them.

#### Contract Management

- Finalise Installment Plans for individual Contracts:

  - When installments in the instalment plan modal are in a read-only state the date picker is still enabled.
  - Ability to edit installments after one payment has already been processed.
- Installment Costs Splitting:
  - Ability to define how the cost of a contract is split across installments.
  - Process payments for installments that have had costs manually split.
  - Reverse payments for installments that have had costs manually split
  - Correct cost split payment values in the Payment Details Report.

Read more by clicking the release version in the heading.

## **January 2026**

### [26.01.1](https://support.vintrace.com/hc/en-us/articles/45193312327572) [Rollout Complete]This is your vintrace version

### General Availability

#### Wine operations

- Additional performance improvements were made to the following transfer operations to continue to bring you a better user experience:

  - Extraction
  - Multi topping
  - Multi transfer (many-to-one)
  - Multi transfer (one-to-many)
  - Press cycle
  - Transfer/Rack/Blend

#### Vessels module

- An issue has been fixed that prevented the tasting sheet from being generated from the Vessels page > Actions > Tasting sheet.
- The Yield label was updated as it was incorrectly displaying the rate unit for US and metric unit customers.

#### Reporting and analytics module

- An issue was fixed for the [Air Displacement Report](https://support.vintrace.com/hc/en-us/articles/32301303790996-Air-Displacement-Report "https://support.vintrace.com/hc/en-us/articles/32301303790996-Air-Displacement-Report") that resulted in the transactions appearing on the incorrect date.

#### Vineyard management module

- Improved grouped task management in the Calendar.
- Introduced a customizable "Block type" property to allow you to track activities and properties related to non-vineyard blocks.
- Updated the font across the entire module to match vintrace font, creating a more consistent and unified visual experience across the platform.
- Improved the localization capability for reports, available vine varieties, available default vineyard activities and several other specifics across the application to allow for cross-language consistency for organizations where users may be using application in one country but in various languages.

### Features in pilot

#### Appellations

- We're continuing to expand available options for tracking wine origin:
  - A new optional column on the Vessels page that allows for viewing and filtering the wine's appellations.
  - A new column to the Wine product page > Fruit tab that allows for viewing the appellations the fruit was sourced from.
  - A new column to the [Bulk Inventory by Allocation Report](https://support.vintrace.com/hc/en-us/articles/32301304206996) that allows for viewing the appellations and percentages of the fruit that the wine was sourced from.

Read more by clicking the release version in the heading.
