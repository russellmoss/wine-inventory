# Version 1.30.0 for iOS

# Major New Features

The following features were added to the vintrace mobile app for iOS.

## Change Language

We added the ability to [change the language used in the vintrace mobile app](https://support.vintrace.com/hc/en-us/articles/32301265677076).

![Mobile - Changing Language.gif](https://support.vintrace.com/hc/article_attachments/32328760043284)

## Filter Work Orders by Winery

We added the ability to [filter work orders on the mobile app](https://support.vintrace.com/hc/en-us/articles/32301291247508) by winery.

![Filter Work Orders by Winery 20240326.png](https://support.vintrace.com/hc/article_attachments/32328786500372)

## Delete Notes

We added the ability to delete notes for [products](https://support.vintrace.com/hc/en-us/articles/360001892115-Product-Notes-and-Attachments#DeletingaNote), [stock items](https://support.vintrace.com/hc/en-us/articles/9428482075791-Stock-Item-Notes#DeletingaNotefromaStockItem), and [blocks](https://support.vintrace.com/hc/en-us/articles/360001526295-Block-and-Vineyard-Assessments#DeletingaNote).

![Delete Note 20240328.png](https://support.vintrace.com/hc/article_attachments/32328769420692)

## Upload Existing Photos

We added the ability to upload existing photos from the phone.

![Adding Note - Photo Options 20240328.png](https://support.vintrace.com/hc/article_attachments/32328755029012)

Before this release, you could only upload a photo taken while using the vintrace mobile app.
