# Version 1.31.0 for iOS

# Major New Features

The following features were added to the vintrace mobile app for iOS.

## Dip Calculator

We added a [dip calculator to vintrace mobile](https://support.vintrace.com/hc/en-us/articles/9603875030671) that lets you calculate from the dip or volume, or calculate the volume between two dips.
