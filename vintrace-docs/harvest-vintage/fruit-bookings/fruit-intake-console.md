# Fruit Intake Console

The Fruit Intake Console lets you [create new bookings, receive fruit from scheduled bookings and unplanned arrivals](https://support.vintrace.com/hc/en-us/articles/32303268370324-Managing-Fruit-Intakes-and-Fruit-Intake-Bookings), and [change the status of bookings](#Changing_Booking_Status).

## Accessing the Fruit Intake Console

To access the Fruit Intake Console, click ![Fruit_Intake_Icon_20200403.png](https://support.vintrace.com/hc/article_attachments/32329003690388) Fruit Intake in the sidebar.

By default, the Fruit Intake Console displays all bookings for the current day and current vintage. You can specify a broader date range, or filter the bookings based on different attributes such as a grower, variety, or region.

The Fruit Intake Console uses different colors to show the status of a fruit booking.

![Status_Colors_20230109.png](https://support.vintrace.com/hc/article_attachments/32329033343380)

|  |  |
| --- | --- |
| COLOR | STATUS |
| White | Scheduled booking, but no fruit received |
| Light green | Underway booking, fruit received |
| Dark green | Completed |
| Purple | Confirmed |

## Changing a Booking Status

To change a booking status:

1. From the Fruit Intake Console, click the down arrow beside Arrive.
2. Select Booking > Edit.

![Arrive_Button_-_Booking_-_Edit_20230109.png](https://support.vintrace.com/hc/article_attachments/32329003813908)

3. Select the booking status from the Booking Status list.

![Update_Scale_Booking_-_Booking_Status_20230109.png](https://support.vintrace.com/hc/article_attachments/32329012029588)

4. Click Save.

## Customising the Fruit Intake Console

To change the columns displayed in the Fruit Intake Console:

1. Click ![Wrench_Icon_20200408.png](https://support.vintrace.com/hc/article_attachments/32329021337364).
2. Click ![Pencil_Icon_20200408.png](https://support.vintrace.com/hc/article_attachments/32329021366164). The Update Booking View Configuration window displays.
3. Do any of the following:

- To add a column, select the column from the Column list.
- To remove a column, select the column from the list on the left and click Remove.
- To reorder the columns displayed, select the column from the list on the left and click the up or down arrows to change its position.

4. Click Save.

You can also customise the default booking time slots and standard working hours for the winery. This makes it easier to schedule multiple bookings over the course of a day.

To do this:

1. Click ![Wrench_Icon_20200408.png](https://support.vintrace.com/hc/article_attachments/32329021337364).
2. Click Organiser Options. The Organiser window displays.

![Organiser_20200408.png](https://support.vintrace.com/hc/article_attachments/32329033292692)

3. Specify the times.
4. To remember the Organiser’s settings for the next time, click the ![Heart_Green_Icon_20200408.png](https://support.vintrace.com/hc/article_attachments/32329021458324) icon.
5. Click Close.
