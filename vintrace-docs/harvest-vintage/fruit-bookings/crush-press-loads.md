# Crush/Press Loads

The Crush/Press Loads feature is disabled by default. To have this feature enabled, contact support.

The Crush/Press Loads feature lets you manage the crush or press of multiple blocks of fruit within one load. The feature is ideal for white-dominant producers that manage presses with complicated blends. When Crush/Press Load is selected for an extraction work order ahead of the fruit coming in, vintrace prints and displays the blocks/fruit required for the load. You can also manage loads and report on planned vs. received bins from the Crush Loads console.

For example, a sparkling wine producer might want to crush 3t of Pinot Noir with 2t of Chardonnay in one load to create your base sparkling wine. You can create separate bookings for each block or blocks of fruit; that is, you could book the arrival of the Pinot Noir as one booking, then the Chardonnay as another. More importantly, you can link the two bookings to the same crush/press load.

## Designating a Crush/Press Load for a Booking

When the Crush/Press Loads feature is enabled, you can designate the crush/press loads that the fruit is destined for when you create a booking. You can designate all of the fruit delivery to a single crush/press load, or across multiple. You may need to have multiple designations if you receive 11t of fruit, but your press can only hold 5t at a time.

When you [create a new booking](https://support.vintrace.com/hc/en-us/articles/32303268370324-Managing-Fruit-Intakes-and-Fruit-Intake-Bookings-#CreatingaNewBooking), the Create Scale Booking window includes a Crush/Press Loads tab.

![Create_Scale_Booking_-_Crush_Press_Loads_Tab_20200410.png](https://support.vintrace.com/hc/article_attachments/32328944126740)

To specify a crush/press load for the booking, click Add Line. You can search for an existing crush/press load, or [add a new one](#h_ebceb853-f2e2-4055-bfa8-aa35e3e6e9d8).

### Keeping the Crusher/Press in sync for Crush/Press Load for a Booking

When linking Crush/Press Loads to [Fruit Bookings](https://support.vintrace.com/hc/en-us/articles/32303268370324-Managing-Fruit-Intakes-and-Fruit-Intake-Bookings) you may want to ensure that the Crusher/Press assigned to each record are the same. To do this you can set the 'Sync Crush Load & Booking Crushers' [default setting](https://support.vintrace.com/hc/en-us/articles/32301350367636-Setting-Up-Defaults) to 'Yes'. ![](https://support.vintrace.com/hc/article_attachments/40496530775572)

When saving a Fruit Booking that has one or more crush/press loads assigned, if the Crusher/Press on the Crush/Press Loads do not match the Crusher/Press set on the booking then a message will be displayed:

![](https://support.vintrace.com/hc/article_attachments/40496545826068)

Similarly, if you are updating the Crush/Press Load, then a message is displayed if the Crushers/Presses do not match, but in this case you have the option to update the Fruit Booking(s)

![](https://support.vintrace.com/hc/article_attachments/40496530775956)

## Accessing the Crush/Press Loads Console

You can access the Crush/Press Loads Console from the Fruit Intake window by clicking Loads.

![Fruit_Intake_Console_-_Loads_Button_20230228.png](https://support.vintrace.com/hc/article_attachments/32328936101652)

You can also access the Crush/Press Loads console from More Options:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328936054548) More Options in the sidebar.
2. From the Harvest tile, click Crush/Press Loads.

![More_Options_-_Crush_Press_Loads_Link_20200410.png](https://support.vintrace.com/hc/article_attachments/32328936077460)

The Crush/Press Loads Console displays.
![Crush/Press Load Console](https://support.vintrace.com/hc/article_attachments/33616606017684)

From the Crush/Press Loads console you can create a crush/press load, or update an existing one.

## Creating a Crush/Press Load

There are several ways to create a crush/press load. You can create a crush/press load when you [create a new booking](#h_245afdd2-3c4b-4c2a-bd9f-d880b7cb9655). You can also do so from the [Crush/Press Loads Console](#h_25d5cb70-507c-419a-9fc8-4cfbda68b06b).

![](https://support.vintrace.com/hc/article_attachments/40496530776340)

Enter the Crusher/Press where the load will be processed, the expected date(s)/time(s), etc. Then link the Fruit Bookings to be included in the load.

- Load# - The system can automatically generate a code for the load number. To configure an automatic naming/numbering convention, contact support.

If the Sub AVAs (Regions) and/or varieties for the bookings do not match then a warning message may be displayed.

![](https://support.vintrace.com/hc/article_attachments/40496545826964)
