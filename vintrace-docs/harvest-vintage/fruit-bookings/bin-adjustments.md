# Bin Adjustments

This functionality is available starting with [vintrace 9.6.1](https://support.vintrace.com/hc/en-us/articles/8457704128655), but not enabled by default. If you would like to use this functionality, please contact our support team.

When this feature is enabled, you’ll be able to [set up bins to allow their tare to be edited](https://vinx2-old.zendesk.com/hc/en-us/articles/undefined#id) during a [fruit intake](https://vinx2-old.zendesk.com/hc/en-us/articles/undefined#editing_tare). The bin adjustment displays on the weight dockets.

## Setting Up a Bin

When this feature is enabled, you’ll be able to specify whether a bin’s tare can be edited during a fruit intake.

We recommend naming your bins so that it’s obvious that they’re adjustment bins.

To allow a bin’s tare to be edited, [set up the bin](https://support.vintrace.com/hc/en-us/articles/4402603405327) and be sure to select the Common Tare checkbox, then select the Can Adjust Tare checkbox.

![Bin Create 20231129.png](https://support.vintrace.com/hc/article_attachments/32328787279124)

The Can Adjust Tare checkbox only displays after the Common Tare checkbox is selected.

## Editing a Bin’s Tare During a Fruit Intake

When a bin’s tare can be edited, you’ll be able to edit the bin’s tare when [receiving fruit](https://support.vintrace.com/hc/en-us/articles/360000814175-Managing-Fruit-Intakes-and-Fruit-Intake-Bookings-#ReceivingFruit).

![Intake Details - Common Tare 20231130.png](https://support.vintrace.com/hc/article_attachments/32328787297428)

When a fruit intake uses a [bin that has an editable tare](https://vinx2-old.zendesk.com/hc/en-us/articles/undefined#id), the weigh scale docket includes an additional line for bin adjustments in the weights summary. The adjustment bin's weight is not included in the Common Tare value.

![Weighmaster Certificate 20231130.png](https://support.vintrace.com/hc/article_attachments/32328789757332)
