# Common Tare

When [common tare is enabled](#h_3f17719d-e346-4fd9-bac3-2e2823804332) and you’ve [added a bin with the common tare code](#h_0941611f-5c72-4789-9b1e-42262baf21c1), vintrace will automatically calculate the tare for you using the weights of individual bins that you enter.

![Intake_Details_-_Common_Tare_20200406.png](https://support.vintrace.com/hc/article_attachments/32328935991188)

## Enabling Common Tare

If you have common tare certificates for the bins and pallets that you’re receiving fruit in, you can use common tare codes in vintrace.

To enable common tare:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328936015764) Set Up in the sidebar.
2. Click General.
3. From the Defaults tile, click Configure. The Winery Setup window displays.
4. Select the Winery tab.
5. From the Fruit Delivery Mode list, select Common Tare.

![Fruit_Delivery_Mode_-_Common_Tare_20200406.png](https://support.vintrace.com/hc/article_attachments/32328929805844)

6. Click Apply.

## Adding Common Tare Codes to a Bin

To add common tare codes to a bin:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328936015764) Set Up in the sidebar.
2. Click Vessels.
3. From the Bins tile, click Configure.
4. Click New Bin. The Bin [Create] window displays.
5. Specify the details for the new bin. Be sure to do the following:

- Name the new bin using the common tare code that you want to print on your docket or weighmaster certificate.
- From the Content Type list, select *Fruit/Parcels*.
- In the Tare field, specify the weight.
- Be sure to check the Common Tare checkbox. This setting and the weight specified in the Tare field allow the bin details to be used as part of a common tare record.

![Bin_-_Create_-_Common_Tare_20200406.png](https://support.vintrace.com/hc/article_attachments/32328936003476)

6. Click Save.
