# Booking Confirmation Report

This functionality will be available to all accounts starting with [vintrace 9.6.1b01](https://support.vintrace.com/hc/en-us/articles/32303261607828).

The Booking Confirmation report lists all the bookings for the selected Grower.

![Booking_Confirmation_Report.jpg](https://support.vintrace.com/hc/article_attachments/32328592294036)

To run the Booking Confirmation report:

1. Click ![Fruit_Intake_Icon_20200403.png](https://support.vintrace.com/hc/article_attachments/32328592302996) Fruit Intake in the sidebar.
2. Specify the filters for the report.

You must specify a grower and winery for the Booking Confirmation report.

3. Click Search.
4. From the Report menu, select Booking Confirmation.

![Fruit_Intake_Console_-_Reports_-_Booking_Confirmation_20230106.png](https://support.vintrace.com/hc/article_attachments/32328579539092)

If you’d prefer to receive a PDF of the Booking Confirmation report, select Booking Confirmation (Email).
