# Configuring Instalment Dates for Grower Contracts (Classic Grower Contract Console)

This article relates to the classic Grower Contract Console. For the Contracts Management module, refer to [Managing Grower Contracts (Contracts Management)](https://support.vintrace.com/hc/en-us/articles/45962805386004).

Configuring an instalment schedule for [grower contracts](https://support.vintrace.com/hc/en-us/articles/32301319829268) ensures that instalments are created for the correct dates. You can specify up to four instalments for grower contracts. The first two instalments are required; the third and fourth instalments are optional.

To configure grower instalment dates:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328607936020) Set Up in the sidebar.
2. Click Grower, or search for *System Policy*.
3. From the System Policy tile, click Configure.
4. Click Edit Grower instalment Dates.

![Winery_Setup_-_System_Policy_-_Edit_Installment_Dates_20221220.png](https://support.vintrace.com/hc/article_attachments/32328592623636)

The Edit Grower Instalment Dates window displays.

5. For the first instalment, do one of the following:

- Specify the day of the month and the month for the first instalment. The bulk instalment action will use the existing rules for whether to pay on the contracted or received amounts.

![Edit_Grower_Installment_Dates_-_Specific_Month_20221221.png](https://support.vintrace.com/hc/article_attachments/32328623934100)

- Enter the day of the month and select *On This Day In the Month Following Fruit Receival* from the list.

![Installment_1_20221220.png](https://support.vintrace.com/hc/article_attachments/32328598740628)

If you select *On This Day In the Month Following Fruit Receival,*no installments can be made unless you've received fruit.

6. For the second instalment, enter the day of the month and select the month from the list.
7. To specify the third and fourth instalments, repeat the previous step.
8. Click Save.

## Creating an Instalment for Grower Contracts

To create an instalment for grower contracts:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328607921044) More Options in the sidebar.
2. From the Accounts tile, click Manage Grower Contracts. The Grower Contract Console displays.
3. Filter the grower contracts by the next instalment date.

![Grower_Contract_Console_20221221.png](https://support.vintrace.com/hc/article_attachments/32328623966868)

You can also hide internal growers by selecting the Hide Internal Growers checkbox.

4. To select all grower contracts listed, select the Select All/None checkbox. You can also select individual grower contracts manually by selecting the checkbox beside the grower contract.

![Grower_Contract_Console_-_Select_All_20221220.png](https://support.vintrace.com/hc/article_attachments/32328598757524)

5. From the Actions menu, select Create Next Installment.

![Grower_Contract_Console_-_Actions_Create_Next_Installment_20221220.png](https://support.vintrace.com/hc/article_attachments/32328592830740)

The Create next installment window displays.![](https://support.vintrace.com/hc/article_attachments/41807394131860)

6. To preview a CSV file of the instalments, click Preview Instalment Costs. To save the instalments, click Save.

## Exporting the Instalments to a CSV File

You can export selected instalments to a CSV file.

To export the instalments to a CSV file:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328607921044) More Options in the sidebar.
2. From the Accounts tile, click Manage Grower Contracts. The Grower Contract Console displays.
3. Filter the grower contracts by the next instalment date.
4. To select all grower contracts listed, select the Select All/None checkbox. You can also select individual grower contracts manually by selecting the checkbox beside the grower contract.
5. From the Reports menu, select Installment Details.
