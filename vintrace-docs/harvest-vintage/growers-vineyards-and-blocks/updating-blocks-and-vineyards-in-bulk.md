# Updating Blocks and Vineyards in Bulk

You can [update multiple blocks](#updating_blocks) and [vineyards](#updating_vineyards) with a spreadsheet by using vintrace's [import/export functionality](https://support.vintrace.com/hc/en-us/articles/32303307646868).

## Updating Blocks in Bulk

Updating your blocks with a CSV file will overwrite your block’s existing details with the details provided in the spreadsheet.

To update blocks in bulk:

1. [Export your blocks from vintrace.](#blocks_exporting)
2. [Update the block data in the spreadsheet.](#blocks_updating)
3. [Import the spreadsheet into vintrace.](#blocks_importing)

### Exporting Blocks from vintrace

To export the blocks:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328915398804) Set Up in the sidebar.
2. Search for *Fruit Sources*.
3. From the Fruit Sources tile, click Configure.
4. Expand the Import/Export Block list by clicking the arrow.
5. Select Export Block to CSV.

![Winery_Setup_-_Fruit_Sources_-_Export_Block_to_CSV_20220902.png](https://support.vintrace.com/hc/article_attachments/32328887138452)

6. Save the file to your computer.

You can now [update the block data in the spreadsheet](#blocks_updating).

### Updating Block Data

To update the block data in the spreadsheet:

1. Open the CSV file that you [downloaded](#blocks_exporting).
2. Edit the block data as needed.

Do NOT edit the values in the VINx2 ID column.

3. Save the file.

You can now [upload the spreadsheet to vintrace.](#blocks_importing)

### Importing Block Data

To import the block data into vintrace:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328915398804) Set Up in the sidebar.
2. Search for *Fruit Sources*.
3. From the Fruit Sources tile, click Configure.
4. Expand the Import/Export Block list by clicking the arrow.
5. Select Import Block.

![Winery_Setup_-_Fruit_Sources_-_Import_Block_20220906.png](https://support.vintrace.com/hc/article_attachments/32328893232660)

The Block Importer window displays.

6. Click the ![Upload_File_20220901.png](https://support.vintrace.com/hc/article_attachments/32328915369236)upload icon.

![Block_Importer_-_Upload_a_File_20220902.png](https://support.vintrace.com/hc/article_attachments/32328893311636)
The Upload window displays.

![Upload_20220901.png](https://support.vintrace.com/hc/article_attachments/32328893505428)

7. Click Choose File.
8. Select your spreadsheet.
9. Click Upload. The selected file’s name displays in the Block Importer window.
10. Select the Update Existing Records checkbox.

![Block_Importer_-_File_Selected_20220906.png](https://support.vintrace.com/hc/article_attachments/32328898409492)

11. Click Import Block. If the Confirm Column Header Matching checkbox was selected, you’ll be asked to confirm the mapping of your file’s columns to vintrace’s fields.

![Block_Importer_-_Column_Matching_US_20220902.png](https://support.vintrace.com/hc/article_attachments/32328893353620)

12. Click Continue. vintrace confirms the number of blocks that were imported.
13. Click Close.

## Updating Vineyards in Bulk

To update your vineyards using a spreadsheet:

1. [Export the vineyards from vintrace.](#vineyards_exporting)
2. [Update the vineyards in the spreadsheet.](#vineyards_updating)
3. [Import the spreadsheet into vintrace.](#vineyards_importing)

### Exporting Vineyards from vintrace

To export the vineyards:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328915398804) Set Up in the sidebar.
2. Search for *Fruit Sources*.
3. From the Fruit Sources tile, click Configure.
4. Expand the Import/Export Block list by clicking the arrow.
5. Select Export Vineyard to CSV.

![Winery_Setup_-_Fruit_Sources_-_Export_Vineyard_to_CSV_20220902.png](https://support.vintrace.com/hc/article_attachments/32328893495060)

6. Save the file to your computer.

You can now [update the vineyard data in the spreadsheet](#vineyards_updating).

### Updating Vineyard Data

To update the vineyard data in the spreadsheet:

1. Open the CSV file that you [downloaded](#vineyards_exporting).
2. Edit the vineyard data as needed.

Do NOT edit the values in the VINx2 ID column.

3. Save the file.

You can now [upload the spreadsheet to vintrace](#vineyards_importing).

### Importing Vineyard Data

To import the vineyard data into vintrace:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328915398804) Set Up in the sidebar.
2. Search for *Fruit Sources*.
3. From the Fruit Sources tile, click Configure.
4. Expand the Import/Export Block list by clicking the arrow.
5. Select Import Vineyard.

![Winery_Setup_-_Fruit_Sources_-_Import_Vineyard_20220902.png](https://support.vintrace.com/hc/article_attachments/32328887296404)
The Vineyard Import window displays.

6. Click the upload icon.

![Vineyard_Importer_-_Upload_a_File_20220902.png](https://support.vintrace.com/hc/article_attachments/32328893333140)
The Upload window displays.

![Upload_20220901.png](https://support.vintrace.com/hc/article_attachments/32328893505428)

7. Click Choose File.
8. Select your spreadsheet.
9. Click Upload. The selected file’s name displays in the Block Importer window.
10. Select the Update Existing Records checkbox.

![Vineyard_Importer_-_Update_Existing_-_File_Selected_20220902.png](https://support.vintrace.com/hc/article_attachments/32328898543124)

11. Click Import Vineyard. If the Confirm Column Header Matching checkbox was selected, you’ll be asked to confirm the mapping of your file’s columns to vintrace’s fields.

![Vineyard_Importer_-_Column_Matching_20220902.png](https://support.vintrace.com/hc/article_attachments/32328915597204)

12. Click Continue. vintrace confirms the number of vineyards that were imported.
13. Click Close.
