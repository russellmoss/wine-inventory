# Managing Vineyards

In vintrace, each [grower](https://support.vintrace.com/hc/en-us/articles/32301351385364) can have one or more vineyards which can have one or more [blocks](https://support.vintrace.com/hc/en-us/articles/32303262299284).

## Viewing a Grower's Vineyards

To view a [grower's](https://support.vintrace.com/hc/en-us/articles/32301351385364) vineyards:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328915667732) More Options in the sidebar.
2. From the Harvest tile, click Block Overview.
3. From the filters on the left, select the grower.
4. Click Search. The selected grower's vineyards and blocks display.

![Block_Overview_-_Search_by_Grower_20210802.png](https://support.vintrace.com/hc/article_attachments/32328887400852)

## Setting Up a Vineyard

To add a vineyard:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328915667732) More Options in the sidebar.
2. From the Harvest tile, click Block Overview.
3. From the New menu in the lower left, select New Vineyard.

![Block_Overview_-_New_-_New_Vineyard_20210802.png](https://support.vintrace.com/hc/article_attachments/32328893568660)

The Vineyard window displays.

![Vineyard_Create_20220902.png](https://support.vintrace.com/hc/article_attachments/32328887489812)

4. Specify the details for the vineyard.

You can [add a sub AVA](https://support.vintrace.com/hc/en-us/articles/5188347025423-Setting-Up-Appellations-AVAs-Sub-AVAs-and-Micro-AVAs#AddingaSubAVA) or [micro AVA](https://support.vintrace.com/hc/en-us/articles/5188347025423-Setting-Up-Appellations-AVAs-Sub-AVAs-and-Micro-AVAs#AddingaMicroAVA) from the Vineyard window.

5. Click Save.

Users with a multi-winery license can specify which vineyards are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/4413475617423) for details.

## Editing a Vineyard

To edit a vineyard:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328915667732) More Options in the sidebar.
2. From the Harvest tile, click Block Overview.
3. Use the filters on the left to search for the vineyard.
4. Select any block belonging to the vineyard.
5. Select the Vineyard/Grower tab.
6. Click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32328887451540) pencil icon beside the vineyard's name.

![Block_Overview_-_Editing_Vineyard_20220902.png](https://support.vintrace.com/hc/article_attachments/32328893633556)

7. Update the vineyard's details as needed.

You can [add a sub AVA](https://support.vintrace.com/hc/en-us/articles/5188347025423-Setting-Up-Appellations-AVAs-Sub-AVAs-and-Micro-AVAs#AddingaSubAVA) or [micro AVA](https://support.vintrace.com/hc/en-us/articles/5188347025423-Setting-Up-Appellations-AVAs-Sub-AVAs-and-Micro-AVAs#AddingaMicroAVA) from the Vineyard window.

8. Click Save.
