# Managing Blocks

vintrace uses a structure of growers, vineyards, and blocks to track your fruit sources. A [grower](https://support.vintrace.com/hc/en-us/articles/32301351385364) can have multiple [vineyards](https://support.vintrace.com/hc/en-us/articles/32301351350420) set up within the system, and each vineyard contains a selection of blocks.

![Diagram_-_Grower_Vineyard_Block_20200414.png](https://support.vintrace.com/hc/article_attachments/32328935021588)

Users with a multi-winery license can specify which blocks are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/4413475617423) for details.

## Setting Up a Block

To add a block:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328954175892) More Options in the sidebar.
2. From the Harvest tile, click Block Overview.
3. From the New menu in the lower left, select New Block.

![Block_Overview_-_New_-_New_Block_20210802.png](https://support.vintrace.com/hc/article_attachments/32328935068948)

The Block window displays.

![Block_Create_20210802.png](https://support.vintrace.com/hc/article_attachments/32328942998292)

4. Specify the details for the block.
5. Click Save.

## Editing a Block

Updating a block’s name will update all historic fruit intakes on the block and show on composition details on existing wines. If you’re updating a block to correct the region/sub AVA on a wine, be sure you’re updating past fruit intake data to allow these details to flow through.

To edit a block:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328954175892) More Options from the sidebar.
2. From the Harvest tile, click Block Overview.

![Harvest_Tile_-_Block_Overview_20200414.png](https://support.vintrace.com/hc/article_attachments/32328959128724)

1. Search for the block that you want to update.
2. Click the block.
3. To update the block’s details or correct a block’s vineyard, select the Block Details tab and click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32328959136148) pencil icon.

![Editing_Block_Details_20200414.png](https://support.vintrace.com/hc/article_attachments/32328935035924)
