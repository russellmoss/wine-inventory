# Custom Field for Block Assessments

This functionality is available starting with [vintrace 9.6.1](https://support.vintrace.com/hc/en-us/articles/8457704128655), but not enabled by default. If you would like to use this functionality, please contact our support team.

We added the ability to include an optional custom field for block assessments that can be used to record an alphanumeric value that’s relevant to your winery.

## Specifying the Custom Field’s Label

To include a custom field, you’ll need to specify the field’s label in your [system defaults](https://support.vintrace.com/hc/en-us/articles/4659960052751).

![Winery Setup - Block Assessment Value Label 20240226.png](https://support.vintrace.com/hc/article_attachments/32328745440148)

The label must be specified for the system. It cannot be specified at the winery or user level.

## Entering the Custom Field’s Value

The specified label is included in the Block Assessments window where you can specify the value.

![Create Block Assessment with Custom Field 20240226.png](https://support.vintrace.com/hc/article_attachments/32328775735956)

## Viewing the Custom Field’s Value

The custom field and its value display in the [block’s Seasonal tab](https://support.vintrace.com/hc/en-us/articles/4404549013391-Block-Overview-Window#SeasonalTab).

![Block Overview - Seasonal - Custom Field 20240226.png](https://support.vintrace.com/hc/article_attachments/32328760433300)

You can also [customize the Vineyards & Blocks page](https://support.vintrace.com/hc/en-us/articles/5648201450383-Customizing-the-Vineyards-Blocks-Page#ChangingtheColumnsDisplayed) to display the custom field as a column and [apply a filter to the column](https://support.vintrace.com/hc/en-us/articles/5648086595983-Vineyards-Blocks-Page#FilteringtheVineyards&BlocksPage).

![Vineyards and Blocks - Including Custom Field Column 20240226.png](https://support.vintrace.com/hc/article_attachments/32328760401940)
