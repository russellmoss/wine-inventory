# Processing Grower Payments (Contracts Management)

This article relates to the Contracts Management module. For the classic Grower Contract Console, refer to [Managing Grower Contract Payments and Fruit Costs (Classic Grower Contract Console)](https://support.vintrace.com/hc/en-us/articles/32303300639124). The Contract Management module is disabled by default. To have this feature enabled, contact support.

See [Managing Grower Contracts (Contracts Management)](https://support.vintrace.com/hc/en-us/articles/43988801346068) for creating instalments/payment plans for contracts.

## **Applying and Waiving Bonuses & Penalties**

The first grower payment for a vintage may be based on a fixed dollar amount, or on the expected, or ‘contracted’, fruit. Otherwise the cost of the fruit is calculated based on the actual fruit received. If any [bonus and/or penalty rules](https://support.vintrace.com/hc/en-us/articles/45962805386004) have been defined, then each rule for each intake must be applied or waived.

1. Click ![](https://support.vintrace.com/hc/article_attachments/43989505218196) Contracts in the sidebar.

The Contract page displays.

Access to this menu is only available to users with the [Can manage grower contract permission](https://support.vintrace.com/hc/en-us/articles/32303349421588).

2. Select the contract you want to process a payment for.
3. Click the ‘Value rules’ button at the top of the Payments section of the contract page.

The Value rules window displays.

![](https://support.vintrace.com/hc/article_attachments/43989516880404)

Each intake with bonus or penalty criteria that has been met can be viewed by clicking on the docket (aka weigh tag) link.

4. Enter a reason for your decision
5. Click either ‘Apply’ or ‘Waive’ as appropriate.

If the bonus or penalty has been applied then the fruit costs are updated with the cost adjustment.

Note that these cost adjustments are not reflected in the payment section of the contract page until payments have been processed.  You can run a ‘Payment Details Report’ for one or more contracts via the main contracts list page to see the most up to date expected payment values.

## **Processing Grower Payments**

The next due payment for a contract can be processed by clicking the Process next payment button at the top of the Payments section for the contract.

![](https://support.vintrace.com/hc/article_attachments/43989505219604)

The payment can be made based on actual costs of fruit received, or based on the contracted amount of fruit expected to be received.

If you want to pay out the remaining costs of the fruit, then select the ‘Last payment’ option. Any later instalments will be cancelled and all outstanding levies (fruit assessments), if any, will be deducted from this payment.

Alternatively payments for multiple contracts can be processed.

1. Go to the main Contracts page and use the search options to locate the contracts you want to process payments for.
   E.g. use the ‘Next payment’, or ‘Fruit arrival’ dates to limit the contracts listed.

![](https://support.vintrace.com/hc/article_attachments/43989505220884)

2. Select the contracts you want to process payments for
3. Select the Process next payment option from the bulk actions menu button

![](https://support.vintrace.com/hc/article_attachments/43989516885908)

4. If required a preview report of the payments can be produced by clicking the ‘Preview payment costs’ link.

![](https://support.vintrace.com/hc/article_attachments/43989505222804)

Where the last payment for the vintage is being processed you will be asked to acknowledge the message.

5. Click Process
