# Managing Growers

Growers are a [type of organization in your address book](https://support.vintrace.com/hc/en-us/articles/32301367488788) that are assigned the [*Grower* role](https://support.vintrace.com/hc/en-us/articles/32303349421588). Each grower can be associated with multiple vineyards.

## Viewing and Editing Growers

To view growers in vintrace:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329200231956) More Options in the sidebar.
2. From the Address Book tile, click Manage Growers. The address book displays growers.
3. To edit a grower, click the grower.

## Setting Up a Grower

To add a grower:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329200231956) More Options in the sidebar.
2. From the Address Book tile, click Manage Growers.
3. From the Add menu in the lower left, select *Organization*.

![Address_Book_-_Manage_Growers_-_Add_Organization_20210802.png](https://support.vintrace.com/hc/article_attachments/32329229321236)

If you want to assign an individual the Grower role, you can select *Individual* from the Add menu.

The Basic Organization Widget window displays with the *Grower* role selected.

![Create_Basic_Organization_Widget_-_Grower_Role_20210802.png](https://support.vintrace.com/hc/article_attachments/32329241528724)

4. Specify the details for the grower.
5. If your winery uses a unique identifier for the grower, you can enter it by doing the following:

- Click Edit beside the Grower role.

![Basic_Organization_Widget_-_Grower_-_Edit_20221011.png](https://support.vintrace.com/hc/article_attachments/32329213222420)

The Configure Grower Settings window displays.

- Enter the grower's identifier in the Auto Code field.

![Configure_Grower_Settings_-_Grower_ID_20221011.png](https://support.vintrace.com/hc/article_attachments/32329213231380)

- Click Ok.

You'll be able to [search for the grower’s identifier by entering it in the Auto Code field of the Address Book window](https://support.vintrace.com/hc/en-us/articles/360001669836-Address-Book-Contacts#FilteringtheAddressBook).

6. Click Save.

After adding a grower, you can [set up a grower contract](https://support.vintrace.com/hc/en-us/articles/32303300639124).
