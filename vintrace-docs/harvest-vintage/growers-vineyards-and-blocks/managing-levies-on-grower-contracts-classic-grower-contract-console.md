# Managing Levies on Grower Contracts (Classic Grower Contract Console)

This article relates to the classic Grower Contract Console. For the Contracts Management module, refer to [Managing Grower Contracts (Contracts Management)](https://support.vintrace.com/hc/en-us/articles/45962805386004).

Australian customers can add levies to their [grower contracts](https://support.vintrace.com/hc/en-us/articles/32303300639124). [Levies are only available for South Australia](#h_01F9M9FAK7Q4Q0EVQ0KQ72AHZZ); levies for other Australian states will be available soon.

## Adding Levies to a Grower Contract

BEFORE YOU BEGIN: You’ll need to [create a booking](https://support.vintrace.com/hc/en-us/articles/32303268370324) that’s linked to the grower contract. The Levies tab will only be available after you save the contract with Defer Payment selected. To add a levy to a contract, be sure to save it with the Defer Payment checkbox selected.  ![Grower_Contract_Console_-_Defer_Payment_and_Levies_Tab_20210702.png](https://support.vintrace.com/hc/article_attachments/32328899238548)

To add a levy to a grower contract before making an instalment:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328899248660) More Options in the sidebar.
2. From the Accounts tile, click Manage Grower Contracts.
3. Click View beside the grower contract.
4. Select the Levies tab.
5. Click Add Levies.

![Grower_Contract_Update_-_Levies_Tab_20210702.png](https://support.vintrace.com/hc/article_attachments/32328888088852)

The Add Levies window displays.

6. Select the country , state, and region. The levies that are listed will depend on the country, state, and region that are selected. The Riverland region is the only levy calculated by a percentage of the fruit’s total cost.

![Add_Levies_-_Levies_Listed_20210702.png](https://support.vintrace.com/hc/article_attachments/32328894167828)

7. Select the levy you want to add to the grower contract.
8. Click OK. The selected levy displays in the Levies tab of the Grower Contract window.

If you need to remove a levy from a grower contract, click the ![Trash_20210702.png](https://support.vintrace.com/hc/article_attachments/32328916453908) icon. Once an [instalment is made with the levy](#making_instalment), it cannot be removed from the contract, regardless of whether the instalment was reversed.

9. Click Save.

You can also add a levy to a grower contract when you [make an instalment](#making_instalment).

## Making an Instalment with a Levy

To make an instalment with a levy:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328899248660) More Options in the sidebar.
2. From the Accounts tile, click Manage Grower Contracts.
3. Do one of the following:

- Expand the View menu beside the grower contract and select Create Next Instalment.

![Grower_Contract_Console_-_View_-_Create_Next_Instalment_20210702.jpg](https://support.vintrace.com/hc/article_attachments/32328899416852)

- Click View beside the grower contract, select the Instalments tab, then click Make Instalment.

![Grower_Contract_Update_-_Instalments_-_Make_Instalment_Button_20210705.png](https://support.vintrace.com/hc/article_attachments/32328877473300)

The Create Next Instalment window displays.

4. Select the option to pay the grower based on the received amount. You will not be able to select a levy if you select the contracted amount as the amount to be paid to the grower is based on the amount of fruit received.

When making the last instalment, be sure to select the Last Instalment checkbox. You'll also need to select the I Understand checkbox to acknowledge that you'll need to create a new grower contract if you receive additional fruit from the grower.

5. Select the levy.

![Create_Next_Installment_-_Levy_Selected_20210705.png](https://support.vintrace.com/hc/article_attachments/32328877541012)

If you need to add a new levy to the grower contract, click Add Levies.

6. Click OK. The Instalments tab displays the instalments made with the levies, grower amount, and total amount.
7. Click Save.

## Running the Remittance Report

To run the [Remittance report](https://support.vintrace.com/hc/en-us/articles/32301315918868):

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328899248660) More Options in the sidebar.
2. From the Accounts tile, click Manage Grower Contracts.
3. Select the checkbox beside the grower contract you want to run the report for.
4. Click Reports.
5. Select Remittance.

![Grower_Contract_Console_-_Remittance_Report_20210702.png](https://support.vintrace.com/hc/article_attachments/32328899320724)

The Net Previous Payment is the paid to date amount of all instalments minus the total levies paid to date. When only one instalment has been made, the Net Previous Payment value will always be $0.00.

![Remittance_Report_-_First_Installment_20210705.png](https://support.vintrace.com/hc/article_attachments/32328888141716)

As additional instalments are made, the Net Previous Payment value will reflect the paid to date amount minus the total levies paid to date.

![Remittance_Report_-_Second_Installment_20210705.png](https://support.vintrace.com/hc/article_attachments/32328899530388)

The Adjustments section of the report will always display below the Net for This Payment value.
