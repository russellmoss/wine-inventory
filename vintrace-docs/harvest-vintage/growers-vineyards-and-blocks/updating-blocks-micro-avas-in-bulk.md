# Updating Blocks' Micro AVAs in Bulk

After you’ve added micro AVAs (either [manually](https://support.vintrace.com/hc/en-us/articles/32301318591252) or [using an import](https://support.vintrace.com/hc/en-us/articles/32301282791828)), you can update the blocks in bulk to reflect their micro AVAs. To do this, you’ll want to use the [vineyard import/export functionality](https://support.vintrace.com/hc/en-us/articles/4404515630223-Updating-Blocks-and-Vineyards-in-Bulk#UpdatingVineyardsinBulk).

1. [Export the vineyards](https://support.vintrace.com/hc/en-us/articles/4404515630223-Updating-Blocks-and-Vineyards-in-Bulk#vineyards_exporting).
2. Specify the micro AVA by adding it to the Micro AVA column in the spreadsheet.

![CSV_Vineyard_Export_-_Micro_AVA_20220902.png](https://support.vintrace.com/hc/article_attachments/32329227523732)

- Do NOT edit the values in the VINx2 ID column.
- Be sure to save the file after editing it.

3. [Import the spreadsheet](https://support.vintrace.com/hc/en-us/articles/4404515630223-Updating-Blocks-and-Vineyards-in-Bulk#vineyards_importing) into vintrace using the vineyard import/export functionality.

After you import the spreadsheet with the micro AVAs, vintrace displays the micro AVA when you view the block from the Block Overview window.

![Block_Overview_-_B1-PN_with_Micro_AVA_20220906.png](https://support.vintrace.com/hc/article_attachments/32329227537812)
