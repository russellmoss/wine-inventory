# Vineyards & Blocks Page

The Vineyards & Blocks page displays a list of blocks. To access the Vineyards & Blocks page, click ![Blocks_20200803.png](https://support.vintrace.com/hc/article_attachments/32328812398612) Blocks in the sidebar.

At the top of the page are [filters](#filtering) that let you refine what's displayed on the page. The ![Fruit_Intake_20220929.png](https://support.vintrace.com/hc/article_attachments/32328849490324) fruit intake icon displays the [Fruit Intake Console](https://support.vintrace.com/hc/en-us/articles/32303330881044). The ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328849506836) plus icon lets you [add a block](#adding_block), [create a booking](https://support.vintrace.com/hc/en-us/articles/360000814175-Managing-Fruit-Intakes-and-Fruit-Intake-Bookings-#CreatingaNewBooking), or receive fruit.

You can [customize the Vineyards & Blocks page](https://support.vintrace.com/hc/en-us/articles/32301313813396) to include additional details.

## Filtering the Vineyards & Blocks Page

At the top of the page are buttons that let you control what blocks and vintage are displayed.

![Filters_20220929.png](https://support.vintrace.com/hc/article_attachments/32328834691476)

You can also apply a column filter by clicking the ![Filter_Gray_20210621.png](https://support.vintrace.com/hc/article_attachments/32328820937876) filter icon beside a column. For example, you can display only those blocks where the variety is Riesling or Sauvignon Blanc.

![Column_Filter_-_Variety_20220929.png](https://support.vintrace.com/hc/article_attachments/32328834714516)

## Printing QR Codes

To print QR codes for a block:

1. Select the blocks. You can select all blocks by selecting the checkbox at the top of the page.

![Selecting_All_Blocks_20220929.png](https://support.vintrace.com/hc/article_attachments/32328849331476)

2. From the Actions menu, select Print QR Codes.

![Actions_-_Print_QR_Codes_20220929.png](https://support.vintrace.com/hc/article_attachments/32328849387284)

3. Select the option you want to use for printing.

## Exporting Block Details

You can export the blocks to Excel (i.e., XLS), PDF, or a comma-separated file (i.e., CSV). When you export the blocks, you'll have the option to export all blocks, only the selected blocks, or the blocks on the displayed page.

To export blocks:

1. Click the ![Download_20200323.png](https://support.vintrace.com/hc/article_attachments/32328834607124) download icon displayed in the lower, right of the Vineyards & Blocks page.
2. Select the format you want to use for the export file.

- Excel — XLS file
- Portrait — PDF file
- Landscape — PDF file

3. Select which blocks you want to export.
4. Specify the location where you want to save the file.

To update blocks in bulk using vintrace’s [importing/exporting data](https://support.vintrace.com/hc/en-us/articles/32303307646868) functionality, refer to our [Updating Blocks and Vineyards in Bulk article](https://support.vintrace.com/hc/en-us/articles/32301319847828).

## Adding a Block

You can also add blocks from the [Block Overview](https://support.vintrace.com/hc/en-us/articles/360001287555-Managing-Blocks#SettingUpaBlock) window and the [mobile app](https://support.vintrace.com/hc/en-us/articles/32303262642196).

To add a block from the Vineyards & Blocks page:

1. Click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328849506836) plus icon.
2. Select Add Block.

![Add_Block_Menu_20220929.png](https://support.vintrace.com/hc/article_attachments/32328820758804)

The Add Block window displays.

![Add_Block_20220929.png](https://support.vintrace.com/hc/article_attachments/32328849445652)

3. Specify the block’s details.
4. Click Save.

You can specify additional block details such as its harvest details, vine details, agriculture, or site details by [editing the block](#editing_block).

## Editing a Block

To edit a block from the Vineyards & Blocks page:

1. Click the block’s name in the Block column. The Block Overview window displays the block’s details.

You can also [access the Block Overview window and edit a block](https://support.vintrace.com/hc/en-us/articles/32303262299284) from the More Options page.

2. From the Block Details tab, click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32328820779156) pencil icon.

![Block_Overview_-_Edit_Icon_20220929.png](https://support.vintrace.com/hc/article_attachments/32328812548372)

The Block window displays.

3. Update the block’s details as needed.
4. Click Save.

You can also [update blocks in bulk](https://support.vintrace.com/hc/en-us/articles/32301319847828) using vintrace’s [importing/exporting data](https://support.vintrace.com/hc/en-us/articles/32303307646868) functionality.
