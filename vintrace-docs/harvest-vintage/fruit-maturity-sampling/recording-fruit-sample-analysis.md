# Recording Fruit Sample Analysis

There are several ways to record a fruit sample analysis. You can:

- [Enter the sample manually](#Entering_Manually).
- Export, update and import a [CSV file](#Using_CSV).
- [Scan the sample’s barcode](#Scanning_Barcode).
- Use the vintrace app

## Entering the Sample Manually

To enter the sample manually, do the following from the Sample Set Console:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328893167124) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Search for the sample set you wish to update using the filter options at the top of the Console.
4. Click Add beside the sample set.

![add.sample.png](https://support.vintrace.com/hc/article_attachments/49846037812116)

The Grape Sampling window displays with the default analysis template and Sample Set pre-selected.

![record.sample.png](https://support.vintrace.com/hc/article_attachments/49846027594772)

5. Enter the analysis results in the right area of the window. You can also select the type of sample from the Type list, and enter information about the rows, vines, and area of the sample area if required.
6. Click Save.

## Using the CSV File

You can manage the arrival of sample sets and familiarise yourself with the codes by [downloading a CSV file](#Downloading_CSV) from vintrace, entering the data into the CSV, then [uploading the CSV file](#Uploading_CSV) to vintrace.

### Downloading the CSV File

The CSV file provides you a work list to use to collect and process samples. You can enter the results into the file, then submit the sample set.

To download the CSV file:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328893167124) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Click CSV.

![Sample_Set_Console_-_CSV_Button_20200413.png](https://vinx2-old.zendesk.com/hc/article_attachments/360004102276/Sample_Set_Console_-_CSV_Button_20200413.png)

### Uploading the CSV File

To upload the CSV file with your sample set data:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328893167124) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Click Upload.

## Scanning the Sample’s Barcode

If you have a barcode scanner (such as a USB device) attached to your computer, a fast way to record the results for a sample set is by clicking the barcode icon from the top bar, then scanning the sample label.

![barcode.scan.png](https://support.vintrace.com/hc/article_attachments/49846027595156)

You can also enter the sample code manually by entering the sample code as it appears on the sample label (e.g. *bs8 in the example below*)

![sample.code.png](https://support.vintrace.com/hc/article_attachments/49846027595412)
