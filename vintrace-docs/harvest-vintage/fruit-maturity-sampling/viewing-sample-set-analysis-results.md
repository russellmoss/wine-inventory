# Viewing Sample Set Analysis Results

Once you have recorded some [maturity sampling data](https://support.vintrace.com/hc/en-us/articles/32301319766292) against a block, there are a number of options for viewing that data. Which option you choose will depend on your needs. For a simple visual representation of a block's maturity progress, the Lab tab on the Block Overview screen is a quick and helpful tool.

## Block Overview Lab Tab

To view maturity data from the [Block Overview](https://support.vintrace.com/hc/en-us/articles/32301319801876-Block-Overview-Window):

1. Click ![vin_blocks_icon.png](https://support.vintrace.com/hc/article_attachments/41450983468180) Blocks from the sidebar.
2. Locate the Block and click to open the Block Overview window.
3. Click the Lab tab.
4. Select the checkboxes next to the metrics you wish to display in the chart.

or

1. Use [Quick Search](https://support.vintrace.com/hc/en-us/articles/32301339319700-Navigating-vintrace#h_01K5B6TPRKDB7T3B0RAQSXXT1R) to find the Block.
2. Click the Lab Tab.
3. Select the checkboxes next to the metrics you wish to display in the chart.

![lab_sample_set.png](https://support.vintrace.com/hc/article_attachments/41450959074324)

The Lab tab also allows you to View Chart to display a larger format, printable chart; as well as to Show Chart Data and Download a CSV of the sampling data of the chart.

## More Sample Set Reports

There are further reports you can generate against a sample set from the Sample Set Console.

To report on [sample data](https://support.vintrace.com/hc/en-us/articles/32301319766292) from the Sample Set Console:

1. Click ![vin_moreoptions_icon.png](https://support.vintrace.com/hc/article_attachments/41450959074708) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Search for the sample set.
4. Select the sample set.
5. Click Reports. You can select any of the following options:

- [Sample Day Sheet](#Sample_Day_Sheet)
- [Sample Analysis Stack Chart](#Sample_Analysis_Stack_Chart)
- [Sample Analysis Dual Axis Chart](#Sample_Analysis_Dual_Axis_Chart)
- [Sample Summary](#Sample_Summary)

NOTE: You can access the same reports by clicking Reports in the sidebar and then clicking Fruit Samples in the Report Category menu, but the block data won't be pre-filled and you'll need to enter it manually.

## Sample Day Sheet

The Sample Day Sheet option produces a PDF or CSV file that groups related samples by grower, vineyard, and block for the provided date range. It’s useful for comparing data across multiple fruit sources within the system for arbitrary periods of time.

To produce a Sample Day Sheet report, specify your filters, then click Generate.

![report_day_sheet.png](https://support.vintrace.com/hc/article_attachments/41450959075092)

*example - Sample Day Sheet*

## Sample Analysis Stack Chart

This chart produces a line chart over time with each metric (e.g., ph, TA) reported on its own separate XY axis in a stacked formation. To produce a Sample Analysis Stack Chart, specify each metric you want to see, then click Show Chart.

![report_analysis_stack.png](https://support.vintrace.com/hc/article_attachments/41450983471892)

*example - Sample Analysis Stack Chart*

## Sample Analysis Dual Axis Chart

This chart allows you to report on two metrics in a single chart providing for a closer correlation. To produce a Sample Analysis Dual Axis Chart, select any two metrics to correlate, then click Show Chart.

![report_dual_axis.png](https://support.vintrace.com/hc/article_attachments/41450983472788)

*example - Sample Analysis Dual Axis Chart*

## Sample Summary

This option provides you with a CSV file containing all analyses on each block. You can choose to filter this by vintage, owner, and grower.
