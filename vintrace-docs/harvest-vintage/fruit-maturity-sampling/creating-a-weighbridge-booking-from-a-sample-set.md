# Creating a Weighbridge Booking from a Sample Set

If you already have a [sample set configured](https://support.vintrace.com/hc/en-us/articles/32301351265172) for a block of fruit, vintrace streamlines the creation of a weighbridge booking. To create a weighbridge booking for a sample set:

1. Click ![vin_moreoptions_icon.png](https://support.vintrace.com/hc/article_attachments/41449754491412) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Search for the sample set.
4. Select the sample set by clicking it to highlight.
5. Click Make Booking.

![maturity_booking.png](https://support.vintrace.com/hc/article_attachments/41449754491924)

The [Scale booking](https://support.vintrace.com/hc/en-us/articles/32303268370324-Managing-Fruit-Intakes-and-Fruit-Intake-Bookings#h_52356d83-df02-4dfe-af54-839f2af65849) page opens ready to be completed.
