# Recording a Tasting Note Against a Sample Set

To record a tasting note to a [sample set](https://support.vintrace.com/hc/en-us/articles/32301351265172) from the Sample Set Console:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/41335982326420) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Search for the sample set.
4. Select the sample set by clicking it to highlight.
5. Click Tasting Note.

![tasting_sample.png](https://support.vintrace.com/hc/article_attachments/41449748908180)

6. Enter the information in the Tasting Note window.

![sample.note.png](https://support.vintrace.com/hc/article_attachments/41336470559252)

7. Click Save.
