# Creating a Sample Set

In vintrace, you'll need to create a sample set for every block that you record fruit samples for. Sample sets act as a grouping mechanism and allow you to record individual samples against a block for an extended period of time.

After you create a sample set, you can [record analysis](https://support.vintrace.com/hc/en-us/articles/32301319766292), [record tasting notes](https://support.vintrace.com/hc/en-us/articles/32301351165204), and [view its analysis results](https://support.vintrace.com/hc/en-us/articles/32301351220372).

NOTE: If you do your [maturity sampling through the vintrace app](https://support.vintrace.com/hc/en-us/articles/32301373156756-Viewing-and-Recording-Fruit-Samples-for-a-Block), the sample set is created automatically and does not need to be duplicated with the steps below.

## Creating a Sample Set

To create a sample set for a single block:

1. Click ![vin_moreoptions_icon.png](https://support.vintrace.com/hc/article_attachments/41449673969556) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples.
3. Click Add Sample Set.

![add_sample.png](https://support.vintrace.com/hc/article_attachments/41449689642260)

The Create Sample Set window displays.

![sample_set.png](https://support.vintrace.com/hc/article_attachments/41449689642516)

4. Specify the details for the sample set.

- Code — The code can be auto-generated by the system, or entered manually. If you’d like assistance auto-generating the code, contact support.
- Grower/Block — Search for a grower, or any combination of grower, variety, region, or block name. You can only use this field where blocks are configured in the system. After selecting a grower/block, the system automatically fills in the grower, variety, and region.
- Specify which [analysis template](https://support.vintrace.com/hc/en-us/articles/32301372281748-Setting-Up-an-Analysis-Template) you plan to use for your maturity sampling. If you have already specified the analysis template as the [default for fruit sampling](https://support.vintrace.com/hc/en-us/articles/32301372281748-Setting-Up-an-Analysis-Template#h_01K5DV2B8W80T92WXZXNYG9211), this will be pre-filled but can be updated to a different template if required by clicking the cross button.

5. Click Save.

## Creating a Sample Set Using Quick Add Function

In most cases, it's quicker to select multiple blocks at a time for sampling. The Quick Add function allows for this.

IMPORTANT: Filling the 'Date flowered' field will assign the same flowering date to all blocks selected. If you do need to record unique information against your blocks, it's best to leave this field blank while creating the sample set, and add the flowering date against individual blocks by [editing their sample set](#h_01K5QWWK9JW60E0JP8P14M3DJ6).

To add several sample sets at once for a specific grower or region:

1. Click ![vin_moreoptions_icon.png](https://support.vintrace.com/hc/article_attachments/41449673969556) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples.
3. Click Quick Add.

![quick_add.png](https://support.vintrace.com/hc/article_attachments/41449689642900)

The Add Multiple Sample Sets window displays.

4. From the Add Multiple Sample Sets window, search for suitable blocks either by region or by grower. Check the blocks that you want to record fruit samples for, then complete the other details as per [creating a sample set](#h_01K3SWBHGGN2RN921KEVPA2V0W) above.

![multi_sample.png](https://support.vintrace.com/hc/article_attachments/41449673971348)

5. Click Save.

NOTE: If a block has previously had a Sample Set created for current vintage, vintrace will display the following error:

![sample_warning.png](https://support.vintrace.com/hc/article_attachments/41449689646356)

In this case, if you do not intend to create duplicates, click Cancel to return to the Add Multiple Sample Sets window and uncheck the duplicate blocks before clicking Save again.

## Editing a Sample Set

You can edit details against an existing sample set if required. For example, to record a flowering date or update a withholding date for the block. To do this:

1. Click ![vin_moreoptions_icon.png](https://support.vintrace.com/hc/article_attachments/41449673969556) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples.
3. Search for the Sample Set and click it to highlight.
4. Click Edit.
5. Specify your updates.
6. Click Save.
