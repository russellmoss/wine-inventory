# Correcting a Sample Set

To make corrections to a [sample set's measurements](https://support.vintrace.com/hc/en-us/articles/32301319766292):

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328864622996) More Options from the sidebar.
2. From the Harvest tile, click Maturity Samples. The Sample Set Console displays.
3. Search for the sample set.
4. Select the sample set by clicking it to highlight.
5. Click Show History.

![show.history.png](https://support.vintrace.com/hc/article_attachments/41336477934356)

The Sample History window displays a list of past samples. Click View to display more details for a sample.

![view.samples.png](https://support.vintrace.com/hc/article_attachments/41336490182804)

From the Grape Sampling window, you can reverse the analysis by clicking Reverse/Undo. To edit the measurements, click Edit.

![sample.update.png](https://support.vintrace.com/hc/article_attachments/41336490185620)
