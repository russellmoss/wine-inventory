# Filling Barrels to Fermentation Height

When you’re ready to fill your barrels to fermentation height, you can create a new barrel group by using the Multi Transfer (One-to-Many), or Transfer to Barrel Group operations. If you use the latter, you may need to select the Transfer to Barrel Group sub-tab in the Racking window.

![Multi_Transfer_or_Transfer_to_Barrel_Group_Operation_20200826.png](https://support.vintrace.com/hc/article_attachments/32328788258324)

To create the barrel group that you want to fill to fermentation height:

1. Click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328762236820) Plus beside the vessel field.

Do NOT click Add Line. You’d only click Add Line if you also wanted to transfer wine into a tank.

The Barrel Group window displays.

2. Select the area of the winery where you’ll be storing the barrels.
3. Do any of the following:

- To search for specific barrels, click Search. From the Search for Barrels window, search for the barrels that you want to include in the barrel group. You can apply a filter to only display empty barrels.

![Search_for_Barrels_-_Empty_20200824.png](https://support.vintrace.com/hc/article_attachments/32328777468948)

- To paste a list of barrels you want to fill, click List. In the Barrel Code List window, enter each barrel’s code. Each barrel code should be entered on a separate line.

![Barrel_Code_List_20200826.png](https://support.vintrace.com/hc/article_attachments/32328746843028)

- If you only have a handful of barrels and want to enter the barrel codes manually, click Add Line.

![Barrel_Group_Create_-_Add_Line_20200827.png](https://support.vintrace.com/hc/article_attachments/32328746690452)

4. Select the I Know the Volume of Wine option.

![Barrel_Group_Create_-_I_Know_Volume_of_Wine_20200824.png](https://support.vintrace.com/hc/article_attachments/32328777524500)

5. In the New Amount field, enter the wine volume.

![Barrel_Group_Create_-_New_Amount_20200824.png](https://support.vintrace.com/hc/article_attachments/32328770991252)

6. Select the Select All checkbox.
7. From the Set as Full list, select *Set as Partial (Remaining)*.

![Barrel_Group_Create_-_Select_All_and_Set_as_Partial_20200824.png](https://support.vintrace.com/hc/article_attachments/32328777195284)

8. Click Apportion Partial. vintrace determines how much will be added to each barrel using the new amount that you entered. In our example, the new amount is 550 gallons and the barrel group that we created includes 10 barrels. Each barrel will be filled with 55 gallons.

![Barrel_Group_Create_-_Apportion_Partial_20200824.png](https://support.vintrace.com/hc/article_attachments/32328790821140)

9. Click Save.
