# Topping Up Barrels Post Fermentation

To top up barrels:

1. [Create a Manual Work Order](https://support.vintrace.com/hc/en-us/articles/32303315610388).
2. Click Add Job.
3. Select Multi Topping.
4. From the Top From section, click ![Select_Specific_Barrel_Icon_20200824.png](https://support.vintrace.com/hc/article_attachments/32329005396244) to select the barrel that you want the cellar to use to top up the rest of the barrels (i.e., select the barrel you want to empty).

![Multi_Topping_-_Select_Top_From_Barrel_Icon_20200824.png](https://support.vintrace.com/hc/article_attachments/32328996183060)

5. From the Barrel Group List window, select the barrel to empty, then click OK.

![Barrel_Group_List_-_Barrel_Selected_20200824.png](https://support.vintrace.com/hc/article_attachments/32329023298580)

6. In the Top Into section, select the barrel group that you're topping up and select the *Increase Volume by Amount* Topping Mode.

![Multi_Topping_-_Topping_Mode_Increase_Volume_20200826.png](https://support.vintrace.com/hc/article_attachments/32329035353492)

7. Click Add to Work Order.

## Completing the Multi Topping Work Order

When you’re ready to complete the work order:

1. From the Top From section of the Multi Topping window, click ![Select_Specific_Barrel_Icon_20200824.png](https://support.vintrace.com/hc/article_attachments/32329005396244). The Barrel Group List window displays with the barrel that you selected previously highlighted.
2. Select the Work Out Volume from the Barrel Details option.
3. Select the appropriate fill for the barrel. For example, if you emptied the barrel, you would select *Empty* from the list.

![Barrel_Group_List_-_Selected_Barrel_-_Remaining_Fill_Level_2020824.png](https://support.vintrace.com/hc/article_attachments/32329013762196)

vintrace updates the Volume Out amount based on the fill level that you specified.

4. Click OK. vintrace detects a self topping and asks you if you’d like to update the topping side.

![Self_Topping_Detected_20200824.png](https://support.vintrace.com/hc/article_attachments/32329035096468)

5. Click Yes.
6. In the Top Into section of the window, select the *Increase Volume by Amount Option* from the Topping Mode list.

![Multi_Topping_-_Topping_Mode_Increase_Volume_20200824.png](https://support.vintrace.com/hc/article_attachments/32329035195284)

7. Click ![Edit_Advanced_Barrel_Fill_Info_Icon_20200824.png](https://support.vintrace.com/hc/article_attachments/32328996234132). This icon only displays after you’ve selected the Increase Volume by Amount topping mode.
8. Select all of the barrels except the one that you used (e.g., all barrels except the one you emptied). You can do this by selecting the Select All checkbox, then manually de-selecting the empty barrel.
9. Click Set as Full.

![Advanced_Barrel_Fill_Info_-_Set_as_Full_20200824.png](https://support.vintrace.com/hc/article_attachments/32329035223828)

This causes the Fill for the selected barrels to be set to *Full*. The amount to be transferred in displays in the Transfer in Volume field.

![Advanced_Barrel_Fill_Info_-_Selected_Set_as_Full_20200824.png](https://support.vintrace.com/hc/article_attachments/32328996394644)

If you filled another vessel that’s not currently included, you can include that vessel by clicking Add Line. A new line is added to the Advanced Barrel Fill Information window where you can specify the details for the vessel including its fill amount.
  ![Advanced_Barrel_Fill_Info_-_Add_Line_20200824.png](https://support.vintrace.com/hc/article_attachments/32328996369428)

10. Click OK.
11. Click the ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32329023477268) Calculator in the lower right corner. The loss amount is calculated.
12. Select a Loss Reason.
