# Changing the Language on the Mobile App

To change the language on the vintrace mobile app:

1. Tap the ![Account Settings Icon 20200914.png](https://support.vintrace.com/hc/article_attachments/32328573085076) in the upper left.
2. Tap Language.
3. Select the language you would like to use for the mobile app.
4. Tap Submit.
