# Scanning Lab Barcodes to View and Record Lab Results on the Mobile App

You can now scan the barcodes for [lab requests](https://support.vintrace.com/hc/en-us/articles/32301356827540) from the mobile app. This lets you quickly view and record your lab results in real time. You can even backdate lab results if needed.

## Viewing Lab Requests

You can also scan the barcodes from the Work orders screen

To view the lab request using the lab’s barcode from the mobile app:

1. From the main screen, tap on the barcode icon.

![Main_Screen_Barcode_Scanner_1.PNG](https://support.vintrace.com/hc/article_attachments/32329145836564)

2. Scan the lab request's barcode.

![Scan_1.PNG](https://support.vintrace.com/hc/article_attachments/32329168849044)

The lab request displays with the Status color coded.

![Analysis_-_Color_Coded_Status_20200918.png](https://support.vintrace.com/hc/article_attachments/32329155475860)

| COLOR | STATUS |
| --- | --- |
| Orange | Outstanding, pending results to be entered |
| Purple | Awaiting approval, all results have been entered and saved as a draft |
| Teal | Completed |

## Saving Lab Requests as a Draft

Saving a lab request as draft is useful if you still need to make additional changes to the request, or need someone else to confirm the results before the request is submitted.

To save a lab request as a draft:

1. From the main screen, tap on the barcode icon.
2. Scan the lab request's barcode.
3. Enter the lab results as needed.
4. Tap on the X icon.

![Analysis_-_Saving_as_Draft_-_X_Icon_20200918.png](https://support.vintrace.com/hc/article_attachments/32329164139412)

You’ll be prompted to save the analysis as a draft.

![Draft_1.PNG](https://support.vintrace.com/hc/article_attachments/32329155328532)

5. Tap Yes.

## Finishing Lab Requests at a Later Time

If you want to check whether all metrics were completed, you can use the Finish Later option. This option does not change the status of the request.

To finish a lab request at a later time:

1. From the main screen, tap on the barcode icon.
2. Scan the lab barcode for the lab request.
3. Enter the lab results as needed.
4. Tap Submit.
5. Tap Finish Later.

![Analysis_-_Incomplete_Analysis_-_Finish_Later_Button_20200918.png](https://support.vintrace.com/hc/article_attachments/32329132713108)

## Recording Lab Results

To record lab results:

1. From the main screen, tap on the barcode icon.
2. Scan the lab request's barcode.
3. Enter the lab results as needed.
4. To update the request date, tap the currently displayed date and time.
5. Tap Submit.

## Correcting Completed Lab Requests

1. From the main screen, tap on the barcode icon.
2. Scan the lab request's barcode.
3. Tap the ![Three_Vertical_Dots_20200918.PNG](https://support.vintrace.com/hc/article_attachments/32329132804244) Actions icon.

![Analysis_-_Actions_Icon_Location_20200918.PNG](https://support.vintrace.com/hc/article_attachments/32329164200468)

4. Tap Make a Correction.

![Analysis_-_Make_a_Correction_20200918.png](https://support.vintrace.com/hc/article_attachments/32329164220436)

5. Edit the request details as needed.
6. Tap Update.
