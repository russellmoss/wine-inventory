# Backdating Jobs on the Mobile App

To backdate jobs on the mobile app:

1. View the job’s Summary screen.
2. Tap Just Now.

![Backdate_1.png](https://support.vintrace.com/hc/article_attachments/32329033544340)

To submit a job with the current date and time, tap Submit.

3. Select the date you want to use, then tap OK.

![Backdate_2.png](https://support.vintrace.com/hc/article_attachments/32328988668692)

4. Select the hour, then tap OK.
5. Select the minute, then tap OK.
6. Tap Submit.

![Backdate_5.png](https://support.vintrace.com/hc/article_attachments/32329012339092)
