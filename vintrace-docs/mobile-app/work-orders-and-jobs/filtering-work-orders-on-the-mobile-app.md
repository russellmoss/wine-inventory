# Filtering Work Orders on the Mobile App

To filter work orders on the mobile app:

1. From the main screen, tap Work Orders.
2. Tap the ![Work Order Filter 20240325.png](https://support.vintrace.com/hc/article_attachments/32328786620180) icon in the upper right.
3. Specify the criteria for the work orders.

![Work Order Filters 20240325.png](https://support.vintrace.com/hc/article_attachments/32328760138388)

4. Tap Apply.
