# Recording Additions on the Mobile App

If you'd like the additive amount to be automatically calculated, be sure to [set up your additives](https://support.vintrace.com/hc/en-us/articles/32301344910740) with metric weightings from the vintrace web application. Refer to our [Using the Additives Calculator article](https://support.vintrace.com/hc/en-us/articles/32301358168468) for details on using the additives calculator.

To add an additive from the mobile app:

1. Search for and view the bulk wine on the mobile app.
2. Tap the +.

![17CH-JX2-FR_-_Plus_Button_20201021.png](https://support.vintrace.com/hc/article_attachments/32329152393748)

3. Tap Addition.

![17CH-JX2-FR_-_Action_-_Addition_20201021.png](https://support.vintrace.com/hc/article_attachments/32329161086100)

The screen to select vessels displays.

![17CH-JX2-FR_-_Addition_-_Select_Vessels_20201021.PNG](https://support.vintrace.com/hc/article_attachments/32329161104660)

4. Select the vessels you want to make the addition to, then tap Next. You can also tap the ![Barcode_Scanner_20200402.png](https://support.vintrace.com/hc/article_attachments/32329115895700) barcode icon and scan the barcode of the vessels to select them.
5. Select the additive. The Rate Unit displays the target metric.
6. If you specified the metric weighting when you [set up the additive,](https://support.vintrace.com/hc/en-us/articles/32301344910740) enter a value in the Rate of Add field; the amount will be automatically calculated. If you didn't specify the metric weighting, you can enter the amount.

![17CH-JX2-FR_-_Addition_-_KMBS_20201021.png](https://support.vintrace.com/hc/article_attachments/32329141011220)

If you have inventory enabled, you can also select the lot and location.

7. Tap Review.
8. Tap Submit.
