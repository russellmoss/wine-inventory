# Creating a Booking on the Mobile App

To create a booking from the mobile app:

1. Search for the block.
2. Tap ![Calendar_Icon_20200909.png](https://support.vintrace.com/hc/article_attachments/32329120194708) Bookings. If there are existing bookings for the block, a number displays beside the icon to indicate the number of bookings.
3. Tap Create a New Booking.

![Adding_Booking_20200909.png](https://support.vintrace.com/hc/article_attachments/32329133876756)

4. Specify the details for the booking.
5. Tap Create. After the booking is created, it’ll display in the list of bookings for the block.

![Bookings_B1-CH_20200909.PNG](https://support.vintrace.com/hc/article_attachments/32329094722836)
