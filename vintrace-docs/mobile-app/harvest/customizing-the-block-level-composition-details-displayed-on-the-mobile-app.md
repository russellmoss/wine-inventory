# Customizing the Block-Level Composition Details Displayed on the Mobile App

The block-level composition details displayed in the mobile app are determined by the [composition details displayed on the Product page](https://support.winery-software.com/hc/en-us/articles/360000814455#CompositionTab) of the vintrace web application.

![Block-Level_Composition_Details_20201021.png](https://support.vintrace.com/hc/article_attachments/32329120990996)

The composition details displayed are specific to each user. If you’d like to change what’s displayed, you’ll need to [customize the vintrace web application’s Product page](https://support.winery-software.com/hc/en-us/articles/360000814455#CompositionTab). After changing the composition details displayed in the web application, you’ll need to log out of the mobile app and log back in to refresh the display.
