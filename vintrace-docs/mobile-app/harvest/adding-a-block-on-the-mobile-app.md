# Adding a Block on the Mobile App

To add a block from the mobile app:

1. From the main screen, tap +.

![Add_a_block_1.jpg](https://support.vintrace.com/hc/article_attachments/32329243793300)

2. Tap Add a Block.

*![Add_a_block_2.jpg](https://support.vintrace.com/hc/article_attachments/32329277098004)*

3. Specify the block’s details.

![Add_a_block_3.jpg](https://support.vintrace.com/hc/article_attachments/32329243826196)

4. Tap Submit.

## Printing QR Codes for Blocks

If you entered the number of rows for the block, the QR codes for each row will be emailed to you. From the email, you’ll be able to select the size of the QR codes, then download them as a PDF that you can print and stick to signposts on your block.

![Add_a_block_4.jpg](https://support.vintrace.com/hc/article_attachments/32329277155988)

When you scan a QR code with the vintrace app, the app displays your block so that you can record labs, create bookings, or receive fruit.
