# Block and Vineyard Assessments on the Mobile App

The Observations tab displays a list of the block’s assessments.

![Observations_Tab_20200914.png](https://support.vintrace.com/hc/article_attachments/32329076557332)

From this tab you can view, add, or edit an assessment, note, or attachment.

## Recording a Block Assessment

To record a block assessment:

1. Search for and view the block.
2. Do one of the following:

- Tap the + in the upper right, then tap Block Assessment.

![Actions_-_Block_Assessment_20200914.png](https://support.vintrace.com/hc/article_attachments/32329062667412)

- Tap the ![Observation_Tab_Icon_20200914.PNG](https://support.vintrace.com/hc/article_attachments/32329062710548) Observations tab, then tap Add Observation > Block Assessment.

![Observations_-_Adding_Assessment_20200914.png](https://support.vintrace.com/hc/article_attachments/32329029790740)

3. Enter the details for the block assessment. You can include a note or attachment (e.g. photo) with the assessment by clicking the paperclip in the upper right.

![Adding_Assessment_20200914.png](https://support.vintrace.com/hc/article_attachments/32329029804052)

4. If you need to change the assessment date, tap Just Now to select the date/time.
5. Tap Submit to save the block assessment, or X to discard it.

## Editing a Block's Assessment

To edit an assessment:

1. View the assessment.
2. Edit the assessment as needed.
3. Tap Update to save your changes, or X to discard them.

## Adding Notes and Photos

To add notes and attachments (e.g., photos) to a block:

1. Search for and view the block.
2. Do one of the following:

- Tap the + in the upper right, then tap Add Note & Photo.

![Actions_-_Add_Note_and_Photo_20200914.png](https://support.vintrace.com/hc/article_attachments/32329052789396)

- Tap the ![Observation_Tab_Icon_20200914.PNG](https://support.vintrace.com/hc/article_attachments/32329062710548) Observations tab, then tap Add Observation > Add Note & Photo.

![Observations_-_Adding_Note_and_Photo_20200914.png](https://support.vintrace.com/hc/article_attachments/32329076813332)

3. If you’ve enabled location tracking, Sharing Location displays in the upper right. Otherwise, you can include a location by toggling Include Location.
4. Enter your notes.
5. To attach photos:

- Tap the camera icon.
- Tap the type of photo you want to attach.
- To attach a new photo, tap Camera. To attach an existing photo, tap Photos.

6. Tap the checkmark to save your notes and photos.

![Adding_Note_and_Attachments_20200914.png](https://support.vintrace.com/hc/article_attachments/32329070916244)

## Editing a Note or Attachment (Photo)

To edit a note or attachment:

1. View the note.
2. Edit the note as needed.
3. To update the location for the note or attachment, tap Edit. If you don’t want to attach a location with the note or attachment, tap Clear.

![Block_notes_10.jpg](https://support.vintrace.com/hc/article_attachments/32329076517524)

4. Tap the checkmark to save your changes, or tap X to discard them.

## Deleting a Note

To delete a note:

1. View the note.
2. Tap the ![Trash 20240328.png](https://support.vintrace.com/hc/article_attachments/32329048171284) icon.

![Delete Note 20240328.png](https://support.vintrace.com/hc/article_attachments/32329071040148)

You'll be asked to confirm that you want to delete the note.

3. Tap Delete to delete the note. Tap Cancel to retain the note.

## Enabling/Disabling Location Tracking

To specify the default location tracking setting:

1. From the main screen in the vintrace app, tap ![Account_Settings_Icon_20200914.png](https://support.vintrace.com/hc/article_attachments/32329076691860).

![Main_-_Account_Settings_Icon_20200914.png](https://support.vintrace.com/hc/article_attachments/32329076707476)

2. Toggle the Auto-Location setting as needed.

![Account_Settings_-_Auto_Location_Toggle_20200914.png](https://support.vintrace.com/hc/article_attachments/32329052841108)
