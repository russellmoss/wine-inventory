# Tagging Blocks from the Mobile App

You can now add tags to your block and search blocks from the app. Tags let you quickly classify blocks with multiple values. They can be used to track certain properties of the block for administrative purposes. You can add one or more tags to a block.

![Block_with_Tags_20200803.png](https://support.vintrace.com/hc/article_attachments/32329193561620)

## Adding a Tag to a Block

To add a tag to a block from the app:

1. View a block from the app.
2. Tap Add Tag.

![App_-_Tagging_Blocks_-_Add_Tag_20200803.png](https://support.vintrace.com/hc/article_attachments/32329188219028)

3. To use an existing tag, search for it. Or, create a new tag by tapping Create.

![Search_or_Add_Tag_20200803.png](https://support.vintrace.com/hc/article_attachments/32329193599892)

4. To add additional tags to the block, click ![Plus_in_White_Circle_20200715.png](https://support.vintrace.com/hc/article_attachments/32329233869332) and repeat step 3.

![Assign_Additional_Tags_20200803.png](https://support.vintrace.com/hc/article_attachments/32329193575956)

## Removing Tags from a Block

To remove a tag from a block, tab the X beside the tag.

![Removing_Tag_from_Block_20200803.png](https://support.vintrace.com/hc/article_attachments/32329173540884)

## Searching for Blocks with a Tag

To search for blocks with a specific tag, enter the tag in the search field on the main screen. Blocks that have been assigned the tag are listed.

![Tagging_blocks_4.jpg](https://support.vintrace.com/hc/article_attachments/32329183451796)

## Viewing Block Tags from the Web

Any tags that have been added to a block display on the Vineyards & Blocks page.

To view a block's tags, click ![Blocks_20200803.png](https://support.vintrace.com/hc/article_attachments/32329202024084) Blocks in the sidebar. Any tags that have been added to a block display in the Tags column.

![Tagging_blocks_5.jpg](https://support.vintrace.com/hc/article_attachments/32329201991572)
