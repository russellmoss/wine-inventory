# Processing Fruit from Different Blocks Together on the Mobile App

To process fruit from multiple blocks, or fruit without a block:

1. From the main screen, tap the Harvest calendar.

![Main_-_Harvest_20200923.png](https://support.vintrace.com/hc/article_attachments/32329138962964)

2. Tap the + in the upper right.

![Harvest_Calendar_Plus_Icon_20200923.PNG](https://support.vintrace.com/hc/article_attachments/32329158974868)

3. From the Actions menu, tap Process Fruit.

![Harvest_Calendar_-_Process_Fruit_20200923.PNG](https://support.vintrace.com/hc/article_attachments/32329150390036)

4. Specify the process and equipment options, then tap Next.

![Process_20200923.PNG](https://support.vintrace.com/hc/article_attachments/32329150418836)

A list of all uncrushed fruit grouped by block name and variety displays. If you have a parcel that doesn’t have a block set against it, it will be grouped by region (sub AVA) - variety.

![Process_all_6.jpg](https://support.vintrace.com/hc/article_attachments/32329159002132)

5. Select the parcels from the different blocks that you want to process, then tap Next.

![Parcels_Selected_20200923.PNG](https://support.vintrace.com/hc/article_attachments/32329114014740)

6. Specify any additives, then tap Next.
7. Specify details for the vessels, then tap Review.
8. Tap Submit.
