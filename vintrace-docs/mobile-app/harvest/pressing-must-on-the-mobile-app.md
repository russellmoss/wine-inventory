# Pressing Must on the Mobile App

You can press off multiple musts and keep your fractions separate using the mobile app.

1. Search for and view the must.
2. Tap the + icon.

![Press_1.png](https://support.vintrace.com/hc/article_attachments/32329099262740)

3. Tap Press Must.

![Press_2.png](https://support.vintrace.com/hc/article_attachments/32329090284308)

The Press Selection and Vessels screen displays.

![Press_3.png](https://support.vintrace.com/hc/article_attachments/32329126161556)

4. Specify the press and vessels, then tap Next.

If you need to add another vessel, tap + Another Vessel. You can edit the press volume for each vessel by tapping the volume.
![Press_4.png](https://support.vintrace.com/hc/article_attachments/32329099275156)

The Select Addition screen displays.

![Press_5.png](https://support.vintrace.com/hc/article_attachments/32329113235732)

5. Select any additives that you want to add to the press, then tap Next. The Destination screen displays.
6. Select each destination vessel’s fraction type, then tap Review.

![Press_6.png](https://support.vintrace.com/hc/article_attachments/32329099385236)

The Press Summary screen displays.

![Press_7.png](https://support.vintrace.com/hc/article_attachments/32329090401300)

7. Verify the details, then tap Submit.
