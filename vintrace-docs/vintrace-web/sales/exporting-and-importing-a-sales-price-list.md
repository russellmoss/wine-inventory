# Exporting and Importing a Sales Price List

When you [export a sales price list](#Exporting) from vintrace, you can make changes to the prices or override tax rates in the file, and add new items. You can then import the CSV file into vintrace to [update an existing sale price list](#Importing_Updates_to_Existing), or [create a new one](#Importing_New_Sales_Price_List).

## Exporting a Sales Price List

You can export a sales price list to a CSV file from the Winery Setup window (Setup Options > Sales > Sales Price Lists):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329278918164) Set Up in the sidebar.
2. Click Sales.
3. From the Sales Price Lists tile, click Configure.
4. Select the price list you want to export.
5. Click Export.

![Update_Sales_Price_List_-_Export_Button_20200810.png](https://support.vintrace.com/hc/article_attachments/32329233092500)

Edit the CSV file as needed, then import it to update an existing price list, or to create a new one.

 ![Exported_Price_List_CSV_20200810.jpg](https://support.vintrace.com/hc/article_attachments/32329216129300)

## Importing Updates to an Existing Sales Price List

Before you import updates to an existing price list, be sure to [export the price list](#Exporting) that you want to update, then edit the CSV file. If you accidentally delete an item from the CSV file, the item will NOT be deleted from the price list in vintrace.

To import a CSV file to update an existing price list:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329278918164) Set Up in the sidebar.
2. Click Sales.
3. From the Sales Price Lists tile, click Configure.
4. Select the price list you want to update with the CSV file.
5. Click Import.

![Update_Sales_Price_List_-_Import_Button_20200810.png](https://support.vintrace.com/hc/article_attachments/32329245743380)

The Price List Importer window displays.

![Price_List_Importer_20200810.png](https://support.vintrace.com/hc/article_attachments/32329278992916)

6. Click Choose File.
7. Select the CSV file that you want to import.
8. Click Import.

## Importing A New Sales Price List

If you need to create a new price list that’s similar to an existing one that you have, you can [export the price list](#Exporting) that you want to copy, edit the CSV file as needed, then import it into the new sales price list. When editing the CSV file, be sure the edit the price and override tax rates as needed. You can also add new items to the price list.

To import a CSV for a new price list:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329278918164) Set Up in the sidebar.
2. Click Sales.
3. From the Sales Price Lists tile, click Configure.
4. Click New Price List.
5. At a minimum, specify the Code, Currency Code, and Tax Policy for the new price list.

![Create_Sales_Price_List_-_Required_Fields_20200810.png](https://support.vintrace.com/hc/article_attachments/32329216176020)

6. Click Import.
7. Select the CSV file that you want to import.
8. Click Import.
