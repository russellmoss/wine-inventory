# Using Programs for Batch Classification

Programs enable you to provide additional details or classifications for a wine batch. For example, you might want to set up a program for a wine batch that’s being used for a particular brand, or to record a designated variety or region. After you [set up the program](#h_01EMHGJG5WQQCTJWEDZT8JGJJJ), you can [associated it with your wine batches](#h_01EMHGJQXSQN5YGNRG43ABJBME). You’ll then be able to [filter the Vessels page and the Bulk Stock Report to locate those wines](#h_01EMHGK3G35P1GTQ2K59AFCPZ0).

## Setting Up a Program

You can set up a program from the Winery Setup window (Setup Options > Work-flow > Programs):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329235836820) Set up in the sidebar.
2. Click Classification.
3. From the Programs tile, click Configure.
4. Click New Program. The Program/Brand window displays.

![Create_Program_Brand_20200819.png](https://support.vintrace.com/hc/article_attachments/32329203788692)

5. Specify the details for the program.
6. Click Save.

## Assigning a Program to a Batch

After you’ve set up a program, you can assign it to a wine batch:

1. Use the Quick Search to locate the wine batch.
2. From the Product page, click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32329235963540) in the Program tile.

![Assigning_to_Program_20201013.png](https://support.vintrace.com/hc/article_attachments/32329235925908)

The Edit Program window displays.

![Edit_Program_20201013.png](https://support.vintrace.com/hc/article_attachments/32329208844180)

3. Select the program.
4. Click Save. The selected program displays on the wine’s Product page.

![16CH-AJG_Program_20201013.png](https://support.vintrace.com/hc/article_attachments/32329203801748)

## Locating Products in the Program

To find products associated with the program, you can [filter the Vessels page](https://support.winery-software.com/hc/en-us/articles/360001550655-The-Vessels-Page#FilteringtheVesselsPage).

![Filtered_by_Program_20201013.png](https://support.vintrace.com/hc/article_attachments/32329208920468)

You can also run the Bulk Stock Report and filter it by the program.

![Bulk_Stock_Report_PDF_-_Program_Filter_20201013.png](https://support.vintrace.com/hc/article_attachments/32329235991188)

To run the Bulk Stock Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329195493396) Reports in the sidebar.
2. Select the Bulk Wine report category.
3. Expand the Advanced Report Options.
4. Click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329236005524) beside Program Filter.

![Bulk_Stock_Report_-_Program_Filter_20201013.png](https://support.vintrace.com/hc/article_attachments/32329208967188)

5. Be sure the Search by Program option is selected.
6. Select the program(s) you want to include in the report.

![Search_by_Program_20201013.png](https://support.vintrace.com/hc/article_attachments/32329203929876)

7. Click OK.
8. Specify any other filters or options for the report.
9. Click Generate.
