# Fixing a Wine's Composition

vintrace enables you to update a wine’s composition. You may need to do this if the wine was loaded with the incorrect composition, the wrong blocks, or missing blocks. When you fix the composition, vintrace updates any blends that have occurred using the wine.

To identify wines that may need their composition corrected, you can run the [Bulk Stock Report](https://support.vintrace.com/hc/en-us/articles/360000814095-Composition-of-Bulk-Wine#h_01ET5Z7FNBF9ESXG8FT32EMB7K) with the Composition Details checkbox selected to a CSV file.

In order to correct a wine’s composition, you’ll need to have the [Advanced Data Management permission](https://support.vintrace.com/hc/en-us/articles/32303349421588#Permissions).

To fix a wine's composition:

1. Use vintrace’s Quick Search to find the wine.
2. From the wine’s Product page, select the Jobs tab.
3. Click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32328880520340) beside the bulk intake job and select View.

![20CH-BULK01_-_View_Bulk_Intake_20200828.png](https://support.vintrace.com/hc/article_attachments/32328898216596)

4. Click Correct.

![Bulk_Intake_-_Correct_Button_20200828.png](https://support.vintrace.com/hc/article_attachments/32328911873300)

5. Select the Composition tab.
6. Edit the composition as needed. As you update the wine’s composition, vintrace calculates the total composition. If you don’t need to include block information, you can click Calculator to use the Composition Calculator.

![Bulk_Intake_-_Composition_20220906.png](https://support.vintrace.com/hc/article_attachments/32328924832916)

7. In the Reason for Correction field, enter the reason for the change.
8. Click Save.
