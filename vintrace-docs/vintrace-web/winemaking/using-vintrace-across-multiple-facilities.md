# Using vintrace Across Multiple Facilities

One vintrace database can be used across multiple winery facilities to enable greater visibility of your company’s operations. You can restrict users’ access to a single facility, or [allow them to switch between different facilities](https://support.vintrace.com/hc/en-us/articles/360000825496-Managing-System-Users#ConfiguringaUserforMultipleFacilities).

If you need to use vintrace across multiple winery facilities, contact [sales@vintrace.com](mailto:sales@vintrace.com) for details.

## Switching Between Wineries

The name of the winery you’re currently viewing displays in the upper right.

![Current_Winery_20201019.png](https://support.vintrace.com/hc/article_attachments/32328955965844)
If you have [access to multiple wineries](https://support.vintrace.com/hc/en-us/articles/360000825496-Managing-System-Users#ConfiguringaUserforMultipleFacilities), you can easily switch between each facility by doing the following:

1. Click the ![User_Profile_Icon_20201019.png](https://support.vintrace.com/hc/article_attachments/32328936419732)user icon in the upper right.
2. Select Switch Winery.

![User_Menu_-_Switch_Winery_20201019.png](https://support.vintrace.com/hc/article_attachments/32328969388436)

The Switch Winery window displays.

![Switch_Winery_20201019.png](https://support.vintrace.com/hc/article_attachments/32328930229908)

3. From the Switch To list, select the facility that you want to switch to. If you want to work across all facilities, select [Reset].
4. Click Switch.
