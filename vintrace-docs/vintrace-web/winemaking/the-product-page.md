# The Product Page

The Product page displays the details for a batch or wine. You can view the Product page by using the [Quick Search](https://support.winery-software.com/hc/en-us/articles/360001922876#QuickSearch), or by selecting a vessel from the [Vessels page](https://support.winery-software.com/hc/en-us/articles/360001550655-The-Vessels-Page).

![19BLEND_20201019.png](https://support.vintrace.com/hc/article_attachments/32329148793620)

The Product page displays the following tabs:

- [General](#h_01EN18PYNVNGR2EHC37D558ZEG)
- [Lab](#h_01EN18Q41K4H6ED7Z8WX1K3Y1E)
- [Adds](#h_01EN18Q8J8KXVKJR7K47RZSN8S)
- [Notes](#h_01EN18QD8YRE5PM2TCZET1GFEQ)
- [Composition](#h_01EN18QJPQ5MS84Z7C3Q3H2CPW)
- [Fruit](#h_01EN18QRJ7VMHW5DA363RTFHZZ)
- [Costs](#h_01EN18QYCA37GHZD744TA6WHRT)
- [Jobs](#h_01EN18R474EY1KJEA1HA24VZPQ)

## General Tab

By default, the General tab displays the product’s latest lab results and details. You can customize the metrics and details displayed by clicking the ![Gear_Green_Icon_20200417.png](https://support.vintrace.com/hc/article_attachments/32329192143380) gear icon. To edit a detail, click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32329200738452) pencil icon.

![General_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329182172564)

You can also [add tags](https://support.vintrace.com/hc/en-us/articles/32303337168148) to the product from the General tab.

## Lab Tab

The Lab tab displays the results for the product. To add a lab result, click + Result.

![Lab_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329192455572)

You can also graph the results for the metric over time by clicking the ![Graph_Icon_20201019.png](https://support.vintrace.com/hc/article_attachments/32329149098772) graph icon.

## Adds Tab

The Adds tab lists all the additives that have been added to the product. You can view the Wine Impact Additions Report for an additive by clicking the additive’s name.

![Adds_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329149128468)

![Wine_Addition_Impact_Report_-_19Blend_-_KMBS_20201019.png](https://support.vintrace.com/hc/article_attachments/32329186908564)

## Notes Tab

The Notes tab lets you add and view comments and file attachments for the product.

![Notes_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329187045268)

## Composition Tab

The Composition tab displays the product’s composition by the selected attribute (e.g., varietal, sub-AVA, and vintage).

![Composition_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329192261652)

You can customize the details displayed by clicking the ![Gear_Green_Icon_20200417.png](https://support.vintrace.com/hc/article_attachments/32329192143380) gear icon.

![Customise_Composition_Details_20220902.png](https://support.vintrace.com/hc/article_attachments/32329149228564)

The composition details displayed on the Product page will be the same [composition details displayed in the mobile app](https://support.vintrace.com/hc/en-us/articles/32303347754004).

## Fruit Tab

The Fruit tab displays all the intakes associated with the product. You can click a docket number to view the intake delivery’s details. To view the block’s details, click the code in the Block column.

![Fruit_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329182300436)

## Costs Tab

The Costs tab displays the product’s running costs and its cost event history.

![Costs_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329172591508)

## Jobs Tab

The Jobs tab displays all jobs associated with the product. To view the work order’s details, click the number in the TWL column.

![Jobs_Tab_20201019.png](https://support.vintrace.com/hc/article_attachments/32329149207700)
