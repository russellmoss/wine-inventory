# Using the Batch Explorer

The Batch Explorer lets you view your current and depleted bulk wines by batch so that you can follow their journey from production to finished goods.

To access the Batch Explorer, click ![BulkWine_SidebarIcon.png](https://support.vintrace.com/hc/article_attachments/32329194853524) Bulk Wine in the sidebar.

At the top of the Batch Explorer page are filters that let you refine what’s displayed on the page.

![Bulk_Wine_List_Page.png](https://support.vintrace.com/hc/article_attachments/32329235409300)

The Status filter lets you include new, active, archived, and/or depleted batches; each status is indicated by a different color.

![Batch_Status_Top_Filter.png](https://support.vintrace.com/hc/article_attachments/32329235434260)

|  |  |  |
| --- | --- | --- |
| **COLOR** | **STATUS** | **DESCRIPTION** |
| Red | New | Batches that haven't been used |
| Green | Active | Batches where wine is currently in production |
| Grey | Archived | Batches that are no longer in use, or have been made inactive |
| White | Depleted | Batches where the wine has been fully dispatched, bottled, changed batch, or fully blended out. |

Icons at the top of the page enable you to access different functions.

|  |  |
| --- | --- |
| **ICON** | **DESCRIPTION** |
| Operations_Icon_20200403.png | Access vintrace operations. |
| Plus.png | Create a batch or [receive wine](https://support.vintrace.com/hc/en-us/articles/32303303281428). |

## Viewing a Bulk Product or Batch

Click a bulk product or batch to view its details.

![Bulk_Product_Page.png](https://support.vintrace.com/hc/article_attachments/32329235589652)

At the top of the page are the following icons:

|  |  |
| --- | --- |
| **ICON** | **DESCRIPTION** |
| Manage_Product_Allocations_Icon_20200930.png | Manage product allocations. |
| Reports_Icon_in_White_Circle_20201016.png | Run the Analysis Day Sheet, Cost, Stock Summary, and Wine/Juice Costing [reports](https://support.winery-software.com/hc/en-us/categories/360000090835-Reporting). |
| Plus.png | Add costs, create a packaging item, or create a work order. |

The page also displays the following tabs:

- In Production — Displays what’s in production for the bulk product or batch. You can customize the columns displayed on the tab, and export the data to Excel or PDF.

![In_Production_Tab.png](https://support.vintrace.com/hc/article_attachments/32329195050644)

You can also perform actions on selected vessels in the tab.

![Actions_Menu.png](https://support.vintrace.com/hc/article_attachments/32329235643412)

- Dispatched — Displays the dispatch history for the bulk product or batch. You can click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329189565844) to access actions for the job.

![Dispatched_Tab.png](https://support.vintrace.com/hc/article_attachments/32329174821012)

- Bottled — Displays the bulk product or batch’s bottling history. Click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329189565844) to view the job or perform a rollback.

![Bottled_Tab.png](https://support.vintrace.com/hc/article_attachments/32329203470228)

- Costs — Displays the costing history for the bulk product or batch.

![Costs_Tab.png](https://support.vintrace.com/hc/article_attachments/32329189553940)
