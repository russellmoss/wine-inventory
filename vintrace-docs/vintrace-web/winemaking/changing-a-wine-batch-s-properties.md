# Changing a Wine Batch's Properties

You can [change a wine batch’s properties](#h_01EKJN7KP74MK2S75SS88VDS92), or [edit its batch code](#h_01EKJN7T0R3JYVNP57HRPHQ4R4).

## Editing a Wine Batch’s Property

To edit a wine batch’s properties:

1. Use vintrace’s Quick Search to find the wine batch.
2. Click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32329029082516)pencil icon beside the property you want to change.

![Editing_Wine_Batch_Property_-_20CHCS-01_20201001.png](https://support.vintrace.com/hc/article_attachments/32329061989652)

A window for editing the property displays. The window’s label and available values will depend on the property you’re editing.

![Edit_Property_Windows_20201001.png](https://support.vintrace.com/hc/article_attachments/32329029099924)

3. Edit the property, then click Save.
   - Some properties may need additional permissions to update, e.g. to update the Program you need the [Advanced data management permission](https://support.vintrace.com/hc/en-us/articles/32303349421588#Permissions).
   - Note that for some properties - e.g. Program -  you may get a warning message![](https://support.vintrace.com/hc/article_attachments/40033777720340)
   - When to proceed: Only to correct an administrative error (e.g., the batch was assigned to the wrong program by mistake at creation).

     When to ignore the warning: Only if there was no program assigned to the batch originally.

     When to avoid: Never use this method to "reclassify" wine from Program A to Program B as part of a standard winemaking process.
   - If you have already tested the change in your sandbox, and/or you are confident that you want to make the change then click OK.

## Editing the Batch Code

Changing the batch code updates all historical references to that code. If you need to move a wine from one batch to another, use the Change Batch operation.

To edit the batch code:

1. Use vintrace’s Quick Search to find the wine batch.
2. Click the ![Change_Batch_Icon_20201001.png](https://support.vintrace.com/hc/article_attachments/32329052083092) change batch icon.

![Editing_Batch_Code_20201001.png](https://support.vintrace.com/hc/article_attachments/32329062041236)

3. Select the Edit the Batch Code for 'x' Vessels in the Batch option, then click OK.

![Change_Batch_-_Edit_Batch_Code_20201001.png](https://support.vintrace.com/hc/article_attachments/32329052096276)

The Simple Wine Batch window displays.

![Update_Simple_Wine_Batch_-_20CHCS-01_20201001.png](https://support.vintrace.com/hc/article_attachments/32329062054548)

4. Edit the batch code, then click Save.
