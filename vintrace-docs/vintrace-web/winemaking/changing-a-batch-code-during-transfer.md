# Changing a Batch Code During Transfer

To change a wine’s batch code during a [transfer](https://support.vintrace.com/hc/en-us/articles/32303355314708):

1. From the To section of the window, click the ![X_in_Gray_Circle_20200330.png](https://support.vintrace.com/hc/article_attachments/32328572073748) remove icon beside the Batch field.

![Racking_-_Removing_Destination_Batch_Code_20201001.png](https://support.vintrace.com/hc/article_attachments/32328566428948)

2. Click the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32328572055956) search icon to select a different batch code, or click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328541385748) add icon to add a new batch code.

![Racking_-_Selecting_or_Adding_New_Batch_Code_20201001.png](https://support.vintrace.com/hc/article_attachments/32328582654484)
