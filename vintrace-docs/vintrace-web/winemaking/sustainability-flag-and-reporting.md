# Sustainability Flag and Reporting

After a vineyard’s sustainability certification number is entered, all past and future bulk wines whose fruit comes from the vineyard will be marked as sustainable.

## Setting a Vineyard as Sustainable

To specify that a vineyard is a sustainable source:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329184474260) More Options in the sidebar.
2. From the Harvest tile, click Block Overview.
3. Search for the vineyard.
4. Select the Vineyard/Grower tab.
5. Click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32329147204500)pencil beside Vineyard.

![Block_Overview_-_Editing_Vineyard_-_Sustainability_Cert_20200929.png](https://support.vintrace.com/hc/article_attachments/32329170357908)

6. In the Sustainable Cert # field, enter the vineyard’s sustainability certification number. In New Zealand, enter the information in the SWNZ Code field.

![Vineyard_Update_-_Sustainable_Cert_20200929.png](https://support.vintrace.com/hc/article_attachments/32329170467732)

7. Click Save.

You can also [create a new vineyard](https://support.vintrace.com/hc/en-us/articles/32301351350420-Managing-Vineyards) and set it as sustainable. Wines produced from fruit that's received from the new vineyard will be marked as sustainable.

TIP: You can display details related to sustainability on the [Composition tab of the product page](https://support.vintrace.com/hc/en-us/articles/32303310460948-The-Product-Page#h_01EN18QJPQ5MS84Z7C3Q3H2CPW) by clicking the ![Gear Green Icon 20200417.png](https://support.vintrace.com/hc/article_attachments/32672928923668) gear icon and including the Sustainable Cert # and Sustainable Source fields in the Selected list.

![Composition - Displaying Sustainable Details 20241203.png](https://support.vintrace.com/hc/article_attachments/32672922371604)

## Reporting on Sustainable Wines

You can run the Bulk Stock Report to report on sustainable wines.

![Bulk_Stock_Report_-_Sustainable_CSV_20200929.png](https://support.vintrace.com/hc/article_attachments/32329165503636)

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329156847892) Reports in the sidebar.
2. Select Bulk Wine. The Bulk Stock Report displays.

![Winery_Reports_-_Bulk_Wine_-_Bulk_Stock_Report_20200929.png](https://support.vintrace.com/hc/article_attachments/32329170503956)

3. Select the Composition Details checkbox.
4. Select any of the following from the list: Sustainable Source, Sustainable Cert #, or Vineyard [Cert#]. For New Zealand, select either SWNZ Code, or Vineyard [SWNZ#].

![Bulk_Stock_Report_-_Composition_Details_20200929.png](https://support.vintrace.com/hc/article_attachments/32329165578004)

5. Specify the filters and options for the report.
6. Click Generate or Email.
