# Viewing a Tank Map

A tank map displays a bird’s-eye view of your [tanks](https://support.vintrace.com/hc/en-us/articles/32301359425428)’ layout.

![Tank_Farm_20201030.png](https://support.vintrace.com/hc/article_attachments/32329204310420)

Refer to [Selecting a Tank Map](#h_01EP7F11SZR7SDMTD23XK3GCDD) to view a different tank map. For details on setting up a tank map, refer to our [Setting Up a Tank Map article](https://support.winery-software.com/hc/en-us/articles/360000826696).

By default, the tank map displays using the [size selected when it was set up](https://support.winery-software.com/hc/en-us/articles/360001842276). You can zoom in and out by using the plus and minus controls in the lower right. Clicking Reset displays the tank map at its default size.

![Zoom_Controls_20201030.png](https://support.vintrace.com/hc/article_attachments/32329204360724)

You can also reposition the tank map, by clicking and dragging it.

![Moving_Tank_Map_20201030.gif](https://support.vintrace.com/hc/article_attachments/32329236595732)

Each tank displays a first circle in the upper right with the number of jobs for the tank (based on the filter’s selected date range). The bar on the left side of the tank’s card provides a visual indication of the tank’s fill level. The icon below the tank’s name indicates its contents.

![Tank_Overview_20201030.png](https://support.vintrace.com/hc/article_attachments/32329195904404)
To view a tank’s details and metrics, hover over the card.

![Tank_Map_-_Tank_Details_T5-03_20201030.png](https://support.vintrace.com/hc/article_attachments/32329190871316)

The details and metrics displayed will depend on [how the tank map was set up](https://support.winery-software.com/hc/en-us/articles/360001842276).

The ![Three_Horizontal_Lines_20201030.png](https://support.vintrace.com/hc/article_attachments/32329196188180) icon in the upper right of the tank's card lets you access different actions for the tank including viewing and editing the tank, creating a new work order, and recording an operation.

![Actions_Menu_20201030.png](https://support.vintrace.com/hc/article_attachments/32329209386004)

## Selecting a Tank Map

To view a tank map:

1. Click ![Tank_Map_20201029.png](https://support.vintrace.com/hc/article_attachments/32329195968532) Tank Map from the sidebar. If the tank map that you want to view isn’t displayed, complete the following steps.
2. Click the ![Filter_20200323.png](https://support.vintrace.com/hc/article_attachments/32329204490516) filter icon.
3. From the View By section, select Tank Map from the first list.
4. From the second list, select the tank map that you want to view.

![View_by_Tank_Map_20201030.png](https://support.vintrace.com/hc/article_attachments/32329204504340)

5. Specify the date range of the jobs that you want to include.
6. Click Search.

## Editing a Tank Map

To edit the tank map’s configuration, click the ![Gear_Green_Icon_20200417.png](https://support.vintrace.com/hc/article_attachments/32329219793044) gear icon in the upper right, then select Edit Tank Map.

![Edit_Tank_Map_20201030.png](https://support.vintrace.com/hc/article_attachments/32329196163860)

Refer to our [Setting Up a Tank Map article](https://support.winery-software.com/hc/en-us/articles/360000826696) for details on configuring a tank map.

## Changing a Tank Map’s Refresh Interval

To change how frequently a tank map is updated to reflect the most recent data:

1. Click the ![Gear_Green_Icon_20200417.png](https://support.vintrace.com/hc/article_attachments/32329219793044) gear icon in the upper right.
2. Select Edit Refresh Setting. The Auto Refresh Options window displays.

![Auto_Refresh_Options_20201030.png](https://support.vintrace.com/hc/article_attachments/32329196075028)

3. Enter the number of minutes that you want the tank map to refresh.
4. Click Save.

## Viewing a Saved Search

You can also view the results of a [saved search](https://support.winery-software.com/hc/en-us/articles/360001842276#SavedSearches) on the Tank Map page. Tanks included in results don’t display in any particular layout.

To view the results of a saved search:

1. Click ![Tank_Map_20201029.png](https://support.vintrace.com/hc/article_attachments/32329195968532) Tank Map from the sidebar.
2. Click the ![Filter_20200323.png](https://support.vintrace.com/hc/article_attachments/32329204490516) filter icon.
3. From the View By section, select Saved Search from the first list.
4. From the second list, select the name of the saved search.

![View_by_Saved_Search_-_Fermenting_20201030.png](https://support.vintrace.com/hc/article_attachments/32329196119828)

5. Specify the date range of the jobs that you want to include.
6. Click Search.

## Highlighting Tanks

You can highlight tanks that meet the filters of a [saved search](https://support.winery-software.com/hc/en-us/articles/360001842276#SavedSearches) in either the tank map, or in the results of a saved search. For example, you might want to highlight tanks that are fermenting on your tank map. Highlighted tanks display a blue border around them. In the example below, tanks T1-13 and T1-15 are highlighted.

![Highlighted_Tanks_20201030.png](https://support.vintrace.com/hc/article_attachments/32329196176916)

In the example below, we’re displaying the tanks that meet the filters in the 2017 saved search. We’re then highlighting the tanks that meet the filters in the Fermenting saved search.

![View_by_Saved_Search_-_Highlight_by_Fermenting_Cropped_20201103.png](https://support.vintrace.com/hc/article_attachments/32329236947476)

To highlight tanks:

1. View the tanks in a tank map, or by using a saved search.
2. From the Highlight By list, select the saved search that will determine which tanks are highlighted.

![Highlight_By_20201030.png](https://support.vintrace.com/hc/article_attachments/32329191116692)

3. Click Search.
