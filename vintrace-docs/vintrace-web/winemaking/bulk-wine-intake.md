# Bulk Wine Intake

## Accessing the Bulk Intake Window

There are a number of ways to access the Bulk Intake operation. You can click the Operations icon, then select Bulk Intake from the following:

- [The Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- [The Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)

You can also add a Bulk Intake job to a [work order](https://support.vintrace.com/hc/en-us/articles/32303315610388).

![Bulk_Intake_-_General_20220722.png](https://support.vintrace.com/hc/article_attachments/32329024539156)

The Bulk Intake window has the following tabs:

- [General](#h_01EKG4WQ13M16AED1ETWXR3NHT)
- [Notes](#h_01EKG4WXM7J2MHTP0Y09DNRGX7)
- [Documents](#h_01EKG4X32VVJW1KF180MPCXT3V)
- [Schedule Details](#h_01EKG4X97YVBX2HTDNJKSXM8FX)

## General Tab

The General tab includes several sub-tabs:

- Delivery Details — Information about the delivery, including its [purchase order](https://support.vintrace.com/hc/en-us/articles/32303315399444).

![Bulk_Intake_-_General_-_Delivery_Details_20220722.png](https://support.vintrace.com/hc/article_attachments/32329015014932)

- Wine Details — The wine’s batch, fraction type, ferment state, and type.

![Bulk_Intake_-_General_-_Wine_Details_20200930.png](https://support.vintrace.com/hc/article_attachments/32329006542740)

- Vessels — Information about the wine’s vessel and batch.

![Bulk_Intake_-_General_-_Vessels_20200930.png](https://support.vintrace.com/hc/article_attachments/32329024419604)

- Tiraged — If you have the Sparkling module and the wine comes tiraged, you can specify the cages and bins for the wine.

![Bulk_Intake_-_General_-_Tiraged_20200930.png](https://support.vintrace.com/hc/article_attachments/32329041076500)

- Composition — Block information for the wine and its percentage of the composition. To calculate the import composition, click Calculate.

![Bulk_Intake_-_General_-_Composition_20220726.png](https://support.vintrace.com/hc/article_attachments/32329036319380)

- Costing & Labs — If you’re using the Inventory module, or have costing enabled without the Inventory module, you can enter the cost and freight information in the Costing & Labs tab. vintrace will track the bulk wine’s cost as it moves into blends.

US customers must select a Tax State and Tax Class so that vintrace can correctly report the bulk wine intake on your TTB Compliance report. If you bring the wine in undeclared, the delivery will NOT appear in Section A. Instead, the volume will be added to Part VII - IN FERMENTERS END OF PERIOD (Gallons).

If you’ve been given analysis data, you can enter it in the Costing & Labs sub-tab.

![Bulk_Intake_-_General_-_Costing_and_Labs_20200930.png](https://support.vintrace.com/hc/article_attachments/32329014887444)

- Additive Summary — If you’ve been given the wine’s rate of addition, you can enter it in the Additive Summary sub-tab. This is important as it tells vintrace the starting point for additions. As you make your own additions to the wine, vintrace will use this as a base when calculating the total additions on wines that you produce from this bulk wine.

![Bulk_Intake_-_General_-_Additive_Summary_20200930.png](https://support.vintrace.com/hc/article_attachments/32329041113492)

## Notes Tab

You can enter notes, or click the to include a [standard note](https://support.vintrace.com/hc/en-us/articles/32301315435028) with the bulk intake. Any notes that you add will be permanently saved with the intake. The cellar can enter information about the delivery in the Operator Notes field.

![Bulk_Intake_-_Notes_20200930.png](https://support.vintrace.com/hc/article_attachments/32329006694164)

## Documents Tab

In the Documents tab you can attach files such as the original BOL that was sent with the wine. These will be permanently saved with the intake.

![Bulk_Intake_-_Documents_20200930.png](https://support.vintrace.com/hc/article_attachments/32329006717332)

## Schedule Details Tab

You can record how long the job took to complete in the Schedule Details tab.

![Bulk_Intake_-_Schedule_Details_20200930.png](https://support.vintrace.com/hc/article_attachments/32329006750740)
