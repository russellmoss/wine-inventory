# Bulk Wine Search

The Bulk Wine Search allows you to quickly find bulk products using criteria such as the batch code, batch owner, and vintage.

## Accessing the Bulk Wine Search

The Bulk Wine Search is accessible from the following operations:

- [Bulk Wine Intake](https://support.vintrace.com/hc/en-us/articles/32303303281428)
- Multi Additions
- Many-to-One Transfer
- One-to-Many Transfer
- Multi Topping

## Using the Search Filters

By default, the Bulk Wine Search displays the following search filters:

- Batch code
- Batch owner
- Vintage
- Wine color
- Designated varietal
- Designated Sub AVA (Region)
- Product state
- Winery (for multi-winery mode only)

![Bulk_Wine_Search_-_Default_Filters_20201009.png](https://support.vintrace.com/hc/article_attachments/32329067564564)

## Adding a Filter

If you want to find your wines using a filter that’s not included in the defaults, you can add other filters.

To add a filter:

1. Click More Options.

![Bulk_Wine_Search_-_More_Options_Link_20201009.png](https://support.vintrace.com/hc/article_attachments/32329067534356)

2. Click Add a Filter.

![Bulk_Wine_Search_-_Add_a_Filter_20201009.png](https://support.vintrace.com/hc/article_attachments/32329067596052)

The additional filters and their descriptions display.

![Bulk_Wine_Search_-_Add_a_Filter_-_Filter_List_20201009.png](https://support.vintrace.com/hc/article_attachments/32329044027668)

3. To add a filter, click its name.
4. Specify the values that you want to include for the filter.
5. Click Search.

If you use the filter(s) often, you can create a [saved search](#h_01EM7BTA97NPR5NMRZSFT0MGRV).

## Saving Search Filters

If you tend to use the same search criteria in the Bulk Wine Search window, you can name and save the search. This enables you to apply the filter to the page without having to re-specify the filter columns and values. For example, you might want to add a saved search for a filter that displays wines in a certain product state.

To create a saved search in the Bulk Wine Search window:

1. Add the filter.
2. Enter a name for the search.

![Bulk_Wine_Search_-_Saved_Search_20201009.png](https://support.vintrace.com/hc/article_attachments/32329038986132)

3. Click Save.

You can also access saved searches from the Winery Reports window’s Custom Reports category. Click the ![Run_Search_Icon_202010109.png](https://support.vintrace.com/hc/article_attachments/32329067604116) to view the results in the Bulk Wine Search window.
![Winery_Reports_-_Custom_Reports_20201009.png](https://support.vintrace.com/hc/article_attachments/32329059557012)

## Changing the Columns Displayed

To change the columns that are displayed in the search results:

1. Click More Options.

![Bulk_Wine_Search_-_More_Options_Link_20201009.png](https://support.vintrace.com/hc/article_attachments/32329067534356)

The columns included in the search results are listed in the Selected column.

![Bulk_Wine_Search_-_Selected_Columns_20201009.png](https://support.vintrace.com/hc/article_attachments/32329026865428)

2. Use the following buttons to change the columns displayed.

|  |  |
| --- | --- |
| Select_Icon_20201009.png | Includes selected column in the search results |
| Select_All_Icon_20201009.png | Includes all columns in the search results |
| Remove_Icon_20201009.png | Removes selected column from the search results |
| Remove_All_Icon_20201009.png | Removes all columns from the search results |

## Using Wines in an Operation

After you’ve located the products, you can use them for the operation that you launched the Bulk Wine Search window from. For example, if you launched the Bulk Wine Search from the Multi Topping window, the selected wines will be used for the Multi Topping operation.

To use the product for an operation

1. Select the checkbox beside the wine.
2. Click Use Selection.

![Bulk_Wine_Search_-_Use_Selection_20201009.png](https://support.vintrace.com/hc/article_attachments/32329059706132)
