# Adding Notes and Attachments to Wine

This feature is only available in the new vintrace.
In order to add product notes and attachments to a wine, you’ll need permission to view and add product notes.

You can view a wine’s notes and attachments from the product page’s Notes tab. The number beside the tab indicates how many notes have been added to the wine. If there are any attachments included with a note, a paperclip displays in the note. You can hover over the paperclip to view how many attachments are included with the note.

![Product_19Blend_-_Number_of_Notes_and_Attachments_Annotated_20200916.png](https://support.vintrace.com/hc/article_attachments/32329277768596)

To add notes and attachments to wine:

1. Display the wine’s product page by either using vintrace’s Quick Search feature, or by selecting the wine from the Vessels page.
2. Select the Notes tab.
3. Click Add Note.

![Product_2019Chard1_-_Notes_Tab_Annotated_20200916.png](https://support.vintrace.com/hc/article_attachments/32329244456980)

The Add Product Note window displays.

4. Enter your notes in the field.
5. To upload an attachment, click Upload File and select the files that you want to attach to the note.

The maximum allowed file size for each file 20Mb.

6. Click Save.

## Notes and Attachments for Blended Wines

If a wine with notes and attachments is blended and its volume is 25% or more of the resulting wine’s total volume, the notes and attachments from the source wine are included in the resulting blend. For example, suppose you have two source wines that both have notes and attachments.

![Notes_8jpg.jpg](https://support.vintrace.com/hc/article_attachments/32329225235348)

![Notes_7.jpg](https://support.vintrace.com/hc/article_attachments/32329252598804)

The two wines are blended into an empty tank. The first wine’s volume is 80% of the blend, and the second wine’s volume is 20%.

The blended wine’s notes will be from the first wine (i.e., the one that makes up 80% of the blend). The notes from the second wine do not make it into the blended wine.

![Notes_10.jpg](https://support.vintrace.com/hc/article_attachments/32329231708564)

If you edit the blended wine’s notes, vintrace updates the note in the source wine.
