# Generating a Wine's History

You can run the Operation History report to generate the history of a wine in either a PDF or Excel spreadsheet.

![Operation_History_Report_-_New_Blend_20200923.png](https://support.vintrace.com/hc/article_attachments/32328953727764)

To run the Operation History report:

1. Click Reports in the sidebar.
2. Select Product History.

![Winery_Reports_-_Product_History_-_Operation_History_20200923.png](https://support.vintrace.com/hc/article_attachments/32328934632468)

3. Specify the filters and options for the report.

A few notes:

- Be sure to enter a reasonable Relevancy Weighting to filter out small blends or toppings from wines. For example, setting the Relevancy Weighting to 2 would filter out the history of wines whose composition of the final blend comprises less than 2%.
- Be sure to select only the operation that you’re interested in seeing. You can easily exclude things such as analysis or additions if you’re only looking for operations that impact your composition.

4. Click Generate or Email.
