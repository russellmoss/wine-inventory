# Editing a Vessel

You can edit a barrel, bin, tank, or other vessel that you’ve added from the Winery Setup window (Set up Options > Vessels):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329120882068) Set Up in the sidebar.
2. Click Vessels.

![Setup_-_Vessels_20200916.png](https://support.vintrace.com/hc/article_attachments/32329095429652)

3. From the vessel type’s tile, click Configure. For example, if you want to edit a barrel, click the Configure button in the Barrels tile.
4. Click on the vessel.
5. Edit the vessel’s details as needed.
6. Click Save.
