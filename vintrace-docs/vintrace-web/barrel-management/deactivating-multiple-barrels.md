# Deactivating Multiple Barrels

When you’ve sold some of your barrels, or deemed them no longer fit for use, you can use a [barrel treatment](#h_42e38444-be75-41c9-8e9a-c96885eea0f9) in vintrace to [set them to inactive](#h_01JDAF90W9JZQ3NYF7AF9M6BR6) and prevent them from being used.

## Creating a Barrel Treatment

To deactivate multiple barrels, you can create a barrel treatment. You can [set up a barrel treatment](https://support.vintrace.com/hc/en-us/articles/32301341352084) from the Winery Setup window (Setup Options > Treatment > Treatment (Barrel)).

To set up a barrel treatment for deactivating barrels:

1. Click ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32329103219092) Set Up in the sidebar.
2. Click Treatments.
3. From the Barrels Treatments tile, click Configure.
4. Click New Barrel Treatments.
5. In the Barrel Treatment Definition window, select the *Deactivate Barrels When Used* option.

![Barrel_Treatment_Definition_-_Sold_Barrels_Deactivated_20200430.png](https://support.vintrace.com/hc/article_attachments/32329057333268)

TIP: If you need to create a barrel treatment to reactivate barrels, [create a separate barrel treatment](https://support.vintrace.com/hc/en-us/articles/32301341352084) with the Reactivate Barrels When Used option selected.

## Deactivating Barrels Using the Barrel Treatment

To apply the [treatment that you created](#h_42e38444-be75-41c9-8e9a-c96885eea0f9) to your barrels you can either:

- Schedule it on a [work order](https://support.vintrace.com/hc/en-us/articles/32303315610388) using the Treatment (Barrel) job.
- Select the Treatment (Barrel) operation and set the Treatment to the treatment that you created.

Whichever option you choose, you’ll need to select the barrels that you want to deactivate.

![Treatment_Barrel_-_Sold_Barrels_20200430.png](https://support.vintrace.com/hc/article_attachments/32329081511956)

## Reporting on Deactivated Barrels

You can report on deactivated barrels by using the Barrel Treatment Report and selecting the [treatment that you created](#h_42e38444-be75-41c9-8e9a-c96885eea0f9) from the Barrel Treatment list.

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329081457428) Reports in the sidebar.
2. Select Operations.
3. Scroll down to the Barrel Treatment Report.
4. From the Barrel Treatment list, select the treatment that you added to deactivate barrels.

![Winery_Reports_-_Operations_-_Barrel_Treatment_Report_-_Sold_Barrels_20200430.png](https://support.vintrace.com/hc/article_attachments/32329081541140)
