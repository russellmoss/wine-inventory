# Rack and Return of Barrels

## Scheduling Rack and Return of Barrel Groups

To schedule the rack and return of a barrel group:

1. [Create a work order](https://support.vintrace.com/hc/en-us/articles/32303315610388).
2. Click Add Job.
3. Select Transfer/Rack/Blend.
4. From the Transfer/Racking list, select *Rack & Return*.

![Racking_-_Rack___Return_20200429.png](https://support.vintrace.com/hc/article_attachments/32328967460372)

5. Select the source barrel group and the temporary vessel.
6. Click Add to Work Order.
7. Click Save As > Ready to save the work order.

## Completing Rack and Return of Barrel Groups

To complete the rack and return:

1. View the work order with the rack and return job.
2. Click Complete.
3. From the Racking window, click the ![Edit_Barrel_Group_Icon_20200429.png](https://support.vintrace.com/hc/article_attachments/32328967420052) Edit Barrel Group icon to configure what the barrel group looks like after the rack and return. In the example below, two barrels were set to empty on return due to lees. The Fill Type for the two barrels has been set to Empty, while the others are full.

![Advanced_Barrel_Fill_Info_-_Rack___Return_Example_20200429.png](https://support.vintrace.com/hc/article_attachments/32328959000340)

4. Click OK. In the Racking window, the temporary vessel’s In amount is automatically calculated based on the volume that left the barrel group and the volume that was lost.

![Rack___Return_Example_-_Temp_Vessel_In_Value_Auto_Calculated_20200429.png](https://support.vintrace.com/hc/article_attachments/32328959034516)

5. Select a Loss Reason.
6. Click Now+Save.
