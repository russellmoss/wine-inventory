# Changing Barrel Locations

## Moving an Individual Barrel

To change the location of a single barrel:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328757240852) Set Up in the sidebar.
2. Search for the barrel.
3. Click the barrel.
4. Select the Location tab.

![Barrel_Update_-_Location_20210615.png](https://support.vintrace.com/hc/article_attachments/32328722135316)

5. Click Add Line.
6. Specify the barrel’s Winery Building, Area, Bay, and/or Position.
7. Click Save.

## Moving Multiple Barrels with a Barrel Treatment

You can [set up a barrel treatment](#h_c5b60cb4-fcf0-446f-be5e-c9ddc347a1b4) to update the storage area, bay, and stack of individual barrels in a single operation, then [apply the treatment to the barrels you want to move](#h_6199a432-036f-4f95-97bc-5ba730b4e300).

### Setting Up a Barrel Treatment

You can set up a barrel treatment from the Winery Setup window (Setup Options > Treatment > Treatment (Barrel)).

To set up a barrel treatment:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328757240852) Set Up in the sidebar.
2. Click Treatments.
3. From the Barrels Treatments tile, click Configure.
4. Click New Barrel Treatments.
5. In the Barrel Treatment Definition window's Treatment tab, specify the details for the new barrel treatment.

![Barrel_Treatment_Definition_-_Move_to_Caves_-_Treatment_20220121.png](https://support.vintrace.com/hc/article_attachments/32328783837716)

6. From the Location tab, click Add Line, then enter the Storage Area, Bay, Stack, or To Stack values.

![Barrel_Treatment_Definition_-_Location_20220121.png](https://support.vintrace.com/hc/article_attachments/32328752475796)

7. Click Save.

### Apply the Treatment to Barrels

After you’ve set up the barrel treatment, you can apply it to the barrels that you want to move.

1. From the Vessels page, click the ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32328757014036) Operations icon.
2. From the Treatments section, click Treatment (Barrel). The Treatment (Barrel) window displays.
3. From the Treatment list, select the barrel treatment that you created.
4. Click Add.

![Treatment_Barrel_-_Move_to_Cave_20200429.png](https://support.vintrace.com/hc/article_attachments/32328752370068)

5. From the Search for Barrels window, search for the barrels that you’d like to move, then click Use Selection. The barrels are added to the Treatment (Barrel) window’s Barrel List.
6. Click Save.
