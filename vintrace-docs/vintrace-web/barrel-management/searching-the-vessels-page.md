# Searching the Vessels Page

You can refine what’s displayed on [the Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924) by using a [saved search](#h_01EH7Q2JVPCQ4872T14MQT5EY2), [creating a new filter](#h_01EH7Q2WN8NQ7GKCY6FK9BH414), or [applying a filter to a displayed column](#h_01EH7Q3S3KZSA3FRM5GW7F1C1Z). Any saved search can be [pinned to the Dashboard](#h_01EH7Q3D8TE93GARKXDK4MHYEG) for easy access. [Saved searches](https://support.vintrace.com/hc/en-us/articles/32301304791316-Configuration-for-Multi-Winery-Support#managing_item_availability) can be filtered/assigned at the winery level through set-up.

## Saved Searches

A saved search is when you [create a filter](#h_01EH7Q2WN8NQ7GKCY6FK9BH414) and give it a name. This enables you to apply the filter to the page without having to re-specify the filter columns and values. It's particularly helpful for filters that you use often. For example, you might want to add a saved search for a filter that displays wines in a certain product state.

Saved searches display at the top of the Vessels page.

![Saved_Searches_20200902.png](https://support.vintrace.com/hc/article_attachments/32328594993044)

You can also [pin a saved search to the Dashboard](#h_01EH7Q3D8TE93GARKXDK4MHYEG) for easy access.

## Creating a New Filter

To create a new filter on the Vessels page:

1. Click the ![Filter_20200323.png](https://support.vintrace.com/hc/article_attachments/32328594766868) at the top of the page.

![Filter_-_Adding_20200901.png](https://support.vintrace.com/hc/article_attachments/32328626087444)

2. Click Add a Filter.
3. Select the column to which you'd like to apply a filter. You can filter the list of columns displayed by entering the characters below the Name column heading. In the example below, entering "ow" returns all column names that include. "ow".

![Search_Filters_-_Select_Property_20200604.png](https://support.vintrace.com/hc/article_attachments/32328600926484)

4. Click the down arrow in the Values column and select the values that you want to include for the filter. The number of values you've selected displays in the Values column.

![Search_Filters_20200604.png](https://support.vintrace.com/hc/article_attachments/32328626158100)

5. You can add filters to additional columns by clicking Add a Filter. To remove a filter, click the ![X_in_Green_Circle_20200604.png](https://support.vintrace.com/hc/article_attachments/32328618232596) X in the Action column of the filter.
6. To apply the filter to the Vessels page, click Apply. The icon on the Vessels page displays ![Filter_Applied_20200323.png](https://support.vintrace.com/hc/article_attachments/32328632640020) to indicate that a filter has been applied. You can click the icon to change the filters, or save the filter for future use. To save the filter for future use, click Save. The filter will be included in the list of filters in the Use a Saved Search list.

## Filtering Displayed Columns

You can also apply a filter to any column displayed on the Vessels page.

To change the columns displayed or their order, refer to our [Customizing the Vessels Page article](https://support.vintrace.com/hc/en-us/articles/32301323976084).

To filter a displayed column:

1. Click the ![Column_Filter_Icon_20200901.png](https://support.vintrace.com/hc/article_attachments/32328610427284)beside the column header.
2. Specify the values you want to include in the filter.
3. Click the ![Checkmark_20200901.png](https://support.vintrace.com/hc/article_attachments/32328610449172) checkmark.

![Filtering_Column_20200901.png](https://support.vintrace.com/hc/article_attachments/32328618285716)

The column header of filtered columns display in different color and the icon at the top of the Vessels page displays ![Filter_Applied_20200323.png](https://support.vintrace.com/hc/article_attachments/32328632640020)to indicate that a filter has been applied.

![Filtered_Column_20200901.png](https://support.vintrace.com/hc/article_attachments/32328626262932)

You can click the ![Filter_Applied_20200323.png](https://support.vintrace.com/hc/article_attachments/32328632640020) to change the filters, or save the filter for future use. Saved filters are included in the list of filters in the Use a Saved Search list.

## Pinning a Saved Search to the Dashboard

To pin a saved search to the My Searches dashlet on the Dashboard, click the ![Heart_White_20200731.png](https://support.vintrace.com/hc/article_attachments/32328601082900) heart in the Search Filters window. The icon changes to a ![Heart_Green_20200731.png](https://support.vintrace.com/hc/article_attachments/32328610491540) to indicate that the search has been pinned to the Dashboard.

![Pinned_Search_on_Dashboard_20200827.png](https://support.vintrace.com/hc/article_attachments/32328618344852)
