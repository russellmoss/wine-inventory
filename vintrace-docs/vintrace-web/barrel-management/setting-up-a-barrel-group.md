# Setting Up a Barrel Group

You can add a barrel group from numerous windows in vintrace including the Extraction, Racking, and Multi Transfer windows.

To add a new barrel group, click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329140142228) beside the Vessel field. You can also create a barrel group from the Winery Setup window (Setup Options > Vessels > Barrel Groups).

From the Barrel Group window, you’ll need to add barrels by doing any of the following:

- To search and select an existing barrel, click Search.
- To type or copy/paste your barrel codes, click List.
- To enter the barrel code, click Add Line, then either enter the barrel code or click ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32329151581204) to search for the barrel.
