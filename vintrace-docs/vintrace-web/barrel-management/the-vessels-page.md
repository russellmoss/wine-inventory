# The Vessels Page

The Vessels page lists your bins, barrels, and tanks. To access the Vessels page, click ![Vessels_Menu_Option_20200402.png](https://support.vintrace.com/hc/article_attachments/32329167181460) Vessels in the sidebar.

At the top of the Vessels page are [filters](#h_3203f361-6320-4d92-bcd5-f92fba66da53) that let you refine what's displayed on the page. You can also apply a [saved search](https://support.vintrace.com/hc/en-us/articles/32301344204308) to the page. The ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32329175773076) Operations icon lets you access a number of different functions in vintrace. The ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329162261908) Plus icon lets you add vessels and work orders, as well as receive fruit and wine. You can [customize the details that are displayed on the Vessels page](https://support.vintrace.com/hc/en-us/articles/32301323976084) by clicking the ![Gear_Green_Icon_20200417.png](https://support.vintrace.com/hc/article_attachments/32329130239892) icon.

![Vessels_Page_Overview_20200603.png](https://support.vintrace.com/hc/article_attachments/32329176042004)

The vessels that meet your filters display both a wine type and vessel type icon in the Wine Type column. The wine type icons display "J" if it's a juice, and "W" if it's a wine. You can hover over the icons to view their description, or refer to the tables below.

The Batch column displays a link that you can click to view the batch's Product page.

You can [customize the Vessels page](https://support.vintrace.com/hc/en-us/articles/32301323976084) to include additional details.

At the bottom of the page is an [Actions menu](#h_e6c748ea-44e9-46b7-bab1-28576b6a3028) and options for [exporting the vessel details](#h_04e8f2f9-b4f3-43f5-98d2-dbf3a92648d6).

## Filtering the Vessels Page

There are several ways that you can refine the scope of what's displayed on the Vessels page using vintrace's filters.

At the top of the page are buttons that let you quickly filter the vessels by whether the vessel is in use or empty, and by the type of vessel. You can also select whether individual barrels or archived vessels are listed.

![Filters_Overview_20200604.png](https://support.vintrace.com/hc/article_attachments/32329144327316)

If you have an existing filter that you'd like to use, select it from the Use a Saved Search list. You can also [create a new filter.](https://support.vintrace.com/hc/en-us/articles/32301344204308)

## Actions Menu

When you select one or more vessels on the page, the Actions menu in the lower left becomes available.

![Actions_Menu_20200604.png](https://support.vintrace.com/hc/article_attachments/32329162584852)

You can select a vessel by selecting the checkbox beside it. To select all vessels on the page, select the checkbox that's displayed to the left of the column headings.

![Select_All_Checkbox_20200604.png](https://support.vintrace.com/hc/article_attachments/32329162442644)

The options available in the Actions menu will depend on the vessels selected.

## Printing Barcodes for Vessels

When you have barcodes on your vessels, you can scan the barcodes any time you need to perform functions such as recording additions, moving barrels, or submitting work orders that include the vessel.

To print barcodes for vessels:

1. Select the vessels you want to print barcodes for.
2. From the Actions menu, select either Print QR Codes, or Print Barcodes.

![Actions_-_Print_Options_20200604.png](https://support.vintrace.com/hc/article_attachments/32329130688916)

3. Select the option you want to use for printing.

## Exporting Vessel Details

You can export the details of the vessels to Excel (i.e., XLS), PDF, or a comma-separated file (i.e., CSV). When you export the vessel details, you'll have the option to export all vessels, only the selected vessels, or the vessels on the displayed page.

To export vessel details:

1. Click the ![Download_20200323.png](https://support.vintrace.com/hc/article_attachments/32329144313364) displayed in the lower, right of the Vessels page.
2. Select the format you want to use for the export file.

- Excel — XLS file
- Portrait — PDF file
- Landscape — PDF file
- Barrel Details — CSV file

3. Select which vessels you want to export (i.e., All, This Page, or Selected).
