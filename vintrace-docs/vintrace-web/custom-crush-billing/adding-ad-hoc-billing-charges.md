# Adding Ad Hoc Billing Charges

It's not always possible to automatically capture charges such as administrative or paperwork fees as part of regular winery work. For these situations, you'll need to [create billable items](https://support.winery-software.com/hc/en-us/articles/360000812935) in vintrace that aren't linked to any operation or action, then add them to a [service order](https://support.winery-software.com/hc/en-us/articles/360000812915) as detailed below.

To add ad hoc billing charges to service orders:

1. Click ![Sidebar - Setup 20241118.png](https://support.vintrace.com/hc/article_attachments/33454237270164) Set Up in the sidebar.
2. Click Client Billing.
3. From the Service Orders tile, click Configure.
4. Click Add Charges beside the service order.

![Winery Setup - Billing - Service Orders - Add Charges Button 20200714.png](https://support.vintrace.com/hc/article_attachments/33454243372180)

The Ad Hoc Charges window displays.

![Create Ad Hoc Charges 20200714.png](https://support.vintrace.com/hc/article_attachments/33454243373844)

5. In the Apply These Charges To section, select Wine Batch, Stock Item, or General Charge. If you selected Wine Batch or Stock Item, add the wine batch or stock item that you want to link the charge to.
6. Add the billing item by either clicking ![Magnifying Glass 20200320.png](https://support.vintrace.com/hc/article_attachments/33454243377684) to search for the billing item or clicking ![Plus in Green Circle 20200319.png](https://support.vintrace.com/hc/article_attachments/33454243380116) to add a new billing item.
7. For each billing item that you add, enter the reason for the charge in the Line Note field.
8. Click Save.
