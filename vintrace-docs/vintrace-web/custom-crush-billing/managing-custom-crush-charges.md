# Managing Custom Crush Charges

You can manage custom crush charges and view invoice history for a service order.

To manage custom crush charges and view a service order’s invoice history:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328842338836) Set Up in the sidebar.
2. Click Client Billing.
3. From the Service Orders tile, click Configure.
4. Select the service order you want to view. You can use the filters at the top of the window to find the service order.
5. Select the service order.
6. Select the Charges tab.

![Update_Service_Order_-_Charges_Tab_20210615.png](https://support.vintrace.com/hc/article_attachments/32328854889876)

You can use the filters at the top of the tab to find charges linked to the service order.

To edit the charges linked to an operation, click the link displayed in the Type column.
