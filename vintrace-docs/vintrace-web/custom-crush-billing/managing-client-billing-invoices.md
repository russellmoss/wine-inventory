# Managing Client Billing Invoices

The Client Billing Invoices window lets you manage client billing invoices. From the Client Billing Invoices window you can:

- [Create invoices](#Creating_invoices)
- [Print invoices](#Printing_invoices)
- [Export invoices](#Exporting_invoices_to_an_external_package)
- [Email invoices](#Emailing_invoices)
- [Reverse invoices](#Reversing_invoices)

To access the Client Billing Invoices window:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329093019028) More Options in the sidebar.
2. From the Client Billing tile, click Invoice Management.

![Client_Billing_Invoices_20200710.png](https://support.vintrace.com/hc/article_attachments/32329108094868)

## Creating Invoices

By default, charges raised in vintrace aren’t automatically added to invoices. This lets you review the charges, make changes, and add more charges before raising the invoice.

Once you invoice for charges, they can no longer be updated.

To create a new invoice:

1. From the Client Billing Invoices window, click New Invoice Run.

![Client_Billing_Invoices_-_New_Invoice_Run_Button_20200708.png](https://support.vintrace.com/hc/article_attachments/32329082155796)

2. Search for the charges to include. If you’d like to separate the invoices and have different charges on different invoices, adjust your search criteria.
3. Click Generate Invoice.
4. Review the invoices details by either clicking the invoice number, or clicking ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329108119700) and selecting View Details.
5. If there aren't any changes to the invoice, click Approve.

## Printing Invoices

The PDF for invoices can be customized to include your detailed remittance advice. The printed invoice displays charges based on the billing item category and a breakdown of the charges by billing/stock item.

![Printed_Invoice_20200713.png](https://support.vintrace.com/hc/article_attachments/32329093127444)

To print an invoice:

1. From the Client Billing Invoices window, select the checkbox beside the invoice. If you want to print all invoices listed, select the checkbox beside the Invoice column header.
2. From the Actions menu in the lower left, select Print Invoices.

![Client_Billing_Invoices_-_Printing_20200713.png](https://support.vintrace.com/hc/article_attachments/32329082224020)

You can also print an invoice while viewing its details.
![View_Details_-_Print_20200713.png](https://support.vintrace.com/hc/article_attachments/32329103972116)

## Exporting Invoices

You can export invoices to an external package such as Excel, MYOB, Quickbooks, or Xero.

To export an invoice:

1. From the Client Billing Invoices window, select the checkbox beside the invoice. If you want to export all invoices listed, select the checkbox beside the Invoice column header.
2. Click ![Download_20200323.png](https://support.vintrace.com/hc/article_attachments/32329118721684).
3. Select the format for the export.

[![Client_Billing_Services_-_Export_Options_20200713.png](https://support.vintrace.com/hc/article_attachments/32329104031252)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2018/12/Client-billing-10.jpg)

4. Select the options for the export.

## Emailing Invoices

In order to email an invoice to a client, be sure that the client’s email address is specified in the address book.

You can email one or more invoices to the client specified in the service order. vintrace will use the email address for the client that’s specified in the address book.

![Service_Order_Client_Email_Address_20200713.png](https://support.vintrace.com/hc/article_attachments/32329082280340)

To email an invoice to a client:

1. From the Client Billing Invoices window, select the checkbox beside the invoice. If you want to email all invoices listed, select the checkbox beside the Invoice column header.
2. From the Actions menu in the lower left, select Email.

![Client_Billing_Invoices_-_Email_20200713.png](https://support.vintrace.com/hc/article_attachments/32329118805780)

You can also email an invoice while viewing its details.
![View_Details_-_Email_20200713.png](https://support.vintrace.com/hc/article_attachments/32329085291924)

## Reversing Invoices

If you need to edit charges on an invoice, you’ll first need to reverse the invoice.

To reverse an invoice:

1. From the Client Billing Invoices window, select the checkbox beside the invoice. If you want to reverse all invoices listed, select the checkbox beside the Invoice column header.
2. From the Actions menu in the lower left, select Reverse.

![Client_Billing_Invoices_-_Reversing_20200713.png](https://support.vintrace.com/hc/article_attachments/32329104064916)

You can also reverse an invoice while viewing its details.
![View_Details_-_Reverse_20200713.png](https://support.vintrace.com/hc/article_attachments/32329082361364)

After you reverse an invoice, all its charges become editable. To edit an invoice’s charges:

1. View the operation that the charges are linked to.
2. Click Charges.
3. Update and save the charges as needed.
4. Re-generate the invoice for the same service order and date range.
