# Declaring Wine from the Product Page

Sometimes you may want to declare a single wine. This is common when you’re doing a rollback and discover that a wine wasn’t declared prior to dispatch. Instead of using the [Wine Declaration Console](https://support.vintrace.com/hc/en-us/articles/32301352553236), you can declare a wine from the wine's [product page](https://support.vintrace.com/hc/en-us/articles/32303310460948).

There are two ways to declare a wine from the product page:

- [From the Tax Class tile](#h_347db02b-f6ff-41b2-91ea-7a2c8c6d56c5)
- [From the Operations menu](#h_775f1c5f-4165-48b4-9141-9e4cf4528808)

## Declaring from the Product Page's Tax Class Tile

To declare the wine from the Tax Class tile:

1. From the Tax Class tile, click the ![Building_Gray_20200415.png](https://support.vintrace.com/hc/article_attachments/32328924570004) building icon.

![Product_Overview_-_Tax_Class_Tile_20200415.png](https://support.vintrace.com/hc/article_attachments/32328911697812)

The Tax Class Details window displays.

2. Click Declare This Wine.

![Tax_Class_Details_-_Declare_This_Wine_Button_20200415.png](https://support.vintrace.com/hc/article_attachments/32328880416788)

The [Wine Declaration Console](https://support.vintrace.com/hc/en-us/articles/32301352553236) displays with the wine. The declaration date will be today's date unless you change it. If there was a valid alcohol percentage, vintrace pre-selects the correct tax class for you.

3. Click Declare. The status changes to Complete.

After closing the Wine Declaration Console, you’ll need to refresh the window to display the tax class and bond under which the wine was declared.

![Product_Overview_-_Tax_Class_Tile_Refreshed_20200415.png](https://support.vintrace.com/hc/article_attachments/32328880443668)

## Declaring from the Operations Menu

To declare the wine from the Operations menu:

1. From the wine's [product page](https://support.vintrace.com/hc/en-us/articles/32303310460948), click the ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32328924625300) operations icon.
2. From the Treatments section, click Treatment (Product).
3. From the Treatment list, select the product treatment.
