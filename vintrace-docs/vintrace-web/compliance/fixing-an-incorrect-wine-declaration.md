# Fixing an Incorrect Wine Declaration

It’s possible for a wine to be accidentally declared with the wrong tax class and/or bond (most common in a custom crush facility). A wine’s tax class and bond can be prematurely declared when there’s a change to the wine’s alcohol percentage.

There are several ways to correct an incorrectly declared wine. The easiest options are to [use a product treatment](https://support.vintrace.com/hc/en-us/articles/32301321116308), or [confirm the wine’s tax class as part of an alcohol analysis](https://support.vintrace.com/hc/en-us/articles/32301366953492).

In some cases, you can also use vintrace’s Rollback operation.

## Rollback Operation

If you have a wine that was incorrectly declared, you can use the [Rollback feature](https://support.vintrace.com/hc/en-us/articles/32303277767188) to undo the operation that caused the premature declaration.

If a rollback is not feasible due to the amount of work performed, [contact support](mailto:support@vintrace.com).
