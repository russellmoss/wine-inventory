# Declaring Wine by Alcohol Analysis

You can also declare a wine when [recording its alcohol analysis](https://support.vintrace.com/hc/en-us/sections/360000176035-Lab-work). You’ll be prompted to confirm the analysis with the wine’s tax class.

![Tax Class Change 20200331.png](https://support.vintrace.com/hc/article_attachments/34686384358164)

You can only declare a wine after recording its alcohol analysis if the Declaration option in your winery setup is enabled.
