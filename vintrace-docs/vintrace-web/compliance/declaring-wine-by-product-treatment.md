# Declaring Wine by Product Treatment

If you want to declare wines by product treatment, you'll first need to [create a product treatment](#Creating_Product_Treatment) for each tax class and reason. When you’re ready to record this treatment on a wine that you want to declare, you can [declare a single wine](#Declaring_Single_Wine), or [multiple wines](#Declaring_Multiple_Wines).

## Creating a Product Treatment

If you want to declare wines by product treatment, you’ll need to [create a product treatment](https://support.vintrace.com/hc/en-us/articles/32301359713428) for each tax class and reason. For the typical winery, most still wines will use the *Not Over 16%* and *16%-21%* classes.

When you set up the product treatment, be sure to select the Change Tax Class checkbox. You’ll also need to select the tax class and the reason for the change. The most common reason is Produced by Fermentation.

![Product_Treatment_Definition_-_Create_-_Change_Tax_Class_20200407.png](https://support.vintrace.com/hc/article_attachments/32328880364564)

## Declaring a Single Wine

If you're declaring a single wine:

1. From the Vessels page, view the wine.
2. Click the ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32328880306964) operations icon.
3. Select Treatment (Product) under Treatments.

![Record_an_Operation_-_Treatment_Product_20200415.png](https://support.vintrace.com/hc/article_attachments/32328924510228)

The Treatment (Product) window displays.

4. From the Treatment list, select the product treatment you created.

## Declaring Multiple Wines

If you're declaring multiple wines:

1. From the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924), select the wines.
2. From the Actions menu that’s located in the lower left, select Work Order > Treatment (Product).

![Actions_-_Work_Order_-_Treatment_Product_20200402.png](https://support.vintrace.com/hc/article_attachments/32328911618324)

The Additional Information Required for Treatment window displays.

![Addl_Info_for_Treatment_20200407.png](https://support.vintrace.com/hc/article_attachments/32328924551444)

3. From the Treatment list, select the product treatment that you created.
4. Click OK and save the work order.
