# Declaring Wine

Until a wine is declared and is considered *produced* or *finished*, it is carried in the bulk wine section of the TTB report (PART VII – IN FERMENTERS END OF PERIOD (Gallons)).

From the [Wine Declaration console](https://support.vintrace.com/hc/en-us/articles/32301352553236), you can [declare wines one-by-one](#h_7bbd0ce3-c85a-474f-8e32-ae93b67ccbc0), or by doing a [mass declaration of multiple wines](#h_bba92898-640f-4da7-b6ab-5079773200db).

The options for declaring wine that are detailed below are done from the Wine Declaration console. However, you can also declare wine when [recording its alcohol analysis](https://support.vintrace.com/hc/en-us/articles/32301366953492), by [using a product treatment](https://support.vintrace.com/hc/en-us/articles/32301321116308), or [from the wine's product page](https://support.vintrace.com/hc/en-us/articles/32301306482708).

## Timing Wine Declaration

The [TTB website has a number of resources](https://www.ttb.gov/wine) and [FAQs](https://www.ttb.gov/wine/wine-faqs) related to wine.

Some wineries wait to declare all finished wines (defined as primary and/or malolactic fermentation has stopped and that remain un-bottled) as of December 31. If you have questions about when to declare a bulk wine, please confer with your Compliance officer, legal adviser or the TTB itself.

All wine must be declared before bottling or blending with an already-declared wine.

## Specifying the Declaration Date

By default, the Declare By in the [Wine Declaration console](https://support.vintrace.com/hc/en-us/articles/32301352553236) defaults to the current date and time. You can select a different date by clicking the ![Declare_Date_20200330.png](https://support.vintrace.com/hc/article_attachments/32328987057684) Calendar icon and selecting the date from the calendar.

![Declaration_Date_20200330.png](https://support.vintrace.com/hc/article_attachments/32329019646100)

## Declaring Individual Wines

If you want to declare wines with different dates, you’ll need to declare them separately by repeating the steps below for each declaration date.

To declare individual wines:

1. From the [Wine Declaration console](https://support.vintrace.com/hc/en-us/articles/32301352553236), search for the wine.
2. [Specify the wine’s declaration date](#Specifying_Declaration_Date).
3. Select the Tax Class for the wine. If the wine has a metric reading that satisfies the metric threshold policy, the Tax Class will automatically be selected.

![Tax_Class_20200330.png](https://support.vintrace.com/hc/article_attachments/32328987087636)

You can specify metric threshold policies from the Set Up console and selecting Lab. Click the Configure button in the Metric Thresholds tile.

4. Click Declare.
5. Repeat steps 3 and 4 for each wine you want to declare.

When a wine has been successfully declared, its status changes to Complete.

## Mass Declaring Wines

Mass declaring wines lets you declare all of the wines listed in the Wine Declaration console that aren’t already completed.

BEFORE YOU BEGIN: We recommend that you remove wines with warnings until the reasons behind the warning are resolved. To remove an individual wine from the list, click the ![X_in_Gray_Circle_20200330.png](https://support.vintrace.com/hc/article_attachments/32329019504020) Remove icon beside the wine. To remove all wines with warnings, click (Remove Entries with Warnings).

![Removing_Wines_20200330.png](https://support.vintrace.com/hc/article_attachments/32329010151700)

To mass declare wines:

1. From the [Wine Declaration console](https://support.vintrace.com/hc/en-us/articles/32301352553236), search for the wine.
2. [Specify the declaration date](#Specifying_Declaration_Date).
3. For each wine in the list, select the Tax Class for the wine. If the wine has a metric reading that satisfies the metric threshold policy, the Tax Class will automatically be selected.
4. Click Declare All.

![Declare_All_Button_20200330.png](https://support.vintrace.com/hc/article_attachments/32328987128852)

A confirmation message displays.

![Declare_All_Confirmation_20200330.png](https://support.vintrace.com/hc/article_attachments/32328987150868)

5. Click OK.

If no errors were encountered, your selected wines will be declared.

If any errors occur during the declaration, you can skip the wines that are causing the problem by either checking the Skip checkbox for the wine and clicking Declare All again.

In the event that you accidentally do any of the following, you'll need to roll back the operation and start again:

- Declared a wine should not have been (not done fermenting and the warning was ignored)
- Used the wrong declaration date
- Discovered the wine was in the wrong bond

However, if you have performed too many complex operations between the “real” declaration date and the one you inadvertently used, then [contact support](mailto:support@vintrace.com).
