# Wine Declaration Console

The Wine Declaration console lets you [search for undeclared wines](#Search_Filters) and declare them. You can declare wines one-by-one, or by doing a mass declaration of multiple wines.

## Accessing the Wine Declaration Console

To access the Wine Declaration console:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328556316948) More Options in the sidebar.
2. From the TTB tile, click Declare Wines.

![Declare_Wines_Link_20200407.png](https://support.vintrace.com/hc/article_attachments/32328536979348)

The Wine Declaration console displays. At the top of the Wine Declaration console are filters for finding wines. You can [find wines by their vintage, owner, and type](#Search_Filters), or by using the [product search](#Product_Search).

## Searching for Wines by Vintage, Owner, or Type

The vintage, owner, or type filters located at the top of the Wine Declaration console will only include wines that are fermented and non-declared.

If you click Search without specifying any filters, the list displays all non-declared wines.

To filter the wines by vintage, owner, or type:

1. From the Wine Declaration console, specify the criteria for your filter. You can click the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32328581018132) search icon to search for existing owners.
2. Click Search.
3. To reset the vintage and type filters, select [Reset…] from the list. To reset the owner filter, click the ![X_in_Gray_Circle_20200330.png](https://support.vintrace.com/hc/article_attachments/32328550940052) remove icon that’s displayed beside the filter.

## Searching for Wines Using the Product Search

Using the product search may include wines that are inappropriate for declaration such as those that are not fermented or still must.

To search for wines using the product search:

1. From the Wine Declaration console, click From Product Search.

![From_Product_Search_Button_20200330.png](https://support.vintrace.com/hc/article_attachments/32328556383508)

The Bulk Wine Search window displays.

2. Specify the criteria for the wine you’d like to find.
3. To add a wine to the Wine Declaration console, check the checkbox beside the wine. To include all wines listed in the Bulk Wine Search console, check the checkbox at the top of the list.

![Bulk_Wine_Search_20200330.png](https://support.vintrace.com/hc/article_attachments/32328564855700)

4. Click Use Selection.

A ![Exclamation_Point_in_Yellow_Triangle_20200811.png](https://support.vintrace.com/hc/article_attachments/32328581117588) warning icon displays beside any wines that have an associated warning. Warnings might include when the wine is not fermented, or is still must. You can hover over the icon to view the warning.

![Viewing_Warning_20200330.png](https://support.vintrace.com/hc/article_attachments/32328581128340)
