# Setting Up a Cost Item

You may need to set up a new cost item when you're [adding storage costs for wines in vessel](https://support.vintrace.com/hc/en-us/articles/32301354452244), or [allocating overhead costs to bulk wine and finished goods](https://support.vintrace.com/hc/en-us/articles/32301338554388).

You can set up a cost item from the Winery Setup window (Setup Options > Costing > Cost Items):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328873480980)Set Up in the sidebar.
2. Click Accounts.
3. From the Cost Items tile, click Configure.
4. Click New Cost Item.

![New_Cost_Item_Button_20200616.png](https://support.vintrace.com/hc/article_attachments/32328856154132)

The Create Cost Item window displays.

![Create_Cost_Item_20200616.png](https://support.vintrace.com/hc/article_attachments/32328861033364)

5. Specify the details for the cost item. Be sure to select the appropriate cost type.
6. Click Save.
