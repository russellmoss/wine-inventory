# Cost Console

The Costs Console makes managing your day-to-day cost operations easier to track and manage. It provides a central place for all costs, including wine and inventory-related costs.

To access the Cost Console:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329091766292) More Options in the sidebar.
2. From the Accounts tile, click Cost console.

At the top of the Cost Console are filters that let you refine what’s displayed in the console.

Entries that display in red text and a ![Warning_20200330.png](https://support.vintrace.com/hc/article_attachments/32329080777748) beside them are restricted, usually from transfers. It’s recommended that you reverse the operation associated with the cost to correct them.

![Restricted_Entries_20200611.png](https://support.vintrace.com/hc/article_attachments/32329117360404)

The Operation and Costing columns display links that let you view the operation and the product that the cost was applied.

![Operation_and_Costing_Links_20200611.png](https://support.vintrace.com/hc/article_attachments/32329080813076)

## Filtering the Cost Console

At the top of the Cost Console are filters that let you find the batch, stock item, or wine that you’re interested in.

![Cost_Console_Filters_20200611.png](https://support.vintrace.com/hc/article_attachments/32329117429780)

If you’d like to search a date range, be sure to select the Range To checkbox and specify an end date.

The cost items included in the list do not include reversed operations by default. To include reversed operations, select the Show Reversed checkbox. Reversed operations will display with a strikethrough.

![Reversed_Operations_20200611.png](https://support.vintrace.com/hc/article_attachments/32329091935508)

## Viewing an Operation’s Cost Impact

To view the individual cost items, click View beside the item.

![View_Button_20200611.png](https://support.vintrace.com/hc/article_attachments/32329084467348)

This displays the Cost Impact window for the cost item.

![Cost_Impact_20200611.png](https://support.vintrace.com/hc/article_attachments/32329056917524)

## Editing Costs

To edit the cost associated with an operation and item:

1. Click the down arrow beside the View button.
2. Select Edit.

![Cost_Console_-_Edit_Button_20200611.png](https://support.vintrace.com/hc/article_attachments/32329092028180)

If the cost event is bound to an operation, a warning displays.

![Warning_-_Cost_Event_Binded_to_Operation_20200611.png](https://support.vintrace.com/hc/article_attachments/32329084171796)

3. Click OK if you want to adjust the cost. The Admin Adjust Associated Operation Cost window displays.

![Admin_Adjust_Associated_Operation_Cost_20200611.png](https://support.vintrace.com/hc/article_attachments/32329081350164)

4. Adjust the costs as needed.
5. Click Save.

## Reversing Costs

To reverse a cost:

1. Select the checkbox beside the events that you want to include. You can select all events listed by selecting the Select All/None checkbox.
2. Click the down arrow beside the Summary button.
3. Select Reverse.

## Fixing a Process Date

To change the date that a cost was applied:

1. Select the checkbox beside the events that you want to include. You can select all events listed by selecting the Select All/None checkbox.
2. Click the down arrow beside the Summary button.
3. Select Fix Date.

![Cost_Console_-_Fix_Date_20200611.png](https://support.vintrace.com/hc/article_attachments/32329056974612)
The Fixup Date window displays.

![Fixup_Date_20200611.png](https://support.vintrace.com/hc/article_attachments/32329102764436)

4. Specify the new date and time.
5. Click OK.

## Adding Ad Hoc Costs

You can add ad hoc costs to a batch, stock item, and lot/batch.

To add ad hoc costs:

1. From the Cost Item list, select what you want to search for.
2. Specify the batch, stock item, or lot/batch.
3. Click ![Add_Ad_Hoc_Cost_to_Batch_Icon_20200611.png](https://support.vintrace.com/hc/article_attachments/32329081077140).

![Cost_Console_-_Add_Ad_Hoc_Cost_20200611.png](https://support.vintrace.com/hc/article_attachments/32329081044116)

The Admin Add Costs window displays.

![Admin_Add_Costs_20200611.png](https://support.vintrace.com/hc/article_attachments/32329117750676)

4. Enter the details for the costs.
5. Click Save.

## Viewing the Cost Summary for Events

You can view a cost summary for selected events from the Cost Console.

![Cost_Summary_20200611.png](https://support.vintrace.com/hc/article_attachments/32329117768980)

To view a cost summary for specific events:

1. Select the checkbox beside the events that you want to include. You can select all events listed by selecting the Select All/None checkbox.
2. Click Summary.

![Cost_Console_-_Summary_20200611.png](https://support.vintrace.com/hc/article_attachments/32329057051156)

## Printing List of Costs

You can print or email the list of costs displayed in the Cost Console as an Excel or PDF file.

![Cost_Console_Report_20200611.png](https://support.vintrace.com/hc/article_attachments/32329081260436)

From the Reports menu located in the lower left, select the option for the file format (i.e., Excel, PDF) and delivery option (i.e., download, email).

![Cost_Console_-_Reports_Button_20200615.png](https://support.vintrace.com/hc/article_attachments/32329081288596)
