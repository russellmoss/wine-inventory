# Tracking Winery Treatment and Operation Costs

You can define your operation types as treatments in vintrace. You can then use these treatments as part of standalone work orders, any transfer, or addition. Each time the treatment is applied to a wine, the cost impact will be recorded.

To set costs against a treatment you’ll need to [set up a cost item](https://support.vintrace.com/hc/en-us/articles/32301359350932). Set the cost item’s Price Per setting to either *Per Gallon/Per Litre*, or *Each* if it’s a fixed cost.

After setting up the cost item, you’ll need to link it to the treatment. You can also override the default cost value at this time.

For example, you could create a cross flow cost item that’s linked to a treatment. When the cross flow treatment is applied to a wine, its cost impact will be recorded.
