# Generating Storage Costs

vintrace has built-in logic to recognize that if a wine was only in barrel for 10 days of the month, only 10 days worth of costs should be added to that wine, even if that wine has now been racked to tank, or bottled. Refer to our [Adding Storage Costs for Wines in Vessel article](https://support.vintrace.com/hc/en-us/articles/32301354452244) to learn more.

To generate storage costs:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328598611348) More Options in the sidebar.
2. From the Accounts tile, click Create Storage Costs.
3. Set the Effective Date to the last date you want storage costs loaded. vintrace will add the costs to all wines up to, and including the effective date. We suggest running this [monthly](https://support.vintrace.com/hc/en-us/articles/32303349485588), and backdating to the end of the previous month.
4. If you need to check the scheduled task work list across all facilities, click Check Job Management Console.
5. Click Preview Storage Costs. Although this step is optional, we recommend you preview prior to saving to ensure the costs are what you’d expect.

![Storage_Costs_Preview_20200610.png](https://support.vintrace.com/hc/article_attachments/32328615891476)

6. If you’ve checked and ensured all outstanding work orders for barrel work has been closed off in the system up to the effective date, enter a note in the Notes/Reason field, then select the I Confirm I Have Checked and Understand checkbox.

![Storage_Cost_Generator_-_Note_and_Confirmation_20200610.png](https://support.vintrace.com/hc/article_attachments/32328623786900)

7. Click Save.
