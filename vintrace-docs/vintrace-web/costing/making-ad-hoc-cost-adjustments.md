# Making Ad Hoc Cost Adjustments

You can adjust the cost of any wine or stock item at any time. More importantly, you can backdate a cost adjustment so it updates all blends, bottling, or other uses AFTER that date. This ripple forward effect is a unique and powerful feature of vintrace’s costing model.

To make an ad hoc cost adjustment:

1. Use vintrace’s Quick Search to find the stock item or wine.

![Quick_Search_20200512.png](https://support.vintrace.com/hc/article_attachments/32328851901332)

2. Do one of the following:

- From the Product page, click ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328836675860), then select Costs.

![Product_-_Plus_Menu_-_Costs_20200622.png](https://support.vintrace.com/hc/article_attachments/32328856971668)

- From the Stock Item Overview window’s Action menu, select Add Cost.

![Stock_Item_Overview_-_Costs_-_Add_Costs_18NVCHJX_20200615.png](https://support.vintrace.com/hc/article_attachments/32328836813844)

The Admin Add Cost window displays.

![Admin_Add_Costs_14BBULK_20200622.png](https://support.vintrace.com/hc/article_attachments/32328814146068)

3. To select an existing cost item, click ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32328814151060)beside the Cost Item field, or click ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328836675860) to [create a new cost item](https://support.vintrace.com/hc/en-us/articles/32301359350932).
4. Enter values in the Quantity and Unit Cost fields.
5. Click ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32328857041428) to calculate the Subtotal.
6. In the Note field, enter an explanation.
7. Click Save.

## Reducing the Cost of Wine

In order to reduce the cost of wine, you can simply apply a negative cost adjustment. To ensure the correct outcome, use the same cost item as the original. In these situations, we recommend always adding a note so that there’s a clear explanation for the price correction.

![Admin_Add_Costs_-_Negative_Unit_Cost_20200622.png](https://support.vintrace.com/hc/article_attachments/32328839860628)

If you’re backdating any cost adjustments, remember that the effective date applies the cost as of that date.
