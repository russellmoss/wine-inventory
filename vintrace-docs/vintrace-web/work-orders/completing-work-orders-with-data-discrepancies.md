# Completing Work Orders with Data Discrepancies

When a cellar hand submits a work order from the mobile app, the vintrace web application shows the job’s status as *Submitted*. As with any work order, you can complete the work order by clicking Complete on the job.

If any of the data submitted by the cellar hand differs from what was scheduled by the winemaker, a warning dialog displays the differences. Below are some of the more common differences.

## New Barrels Scanned

If a cellar hand scanned barrels from the mobile app that aren’t in the web application, a warning displays. The warning displays the barrel code and an options for addressing the discrepancy.

To select an existing barrel in vintrace, you can click the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32329028483604) search icon, or typing and selecting the barrel.

To [create a new barrel](https://support.vintrace.com/hc/en-us/articles/32303355021588), click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329051451540) add icon. Barrels that you create will be copied to the barrel group.

## Not All Scheduled Barrels are Confirmed

When a cellar hand doesn’t check all barrels that were specifically scheduled to be emptied or filled, a warning displays the barrels that weren’t confirmed.

In this situation, you’ll need to check with the cellar hand to learn why the barrels weren’t confirmed. When you save the operation, the barrels that weren’t selected from the mobile app will NOT be emptied.

## Different Additive Amounts

If a cellar hand enters a different additive amount from what was scheduled, a warning displays. In this situation, you’ll need to consult the cellar hand to verify why the amount differs. Saving the operation will use the value entered by the cellar hand.
