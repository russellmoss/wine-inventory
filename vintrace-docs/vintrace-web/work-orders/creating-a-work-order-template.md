# Creating a Work Order Template

There are two options for creating a work order template in vintrace.

|  |  |
| --- | --- |
| **OPTION** | **WHEN TO USE** |
| [Create a template from an existing work order](#h_b5462da2-6ccd-4995-9b15-800c80a28afa) | If you have an existing work order that includes the jobs and details that you want to use for future work orders. |
| [Manually create a work order template](#h_317bcf6d-4207-4372-a713-cb2336b8b5c3) | If you want to set up your work order templates ahead of time so that they’re ready the next time you need them. |

For information on using a template to create a work order, refer to [Creating a Work Order Using a Template](https://support.winery-software.com/hc/en-us/articles/360000822296-Using-Work-order-templates) article.

Users with a multi-winery license can specify which work order templates are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/4413475617423) for details.

## Creating a Template from an Existing Work Order

The simplest way to create a work order template is to use a pre-existing work order. If you've created a work order for a standard set of operations that include all the required notes and instructions, you can create your template from this.

To create a template from an existing work order:

1. Edit the work order.
2. Click the Save As Template link that’s in the lower right corner of the window.

![Work_Order_Footer_-_Save_As_Template_Link_20210723.png](https://support.vintrace.com/hc/article_attachments/32328988316436)

The [Create] Work Order Template window displays.

3. Enter a name and description for the new work order template.
4. Click Next.
5. Click Save Work Template.

## Creating a Template Manually

To create a work order template manually:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329011628052) Set Up in the sidebar.
2. Select Work Orders from the list.
3. From the Work Order Templates tile, click Configure.

![Work_Order_Templates_-_Configure_Button_20200323.png](https://support.vintrace.com/hc/article_attachments/32329033045652)

4. Click New Work Order Template.

![New_Work_Order_Template_Button_20200323.png](https://support.vintrace.com/hc/article_attachments/32329003462804)

The [Create] Work Order Template window displays.

5. Enter a name and description for the work order template.
6. To add existing work orders (and its jobs) to your template, use the search bar to find the work order. To add a blank work order to your template, click Add Work Order.
7. Click Next. The Details tab displays.
8. Click Save Work Template.
