# Creating a Work Order Using a Template

When you use a template to create a work order, the work order includes the jobs, instructions, notes and categories that were saved with the template.

Work orders can be created from the following pages:

- The details page of a vessel
- The [Job Calendar page](https://support.vintrace.com/hc/en-us/articles/32303319976084)
- The [Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)

To create a work order from an existing template:

1. Click ![Plus_Sign_20200320.png](https://support.vintrace.com/hc/article_attachments/32328998365716). The New Work Order window displays.

    ![New_Work_Order_20200319.png](https://support.vintrace.com/hc/article_attachments/32328998343316)
2. Select the template from the list. The Use a Work Order Template window displays the jobs that have been added to the work order’s template.
3. On the Job Details tab, specify the details of your batches and tanks for the work order:

- Click the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32329042554132) to find and use an existing value.
- Click the ![Plus_Sign_20200320.png](https://support.vintrace.com/hc/article_attachments/32328998365716) to add a new value.

4. Click Next when you’ve finished specifying the details.
5. On the Schedule tab, select the date and time that you want the work to start. If you need to change anything from the template for this particular work order, click Edit. This includes adding jobs, or changing the existing jobs.
6. Click Create Work Order.
