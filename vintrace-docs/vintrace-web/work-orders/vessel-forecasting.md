# Vessel Forecasting

This functionality is available starting with version 9.20.1, and is available in our **Winery Enterprise** plan. If you would like to use this functionality, please contact our support team.

You can show a vessel’s forecasted batch and volume when scheduling operations.

Select the 'Show vessel forecasting' checkbox, then select a vessel and future date to view the forecasted state of that vessel. This is based on any pending scheduled jobs.

The forecasted volume and batch will display in purple.

![Racking - Show Vessel Forecasting 20240709.png](https://support.vintrace.com/hc/article_attachments/32328566185364)

We also have a [vessel forecasting report](https://support.vintrace.com/hc/en-us/articles/32301300975124) available, to view the operations that resulted in a vessel's forecasted batch and volume.

To set the 'Show vessel forecasting' default:

1. Click![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328582411796)Set Up in the sidebar.
2. Click General.
3. From the Defaults tile, click Configure.
4. Select this option in the System or Winery tabs.

![Vessel Forecasting Setting.png](https://support.vintrace.com/hc/article_attachments/32328557713300)

5. Click Apply.
