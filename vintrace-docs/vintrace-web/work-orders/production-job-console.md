# Production Job Console

The Production Job console shows a list of your upcoming Production Jobs. You can also calculate the required raw materials and create purchase orders from this console.

## Accessing the Production Job Console

To access the Production Job console:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329074411540)More Options in the sidebar.
2. From the Planning tile, click Production Jobs.

![Production_Jobs_Link_20200423.png](https://support.vintrace.com/hc/article_attachments/32329091450900)

The Production Job console displays.

![Production_Job_20200326.png](https://support.vintrace.com/hc/article_attachments/32329116768020)

## Filtering the Production Jobs Console

The left side of the console displays a filters panel that lets you control which productions jobs are listed.

![Production_Jobs_Filters_20200326.png](https://support.vintrace.com/hc/article_attachments/32329116803860)

You can filter the jobs by:

- Product
- Stock item
- Program
- Party
- Start date
- End date
- Status

To filter the production jobs:

1. Specify the criteria for your filter. You can use the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32329101900052) to search for existing values.
2. Click Search.

You can remove all filters by clicking Reset.

## Creating a New Production Job

To create a new production job:

1. Click New Production Job. The [CREATE] Production Job window displays.

![Create_Production_Job_20200326.png](https://support.vintrace.com/hc/article_attachments/32329091416468)

2. Specify the following details for the production job:

- Name - A meaningful name for the production job.
- Start - The date the production job will begin.
- End - The date the production job will end.

3. Click Add. A new line is added to the production job's Expected Stock list.

![Adding_Expected_Stock_20200326.png](https://support.vintrace.com/hc/article_attachments/32329102022548)

4. Enter the stock item and the number of units you expect to produce for the production job.
5. Repeat steps 3 and 4 for each stock item you want to add to the production job.
6. Click Save.

## Editing a Production Job

To edit an existing production job, click Edit.

![Edit_Button_20200326.png](https://support.vintrace.com/hc/article_attachments/32329116833044)

## Viewing the Materials for a Production Job

To view the materials required for a production job:

1. Select the checkbox beside the production job.
2. Click Required Materials.

![Viewing_Required_Materials_20200326.png](https://support.vintrace.com/hc/article_attachments/32329116875540)

The Required Materials window displays the materials for the production job. Items highlighted in green have the required amount of stock on hand. Items highlighted in red have insufficient stock for the production job.

## Creating Purchase Orders

You can create purchase orders for any materials in your production job. To create a purchase order for a production job:

1. [View the materials for the production job.](#h_9c6e9c60-50c0-4766-9ad1-4cd572d6cd12)
2. Select the materials that you want to create a purchase order for. If you want to create purchase orders for all materials that have insufficient stock, click Select All Materials We Require More Of.
3. The Vendor will be automatically populated if you have a preferred vendor specified for the item. Otherwise, you'll need to select the vendor for the item.
4. Click Create Purchase Order(s).

A purchase order is created for each item and vendor selected. The purchase orders display in the Purchase Order console where you can manage them as needed.
