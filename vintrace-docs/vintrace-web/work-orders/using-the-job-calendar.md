# Using the Job Calendar

The Job Calendar console allows you to see all your fruit bookings, work orders, inventory notes, production jobs, and general tasks for a particular time frame in a calendar view.

## Accessing the Job Calendar

To access the Job Calendar console, click ![Calendar_Icon_20200423.png](https://support.vintrace.com/hc/article_attachments/32328904159892) Calendar in the sidebar.

The Job Calendar defaults to showing jobs for the current month with the current date shaded in gray.

## Changing the Job Calendar's View

You can change the time period displayed on your calendar (i.e., month, week, or day) by using the controls in the top right. Click the arrows to display the previous or next month, week, or day.

![Display_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328921483284)

## Filtering the Job Calendar

The filter lets you find work on certain batches or vessels.

To filter the Job Calendar:

1. Click ![Filter_20200323.png](https://support.vintrace.com/hc/article_attachments/32328949361940). A filter panel displays.

![Filter_Panel_20200323.png](https://support.vintrace.com/hc/article_attachments/32328904125588)

2. Specify the criteria for your filter. You can use the ![Magnifying_Glass_20200320__1_.png](https://support.vintrace.com/hc/article_attachments/32328925235732) to search for existing values.
3. Click Search.

## Creating Work Orders or Appointments from the Job Calendar

To create a new work order or appointment from the Job Calendar:

1. Click the day that you want to add the work order or appointment to. The Create Event window displays.

![Create_Event_20200323.png](https://support.vintrace.com/hc/article_attachments/32328921535252)

2. Click New Work Order or New General Appointment.
