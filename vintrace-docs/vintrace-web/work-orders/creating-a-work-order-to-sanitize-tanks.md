# Creating a Work Order to Sanitize Tanks

If you don't need to create a work order to sanitize a tank, you can adjust the sanitation state by clicking the Sanitize checkbox on transfer operations.

![Racking_-_Sanitize_Checkbox_20221109.png](https://support.vintrace.com/hc/article_attachments/32328872056596)

When you want to write a work order to sanitize tanks, you’ll want to add an equipment treatment job.

Equipment treatment jobs can also be used when you want to create a work order to clean or check a tank.

To add a work order to sanitize tanks:

1. [Create a work order](https://support.vintrace.com/hc/en-us/articles/32303315610388). When you get to the step to add a job to the work order, follow the steps below.
2. From the Work Order window click Add Job.

![Create_Work_Order_-_Add_Job_Button_20211001.png](https://support.vintrace.com/hc/article_attachments/32328886214804)

3. Select Treatment (Equipment).

![Add_Job_-_Treatment_Equipment_20211001.png](https://support.vintrace.com/hc/article_attachments/32328872120980)

The Treatment (Equipment) window displays.

4. From the Treatment list, select *Sanitise Tank*.

![Treatment_Equipment_-_Treatment_List_-_Sanitize_20211001.png](https://support.vintrace.com/hc/article_attachments/32328863801876)

5. To specify the tanks that you want sanitized, click the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32328863809556) search icon beside the Add Equipment field to find and select the tanks. The selected tanks display in the Equipment List.

![Treatment_Equipment_-_Specifying_Tanks_20211001.png](https://support.vintrace.com/hc/article_attachments/32328847367700)

If you don’t want to specify the tanks to be sanitized, the person [submitting the job will be able to scan or select the tanks](https://support.vintrace.com/hc/en-us/articles/32301315087892).

6. Specify any other details that you want to include in the job.
7. Click Add to Work Order. The Work Order window displays with the job that you added.

![Create_Work_Order_with_Equipment_Treatment_Job_20211001.png](https://support.vintrace.com/hc/article_attachments/32328847274516)

8. Click Save As…

- Draft - you’re not ready for your staff to complete the work. The work order will be saved for the future.
- Ready - the work order is ready for your staff to complete.

After saving the work order as Ready, your cellar hands will be able to [view and submit it from the mobile app](https://support.vintrace.com/hc/en-us/articles/32301315087892). For details on completing submitted equipment treatment jobs, refer to our [Completing an Equipment Treatment Job article](https://support.vintrace.com/hc/en-us/articles/32301315044500).
