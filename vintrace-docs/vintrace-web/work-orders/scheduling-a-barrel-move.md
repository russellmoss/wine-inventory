# Scheduling a Barrel Move

This functionality is available starting with [vintrace 9.4.3](https://support.vintrace.com/hc/en-us/articles/32303276816020), but not enabled by default. If you would like to use this functionality, please contact our support team.

To schedule a barrel move:

1. [Create a work order.](https://support.vintrace.com/hc/en-us/articles/360000812015-Creating-a-Work-Order-Manually)
2. From the Work Order window, click Add Job.
3. Select Move Barrels.

![Create Work Order - Move Barrels 20241024.png](https://support.vintrace.com/hc/article_attachments/32328564931732)

4. In the Relocate Bulk Wine window, specify the details for the barrel move.
5. Click Add to Work Order.
6. Set the work order’s Scheduled For setting to the date and time that you want the barrel move to occur.

![Create Work Order - Scheduled For 20241024.png](https://support.vintrace.com/hc/article_attachments/32328551003668)

7. Click Save As and specify whether you want to save the work order as draft or ready.
