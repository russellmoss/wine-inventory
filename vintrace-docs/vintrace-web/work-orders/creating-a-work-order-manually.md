# Creating a Work Order Manually

When you create a work order manually, the work order does not include any jobs,  instructions, notes or categories.

Work orders can be created from the following pages:

- The [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- A wine's [Product page](https://support.vintrace.com/hc/en-us/articles/32303310460948)
- The [Job Calendar page](https://support.vintrace.com/hc/en-us/articles/32303319976084)
- The [Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)

You can [assign a winery specific prefix](https://support.vintrace.com/hc/en-us/articles/32301350503572) to your work orders.

To create a work order manually:

1. Click ![Plus_Button_20200319.png](https://support.vintrace.com/hc/article_attachments/32329020671380). The New Work Order window displays.
2. Click Create a Manual Work Order.

![Create_Manual_Work_Order_Button_20200319.png](https://support.vintrace.com/hc/article_attachments/32329003058580)

The Work Order window displays.

![Create Work Order](https://support.vintrace.com/hc/article_attachments/32723047056276)

3. Specify the following details for the work order.

- Scheduled For — The date you want the work to start. You can click the ![Jobs_Calendar_Icon_20200319.png](https://support.vintrace.com/hc/article_attachments/32329032709396) icon to display the jobs calendar.
- Assigned To — The person you want to assign the work order to. If the work can be done by anyone on the cellar crew, leave this unassigned.
- Issued By — The person who is assigning the work.
- Expected Completion - The date that the work order is expected to be completed.
- Priority - The priority of this work order; 1 - High, 2 - Medium, or 3 - Low.
- Category — The work order category provides a way for you to organize your work orders. A category can be anything that you’d find helpful (e.g., Additions, Contract Work, etc…). Categories can be color-coded to make them easier for you to recognize.
- Summary — A brief summary of the work you want done. If you leave the summary blank, vintrace will add a summary based on the type of work that you’re scheduling.
- Owner — The owner is automatically set if the work order is related to a single owner/client.
- Bond — The bond associated with the work order.
- Notes — You can enter any general instructions for the work order. You can also select one of the [standard notes](https://support.vintrace.com/hc/en-us/articles/32301315435028) (i.e., pre-defined instructions) by clicking ![Standard_Note_Icon_20200319.png](https://support.vintrace.com/hc/article_attachments/32328987991828).

4. Do the following for each job you want to add to the work order:

1. Click Add Job.
2. Select the type of job you want to add to the work order.
3. Specify the details for the job.
4. Click Add to Work Order.

5. To save the work order as a Word document or PDF file that can be printed, click the ![Heart_White_20200731.png](https://support.vintrace.com/hc/article_attachments/32328988070804) heart icon beside Print Work Order on Save in the lower left.

![Work_Order_-_Printing_and_Print_Options_20210723.png](https://support.vintrace.com/hc/article_attachments/32328994295828)

You can specify the [work order print template](https://support.vintrace.com/hc/en-us/articles/32303293741588) and file format by clicking the ![Wrench_20201118.png](https://support.vintrace.com/hc/article_attachments/32328988009108)wrench icon. You can click Remember My Settings and vintrace will remember the selected template and file format for future work orders.  ![Print_Options_20210723.png](https://support.vintrace.com/hc/article_attachments/32328994213780)

6. Click Save As…

- Draft - you’re not ready for your staff to complete the work. The work order will be saved for the future.
- Ready - the work order is ready for your staff to complete.

To save the work order as a template, click Save As Template. Refer to our [Creating a Template from Existing Work Order](https://support.winery-software.com/hc/en-us/articles/360000933976-Creating-and-using-work-order-templates#CreatingaTemplatefromanExistingWorkOrder) article to learn more.
