# Swapping Destination Vessels Using the Equipment Schedule

To change the destination vessel:

1. Edit the work order.
2. Edit the job.
3. Select the Schedule Details tab.
4. Click the ![Equipment_Schedule_20200323.png](https://support.vintrace.com/hc/article_attachments/32329109750932) to view the vessel's schedule and any jobs associated with it.
5. From the Equipment Schedule window, use the View By list to view the schedule for the vessels you're interested in.
6. Click Use beside the destination vessel.

![Equipment_Schedule_-_Use_Button_20200324.png](https://support.vintrace.com/hc/article_attachments/32329134262676)

The Schedule Details and General tabs reflect the vessel that you selected.
