# Creating Equipment Schedules from Work Orders

The [Equipment Schedule](https://support.vintrace.com/hc/en-us/articles/32303343324308) allows you to see what vessels are being used each day, making it easier for you to plan work.

You can create schedules for your vessels directly from your transfer, extraction, multi transfers, press cycle, and bulk intake work orders.

Before you begin, be sure you’ve created a work order for a vessel.

1. Edit the work order.
2. Edit the job.
3. Select the Schedule Details tab.
4. In the How Long Do You Think the Job Will Take field, enter the number of hours and minutes you think it’ll take to complete the job. If you’re unsure of the current schedule for the vessel, click the ![Equipment_Schedule_20200323.png](https://support.vintrace.com/hc/article_attachments/32329044353300) icon to view the vessel’s schedule and any jobs associated with it.
5. Click Add to Work Order.
