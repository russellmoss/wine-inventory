# Exporting Jobs Details

This functionality is available starting with [vintrace 9.9.3](https://support.vintrace.com/hc/en-us/articles/32301318965908), but not enabled by default. If you would like to use this functionality, please contact our support team.

You can export the details of jobs listed on the [Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972) to an XLS file.

The report will group the jobs on a work order by category.

Only work orders with a status of Ready, In Progress or Submitted will be included in the export.

To export the job details:

1. [Filter the Job Management page](https://support.vintrace.com/hc/en-us/articles/360000812055-Job-Management-Console#FilteringtheJobManagementConsole) as needed.
2. Click the ![Download 20200323.png](https://support.vintrace.com/hc/article_attachments/32329220456852) download icon that is located in the lower right.
3. Select Work Order Job Details.
4. Select one of the export options:

- All - exports all work orders including those that are displayed on other pages.
- This Page - exports work orders that are displayed on the page.
