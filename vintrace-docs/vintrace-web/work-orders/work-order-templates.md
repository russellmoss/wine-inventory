# Work Order Templates

If you find yourself repeating similar processes for different products, you might want to create a work order template. Work order templates are re-usable starting points for cellar operations that occur often.

Because work order templates include the jobs, instructions, notes and categories, they speed up the process for creating work orders. You can create work order templates manually, or from completed or pending work orders. These templates can include one or more jobs, or one or more work orders.

For information on creating a work order using a template, refer to [Creating a Work Order Using a Template](https://support.vintrace.com/hc/en-us/articles/32303304944532).

- You can [assign a winery specific prefix](https://support.vintrace.com/hc/en-us/articles/32301350503572) to your work orders.
- Users with a multi-winery license can specify which print templates are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/32301304791316) for details.
