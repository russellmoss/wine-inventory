# Setting Up a Case Stock Item

Before you add a case stock item, you’ll want to [set up your single bottle stock item](https://support.vintrace.com/hc/en-us/articles/32301345671956). You can set up a case stock item [prior to bottling](#h_6a768551-d022-4081-a833-0da050fd1fdb), or [at the time of bottling](#h_06cc3fd6-7eb0-471c-a17b-cd8ac41c4cff).

## Setting Up Prior to Bottling

If you want to set up the case stock item prior to bottling, you can either:

- Click ![Inventory_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328895303444) Inventory in the sidebar, then click New Item.
- Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328888982164) Set Up in the sidebar, click Wine Items, then click Configure in the Case x12 Items tile. If your case contains a different number of bottles, click Configure in the appropriate tile.

Next, follow the [steps below](#h_53dadbf1-562c-49a8-8f84-245b9701814e).

## Setting Up During Bottling

If you want to set up the case stock item at the time of [bottling](https://support.vintrace.com/hc/en-us/articles/32303327186836), click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328888992276) beside the Stock Item field in the Packaging window.

Next, follow the [steps below](#h_53dadbf1-562c-49a8-8f84-245b9701814e). If you add your case stock item during bottling, the new stock item will be automatically used in the Packaging window.

## Adding a Case Stock Item

To set up your case stock item:

1. From the Stock Type list, select *Case x12* (or the appropriate option if your case contains something other than 12 bottles).
2. Specify the details for the case. Be sure to enter a stock code that differentiates the case from the single bottle. For example, the stock code for a 12-bottle case might be *11vintrace RED/CS/750/X12*.
3. Create the [Bill of Materials](https://support.vintrace.com/hc/en-us/articles/32303320516372) for the case. The Bill of Materials will include your [single bottle stock item](https://support.vintrace.com/hc/en-us/articles/32301345671956). You can also include a carton and insert for the case.

![Create_Stock_Item_-_Case_x12_20200518.png](https://support.vintrace.com/hc/article_attachments/32328878306196)

4. Click Save.

Users with a multi-winery license can specify which stock items are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/4413475617423) for details.
