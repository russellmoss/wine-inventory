# Configuring Defaults on Stock Items

You can set the default reorder code and order price for your stock items so that when you create a purchase order for the item, these fields will be pre-populated. You can set up these defaults from the Winery Setup window (Setup Options > General Stock Item or Wine Stock Item).

To set the default reorder code and order price:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329037770644) Set Up in the sidebar.
2. Click Wine Items.
3. Click Configure for the item that you want to specify defaults for.
4. Click the item.
5. In the Order Price and Vendor Code fields, enter the default values.

![Default_Order_Price_and_Vendor_Code_20200508.png](https://support.vintrace.com/hc/article_attachments/32329066641940)

6. Click Save.

When you create a purchase order for the stock item, these values will be automatically included.

![Stock_Purchase_Order_-_Default_Vendor_Code_and_Price_20200508.png](https://support.vintrace.com/hc/article_attachments/32329058563604)
