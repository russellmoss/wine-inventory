# Bottled Wine and Dry Goods Details

This article is relevant to subscriptions that include the [Bottling, Inventory and Warehousing module](http://www.winery-software.com/features).

The Stock Item Overview window provides important information about the stock, including its configuration, lots, stock levels, cost information, and history.

There are several ways to view the Stock Item Overview window for an item:

- Enter the stock code in the search bar.
- Search for the item from the Stock Search window by clicking ![Inventory_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328989006612) Inventory from the sidebar.

The Stock Item Overview window displays the following tabs across the top:

- [Stock Item](#h_40921282-4d5b-48c8-a492-40216036403d)
- [Lots](#h_dc9092b7-0555-4799-b0e8-88cdf59399f0)
- [Distribution](#h_e6852dc5-76af-42e0-8ace-c63a3c9efa45)
- [History](#h_d23f2e50-b0a4-4b53-af0a-22664527b54e)
- [Costs](#h_1e050cfb-0a16-44b0-936e-c9b34b0d51e7)
- [Components](#h_07e9c805-873c-4e2b-972d-f8693a16c6ff)
- [Tasks](#h_a32059bf-f484-4d49-bc1f-bc5d719891cb)

There are also several [menus](#h_f49a4d21-7178-4b06-8a40-722b927504b6) at the bottom of the window.

## Stock Item Tab

Information such as the stock code, category, tax class, and bill of materials. A chart depicting the level for this stock item over time also displays.

![Stock_Item_Overview_-_Stock_Item_20200505.png](https://support.vintrace.com/hc/article_attachments/32329022076564)

## Lots Tab

If you have lot/batch tracking enabled for the item, this tab lists all unique lots related to this stock item. The Lots tab also displays certifications that are used to track export market certifications for your lots. Click Add to define a new lot code and associate any certifications.

![Stock_Item_Overview_-_Lots_20200505.png](https://support.vintrace.com/hc/article_attachments/32329022104340)

## Distribution Tab

The locations and stock levels for the stock item.

![Stock_Item_Overview_-_Distribution_20200505.png](https://support.vintrace.com/hc/article_attachments/32329012747412)

## History Tab

The operation history for the stock item, including quantity adjusted, the work order (TWL#), and a running balance. By default, the system displays the last 3 months. You can expand the date ranges or remove a date by clicking the X beside the date.

If you have the necessary permission, you can change the date of the operation or reverse a historical operation.

![Stock_Item_Overview_-_History_-_View_Menu_20200505.png](https://support.vintrace.com/hc/article_attachments/32328995294996)

## Costs Tab

This tab displays the current cost breakdown for the stock item. The cost event history defaults to the last 3 months. To display all costing events, click the X beside the Show After field, then click Update. Viewing all costing events can be important if you’re trying to track costing issues.

![Stock_Item_Overview_-_Costs_20200505.png](https://support.vintrace.com/hc/article_attachments/32329012679060)

To manually adjust the cost, select the Actions > Add Costs menu option. To export the costing information to PDF, select the Reports > Costing Report menu option. The cost events display each cost change to the product including a running balance. You can click View to view the operation that made the change.

## Components Tab

The Raw Stock Components section at the top of the Components tab displays all the raw components down the bill of materials hierarchy for the stock item. For finished goods, you can click the bulk product (i.e., those that end with, “/BLK”) to view its information in the Product Overview window.

The Assembled Components section displays all the items that have been built, including this item, in its bill of materials.

![Stock_Item_Overview_-_Components_20200505.png](https://support.vintrace.com/hc/article_attachments/32329022208532)

## Tasks Tab

This tab displays a summary of outstanding scheduled work orders related to this stock item. The work order number (TWL#) and any stock levels committed (reserved) as part of the work order are listed.

![Stock_Item_Overview_-_Tasks_20200505.png](https://support.vintrace.com/hc/article_attachments/32329012704148)

## Menus

The following menus are available at the bottom of the Stock Item Overview window.

- New
- Operations
- Actions
- Reports

The options that are available in each menu will depend on the stock item.

![Stock_Item_Overview_-_Menus_20200505.png](https://support.vintrace.com/hc/article_attachments/32329022265748)
