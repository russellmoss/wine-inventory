# Manufacture Operation

The Manufacture operation lets you build complex stock items from packaged goods.

The difference between packaging and manufacturing is that packaging involves, at a minimum, bulk wine and is based on stock item(s) built into the Bill of Materials.

The manufacture operation is based on [Bill of Materials](https://support.vintrace.com/hc/en-us/articles/32303320516372) being combined such as single bottle packagings into cases, or building palletized packaged goods for shipping. You could also use the Manufacture operation to:

- [Label shiners (cleanskins)](https://support.vintrace.com/hc/en-us/articles/32303309107476)
- [Add components to an existing packaging](#h_fe252338-4218-4104-93fb-69eff262298f)
- [Palletize cases for dispatch](#h_57a39b82-bd21-46aa-ab26-f4271188f874)

## Accessing the Manufacture Operation

There are a number of ways to access the Manufacture operation.

- Click Jobs in the sidebar, click the ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32329153015316) Operations icon, then select Manufacture.
- Click Inventory in the sidebar, search for the item, then select Manufacture from the Operations menu in the lower left.

## Adding Components to an Existing Packaging

Once a [Bill of Materials](https://support.vintrace.com/hc/en-us/articles/32303320516372) is set up, it can only be changed if it hasn’t been packaged.

Suppose you need to add components to a packaging, but don’t want to reverse the packaging. For example, you might have cases in temporary cartons that are awaiting the arrival of printed cartons and special dividers. Or, you might need to add stickers such as export neckbands or special award stickers.

If you haven’t already done so, [create the stock items](https://support.vintrace.com/hc/en-us/articles/32303296023316) for the components that you want to add to the packaging. For example, suppose we want to add a “Gold Medal Award” sticker to 90 cases of our [single bottle stock item](https://support.vintrace.com/hc/en-us/articles/32301345671956). We created a stock item where the Bill of Materials includes:

- Case of the already packaged wine
- 12 Gold Medal Award stickers (1 for each bottle in the case)

![Update_Stock_Item_-_Case_with_Gold_Award_20200520.png](https://support.vintrace.com/hc/article_attachments/32329129812500)

We then use the Manufacture operation to add the stickers to 90 cases of our wine.

![Action_Manufacture_-_Case_with_Gold_Award_Sticker_20200520.png](https://support.vintrace.com/hc/article_attachments/32329161880980)

## Palletizing Cases for Dispatch

Oftentimes when you ship out cases, they’re put on pallets. Manufacturing to a pallet is very similar to manufacturing a case.

To do this, [create a stock item](https://support.vintrace.com/hc/en-us/articles/32303296023316) for the pallet itself. You can create one type, or one for each pallet that your winery might use for different dispatches.

![Create_Stock_Item_-_Wooden_Pallet_20200528.png](https://support.vintrace.com/hc/article_attachments/32329175427220)

Suppose we want to palletize 56 12-pack cases (1 pallet) of stock item our [case stock item](https://support.vintrace.com/hc/en-us/articles/32301360537876). To do this, we’d add a new stock item with the Stock Type set to Pallet (Full). The [Bill of Materials](https://support.vintrace.com/hc/en-us/articles/32303320516372) for this stock item will include the pallet and our 12-pack cases.

![Create_Stock_Item_-_Pallet_of_Cases_20200528.png](https://support.vintrace.com/hc/article_attachments/32329129836308)

From the Action Manufacture window, we’d add our pallet of 56 cases.

![Action_Manufacture_-_Pallet_of_Cases_20200528.png](https://support.vintrace.com/hc/article_attachments/32329161860628)
