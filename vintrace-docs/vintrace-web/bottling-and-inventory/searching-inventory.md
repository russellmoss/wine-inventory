# Searching Inventory

There are a number of ways to find stock items in your inventory:

- Click ![Inventory_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329149367188) Inventory in the sidebar to display the Stock Search window. This window provides numerous options for filtering your physical inventory (except bulk wine).

![Stock_Search_20200508.png](https://support.vintrace.com/hc/article_attachments/32329192577684)

- From the Stock Item Overview window that’s displayed when you’re viewing a stock item.

![Stock_Item_Overview_-_Search_20200511.png](https://support.vintrace.com/hc/article_attachments/32329192603540)

- From the Quick Search.

If you specify more than one search filter, the search returns only those items that meet all the conditions.

You can [save searches](#h_6edabcdb-2f80-47d0-9dca-38963ee03f42) so that they display on the Dashboard page, [customize the name of the search, and specify ranges](#h_273934b4-f959-4fcf-a69f-9c0bf514f58b) so that you have a visual indicator of your stock levels.

## Saving a Search

If you have stock levels for items that need to be regularly monitored, you can save the search so that it displays in the My Searches dashlet on the Dashboard page.

 ![My_Searches_Dashlet_20200511.png](https://support.vintrace.com/hc/article_attachments/32329192620948)

To save a search for an item:

1. Search for the item.
2. View the item’s details in the Stock Item Overview window.
3. From the Action menu, select Track on My Searches Dashlet.

![Stock_Item_Overview_-_Actions_-_Save_Search_20200511.png](https://support.vintrace.com/hc/article_attachments/32329182658068)

## Renaming a Saved Search and Adding Visual Indicators

You can change the name of the saved search and add visual warnings for stock levels from the Winery Setup window (Setup Options > Other > Saved Searches). This is useful when you want to set up inventory alerts, or low stock level notifications.

To rename a saved search and add visual indicators:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329149456276) Set Up in the sidebar.
2. Click Other.
3. From the Saved Searches tile, click Configure.
4. Locate and click the search that you want to rename, or add a visual warning to.
5. In the Name field, enter the name that you want displayed in the Dashboard for this search.
6. To add visual indicators, enter the range then select the level (i.e., Optimal, Normal, Warn, Critical) from the list.

![Saved_Search_20200511.png](https://support.vintrace.com/hc/article_attachments/32329192640020)

The colors used for the statuses are:

- Optimal — Green
- Normal — Blue
- Warn — Orange
- Critical — Red

7. Click Save.
