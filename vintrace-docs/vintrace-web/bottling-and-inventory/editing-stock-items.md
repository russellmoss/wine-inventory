# Editing Stock Items

You can edit stock items in vintrace from either the [Setup page](#h_b819e061-0b48-4184-9500-e08bf0585d12), or from the [Stock Item Overview window](#h_11d7bf00-5c7f-4c8d-9c2a-0753d4503c65).

## From the Setup Page

You can edit stock items from the Winery Setup window (Setup Options > General Stock Items and Setup Options > Wine Stock Items).

To edit a stock item from the Winery Setup window:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329105617300) Set Up in the sidebar.
2. In the Search field, enter the item you want to update.

![Search_Field_20200504.png](https://support.vintrace.com/hc/article_attachments/32329134365716)

## From the Stock Item Overview Window

To edit stock items from the Stock Item Overview window:

1. Click ![Inventory_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329105848084) Inventory from the sidebar.
2. From the Stock Search window, search for the item you want to update.
3. Click the item you want to update.
4. Click the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32329087037076) pencil icon to edit the item.

![Stock_Item_Overview_-_Edit_Icon_20220719.png](https://support.vintrace.com/hc/article_attachments/32329134506132)

## Changing an Item’s Unit or Tax Class

Most of the updates made to a stock item are to its stock code, name, description, and category. These fields are available on the Sales tab and can be edited if the item’s Sell This Item checkbox is selected.

![Sales_-_Sell_This_Item_20200504.png](https://support.vintrace.com/hc/article_attachments/32329109918868)

However, the item’s unit or tax class are on the General tab and cannot be changed by default.

![General_-_Unit_and_Tax_Class_Not_Editable_20200504.png](https://support.vintrace.com/hc/article_attachments/32329087093396)

In order to change either of these values, you’ll need to make a copy of the existing item, then re-add the item with the correct unit or tax class.

To do this:

1. Edit the stock item.
2. Rename the item. You can do this by adding *Inactive* to the end of its name.
3. Select the Inactive checkbox.

![Inactive_Stock_Item_20200504.png](https://support.vintrace.com/hc/article_attachments/32329095320212)

4. Click Save.
5. Create a new item with the correct unit or tax class.
