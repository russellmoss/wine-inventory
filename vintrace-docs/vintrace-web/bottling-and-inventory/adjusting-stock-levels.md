# Adjusting Stock Levels

As your inventory stock increases and decreases, you can make adjustments against one or more stock items at one time.

1. Click ![Inventory_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328992887188) Inventory from the sidebar.
2. From the Stock Search window, search for the item you want to update.
3. Click the item.
4. From the Operations menu located in the lower left, select Adjustment.

![Stock_Item_Overview_-_Operations_Menu_-_Adjustment_20200505.png](https://support.vintrace.com/hc/article_attachments/32328980855188)

5. From the Action Adjustment window:

- Enter the number lost or gained in the Loss/Gain using “+” or “-”, then click the ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32329019301396)Calculator icon to calculate the Actual number.
- Enter a number in the Actual field, then click the ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32329019301396)Calculator icon to calculate the Loss/Gain number.

6. From the Reason list, select the reason for the adjustment. If you need to add a new reason, click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328980843412) beside the Reason list.
7. To maintain the current unit cost, select the Keep Unit Cost checkbox.

![Action_Adjustment_20200505.png](https://support.vintrace.com/hc/article_attachments/32328986694932)

8. Click Save.

If you’re doing a full physical inventory, you may want to use the Stock Take option that allows you to do a count and adjust the list at one time. Refer to [this article](https://support.vintrace.com/hc/en-us/articles/32303335157140) to learn more.
