# Setting Up a Gyro Cage

To set up your gyro cages in vintrace, you can either:

- [Add the gyro cages manually](#adding_manually)
- [Import gyro cages using a spreadsheet](#importing)

## Adding Gyro Cages Manually

To add a gyro cage:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328900703764) Set Up in the sidebar.
2. Click Sparkling.
3. From the Gyro Cages tile, click Configure.
4. Click New Gyro Cage. The Gyro Cage window displays.

![Gyro_Cage_Create_20210617.png](https://support.vintrace.com/hc/article_attachments/32328889248276)

5. Specify the gyro cage’s details.
6. Click Save.

## Importing Gyro Cages With a Spreadsheet

You can add gyro cages by importing a spreadsheet with the details.

You can download a spreadsheet with the various gyro cage fields from the Winery Setup window (Setup Options > Sparkling > Gyro Cages) by clicking Import/Export, then clicking Download Headers.

![Gyro_Cages_-_Downloading_Headers_20210617.png](https://support.vintrace.com/hc/article_attachments/32328878583572)

Edit the spreadsheet to include the gyro cages that you want to import. Be sure to save your changes.

To import the gyro cages:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328900703764) Set Up in the sidebar.
2. Click Sparkling.
3. From the Gyro Cages tile, click Configure.
4. Click Import/Export.
5. From the Gyro Cages Importer window, click Upload a File.

![Gyro_Cages_Importer_-_Upload_a_File_20210617.png](https://support.vintrace.com/hc/article_attachments/32328895685524)

6. Click Choose File.
7. Navigate to your spreadsheet’s location.
8. Click Upload.
