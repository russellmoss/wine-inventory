# Reporting on Sparkling Wine

You can report on your wines using the [Bulk Stock report](#bulk_stock_report), the [Operation Throughput report](#operation_throughput_report), or from the [Vessels page](#vessels_page).

## Bulk Stock Report

To run the Bulk Stock report for your sparkling wine:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329242650644)Reports in the sidebar.
2. Select the Bulk Wine report category in the Winery Reports window.
3. Expand the Advanced Report Options.
4. Select a value for the Tiraged filter.

![Winery_Reports_-_Bulk_Stock_Report_20210622.png](https://support.vintrace.com/hc/article_attachments/32329229838740)

5. Click Generate.

![Bulk_Stock_Report_20210622.png](https://support.vintrace.com/hc/article_attachments/32329229771924)

## Operation Throughput Report

The [Operation Throughput report](https://support.vintrace.com/hc/en-us/articles/32301321300756) shows operations performed during the specified date range.

![Operation_Throughput_Report_20210622.png](https://support.vintrace.com/hc/article_attachments/32329213640724)

To run the Operation Throughput report for your sparkling wine:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329242650644) Reports in the sidebar.
2. Select the Bulk Wine report category in the Winery Reports window.
3. From the Operation list, select *Tirage*, *Riddling*, or *Tirage Admin*.

![Winery_Reports_-_Operation_Throughput_20210622.png](https://support.vintrace.com/hc/article_attachments/32329250966548)

4. Click Generate.

## Vessels Page

You can include the Sparkling State column to the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924), then apply a [column filter](https://support.vintrace.com/hc/en-us/articles/32301336063764) to the column.

![Vessels_Page_-_Sparkling_State_Column_20210622.png](https://support.vintrace.com/hc/article_attachments/32329242187156)
