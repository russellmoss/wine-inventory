# Setting Up a Gyro

To add a gyro:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328909479828)Set Up in the sidebar.
2. Click Sparkling.
3. From the Gyros tile, click Configure.
4. Click New Gyro. The Gyro window displays.

![Gyro_Create_20210617.png](https://support.vintrace.com/hc/article_attachments/32328878626324)

5. Specify the gyro’s details.
6. Click Save.
