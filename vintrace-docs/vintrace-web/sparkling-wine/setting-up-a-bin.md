# Setting Up a Bin

To set up your bins in vintrace, you can either:

- [Add the bins manually](#adding_manually)
- [Import bins using a spreadsheet](#importing)

## Adding Bins Manually

To add a bin manually:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329196578068) Set Up in the sidebar.
2. Click Vessels.
3. From the Bins tile, click Configure.
4. Click New Bin. The Bin window displays.

![Bin_Create_20221024.png](https://support.vintrace.com/hc/article_attachments/32329225660052)

5. Select a suitable Content Type for the bin: *Wine/Juice* for holding liquid (in volume) or *Fruit/Parcels* for receiving fruit (in weight).

![Bin_Create_-_Content_Type_Wine_Juice_20221025.png](https://support.vintrace.com/hc/article_attachments/32329220680340)

6. Click Save.

## Importing Bins With a Spreadsheet

You can add bins by importing a spreadsheet with the bins’ details.

You can download a spreadsheet with the various bin fields from the Winery Setup window (Setup Options > Vessels > Bins) by clicking Import/Export, then clicking Download Headers.

![Bins_-_Downloading_Headers_20210617.png](https://support.vintrace.com/hc/article_attachments/32329210296084)

Edit the spreadsheet to include the bins that you want to import. Be sure to save your changes.

To import the bins:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329196578068)Set Up in the sidebar.
2. Click Vessels.
3. From the Bins tile, click Configure.
4. Click Import/Export.
5. From the Bins Importer window, click Upload a File.

![Bins_Importer_-_Upload_a_File_20210617.png](https://support.vintrace.com/hc/article_attachments/32329225641364)

6. Click Choose File.
7. Navigate to your spreadsheet’s location.
8. Click Upload.
