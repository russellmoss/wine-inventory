# Setting Up a Gyro Cycle

To add a gyro cycle:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328889186196) Set Up in the sidebar.
2. Click Sparkling.
3. From the Gyro Cycles tile, click Configure.
4. Click New Gyro Cycle. The Gyro Cycle window displays.

![Gyro_Cycle_Create_20210617.png](https://support.vintrace.com/hc/article_attachments/32328889218068)

5. Specify the gyro cycle’s details.
6. Click Save.
