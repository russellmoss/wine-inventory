# Recording Loss for a Bin Group

To record loss for a bin group, you’ll need to use the Tirage Admin. You can access the Tirage Admin from the following:

- [The Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- [The Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)
- [The Product page](https://support.vintrace.com/hc/en-us/articles/32303310460948)

![Tirage_Admin_-_General_20210621.png](https://support.vintrace.com/hc/article_attachments/32328899854356)

You can also add a Tirage Admin job to a [work order](https://support.vintrace.com/hc/en-us/articles/32303315610388).

To record loss for a bin group:

1. In the New Bin Group Details section of the Tirage Admin window, click Copy.
2. Enter the number of bottles for each cage

![Tirage_Admin_-_Recording_Loss_-_Number_of_Bottles_20210621.png](https://support.vintrace.com/hc/article_attachments/32328888546068)

3. Click the ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32328888527508) calculator icon. The Loss Units and Loss Volume are calculated.

![Tirage_Admin_-_Calculating_Loss_Units_and_Volume_20210621.png](https://support.vintrace.com/hc/article_attachments/32328888518804)

4. Click Save.
