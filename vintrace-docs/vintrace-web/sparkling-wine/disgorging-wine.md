# Disgorging Wine

To disgorge wine in vintrace, you’ll need to use the [packaging operation](https://support.vintrace.com/hc/en-us/articles/32303327186836). You can access the packaging operation from the following:

- [The Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- [The Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)
- [The Product page](https://support.vintrace.com/hc/en-us/articles/32303310460948)

![Packaging_-_General_20210622.png](https://support.vintrace.com/hc/article_attachments/32329250816532)

You can also add a packaging job to a [work order](https://support.vintrace.com/hc/en-us/articles/32303315610388).

To disgorge wine:

1. In the Packaging Details section of the window, select the stock item.

If you haven’t already created a Single x1 stock item for your disgorged wine, you can set one up by clicking the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329242542228) icon beside the Stock Item field. ![Packaging_-_Adding_Stock_Item_20210622.png](https://support.vintrace.com/hc/article_attachments/32329241868692) Adding the stock item from the Packaging window will include the wine and bottle in the Bill of Materials. You’ll need to add wires, hoods, and any other items that are needed.  ![Create_Stock_Item_20210622.png](https://support.vintrace.com/hc/article_attachments/32329223690004)

2. Specify the amount that you want to disgorge in the Quantity field.

Notice that the quantity for the bottle is 0 since the wine is already in bottle. If you set the bottle to be reused when you set up your tirage stock item, notice that the quantity used is 0. ![Packaging_-_Bottle_Reuse_20210622.png](https://support.vintrace.com/hc/article_attachments/32329242525204)

3. Click the ![Forklift_20200511.png](https://support.vintrace.com/hc/article_attachments/32329242583700) forklift icon and specify where you want to route it to.
4. Select a loss reason.
5. Click Save.

After the wine is packaged, it will no longer display in the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924), but will display in the [Inventory Stock search](https://support.vintrace.com/hc/en-us/articles/32303350682388).

![Stock_Search_-_Unlabelled_Wine_20210622.png](https://support.vintrace.com/hc/article_attachments/32329242565268)

Use the [manufacture operation](https://support.vintrace.com/hc/en-us/articles/32303341990548) when you’re ready to label it.
