# Setting Up Tirage Stock Items

Setting up a tirage stock item lets you track the components that are used when you put wines into [tirage](https://support.vintrace.com/hc/en-us/articles/32301315744404).

To add a tirage stock item:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329242755604) Set Up in the sidebar.
2. Click Sparkling.
3. From the Tirage Items tile, click Configure.
4. Click New Tirage Stock Item. The Tirage Item window displays.

![Create_Tirage_Item_20210618.png](https://support.vintrace.com/hc/article_attachments/32329213673108)

5. Specify the tirage item’s details. Below is a description of some of the fields in the General tab.

- Unit — Select *Single x1*.
- Stock Category — Select your sparkling wine category.
- Bill of Materials — Select the items that will be used (e.g., bottle, crown seal, etc.) during tirage. The bottle in the BOM needs to be set as the Glass/Container type, as this is the only category that allows you to select ‘Packet Type’ and specify the Size (mL). After selecting the item, click the ![Checkmark_in_Gray_Circle_20210618.png](https://support.vintrace.com/hc/article_attachments/32329213699476)checkmark beside the item to verify it. This will cause the item to be listed in the Reusable Tirage Items section.

![Create_Tirage_Item_Example_20210618.png](https://support.vintrace.com/hc/article_attachments/32329275872532)

6. In the Reusable Tirage Items section, select the items that will continue to be used when the wine is disgorged. Selecting the reusable items tells vintrace that the item will continue to be used and that it’s not necessary to pull another item from inventory.
7. Click Save.
