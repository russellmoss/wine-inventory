# Riddling Wine

After a wine is tiraged, the Riddling operation becomes available. The riddling operations is used to [start](#starting_riddling) and [stop](#stopping_riddling) riddling.

You can access the Riddling operation from the following:

- [The Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- [The Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)
- [The Product page](https://support.vintrace.com/hc/en-us/articles/32303310460948)

![Riddling_-_General_20210621.png](https://support.vintrace.com/hc/article_attachments/32328894912916)

You can also add a Riddling job to a [work order](https://support.vintrace.com/hc/en-us/articles/32303315610388).

## Starting Riddling

To start riddling a wine:

1. In the Current Details section of the Riddling window, specify the details for the tirage group and batch that you want to riddle. If you accessed the riddling operation from a wine’s product page, the tirage group and batch will be automatically filled in.
2. Confirm that the Mode is set to *Start Riddling*.
3. If you don’t need to select the specific [cages](https://support.vintrace.com/hc/en-us/articles/32301351756308) to use, click [Quick Fill](#quickfill). Otherwise, click Add and select the cages that you want to use.
4. Specify the number of bottles in each cage.
5. If there was any loss, click the ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32328894964500) calculator icon to record the loss units and volume, then select a reason from the Loss Reason list.

![Riddling_-_General_-_Loss_20210621.png](https://support.vintrace.com/hc/article_attachments/32328878020372)

6. From the Gyro Options section of the window, select the [gyro cycle](https://support.vintrace.com/hc/en-us/articles/32301315793428).
7. Select the [gyros](https://support.vintrace.com/hc/en-us/articles/32301305763988) you want to use.
8. Click Save.

The wine’s Sparkling State on the product page will show *Riddling*. Any loss will be reflected in the number of units displayed.

![Sparkling_State_-_Riddling_20210621.png](https://support.vintrace.com/hc/article_attachments/32328888742676)

## Using Quick Fill to Select Cages

When you’re not concerned about the specific cages to use for riddling, you can use the Quick Fill option.

To use quick fill:

1. On the General tab, click Quick Fill

![Riddilng_-_General_-_Quick_Fill_20210621.png](https://support.vintrace.com/hc/article_attachments/32328900047508)

The Quick Fill Bins window displays.

![Quick_Fill_Bins_-_Riddling_20210621.png](https://support.vintrace.com/hc/article_attachments/32328895004052)

2. In the # Bottles field, enter the number of bottles you want to put into riddling.
3. Select a Fill Area.
4. Click OK.

## Stopping Riddling

To stop riddling, you’ll again use the riddling operation:

1. Confirm that the Mode is set to *Stop Riddling*.
2. If you want to use the existing cages, select the Use Existing Cages checkbox.

![Riddling_-_Stopping_20210621.png](https://support.vintrace.com/hc/article_attachments/32328900108820)

3. Click Save.

The wine’s Sparkling State on the product page will show *Riddled*.

![Sparkling_State_-_Riddled_20210621.png](https://support.vintrace.com/hc/article_attachments/32328878062740)
