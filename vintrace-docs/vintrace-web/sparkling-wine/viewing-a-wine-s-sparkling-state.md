# Viewing a Wine's Sparkling State

A wine’s sparkling state displays on the General tab of the [product page](https://support.vintrace.com/hc/en-us/articles/32303310460948).

![Product_Page_-_General_-_Sparkling_State_20210621.png](https://support.vintrace.com/hc/article_attachments/32328888617108)

If you want to view the date and time of the operation, you can click the ![Clipboard_Icon_20210621.png](https://support.vintrace.com/hc/article_attachments/32328894797716) icon displayed in the upper right order of the Sparkling State tile.

![Viewing_Sparkling_State_Dates_20210621.png](https://support.vintrace.com/hc/article_attachments/32328888634516)

You can also view the date and time of the operations from the Jobs tab of the product page.
