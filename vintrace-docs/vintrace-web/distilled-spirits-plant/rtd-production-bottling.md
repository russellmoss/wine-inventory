# RTD Production: Bottling

Prior to recording the Bulk Dispatch operation, change the batch and ensure its designated variety is set to Cocktails and Mixed Drinks so that it’s correctly reported in Part IV of the Processing Report.

![Diagram - Ready to Drink - Bottling 20231101.png](https://support.vintrace.com/hc/article_attachments/32328761624980)

When you’re ready to bottle the blend, you’ll need to record a [bulk dispatch operation](https://support.vintrace.com/hc/en-us/articles/32301314615700). When recording the operation, be sure to select a dispatch type that’s linked to the Bottled or Packaged DSP bond item
