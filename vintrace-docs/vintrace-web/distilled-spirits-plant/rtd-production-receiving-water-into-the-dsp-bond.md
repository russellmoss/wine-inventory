# RTD Production: Receiving Water into the DSP Bond

This functionality is available starting with [vintrace 9.4.3](https://support.vintrace.com/hc/en-us/articles/6196990653071), but not enabled by default. If you would like to use this functionality, please contact our support team.

Be sure to complete the [configuration steps for a Distilled Spirits Plant (DSP)](https://support.vintrace.com/hc/en-us/articles/8224707977487) before starting this workflow.

![Diagram - Ready to Drink - Receiving Water 20230904.png](https://support.vintrace.com/hc/article_attachments/32329238523412)

To receive water into a DSP bond, record a [bulk intake operation](https://support.vintrace.com/hc/en-us/articles/360000910255). When recording the bulk intake:

- The destination vessel will be in the Processing account.
- On the Vessels tab and Costing & Labs tab of the Bulk Intake window, set the alcohol percentage to 0%.
- On the Costing & Labs tab of the Bulk Intake window, set the DSP Account to Processing and set the DSP tax class to a water DSP tax class.
