# Producing Ready-to-Drink Products

This functionality is available starting with [vintrace 9.4.3](https://support.vintrace.com/hc/en-us/articles/32303276816020), but not enabled by default. If you would like to use this functionality, please contact our support team.

Be sure to complete the [configuration steps for a Distilled Spirits Plant (DSP)](https://support.vintrace.com/hc/en-us/articles/32301312232852) before starting these workflows.

There are several workflows related to the producing ready-to-drink products:

- [Receiving water into the DSP bond](https://support.vintrace.com/hc/en-us/articles/32301281110804)
- [Receiving spirits into a vessel containing water](https://support.vintrace.com/hc/en-us/articles/32301302138260)
- [Moving flavors from a bonded winery](https://support.vintrace.com/hc/en-us/articles/32301312037396)
- [Blending flavors into diluted spirits](https://support.vintrace.com/hc/en-us/articles/32301280191636)
- [Bottling](https://support.vintrace.com/hc/en-us/articles/32301301894036)
