# RTD Production: Blending Flavors into Diluted Spirits

![Diagram - Ready to Drink - Blending Flavors 20231101.png](https://support.vintrace.com/hc/article_attachments/32328774200596)

To blend flavors into diluted spirits, record a many-to-one transfer. The transfer will result in an Alcohol, Flavor, Materials, Dumped tax event in the Processing Report.

The proof gallons will be tracked if the flavors have an alcohol percentage recorded.
