# Fulfilling Allocations

If you have multiple allocations, or there’s un-allocated volume on a wine, you can select which allocation to fulfill for a Bulk Dispatch or Packaging operation. For these operations, the Proportional Allocational option is selected by default. This auto-calculates the fulfilled amount of allocated and un-allocated volume according to their ratio.

![Bulk_Dispatch_and_Packaging_-_Proportional_Allocation_20201014.png](https://support.vintrace.com/hc/article_attachments/32328862843796)

If you’d prefer to use a different allocation option, click Proportional Allocation. The Manage Allocation window displays where you can select a different option.

![Manage_Allocation_20201014.png](https://support.vintrace.com/hc/article_attachments/32328891328404)
