# Changing an Allocated Product’s Status

This functionality is available starting with [vintrace 9.9.1](https://support.vintrace.com/hc/en-us/articles/32303283058068), but not enabled by default. If you would like to use this functionality, please contact our support team.

In order to change an allocated product’s status, you will need the [Can Adjust Allocation Product Status permission](https://support.vintrace.com/hc/en-us/articles/360000813755-Roles-and-Permissions#Permissions).

You can change an allocated product’s status and produced volume. Completed products are automatically excluded from the [Allocations table](https://support.vintrace.com/hc/en-us/articles/32301328118932) so that you can focus on active products.

To change an allocated product’s status:

1. Click ![Products Icon 20200715.png](https://support.vintrace.com/hc/article_attachments/32328753634196) products in the sidebar.
2. Click the product’s name. The [product’s allocation details page](https://support.vintrace.com/hc/en-us/articles/32301350652948) displays.
3. Click the ![Pencil Gray Icon 20200414.png](https://support.vintrace.com/hc/article_attachments/32328758564500) pencil icon displayed in the Status tile.

![Status Tile - Edit Icon 20240430.png](https://support.vintrace.com/hc/article_attachments/32328785360660)

The Edit Product Status window displays.

![Edit Product Status 20240430.png](https://support.vintrace.com/hc/article_attachments/32328768015252)

4. Change the status and specify the other details.
5. Click Save.
