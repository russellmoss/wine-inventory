# Product Allocation Details Page

The Product Allocation Details page displays when you click a product on the [Product Allocations page](https://support.vintrace.com/hc/en-us/articles/32301319185940).

![Product_Allocation_Details_-_General_20220112.png](https://support.vintrace.com/hc/article_attachments/32328664216724)

The page displays the following tabs:

- General
- In Production
- Produced
- Stock Allocation - This is only available if you have an inventory module licence
- Stock Items - This is only available if you have an inventory module licence
- Notes
- History

The General tab displays the following tiles. You can edit any of these values by clicking the ![Pencil_Gray_Icon_20200414.png](https://support.vintrace.com/hc/article_attachments/32328664230292) pencil icon in the tile.

You can add a Recent Note by clicking the ![Grey_Plus_Icon.jpg](https://support.vintrace.com/hc/article_attachments/32328629397268) plus icon in the tile.

- Winery - This is only available if you have a multi-winery license and have permission to access other wineries.
- Owner
- Variety
- Sub AVA - Specifying a sub AVA also populates the Appellation tile.
- Appellation
- Chief Winemaker
- Recent Note
- Account
- Item Code - This value must be unique.

![Product_Allocation_Details_-_General_20220112.png](https://support.vintrace.com/hc/article_attachments/32328638354836)

If you’d prefer to display the item code instead of the vintage and product name, contact [support@vintrace.com](mailto:support@vintrace.com).
