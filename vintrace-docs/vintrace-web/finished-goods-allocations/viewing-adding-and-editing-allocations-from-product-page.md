# Viewing, Adding and Editing Allocations from Product Page

The Allocations tab on the wine's [product page](https://support.vintrace.com/hc/en-us/articles/32303310460948) displays all of the products that a wine has been allocated to.

You can add and edit allocations from this tab.

In order to view the Allocations tab, you will need the [Can View Product Allocations permission](https://support.vintrace.com/hc/en-us/articles/32303349421588).

In order to add and edit an allocation, you will need the [Can Manage Product Allocations permission](https://support.vintrace.com/hc/en-us/articles/32303349421588).

![Allocations Tab 20250411.jpg](https://support.vintrace.com/hc/article_attachments/37947519613332)

## Editing allocations

To remove an allocation, enter 0 in the Allocation field and click Save.

1. Display the wine’s product page by either using vintrace’s Quick Search feature, or by selecting the wine from the [Vessels page](https://support.winery-software.com/hc/en-us/articles/360001550655-The-Vessels-Page).
2. Select the Allocations tab.
3. Click Edit.

![Allocations Tab Edit 20250411.jpg](https://support.vintrace.com/hc/article_attachments/37947536489108)

The Allocation field can be edited.

The pagination and allocation summary are not shown when editing allocations.

4. In the Allocation field, enter the updated allocation volume.
5. Click Save.

![Allocations Tab Save 20250411.jpg](https://support.vintrace.com/hc/article_attachments/36568996869524)

The Allocation volumes and summary are updated.

![Allocations Tab Updates 20250411.jpg](https://support.vintrace.com/hc/article_attachments/37947536490004)

## Adding allocations

1. Display the wine’s product page by either using vintrace’s Quick Search feature, or by selecting the wine from the [Vessels page](https://support.winery-software.com/hc/en-us/articles/360001550655-The-Vessels-Page).
2. Select the Allocations tab.
3. Click Add Product.

![Allocations Tab Add Product 20250529.jpg](https://support.vintrace.com/hc/article_attachments/37947519616532)

4. Select the allocation product from the search.

![Allocations Tab Search 20250530.jpg](https://support.vintrace.com/hc/article_attachments/37947519617812)

The selected allocation product is added to the bottom.

5. In the Allocation field, enter the volume to be allocated to the vessel.
6. Click Save.

![Allocations Tab Save Add Product 20250530.jpg](https://support.vintrace.com/hc/article_attachments/37947519619092)

The Allocation volume and summary are updated.

![Allocations Tab Add Product Updates 20250530.jpg](https://support.vintrace.com/hc/article_attachments/37947536493972)
