# Exporting and Importing Product Allocations

In order to export and import product allocations using a CSV file, you will need the [Can Manage Product Allocations permission](https://support.vintrace.com/hc/en-us/articles/360000813755-Roles-and-Permissions#Permissions).

This functionality allows importing product allocations across all wineries. Users will also need the [All Winery Access permission](https://support.vintrace.com/hc/en-us/articles/360000813755-Roles-and-Permissions#Permissions).

To update your product allocations using a CSV, you’ll need to complete the following steps:

1. [Export a CSV file of your product allocations from vintrace](#exporting).
2. Update the CSV to reflect the changes you want to make.
3. [Import the CSV file into vintrace](#importing).

## Exporting Product Allocations from vintrace

To export product allocations from vintrace:

1. Click ![Products Icon 20200715.png](https://support.vintrace.com/hc/article_attachments/32329238064148) products in the sidebar.
2. Click the ![Manage Allocations Icon 20220111.png](https://support.vintrace.com/hc/article_attachments/32329196898836) manage allocations icon.
3. Click the Manage Allocations link.

![Manage Allocations Link 20240430.png](https://support.vintrace.com/hc/article_attachments/32329196929300)

The Manage Allocations window displays.

![Manage Allocations 20240430.png](https://support.vintrace.com/hc/article_attachments/32329238078356)

4. Click Export Existing Records.
5. Specify the location where you would like to save the CSV file.

## Importing Product Allocations into vintrace

After updating the CSV file that you exported, you can import the file into vintrace to update your product allocations.

To import your product allocations into vintrace:

1. Click ![Products Icon 20200715.png](https://support.vintrace.com/hc/article_attachments/32329238064148) products in the sidebar.
2. Click the ![Manage Allocations Icon 20220111.png](https://support.vintrace.com/hc/article_attachments/32329196898836) manage allocations icon.
3. Click the Manage Allocations link.

![Manage Allocations Link 20240430.png](https://support.vintrace.com/hc/article_attachments/32329196929300)

The Manage Allocations window displays.

4. Click Choose File.
5. Specify the CSV file containing your updated product allocations.
6. Click Check Import File.
7. If the Import Validity Check window displays any errors, correct those errors in the CSV, save the CSV, then repeat steps above. When no errors are found, click Import.

![Import Validity Check - Import Button 20240430.png](https://support.vintrace.com/hc/article_attachments/32329225887124)

8. Click OK after the import completes successfully.
