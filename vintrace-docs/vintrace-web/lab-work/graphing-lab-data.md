# Graphing Lab Data

You can graph multiple metrics for multiple wines in a single graph.

![Chart_3.png](https://support.vintrace.com/hc/article_attachments/32329067406996)

Lab data can be graphed from the [Vessels page](#Graphing_From_Vessels_Page) and from the [Ferment Console](#Graphing_From_Ferment_Console).

## Graphing From the Vessels Page

To graph lab data from the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924):

1. Click ![Vessels_Menu_Option_20200402.png](https://support.vintrace.com/hc/article_attachments/32329067441556) Vessels in the sidebar.
2. Search for the wine(s) for which you want to graph.
3. Select the checkbox beside the wine(s). To select all wines listed, select the checkbox beside the left-most column heading.
4. From the Actions menu in the lower left, select Chart Lab Results.

![Actions_-_Chart_Lab_Results_20200728.png](https://support.vintrace.com/hc/article_attachments/32329049619988)

5. From the Analysis Chart Options window, specify the date range that you want to include in the chart, select the metrics, then click OK.

![Analysis_Chart_Options_20200728.png](https://support.vintrace.com/hc/article_attachments/32329038763668)

## Graphing From the Ferment Console

To graph lab data from the Ferment Console:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329026684180) More Options in the sidebar.
2. Specify your search criteria.
3. Select the vessels you want to graph by clicking the row. To select all vessels listed, click All. The row of the selected vessels will be highlighted.
4. Click Chart at the bottom of the window.

![Ferment_Console_-_Graphing_20200728.png](https://support.vintrace.com/hc/article_attachments/32329049661204)

5. From the Analysis Chart Options window, specify the date range that you want to include in the chart, select the metrics, then click OK.
