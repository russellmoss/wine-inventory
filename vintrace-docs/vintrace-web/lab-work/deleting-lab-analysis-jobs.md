# Deleting Lab Analysis Jobs

Deleting an analysis job deletes it permanently so be sure that you’re deleting the correct job. You can delete analysis jobs from the Lab Console, or from a work order.

## Deleting an Analysis Job from the Lab Console

To delete an individual analysis job from the Lab Console:

1. Click ![Lab_Menu_Option_20200403.png](https://support.vintrace.com/hc/article_attachments/32328826587796) Lab in the sidebar.
2. Click the down arrow beside the View button.
3. Select Delete.

![Lab_Console_-_View_-_Delete_20200803.png](https://support.vintrace.com/hc/article_attachments/32328796685076)

## Deleting Multiple Analysis Jobs from the Lab Console

To delete multiple jobs from the Lab Console:

1. Click ![Lab_Menu_Option_20200403.png](https://support.vintrace.com/hc/article_attachments/32328826587796) Lab in the sidebar.
2. Search for the analysis jobs you want to delete.

Be sure that the jobs listed on the page are the ones you want to delete.

3. From the Actions menu, select Delete This Page.

![Lab_Console_-_Actions_-_Delete_This_Page_20200803.png](https://support.vintrace.com/hc/article_attachments/32328796535828)

## Deleting Analysis Jobs on a Work Order

To delete analysis jobs on a work order:

1. Click ![Jobs_Menu_20200803.png](https://support.vintrace.com/hc/article_attachments/32328810113172) Jobs in the sidebar.
2. From the Status list, select the status of the work order that includes the analysis job you want to delete.
3. From the Operation Type list, select *Analysis*.

![Operations_List_-_Analysis_20200803.png](https://support.vintrace.com/hc/article_attachments/32328818286484)

4. Click Search.
5. Select the work order(s) you want to delete.
6. From the Actions menu, select Cancel Selected.

![Actions_-_Cancel_Selected_20200803.png](https://support.vintrace.com/hc/article_attachments/32328818416660)

A confirmation message displays.

![Cancel_Confirmation_20200803.png](https://support.vintrace.com/hc/article_attachments/32328810208532)

7. Click Yes to cancel the job.
