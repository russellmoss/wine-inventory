# Setting Up an Additive

You can set up new additives from the Winery Setup window (Setup Options > Production > Additives):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328638627092) Set Up in the sidebar.
2. Click Winemaking.
3. From the Additives tile, click Configure.
4. Click New Additive. The Additive window displays.

![Additive_20200730.png](https://support.vintrace.com/hc/article_attachments/32328603940116)

5. Specify the details for the new additive. Below are descriptions for some of the fields:

- Cause Treatment — You can link the additive to automatically start a treatment on the wine. Often this is used to link yeast with a Start Ferment treatment that automatically starts the ferment of a wine in vintrace.
- Allergen — If the additive contains a known allergen, select the Allergen checkbox. In the field beside the checkbox, enter the description of the allergen that you'd like displayed on the [Label Integrity Statement](https://support.vintrace.com/hc/en-us/articles/32301330173972).

![Additive_-_Allergen_Description_on_Label_Integrity_Statement_20200917.png](https://support.vintrace.com/hc/article_attachments/32328629647508)

- Linked Stock Item — If you have the Inventory module, you can track stock levels by specifying the item when you create the additive.
- Cost Item — If you’re tracking costing for this item independently from maintaining stock levels, you can specify the cost item that will be added to the wine’s cost when this additive is applied. Refer to our [Managing Chemical Addition Costs article](https://support.vintrace.com/hc/en-us/articles/32301356440084) to learn more. For information on metric weightings, refer to our [Setting Up Metric Weightings for Additions Calculator article](https://support.vintrace.com/hc/en-us/articles/32301358168468).

6. Click Save.

Users with a multi-winery license can specify which additives are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/32301304791316) for details.
