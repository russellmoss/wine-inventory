# Setting the Default Analysis Template for Ferments

You can specify the default analysis template to use for [ferments](https://support.vintrace.com/hc/en-us/articles/32303278530708). When you select a default analysis template for ferments, that template will be automatically selected when you start a ferment.

![Start_Ferment_with_Default_Analysis_Template_20200731.png](https://support.vintrace.com/hc/article_attachments/32329153262228)

If you haven't created an analysis templates for ferments, refer to our [Setting Up an Analysis Template article](https://support.vintrace.com/hc/en-us/articles/32301372281748) for details.

You can specify the default analysis for ferments from the Winery Setup window (Setup Options > Production > Analysis Templates):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329153270676) Set Up in the sidebar.
2. Click Lab.
3. From the Analysis Template tile, click Configure.
4. Select the analysis template that you want to use as the default for ferments.
5. From the Default For list, select Ferment.

![Analysis_Template_Update_-_Ferment_-_Default_For_Ferment_20200731.png](https://support.vintrace.com/hc/article_attachments/32329162031508)

6. Click Save.

If you select a different analysis template when recording the analysis of a wine that’s currently fermenting, you’ll have the option to use the selected template as the default for subsequent analysis.
