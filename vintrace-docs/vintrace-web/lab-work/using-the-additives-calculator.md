# Using the Additives Calculator

The additives calculator determines how much of an additive is required to reach the desired metric level. It’s commonly used for KMBS or PMS, CUSO4, and a variety of other sulfur additions.

Before you can use the additives calculator, be sure do the following:

1. [Set up the metric](https://support.vintrace.com/hc/en-us/articles/32301345260948).
2. [Set up the additive](https://support.vintrace.com/hc/en-us/articles/32301344910740) that you want to associate with the metric. In the example below, KMBS is the additive and Free SO2 is the metric. When selecting the additive’s metric, be sure to specify the weighting for the metric and select the rate of addition in the Rate Unit field.

![Additive_Update_-_Rate_Unit_and_Metric_20200803.png](https://support.vintrace.com/hc/article_attachments/32328900651284)

The additives calculator is available with the Additions operation. There are a number of ways to access the Additions operation. You can click the ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32328889106964) Operations icon, then select Additive from the following:

- The Product page
- [The Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- The Job Management page

To use the additives calculator:

1. From the Metric list, select the metric you want to use in the calculation.
2. If more than one additive uses the metric, select the additive from the Additive list.
3. Enter the target rate.
4. Enter the current rate if the metric hasn’t been tested or has been invalidated. and the desired rate.

vintrace calculates the difference between the current rate and the target rate. If you only enter the target rate, vintrace assumes you’re going from zero to your target and calculates a higher addition. Be sure to pay attention to this to prevent adding too much sulfur or copper.

5. Click the ![Calculator_Icon_20200410.png](https://support.vintrace.com/hc/article_attachments/32328918158612) that’s displayed beside the Required Amount field.
6. Confirm that the amount looks correct.
7. If you’d like to use the amount, click Use This Value to copy the value to the Amount field.

![Additive_-_Use_This_Value_20200803.png](https://support.vintrace.com/hc/article_attachments/32328900636308)
