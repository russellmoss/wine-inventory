# Cap Management

Setting up a cap management in vintrace involves four steps:

1. [Creating an analysis template.](#Creating_Analysis_Template)
2. [Requesting an analysis.](#Requesting_Analysis)
3. [Printing the worksheet.](#Printing_Worksheet)
4. [Recording the results.](#Recording_Results)

## Creating an Analysis Template

You’ll need to [create an analysis template](https://support.vintrace.com/hc/en-us/articles/32301372281748) for each different regime that you have. This can be a combination of analysis metrics (e.g., Brix and Temp), and cap management techniques (e.g., punch down, pump-over) that are set up as dummy metrics in the system. The technique dummy metrics should be simple yes/no metrics. You can also include dummy metrics that include duration information set up as units. These will print as columns on a worksheet for your cellar staff.

When creating your analysis template, be sure to leave the Print Labels on Bulk Lab Request checkbox unchecked as it’s not necessary to print lab labels when you generate a cap management worksheet for the day.

![Analysis_Template_Update_-_Print_Labels_Unchecked_20200721.png](https://support.vintrace.com/hc/article_attachments/32329066213268)

## Requesting an Analysis

To locate the wines that you want to produce a worksheet for:

1. Click ![Vessels_Menu_Option_20200402.png](https://support.vintrace.com/hc/article_attachments/32329058033044) Vessels from the sidebar.
2. Use the [filters on the Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924#FilteringtheVesselsPage) to locate the wines.
3. Select the checkbox beside the vessel(s). To select all vessels listed, select the checkbox beside the left-most column heading.
4. From the Actions menu in the lower left, select Lab Work.
5. From the Request a Lab Sample/Analysis window, select the analysis template that you created.

![Request_a_Lab_Sample_Analysis_-_Analysis_Template_Pump_Over_AM_PM_20200721.png](https://support.vintrace.com/hc/article_attachments/32329025472788)

6. Click OK.

## Printing the Worksheet

To print the worksheet for your cellar staff:

1. Click ![Lab_Menu_Option_20200403.png](https://support.vintrace.com/hc/article_attachments/32329042323092) Lab in the sidebar. The Lab Console displays.
2. From the Template list in the Lab Console, select the analysis template.
3. Click Search. The lab requests display.
4. If you’re supplying the cellar staff with duration instructions, enter the amounts for each tank.

Once a batch has had its cap management lab performed, the previous day’s duration and values will display beside each tank in the Lab Console. This should make it easy to see how the values have changed.

5. To save the information, you can either select Update from the View menu of each request, or select Update This Page from the Actions menu to update all lab requests listed.

![Lab_Console_-_Actions_-_Update_This_Page_20200721.png](https://support.vintrace.com/hc/article_attachments/32329066174740)

6. From the Export menu, select PDF (This Page).

![Lab_Console_-_Export_-_PDF_This_Page_20200721.png](https://support.vintrace.com/hc/article_attachments/32329058104980)

A PDF of the worksheet displays that you can print.

![Lab_Worksheet_Pump_Over_AM_PM_20200721.png](https://support.vintrace.com/hc/article_attachments/32329058046228)

## Recording Results

When the cellar staff returns the worksheet, you’ll need to record the results by doing the following:

1. Click ![Lab_Menu_Option_20200403.png](https://support.vintrace.com/hc/article_attachments/32329042323092) Lab in the sidebar. The Lab Console displays.
2. From the Template list in the Lab Console, select the analysis template, then select date that matches the worksheet.
3. Click Search. The lab requests display.
4. Enter the details including readings, durations, and whether a pump-over or punch-down was performed.
5. To save the information, you can either select Update from the View menu of each request, or select Update This Page from the Actions menu to update all lab requests listed.
