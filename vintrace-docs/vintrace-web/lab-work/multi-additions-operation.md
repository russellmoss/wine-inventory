# Multi Additions Operation

The Multi Additions operation lets you apply one or more additives to multiple vessels.

## Accessing the Multi Additions Operation

There are a number of ways to access the Multi Additions operation. You can click the ![Operations_Icon_20200407.png](https://support.vintrace.com/hc/article_attachments/32328594619540) Operations icon, then select Multi Additions from the following:

- [The Product page](https://support.vintrace.com/hc/en-us/articles/32303310460948)
- [The Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)
- [The Job Management page](https://support.vintrace.com/hc/en-us/articles/32303318317972)

![Multi_Additions_20200720.png](https://support.vintrace.com/hc/article_attachments/32328600736276)

## Performing a Multi Additions Operation

1. Select an additive template by clicking the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32328625909524) beside the Additive Template field.
2. Click Modify.

![Multi_Additions_-_Additive_Template_20200720.png](https://support.vintrace.com/hc/article_attachments/32328625943316)

3. If you need to change the additives and/or values for the current operation, you can do so and it will NOT affect the selected additive template. For example, you can add additives for this operation and it won’t affect the additive template that you selected in step 1. If you find yourself making the same changes to templates, you may want to [create a new additive template](https://support.vintrace.com/hc/en-us/articles/32301359803412).

![Temporary_Template_20200720.png](https://support.vintrace.com/hc/article_attachments/32328632443796)

4. Click Search for Wines. The Selecting Wines window displays.
5. Click Add Wines from Product Search.
6. From the Bulk Wine Search window, use the filters at the top of the window to find the wines.

If you’re using an additive template that has a target rate, you can search for wines that are under the target rate by clicking More Options, then clicking Add a Filter. In the example below, a Metric Value Range filter has been added for Free SO2.

![Bulk_Wine_Search_-_Metric_Value_Range_Free_SO2_20200720.png](https://support.vintrace.com/hc/article_attachments/32328618115476)

7. To include a wine in the operation, select the checkbox beside it. To select all wines listed, select the checkbox beside the Status column heading.
8. Click Use Selection.

![Bulk_Wine_Search_20200702.png](https://support.vintrace.com/hc/article_attachments/32328610215828)

The selected wines display in the Selecting Wines window.

9. Click Continue. The Additives Total Amount Summary window displays the total amount of additive calculated from the selected vessels and template. If you have the Inventory module, click ![Forklift_20200511.png](https://support.vintrace.com/hc/article_attachments/32328600842004) to choose where the additive stock item is routed; otherwise, select which lot to use.

![Additives_Total_Amount_Summary_-_Route_Stock_From_20200720.png](https://support.vintrace.com/hc/article_attachments/32328600828308)

If there were any errors in the calculation, then it won’t contribute towards the total amount.

10. Click OK or Skip based on what you’d like to do.

- To pre-fill the additive’s information for the selected wines, click OK. This option will notify you of any errors. If any errors are found, modify or remove the additives from the Multi Additions window.
- To use the additive template to pre-fill the additive’s information for the selected wines, click Skip. This does not fill in the calculated amount. This option is useful when you’re scheduling an operation, but don’t know the correct rates or amounts.

11. Add, edit or remove wines and additions in the Multi Additions section of the window.

- To add more wines, click Add Line or Add From Search.
- To apply additional additives, click Add Additive for the wine you want to apply the additive.

12. Click Verify. This checks each entry and highlights any errors.

You can hide the wines that have been verified by clicking Hide Verified. This is useful when there are a large number of wines since it only displays wines that need to be modified. You can display all wines by clicking Show All.

![Multi_Additions_with_Wines_20200720.png](https://support.vintrace.com/hc/article_attachments/32328626022036)

13. Click Save. If all required information is entered and verified, you’ll be asked to confirm the operation.
14. Click OK.
