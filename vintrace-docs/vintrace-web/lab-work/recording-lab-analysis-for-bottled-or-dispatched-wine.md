# Recording Lab Analysis for Bottled or Dispatched Wine

To record an analysis on a wine that’s been bottled or dispatched:

1. Use the Quick Search to find the wine. The Stock Item Overview window displays.
2. From the Actions menu at the bottom of the window, select Record an Analysis.

![Stock_Item_Overview_-_Actions_-_Record_Analysis_20200730.png](https://support.vintrace.com/hc/article_attachments/32328883518356)

3. If the wine has been packaged, select the packaging that you want to record the analysis for.

![Analysis_-_Select_Packaging_for_Wine_Batch_20200730.png](https://support.vintrace.com/hc/article_attachments/32328843359252)

The Analysis window displays.

![Analysis_20200730.png](https://support.vintrace.com/hc/article_attachments/32328860486292)

4. Specify the details for the analysis.
5. Click Save.
