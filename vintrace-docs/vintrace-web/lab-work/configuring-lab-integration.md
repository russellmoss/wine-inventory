# Configuring Lab Integration

The steps detailed below apply to WineScan, Priority ERP, Konelab, OenoFoss, [ETS](https://support.vintrace.com/hc/en-us/articles/32301324029076), [Baker Labs](https://support.vintrace.com/hc/en-us/articles/32303294416660), Thermo Scientific Gallery, ChemWell, [Admeo/BioSystems Y15](https://support.vintrace.com/hc/en-us/articles/32303293999892) and SPICA, and [Anton Paar DMA 35](https://support.vintrace.com/hc/en-us/articles/32301304300692).

Before you can request an analysis from a lab, you'll need to [add the lab to the vintrace address book](#h_01EEZR5EF62CV6AHGDEYD7FC7B). If the metric names used by the lab differ from the metric names used in vintrace, you'll also need to [map the names](#Mapping_Metric_Names).

## Adding the Lab to the Address Book

In order to integrate your lab with vintrace, you'll need to add the lab to the address book. To add a lab to the address book:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328586691732) More Options in the sidebar.
2. From the Address Book tile, click Open Address Book.
3. From the Add menu in the lower left, select *Organization*.

![Address](https://support.vintrace.com/hc/article_attachments/32328586809364)

4. Specify the details for the lab. Be sure to select the Laboratory role.

![Create_Basic_Organization_Widget_-_OenoFoss_-_Laboratory_Role_20200727.png](https://support.vintrace.com/hc/article_attachments/32328589277332)

5. Click Save.

For details on requesting and viewing analysis from a lab, refer to our [Requesting Analysis and Viewing Results](https://support.vintrace.com/hc/en-us/articles/32301356827540) article.

## Mapping Metric Names to vintrace

If you plan to [import results from the lab to vintrace](https://support.vintrace.com/hc/en-us/articles/32301343026964), you'll need to determine if the metric names used by the lab are different from the metric names used in vintrace. You can do this by looking at the column headers in the lab's output.

If the metric names are different, you’ll need to map the names between the systems. Refer to our [Mapping a Lab's Metric Names](https://support.vintrace.com/hc/en-us/articles/32301340432788) article to learn more.
