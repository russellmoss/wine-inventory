# Setting Up Sulphur Alarms and Reminders

You can create and save search filters that list wines that haven’t had a metric tested for a specified number of days. You can then add the search to your vintrace Dashboard so that you’re reminded when there are one or more wines that meet your search criteria.

For example, you can create a filter that lists wines that haven’t had their sulfur tested in the last 30 days.

![Search_Filters_-_FSO2_Not_Tested_30_Days_20200731.png](https://support.vintrace.com/hc/article_attachments/32328877576596)

## Adding a Filter

To add a filter:

1. Click Vessels in the sidebar.
2. Click ![Filter_20200323.png](https://support.vintrace.com/hc/article_attachments/32328894420500).
3. Click Add a Filter.

![Search_Filters_-_Add_a_Filter_Button_20200731.png](https://support.vintrace.com/hc/article_attachments/32328894454676)

4. Select Lab Date from the list.

![Adding_Filter_-_Column_List_-_Lab_Date_20200731.png](https://support.vintrace.com/hc/article_attachments/32328888268564)

5. From the Metric list, select the metric that you want to monitor.
6. From the second list, select Not Tested.
7. Enter the number of days after which the wine will be included if it hasn’t been tested.

![Search_Filters_-_Lab_Date_-_Not_Tested_30_Days_20200731.png](https://support.vintrace.com/hc/article_attachments/32328877591828)

8. Do either of the following:

- To apply the filter to the Vessels page, click Apply. The icon on the Vessels page displays ![Filter_Applied_20200323.png](https://support.vintrace.com/hc/article_attachments/32328877606292) to indicate that a filter has been applied. You can click the icon if you need to change the filters, or if you want to save the filter for future use.
- To save the filter for future use, click Save, enter a name for the filter, then click ![Checkmark_in_Green_Box_20200731.png](https://support.vintrace.com/hc/article_attachments/32328899678612).

![Save_Search_-_SO2_Not_Tested_in_30_Days_20200731.png](https://support.vintrace.com/hc/article_attachments/32328908075796)

The filter will be included in the list of filters in the Use a Saved Search list.

## Adding the Search to Your Dashboard

To add the search to your My Searches dashlet on the Dashboard, click the ![Heart_White_20200731.png](https://support.vintrace.com/hc/article_attachments/32328888285204) in the Search Filters window.

![Search_Filters_-_Save_to_Dashboard_20200804.png](https://support.vintrace.com/hc/article_attachments/32328917012500)

The icon changes to a ![Heart_Green_20200731.png](https://support.vintrace.com/hc/article_attachments/32328888348692) to indicate that the search has been saved to the Dashboard.

You can configure the visual alert ranges that are used for the search.
