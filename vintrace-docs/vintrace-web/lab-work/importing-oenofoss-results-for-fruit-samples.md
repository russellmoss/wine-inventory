# Importing OenoFoss Results for Fruit Samples

When you process lab samples on your OenoFoss, you’ll need to enter a code to reference the sample. The code you enter must match the designated sample set code that was allocated to the block for the year in vintrace.

The sample set code allows vintrace to match the analysis results from OenoFoss against your blocks. It’s important to create your sample sets in vintrace ahead of harvest each year before trying to upload results from OenoFoss. For details on creating and managing your sample sets in vintrace, refer to our [Fruit Sampling article](https://support.vintrace.com/hc/en-us/articles/32303291530132).

## Finding the Sample Set Code

To find the sample set code:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328616539284) More Options in the sidebar.
2. From the Harvest tile, click Maturity Samples. Search for the fruit sample set. The sample set code displays in the Code column.

![Sample_Set_Console_-_Code_Column_20200728.png](https://support.vintrace.com/hc/article_attachments/32328616531988)

## Importing OenoFoss Results for Fruit Samples

Before you can import OenoFoss lab results into vintrace, be sure to map the metric names. For details, refer to our [Mapping OenoFoss or Winescan Metric Names article](https://support.vintrace.com/hc/en-us/articles/32301340432788).

Be sure that your OenoFoss file is in CSV format.

To import OenoFoss results for fruit samples:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328616539284) More Options in the sidebar.
2. From the Harvest tile, click Maturity Samples.
3. Click Upload.

![Sample_Set_Console_-_Upload_Button_20200728.png](https://support.vintrace.com/hc/article_attachments/32328608704148)

The Fruit Samples Importer window displays.

You can download a sample OenoFoss file and compare it to the data that you want to upload. To download the sample OenoFoss file from the Fruit Samples Importer window, click Download the Import File Format.

4. From the Import Type list, select Fruit Samples (OenoFoss).

![Fruit_Samples_Importer_-_Import_Type_OenoFoss_20200728.png](https://support.vintrace.com/hc/article_attachments/32328624537876)

5. Click Choose File.
6. Select the file to upload.
7. Click Check Import File to check the file for any errors.
