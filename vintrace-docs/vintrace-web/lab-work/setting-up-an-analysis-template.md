# Setting Up an Analysis Template

Analysis templates can save you time when you need to record metrics or request an analysis. When you add an analysis template, you pre-define the metrics that are included in the analysis. Some harvest/vintage operations also allows [default templates](#h_01K5DV2B8W80T92WXZXNYG9211) to be assigned for quick access.

![Analysis_Template_Metrics_20200826.png](https://support.vintrace.com/hc/article_attachments/32328887003412)

The analysis template can also be used to change the [product state](https://support.vintrace.com/hc/en-us/articles/32301350848916). If you’re using vintrace for costing, you can also track the cost amount for the analysis.

To create an analysis template:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328864635412)Set Up in the sidebar.
2. Click Lab.
3. From the Analysis Template tile, click Configure.
4. Click New Analysis Template. The Analysis Template window displays.

![Analysis_Template_Create_20200721.png](https://support.vintrace.com/hc/article_attachments/32328893069588)

5. Enter the details for the analysis template.

You can automatically change a wine's [product state](https://support.vintrace.com/hc/en-us/articles/32301350848916) when the analysis template is used by selecting the product state from the list. For example, the *Juice Panel* analysis template can change the wine's product state to *TTA/H2O Add*.
![Analysis_Template_-_Juice_Panel_20211028.png](https://support.vintrace.com/hc/article_attachments/32328887016980)

6. Click Save.

## Default Analysis Templates

You can also assign templates as the default for certain events surrounding vintage/harvest. In the drop down menu 'Default for', you may assign a template as the default for fruit sampling or receival, and for ferment rounds. Once a template is set as the default, it will pre-fill the template field on the relevant analysis operation.

![default.analysis.png](https://support.vintrace.com/hc/article_attachments/41336796997396)

Users with a multi-winery license can specify which analysis templates are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/4413475617423) for details.
