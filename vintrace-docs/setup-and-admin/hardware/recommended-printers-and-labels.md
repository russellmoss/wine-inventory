# Recommended Printers and Labels

We recommend the DYMO Label Writer 550 and 550 Turbo for the following label uses:

- fruit sample requests
- lab requests
- scale bookings

Only official DYMO labels that were manufactured after April 2020 will print in the 550 series of printers. Any 3rd party/generic labels will not print in these printers.

The previously recommended DYMO Label Writer 450, 450 Turbo or Twin Turbo have been discontinued.

The printer and labels can be sourced from the links or suppliers below.

### Barrel Barcode Labels

For barrel barcode labelling, the best we can recommend is a High-Tac adhesive Polypropylene label from Zebra [(PolyPro 300T)](https://www.barcodesinc.com/zebra/part-18929-r.htm). These are UV and water-resistant, and are compatible with the Zebra 400 series printers. By default our barcode labels are suited to fit 4'' x 2.5" labels.

DYMO also offers a range of durable labels that are chemical and water-resistant, but do not have the UV protection the Zebra 3000T PolyPro offers. This could cause fade and loss of adhesive if left outdoors for extended periods. Our default barcode labels are designed to fit the following DYMO labels.

- [DYMO® LW Durable 2-1/4" x 1-1/4"](https://www.labelcity.com/Dymo-LabelWriter-Durable-Labels-1933084) (57mm x 32mm)
- [DYMO® LW Durable 2-5/16" x 4"](https://www.labelcity.com/Dymo-LabelWriter-Durable-Labels-1933088) (59mm x 102mm)
