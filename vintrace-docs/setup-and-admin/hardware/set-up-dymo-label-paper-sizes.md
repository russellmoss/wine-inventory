# Set up DYMO label paper sizes

To adjust the paper size on your DYMO label printer while using a Windows operating system, open up **Printers** under the **Control Panel**. You should see something similar to below listing your printers.

[![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/1.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/1.jpg)

If you right click on the printer and open up its **Printing Preferences**, then open up the **Advanced Options** you should see something similar to below.

[![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/3.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/3.jpg)

Then you can change the paper size option to the labels you have available, we will be using 99014 shipping labels in this example.
 [![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/5.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/5.jpg)

Once that is changed click **OK** on all the windows until you are back at the printer list.

Then open up the **Properties** after right clicking your printer.

[![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/6.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/6.jpg)

When you have the **Properties** open, select the **Advanced** tab, then open up the **Printing Defaults**.

[![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/7.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/7.jpg)

This brings up a window similar to before.

[![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/8.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/8.jpg)

Click the **Advanced** button and change the paper size like before.

[![](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/9.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2012/06/9.jpg)

When that is done click **OK** until you are back at the printers window and that should have saved the new print label sizes on the printer.
