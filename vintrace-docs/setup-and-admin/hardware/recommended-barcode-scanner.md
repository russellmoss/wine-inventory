# Recommended Barcode Scanner

The vintrace app can be used on modern [iOS](https://itunes.apple.com/au/app/vintrace/id1132479219?mt=8) or [Android](https://play.google.com/store/apps/details?id=com.vinx2.vintrace) mobile devices with the inbuilt camera to provide barcode scanner functionality. We recommend using flagship brands for best hardware support.

For more industrial and robust solution with long range scanning capability we recommend the following Zebra scanners:

- [MC3300ER](https://www.zebra.com/ap/en/products/mobile-computers/handheld/mc3300.html)
- [MC9300ER](https://www.zebra.com/ap/en/products/mobile-computers/handheld/mc9300.html)

## Legacy Barcode Scanner

The Motorolla MC9200 series running Windows Mobile 6.5 are no longer supported devices and not recommended for use with vintrace.

Please switch to using the recommended Android or iOS devices, or [contact us](https://support.vintrace.com/hc/requests/new) for more information or any questions.
