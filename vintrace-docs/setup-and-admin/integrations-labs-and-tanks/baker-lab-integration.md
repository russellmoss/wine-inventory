# Baker Lab Integration

Vintrace has a direct link to Baker Lab that enables their customers to get their analysis data without having to download or upload files. If you’re using Baker Lab, you’ll need to complete the following steps:

1. [Add Baker Lab to your address book.](#h_01ERN9ZXERFX3WYP1GAQK323JX)
2. [Set up your Baker Lab client ID and web service.](#h_01ERNA036QVM09AGYZ2MSRSBQD)
3. [Map Baker Lab's metric names to vintrace.](#h_01ERNA08F1NKXB2GTBGH299KVS)

## Adding Baker Lab to Your Address Book

To add Baker Lab to your [Address Book](https://support.vintrace.com/hc/en-us/articles/32301367488788):

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329285726740) More Options in the sidebar.
2. From the Address Book tile, click Open Address Book.
3. From the Add menu in the lower left, select Organization.

![Address](https://support.vintrace.com/hc/article_attachments/32329285741332)

4. Specify the details for the lab. Be sure to select the Laboratory role.

![Update](https://support.vintrace.com/hc/article_attachments/32329255980564)

5. Click Save.

## Setting Up Your Baker Lab Client ID and Web Service

To set up your Baker Lab client ID and web service:

1. View Baker Lab in the vintrace address book.
2. Click Edit beside the Laboratory role.

![Update](https://support.vintrace.com/hc/article_attachments/32329262269844)

3. Click Configure Client IDs.

![Configure_Laboratory_Settings_-_Configure_Client_IDs_-_Bakers_Lab_20201203.png](https://support.vintrace.com/hc/article_attachments/32329247786004)

4. Enter your Baker Lab client ID in the Client ID column.

![Configure](https://support.vintrace.com/hc/article_attachments/32329247712660)

5. Click OK.
6. Click Configure 3rd Party Lab Interface.

![Configure_Laboratory_Settings_-_Configure_3rd_Party_Lab_-_Bakers_Lab_20201203.png](https://support.vintrace.com/hc/article_attachments/32329247608084)

7. In the Web Service Connection Parameters window, specify the following:

- Web Service Type — Select *Baker Wine & Grape Analysis*.
- Web Service URL — The URL will be automatically entered when you select the service type.
- Web Service Username — Enter the login credentials provided by Baker Lab.
- Web Service Password — Enter the login credentials provided by Baker Lab.

![Web](https://support.vintrace.com/hc/article_attachments/32329281340052)

8. Click OK.
9. Click OK to close the Configure Laboratory Settings window.
10. Click Save.

## Mapping Baker Lab Metric Names to vintrace

If you plan to import results from Baker Lab to vintrace, you'll need to map the Baker Lab metric names to the metric names used by vintrace. Refer to our [Mapping a Lab's Metric Names article](https://support.vintrace.com/hc/en-us/articles/32301340432788) to learn more.
