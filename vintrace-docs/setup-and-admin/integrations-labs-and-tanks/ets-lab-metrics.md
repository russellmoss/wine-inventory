# ETS Lab Metrics

If you're using ETS for your lab analyses, you'll need to [map the lab's metric names to the metric names used by vintrace](https://support.vintrace.com/hc/en-us/articles/32301340432788).

The most commonly requested ETS analyses are listed below. You can copy and paste these metric names when you're setting up your mapping. These metrics can be used when you have a [direct connection to ETS](https://support.vintrace.com/hc/en-us/articles/32301324029076), or if you're [importing a CSV file](https://support.vintrace.com/hc/en-us/articles/32301343026964).

The list below is subject to change by ETS without prior notice. Refer to [ETS' website](https://www.etslabs.com/analyses) for the most current list.
