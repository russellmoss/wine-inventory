# Admeo Bio Systems Y15 Integration

If you plan to import lab results from Admeo/Bio Systems Y15 into vintrace, you’ll need to do the following:

1. [Add Admeo to your address book.](#Address_book)
2. [Map Admeo’s metric names.](#Mapping)

Once you’ve completed these steps, you’ll be able to [request analysis from Admeo/Bio Systems](#Requesting_Analysis), and [import the lab results](#Importing_Results) into vintrace.

## Adding Admeo to Your Address Book

To add Admeo/Bio Systems to your [Address Book](https://support.vintrace.com/hc/en-us/articles/32301367488788):

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329246921364) More Options in the sidebar.
2. From the Address Book tile, click Open Address Book.
3. From the Add menu in the lower left, select Organization.

![Address_Book_-_Add_Menu_-_Organization_20200805.png](https://support.vintrace.com/hc/article_attachments/32329246955924)

4. Specify the details for the lab. Be sure to select the Laboratory role.

![Create_Basic_Organization_Widget_-_Admeo_-_Laboratory_Role_20201204.png](https://support.vintrace.com/hc/article_attachments/32329246983188)

5. Click Save.

## Mapping Admeo’s Metric Names

You’ll need to map the metric names used by Admeo/Bio Systems Y15 to the metric names used by vintrace.

Refer to our [Mapping a Lab's Metric Names article](https://support.vintrace.com/hc/en-us/articles/32301340432788) to learn more.

## Requesting Analysis

You can request a lab analysis from Admeo/Bio Systems by following the steps detailed in our [Requesting Lab Analysis and Viewing Results article](https://support.vintrace.com/hc/en-us/articles/32301356827540). The Sample ID that you enter into the Admeo/Bio System Y15 is the Lab ID that’s generated by the lab request.

![Lab_Request_Label_-_Lab_ID_20200727.png](https://support.vintrace.com/hc/article_attachments/32329247000084)

## Importing Lab Results

You can import the lab results from Admeo/Bio System Y15 into vintrace.

[![Import 9](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2017/07/Import-9.jpg)](https://s3-us-west-2.amazonaws.com/vintrace-support-site-content/uploads/2017/07/Import-9.jpg)

Refer to our [Importing Lab Results](https://support.vintrace.com/hc/en-us/articles/32301343026964-Exporting-and-Importing-Lab-Results#Importing_Lab_Work) article for details on importing the results.
