# Customizing a Printed Work Order Template

Before you can customize a printed work order template, you’ll need to [enable custom print templates](https://support.vintrace.com/hc/en-us/articles/32303341457684).

To customize a printed work order template, you’ll need to do the following:

1. [Download the templates](https://support.vintrace.com/hc/en-us/articles/32301356341396) you want to customize
2. Using MS Word, [customize the job-specific print template(s)](https://support.vintrace.com/hc/en-us/articles/32303308638740).
3. [Set up a custom work order print template](#Setting_Up_Custom_Work_Order_Print_Template).

After you’ve [set up a custom work order print template](#Setting_Up_Custom_Work_Order_Print_Template), you’ll want to decide when you want to use it for printed work orders. You can [use the customized template by default](#Changing_Default), or [select it on a case-by-case basis depending on the work order](#Changing_Work_Order_Template).

## Setting Up a Custom Work Order Print Template

You can set up a custom work order print template from the Winery Setup window (Setup Options > Templates > Custom Print Templates):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328785732628) Set Up in the sidebar.
2. Click Work Orders.
3. From the Custom Print Templates tile, click Configure.
4. Click New Print Template. The Print Template window displays.

![Create_Print_Template_20201118.png](https://support.vintrace.com/hc/article_attachments/32328754223764)

5. Enter a name for the print template.
6. For each job-specific template that you want to include in the work order print template, select the section the template will be used for, then upload the Microsoft Word file. To add a line for additional job types and their template, click Add Line.

![Update_Print_Template_-_Uploading_Templates_20201118.png](https://support.vintrace.com/hc/article_attachments/32328785575060)

7. Click Save.

## Changing the Default Work Order Print Template

If you want all work orders to use your customized job template by default, you can change the system’s default templates from the Winery Setup window (Setup Options > Templates > Default Print Templates):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328785732628) Set Up in the sidebar.
2. Click General.
3. From the Defaults tile, click Configure. The System tab displays by default.
4. From the Default Custom Template list, select the work order print template that you want to use by default. The setting is located in the Custom Templates section on the left of the System tab.

![Winery_Setup_-_Default_Custom_Template_20201118.png](https://support.vintrace.com/hc/article_attachments/32328754260884)

5. Click Apply.

## Changing a Work Order’s Print Template

If you prefer to use your custom work order print template on a case-by-case basis, you can select it from the work order.

1. From the Work Order window, click the ![Wrench_20201118.png](https://support.vintrace.com/hc/article_attachments/32328754141332) print options icon.

![Work_Order_-_Print_Options_Icon_20201118.png](https://support.vintrace.com/hc/article_attachments/32328754062356)

2. Click the ![Magnifying_Glass_20200320.png](https://support.vintrace.com/hc/article_attachments/32328759188372)search icon in the Using This Template field.

![Print_Options_-_Search_Icon_20201118.png](https://support.vintrace.com/hc/article_attachments/32328754167700)

3. Select the work order print template that you’d like to use for the work order.

## Troubleshooting

If a tag is not displaying on your printed work order, be sure that the tag is:

- Spelled correctly.
- Located in the correct position.
- Supported for the template you are using.
