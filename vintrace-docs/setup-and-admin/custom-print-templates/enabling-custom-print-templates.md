# Enabling Custom Print Templates

Only local vintrace administrators, or senior members of the winemaking/cellar team should enable custom print templates.

Enabling custom print templates allows you to view and [download the job-specific templates](https://support.vintrace.com/hc/en-us/articles/32301356341396) so that you can [customize them to suit your needs](https://support.vintrace.com/hc/en-us/articles/32303308638740). You can then [create a custom work order print template](https://support.vintrace.com/hc/en-us/articles/32303295878932) that you can use as the default, or select for specific work orders.

You can enable custom print templates from the Winery Setup window (Setup Options > Workflow > Defaults):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329142023828)Set Up in the sidebar.
2. Click General.
3. From the Defaults tile, click Configure. The System tab displays by default.
4. From the Printed Work Order Format list, select *MS Word (v6+)*. The setting is located in the Production section on the left of the System tab.

![Winery_Setup_-_Workflow_-_Defaults_-_Printed_Work_Order_Format_20201117.png](https://support.vintrace.com/hc/article_attachments/32329139810708)

The *Legacy MS Word* and *PDF (original)* options are available, but do not allow any customization.

5. Click Apply.

Now that you’ve enabled custom print templates, you should be able to [download the default print templates](https://support.vintrace.com/hc/en-us/articles/32301356341396), [customize the template](https://support.vintrace.com/hc/en-us/articles/32303308638740), and add them to a [custom work order print template](https://support.vintrace.com/hc/en-us/articles/32303295878932).
