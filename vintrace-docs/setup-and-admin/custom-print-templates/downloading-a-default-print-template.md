# Downloading a Default Print Template

After you’ve [enabled custom print templates](https://support.vintrace.com/hc/en-us/articles/32303341457684), you can download the DOCX file for the default templates from vintrace. Although you could create a print template from scratch, we recommend that you download the default print template from vintrace and [customize it as needed](https://support.vintrace.com/hc/en-us/articles/32303308638740). This approach is particularly helpful if you only want to make minor adjustments to the default templates.

You can download a default template from the Winery Setup window (Setup Options > Templates > Default Print Templates):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328814278804) Set Up in the sidebar.
2. Click Work Orders.
3. From the Default Print Templates tile, click Configure. A list of sections and their corresponding files display.

![Winery_Setup_-_Default_Print_Templates_20201112.png](https://support.vintrace.com/hc/article_attachments/32328857071252)

4. Click the name of the default template’s DOCX file to download it.

For details on customizing the template, refer to our [Customizing Job-Specific Print Templates article](https://support.vintrace.com/hc/en-us/articles/32303308638740).
