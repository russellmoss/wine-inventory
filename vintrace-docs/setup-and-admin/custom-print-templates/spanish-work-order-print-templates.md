# Spanish Work Order Print Templates

vintrace includes a default work order print template with Spanish labels.

![Spanish_Work_Order_20210803.png](https://support.vintrace.com/hc/article_attachments/32329223541908)

To use the Spanish work order print template:

1. From the [Work Order window](https://support.vintrace.com/hc/en-us/articles/32303315610388), click the ![Wrench_20201118.png](https://support.vintrace.com/hc/article_attachments/32329223488276)wrench icon.

![Work_Order_-_Wrench_Icon_20210803.png](https://support.vintrace.com/hc/article_attachments/32329242417940)

2. Select Spanish Templates from the work order's print options.

![Print_Options_-_Spanish_Templates_20210723.png](https://support.vintrace.com/hc/article_attachments/32329213267092)

If you plan to use the Spanish work order template, we recommend that you [create the standard notes](https://support.vintrace.com/hc/en-us/articles/32301315435028) that display in the General Instructions field (i.e., Instrucciones Generales) in both English and Spanish.
![Standard_Note_in_Spanish_20210803.png](https://support.vintrace.com/hc/article_attachments/32329229528468)
