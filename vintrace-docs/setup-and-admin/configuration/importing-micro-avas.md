# Importing Micro AVAs

Instead of [manually adding each micro AVA](https://support.vintrace.com/hc/en-us/articles/5188347025423-Setting-Up-Appellations-AVAs-Sub-AVAs-and-Micro-AVAs#AddingaMicroAVA), you can use vintrace’s [data import functionality](https://support.vintrace.com/hc/en-us/articles/32303307646868) to add the micro AVAs using a spreadsheet. To do this, you’ll need to do the following:

1. [Download the micro AVAs headers spreadsheet from vintrace.](#downloading)
2. [Enter the micro AVAs in the spreadsheet.](#entering_data)
3. [Import the micro AVAs into vintrace.](#uploading)

## Downloading the Micro AVAs Header File

The header file is a spreadsheet (i.e., CSV file) that contains the column headings for the data that you’ll need to enter in order to create the micro AVAs in vintrace.

To download the micro AVAs header file:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328858318740) Set Up in the sidebar.
2. Search for *Micro AVA* or *AVA*.
3. From the Micro AVAs tile, click Configure.
4. Click Import/Export.

![Winery_Setup_-_Micro_AVA_-_Import_Export_Button_20220901.png](https://support.vintrace.com/hc/article_attachments/32328837601044)

5. Click Download Headers.

![Micro_AVA_Importer_-_Download_Headers_20220901.png](https://support.vintrace.com/hc/article_attachments/32328881985684)

6. Save the file to your computer.

You can now [enter the micro AVAs in the spreadsheet](#entering_data).

## Entering Micro AVAs in the Spreadsheet

To enter micro AVAs in the spreadsheet:

1. Open the micro AVAs header file that you downloaded.
2. Enter the data for the micro AVAs. Be sure to enter the names of the micro AVAs in the Name column.

![CSV_Micro_AVAs_-_US.png](https://support.vintrace.com/hc/article_attachments/32328837621652)

You’ll also want to enter the micro AVA’s sub AVA and AVA in the appropriate columns.

3. Save the file.

You can now [import the micro AVAs into vintrace](#uploading).

## Importing the Micro AVAs

To upload the spreadsheet with your micro AVA into vintrace:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328858318740) Set Up in the sidebar.
2. Search for *Micro AVA* or *AVA*.
3. From the Micro AVAs tile, click Configure.
4. Click Import/Export.
5. Click the upload icon.

![Micro_AVA_Importer_-_Upload_a_File_20220901.png](https://support.vintrace.com/hc/article_attachments/32328837649172)

The Upload window displays.

![Upload_20220901.png](https://support.vintrace.com/hc/article_attachments/32328881894804)

6. Click Choose File.
7. Select your spreadsheet.
8. Click Upload. The selected file’s name displays in the Micro AVA Importer window.

![Micro_AVA_Importer_-_File_Selected_20220901.png](https://support.vintrace.com/hc/article_attachments/32328840704532)

9. Click Import Micro AVA.
10. If the Confirm Column Header Matching checkbox was selected, you’ll be asked to confirm the mapping of your file’s columns to vintrace’s fields.

![Micro_AVA_Importer_-_Column_Matching_US_20220901.png](https://support.vintrace.com/hc/article_attachments/32328837667860)

11. Click Continue. vintrace confirms the number of micro AVA that were imported.
12. Click Close.

The micro AVAs that you imported should now be listed in the Winery Setup window and accessible in vintrace.
