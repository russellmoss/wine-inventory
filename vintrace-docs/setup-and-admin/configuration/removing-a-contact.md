# Removing a Contact

De-activating a contact prevents them from being included in any of vintrace’s lists. For example, if you de-activate an organization that's a grower, they'll no longer be included in any Grower lists in vintrace.

If the contact is a system user who has [permissions](https://support.vintrace.com/hc/en-us/articles/32303349421588) to log into vintrace, de-activating their user account prevents them logging into vintrace and frees up a user license.

To de-activate a contact:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329127684628) More Options in the sidebar.
2. From the Address Book tile, click Open Address Book.
3. Select the contact that you want to de-activate.
4. Select the Inactive checkbox.

![Address_Book_-_Deactivating_Contact_20201106.png](https://support.vintrace.com/hc/article_attachments/32329141988756)

5. Click Save.

If needed, you can [re-activate a contact](#Reactivating_Contact) at a later time.

## Re-activating a Contact

If needed, you can re-activate a contact at a later time. To re-activate a contact:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329127684628) More Options in the sidebar.
2. From the Address Book tile, click Open Address Book. Inactive contacts are not displayed in the Address Book by default.
3. To display inactive organizations and users, de-select the Show Active Contacts Only checkbox.
4. Select the contact you want to re-activate.
5. De-select the Inactive checkbox.
6. Click Save.
