# Bulk Dispatch Reasons

Available starting with vintrace 9.4.2.

This functionality is not enabled by default. If you’re interested in using this functionality, contact support to have it enabled for your account.

When the bulk dispatch reasons functionality is enabled, you’ll be able to set up dispatch reasons and specify the reason in inter-winery bulk dispatch operations. The reason is included in the Bill of Lading and in the Stock Dispatch Report’s CSV file. You’ll also be able to search dispatches using the dispatch reason.

## Setting Up Bulk Dispatch Reasons

To set up a bulk dispatch reason:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328797276820) Set Up in the sidebar.
2. Click Winemaking, or search for *Bulk Dispatch Reason*.
3. From the Bulk Dispatch Reason tile, click Configure.
4. Click New Bulk Dispatch Reason. The Bulk Dispatch Reason window displays.

![Bulk_Dispatch_Reason_Create_20230202.png](https://support.vintrace.com/hc/article_attachments/32328818998292)

5. Enter a name for the reason. You may also enter an optional description.
6. Click Save.

After you’ve added a bulk dispatch reason, you can select the reason when you record a [Bulk Dispatch (Inter-Winery) operation](https://support.vintrace.com/hc/en-us/articles/32301313513620).

![Bulk_Dispatch_Inter_Winery__Dispatch_Reason_20230202.png](https://support.vintrace.com/hc/article_attachments/32328827221780)

The Bill of Lading includes the dispatch reason.

![Bill_of_Lading_20230202.png](https://support.vintrace.com/hc/article_attachments/32328819020180)

You’ll also be able to filter the [dispatch search](https://support.vintrace.com/hc/en-us/articles/32301313789460) by the reason.

![Dispatch_Search_20230202.png](https://support.vintrace.com/hc/article_attachments/32328827147540)

The dispatch reason will be included in the [Stock Dispatch Report’s](https://support.vintrace.com/hc/en-us/articles/32301330369684) CSV file.

![Stock_Dispatch_Report_CSV_20230202.png](https://support.vintrace.com/hc/article_attachments/32328803138708)
