# Setting Up AVAs, Sub AVAs, and Micro AVAs

vintrace enables you to define the following geographic areas:

- [AVAs](#ava)
- [Sub AVAs](#sub_ava)
- [Micro AVAs](#micro_ava)

![Diagram_-_AVA_SubAVA_MicroAVA_-_US_20220901.png](https://support.vintrace.com/hc/article_attachments/32329207357332)

## Adding an AVA

To add an AVA:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329207224340) Set Up in the sidebar.
2. Search for *AVA*.
3. From the AVAs tile, click Configure.

![Set_Up_-_AVAs_Tile_20220722.png](https://support.vintrace.com/hc/article_attachments/32329227758740)

4. Click *New AVA*. The AVA window displays.

![](https://support.vintrace.com/hc/article_attachments/48190115777428)

5. Enter the name of the AVA.
6. Click Save.

## Adding a Sub AVA

To add an appellation or sub AVA:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329207224340) Set Up in the sidebar.
2. Search for *Sub AVA* or *AVA*.
3. From the Sub AVAs tile, click Configure.

![Set_Up_-_Sub_AVAs_Tile_20220722.png](https://support.vintrace.com/hc/article_attachments/32329198935316)

4. Click New Sub AVA. The Sub AVA window displays.

![Sub_AVA_Create_20220722.png](https://support.vintrace.com/hc/article_attachments/32329207285140)

5. Specify the details for the sub AVA. If you need to add a new AVA, click the ![Plus_in_Green_Circle_Small_20220722.png](https://support.vintrace.com/hc/article_attachments/32329227864084) add icon beside the AVA field.
6. Click Save.

## Adding a Micro AVA

If you'd prefer to import the micro AVAs using a spreadsheet, refer to our [Importing Micro AVAs article.](https://support.vintrace.com/hc/en-us/articles/32301282791828)

To add a micro AVA:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329207224340) Set Up in the sidebar.
2. Search for *Micro AVA* or *AVA*.
3. From the Micro AVAs tile, click Configure.

![Set_Up_-_Micro_AVAs_Tile_20220722.png](https://support.vintrace.com/hc/article_attachments/32329207299348)

4. Click New Micro AVA. The Micro AVA window displays.

![Micro_AVA_Create_20220722.png](https://support.vintrace.com/hc/article_attachments/32329211992084)

5. Specify the details for the micro AVA. If you need to add a new sub AVA, click the ![Plus_in_Green_Circle_Small_20220722.png](https://support.vintrace.com/hc/article_attachments/32329227864084) add icon beside the Sub AVA field.
6. Click Save.

- When you select a micro AVA in vintrace, its associated sub AVA will be automatically populated. In the example below, selecting the Calistoga micro AVA automatically populates the Sub AVA field with Napa Valley.

![Sub_AVA_Auto_Populated_20220901.gif](https://support.vintrace.com/hc/article_attachments/32329227925012)

- If you clear a sub AVA field, the micro AVA field will be automatically cleared. In the example below, clearing Napa Valley from the Sub AVA field also clears Calistoga from the Micro AVA field.

![Clearing_Sub_AVA_20220901.gif](https://support.vintrace.com/hc/article_attachments/32329207380372)

- When you select a sub AVA in vintrace, the available micro AVAs will be automatically filtered to display only those that are in the sub AVA.
