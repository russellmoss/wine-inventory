# Exporting a Tank's Dip Chart

To export a [tank’s dip chart](https://support.vintrace.com/hc/en-us/articles/32301385548308) to a CSV file:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328828826132) Reports in the sidebar.
2. Select Equipment.
3. Scroll down to the Dip Table (CSV) report.

![Winery_Reports_-_Dip_Table_CSV_20200923.png](https://support.vintrace.com/hc/article_attachments/32328828798740)

4. Specify the filters for the report.
5. Click Generate or Email.

If you have access to [edit the tank](https://support.vintrace.com/hc/en-us/articles/32301359425428), you can also export the dip chart from the Tank window in the Winery Setup.
