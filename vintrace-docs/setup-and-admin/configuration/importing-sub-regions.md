# Importing Sub-Regions

Instead of [manually adding each sub-region](https://support.vintrace.com/hc/en-us/articles/5415006429455-Setting-Up-Geographic-Indicators-Regions-and-Sub-Regions#AddingaSub-Region) to vintrace, you can use vintrace’s [data import functionality](https://support.vintrace.com/hc/en-us/articles/32303307646868) to add the sub-regions using a spreadsheet. To do this, you’ll need to do the following:

1. [Download the sub-regions headers spreadsheet from vintrace.](#downloading)
2. [Enter the sub-regions in the spreadsheet.](#entering_data)
3. [Import the sub-regions into vintrace.](#uploading)

## Downloading the Sub-Regions Header File

The header file is a spreadsheet (i.e., CSV file) that contains the column headings for the data that you’ll need to enter in order to create the sub-regions in vintrace.

To download the sub-regions header file:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328853093524) Set Up in the sidebar.
2. Search for *Region*.
3. From the Sub Regions tile, click Configure.
4. Click Import/Export.

![Winery_Setup_-_Sub_Region_-_Import_Export_Button_20220901.png](https://support.vintrace.com/hc/article_attachments/32328837526932)

5. Click Download Headers.

![Sub_Region_Importer_-_Download_Headers_20220901.png](https://support.vintrace.com/hc/article_attachments/32328865188244)

6. Save the file to your computer.

You can now enter the sub-regions in the spreadsheet.

## Entering Sub-Regions in the Spreadsheet

To enter sub-regions in the spreadsheet:

1. Open the sub-regions header file that you downloaded.
2. Enter the data for the sub-regions. Be sure to enter the name of the sub-regions in the Name column.

![CSV_Sub_Regions_20220901.png](https://support.vintrace.com/hc/article_attachments/32328865286804)

You’ll also want to enter the sub-region’s region and geographical indicator in the appropriate columns.

3. Save the file.

You can now upload the spreadsheet to vintrace.

## Importing the Sub-Regions

To upload the spreadsheet with your sub-regions into vintrace:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328853093524) Set Up in the sidebar.
2. Search for *Regions*.
3. From the Sub Regions tile, click Configure.
4. Click Import/Export.
5. Click the ![Upload_File_20220901.png](https://support.vintrace.com/hc/article_attachments/32328881779092) upload icon.

![Sub_Region_Importer_-_Upload_a_File_20220901.png](https://support.vintrace.com/hc/article_attachments/32328881767956)

The Upload window displays.

![Upload_20220901.png](https://support.vintrace.com/hc/article_attachments/32328858227476)

6. Click Choose File.
7. Select your spreadsheet.
8. Click Upload. The selected file’s name displays in the Sub Region Importer window.

![Sub_Region_Importer_-_File_Selected_20220901.png](https://support.vintrace.com/hc/article_attachments/32328858183700)

9. Click Import Sub Region.
10. If the Confirm Column Header Matching checkbox was selected, you’ll be asked to confirm the mapping of your file’s columns to vintrace’s fields.

![Sub_Region_Importer_-_Column_Matching_20220901.png](https://support.vintrace.com/hc/article_attachments/32328840634388)

11. Click Continue. vintrace confirms the number of sub-regions that were imported.
12. Click Close.

The sub-regions that you imported should now be listed in the Winery Setup window and accessible in vintrace.
