# Setting Up Default Fruit Booking Duration

Available starting with [vintrace 9.4.2](https://support.vintrace.com/hc/en-us/articles/32303276924308).

A default fruit booking duration can be specified for each winery. The specified duration is used by default for [new scheduled bookings](https://support.vintrace.com/hc/en-us/articles/32303268370324).

![Default_Booking_Duration_20230106.png](https://support.vintrace.com/hc/article_attachments/32329221585940)

To set up a default fruit booking duration:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329206487316) Set Up in the sidebar.
2. Search for *Winery*, or select Locations.
3. From the Winery tile, click Configure.
4. Select the winery.
5. In the Default Booking Duration field, enter the hours and minutes to be used by default for new scheduled bookings.

![Winery_Create_-_Default_Booking_Duration_20230106.png](https://support.vintrace.com/hc/article_attachments/32329221569044)

6. Click Save.
