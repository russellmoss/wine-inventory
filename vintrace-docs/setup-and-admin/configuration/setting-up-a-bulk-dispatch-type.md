# Setting Up a Bulk Dispatch Type

To set up a custom dispatch type:

1. Click ![Setup](https://support.vintrace.com/hc/article_attachments/32328817056276) Set Up in the sidebar.
2. Click Winemaking, or search for *Bulk Dispatch Type*.
3. From the Bulk Dispatch Types tile, click Configure.
4. Click New Custom Dispatch Type. The Custom Dispatch Type window displays.

![Custom](https://support.vintrace.com/hc/article_attachments/32328825539092)

5. Specify the details for the dispatch type.
6. Click Save.
