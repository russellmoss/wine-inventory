# Setting Up a Standard Note

You can add your standard operating procedures (SOPs) to vintrace by setting up a standard note. By adding a standard note for an SOP, you can quickly include the SOP in your work orders. Instead of typing out each SOP, you can just select the standard note and vintrace will fill in the text that’s associated with it.

For example, you could [create a standard not](#setting_up)e for your inoculation procedures.

![Standard_Note_-_Inoculation_20210730.png](https://support.vintrace.com/hc/article_attachments/32328893970324)

When you want to [include the SOP in a work order](#using_in_work_order), you select the standard note and vintrace includes your SOP in the work order.

![Work_Order_with_Standard_Note_20200902.png](https://support.vintrace.com/hc/article_attachments/32328907377940)

## Setting Up a Standard Note

To set up a standard note:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328877162516) Set Up in the sidebar.
2. Click Work Orders.
3. From the Standard Notes tile, click Configure.
4. Click New Standard Note. The Standard Note window displays.

![Standard_Note_Create_20210730.png](https://support.vintrace.com/hc/article_attachments/32328899142420)

5. Specify the details for the standard note. The Name/Code field is the identifier for the standard note. In the field below the Name/Code field, enter your standard operating procedure.
6. Click Save.

Users with a multi-winery license can specify which standard notes are available at each winery. Refer to our [Configuration for Multi-Winery Support article](https://support.vintrace.com/hc/en-us/articles/32301304791316) for details.

## Including a Standard Note in a Work Order

To use a standard note in a work order:

1. [Create a work order](https://support.vintrace.com/hc/en-us/articles/32303315610388).
2. Click the ![Add_Standard_Note_Icon_20210730.png](https://support.vintrace.com/hc/article_attachments/32328907419540) icon.

![Create_Work_Order_-_Add_Standard_Note_Icon_20210730.png](https://support.vintrace.com/hc/article_attachments/32328899219092)

3. Use the field that displays to search for the standard note you want to include.

![Create_Work_Order_-_Search_Standard_Note_20210730.png](https://support.vintrace.com/hc/article_attachments/32328887888660)

You can also add a standard note by clicking the ![Plus_Sign_20200320.png](https://support.vintrace.com/hc/article_attachments/32328907438484) add icon.

4. Select the standard note. The standard operating procedures display in the Work Order window.
