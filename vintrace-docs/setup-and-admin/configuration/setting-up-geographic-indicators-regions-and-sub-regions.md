# Setting Up Geographic Indicators, Regions, and Sub-Regions

vintrace enables you to define the following geographic areas:

- [Geographic indicators](#geo_ind)
- [Regions](#region)
- [Sub-Regions](#sub_region)

![Diagram_-_Geo_Indicator_Region_Sub_Region_-_Australia_20220901.png](https://support.vintrace.com/hc/article_attachments/32329198700180)

## Adding a Geographic Indicator

To add a geographic indicator:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329227550228) Set Up in the sidebar.
2. Search for *Geo*.
3. From the Geographical Indicator tile, click Configure.

![Set_Up_-_Geographical_Indicator_Tile_20220830.png](https://support.vintrace.com/hc/article_attachments/32329207041044)

4. Click New Geographic Indicator. The Geographic Indicator window displays.

![Geographic_Indicator_20220830.png](https://support.vintrace.com/hc/article_attachments/32329207056404)

5. Enter the geographic indicator's name.
6. Click Save.

## Adding a Region

To add a region:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329227550228) Set Up in the sidebar.
2. Search for *Region*.
3. From the Region tile, click Configure.

![Set_Up_-_Regions_Tile_20220830.png](https://support.vintrace.com/hc/article_attachments/32329239933076)

4. Click New Region. The Region window displays.

![Region_Create_20220830.png](https://support.vintrace.com/hc/article_attachments/32329227628564)

5. Specify the details for the region. If you need to add a new geographical indicator, click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329227665684) add icon beside the G.I. field.
6. Click Save.

## Adding a Sub-Region

If you'd prefer to import the sub-regions using a spreadsheet, refer to our [Importing Sub-Regions article](https://support.vintrace.com/hc/en-us/articles/32301313854612).

To add a sub-region:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329227550228) Set Up in the sidebar.
2. Search for *Region*.
3. From the Sub Region tile, click Configure.

![Set_Up_-_Sub_Regions_Tile_20220830.png](https://support.vintrace.com/hc/article_attachments/32329227659028)

4. Click New Sub Region. The Sub Region window displays.

![Sub_Region_Create_20220830.png](https://support.vintrace.com/hc/article_attachments/32329198665108)

5. Specify the details for the sub-region. If you need to add a new region, click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329227665684) add icon beside the Region field.
6. Click Save.

- When you select a sub-region in vintrace, its associated region will be automatically populated.
- If you clear a region field, the sub-region field will be automatically cleared.
- When you select a region in vintrace, the available sub-regions will be automatically filtered to display only those that are in the region.
