# Managing Owner/Client Logins (SSO enabled)

An owner (client) login is a contact who’s associated with an owner organization. Owner logins can view or run reports on their wines. Owner logins can only view data in vintrace.

Only users with the [Local vintrace Administrator permission](https://support.vintrace.com/hc/en-us/articles/360000813755-Roles-and-Permissions#LocalvintraceAdministrators) can manage owner logins.

## Viewing the Owner Logins Page

To view the Owner Logins page:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328861091348) More Options in the sidebar.
2. From the Address Book tile, click Manage Owner Logins.

![More_Options_-_Manage_Owner_Logins_20220227.png](https://support.vintrace.com/hc/article_attachments/32328856358676)

The Owner Logins page displays.

![Owner_Logins_20220227.png](https://support.vintrace.com/hc/article_attachments/32328856235924)

## Setting Up an Owner Login

Before you add an owner login, be sure that you’ve added their organization to your address book.

To add an owner login:

1. From the Owner Logins page, click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32328884230420) add icon.

![Owner_Logins_-_Add_Icon_20220227.png](https://support.vintrace.com/hc/article_attachments/32328861168916)

The Invite Client with Login form displays.

![Invite_Client_with_Login_20220411.png](https://support.vintrace.com/hc/article_attachments/32328844395284)

2. Specify the owner’s information.
3. Click Invite Client.

## Resending an Invitation

To resend an invitation to a pending owner login:

1. From the Owner Logins page, click the ![Three_Vertical_Dots_-_Carafe_20220227.png](https://support.vintrace.com/hc/article_attachments/32328844255892) beside the owner login.
2. Select Resend.

## Editing an Owner Login

To edit a owner login’s information:

1. From the Owner Logins page, click the ![Three_Vertical_Dots_-_Carafe_20220227.png](https://support.vintrace.com/hc/article_attachments/32328844255892) beside the owner login.
2. Select Edit.

![Owner_Logins_-_Edit_20220301.png](https://support.vintrace.com/hc/article_attachments/32328873680404)

3. Edit the owner login as needed.
4. Click Save.

## Disabling an Owner Login

Disabling an owner login prevents them from logging into vintrace. To disable a an owner login:

1. From the Owner Logins page, click the ![Three_Vertical_Dots_-_Carafe_20220227.png](https://support.vintrace.com/hc/article_attachments/32328844255892) beside the owner login.
2. Select Disable.

![Owner_Login_-_Disable_20220301.png](https://support.vintrace.com/hc/article_attachments/32328861214996)

## Re-enabling an Owner Login

To re-enable a disabled user’s account:

1. From the Owner Logins page, click the ![Three_Vertical_Dots_-_Carafe_20220227.png](https://support.vintrace.com/hc/article_attachments/32328844255892) beside the owner login.
2. Select Re-enable.
