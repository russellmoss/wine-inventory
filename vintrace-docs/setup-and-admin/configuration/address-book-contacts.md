# Address Book Contacts

## Types of Contacts

There are four types of contacts that you can add to your Address Book:

- Individual — A person that can be [assigned as a lab tech, operator, or winemaker](https://support.vintrace.com/hc/en-us/articles/32303349421588). Unlike system users, individuals cannot log into vintrace.
- Organization — A company that can be [assigned as a carrier, cooper, customer, distributor, grower, harvester, laboratory, owner, or vendor](https://support.vintrace.com/hc/en-us/articles/32303349421588).
- [Owner Login](https://support.vintrace.com/hc/en-us/articles/32303334987796) — A person associated with an owner organization. Owner logins can view or run reports on their wines. Owner logins can only view data in vintrace.
- [System User](https://support.vintrace.com/hc/en-us/articles/32303348674196) — A person who can be [assigned as a lab tech, operator, or winemaker](https://support.vintrace.com/hc/en-us/articles/32303349421588). Unlike an individual, a system user can log into vintrace to [perform tasks that are based on the permissions that they’re granted](https://support.vintrace.com/hc/en-us/articles/32303349421588).

For information on setting up Alternating Proprietorships (AP users), refer to our [Setting Up AP02 Licenses article](https://support.vintrace.com/hc/en-us/articles/32303336930708).

## Setting Up a Contact

To add a contact to the address book:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32328933304724) More Options in the sidebar.
2. From the Address Book tile, click Open Address Book.
3. From the Add menu in the lower left, select the type of contact you want to add.

![Address_Book_-_Add_Menu_20200727.png](https://support.vintrace.com/hc/article_attachments/32328927197076)

4. Specify the details for the contact.
5. Click Save.

## Filtering the Address Book

You can search for contacts in the address book by using the filters at the top of the page.

![Address_Book_-_Filters_20221010.png](https://support.vintrace.com/hc/article_attachments/32328927228564)

These filters include the following:

- Category - Filter the contacts to show individuals, organizations, or both. Selecting a category will control whether the Type (Individual) or Type (Organization) filters are available.
- Type (Individual) - Filter the individuals listed by their role. To remove the filter, select *[Reset]*.
- Type (Organisation) - Filter the organizations listed by their role. To remove the filter, select *[Reset]*.
- Auto Code - Filter the contacts by their auto code. Be sure to click Search after entering an auto code.

To remove all search criteria and display all contacts, click Reset.
