# Setting Up a Winery

To set up a winery:

1. Click ![Setup](https://support.vintrace.com/hc/article_attachments/32328780495508) Set Up in the sidebar.
2. Click Locations, or search for *Winery*.
3. From the Winery tile, click Configure.
4. Click New Winery. The Winery window displays.

![Winery Create 20230904.png](https://support.vintrace.com/hc/article_attachments/32328780474388)

5. Specify the details for the winery.
6. Click Save.
