# Managing System Users

Users with the [Local vintrace Administrator permission](https://support.vintrace.com/hc/en-us/articles/32303349421588-Roles-and-Permissions) can [add](#Adding), edit, and [remove](#Removing) vintrace users.

System users are the individuals who will use vintrace to perform tasks and operations for your winery. All contacts - including system users - are managed in the vintrace [Address Book](https://support.vintrace.com/hc/en-us/articles/32301367488788).

## Viewing System Users

To view your system users:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329123482516) More Options in the sidebar.
2. From the Address Book tile, click Manage System Users.

If you’re already viewing the Address Book, you can also click Manage System Users.

3. To view a specific user’s details, select the user.

The System Administrator account is only used by the vintrace support team.

## Adding a System User

To add a system user:

1. [Add the user to your address book](https://support.vintrace.com/hc/en-us/articles/32301367488788-Address-Book-Contacts) as a system user.

![Address_Book_-_Add_Menu_-_System_User_20201104.png](https://support.vintrace.com/hc/article_attachments/32329124974612)

2. Specify the details for the system user.

- Be sure to enter a username and password that the user will use to initially log into vintrace. The user will be able to [change their password](https://support.vintrace.com/hc/en-us/articles/32303262431892) after they’ve logged in.
- At a minimum, we recommend that you assign the user with the [Operator role](#h_01EPQFRFZCJK9NYAN2WMBRDHPB) so that work can be assigned to them.
- Select the appropriate [permissions](https://support.vintrace.com/hc/en-us/articles/32303349421588-Roles-and-Permissions) for the user.

![Address Book - Adding System User 20240401.png](https://support.vintrace.com/hc/article_attachments/32329098208660)

If you [use vintrace across multiple wineries](https://support.vintrace.com/hc/en-us/articles/32301304791316-Configuration-for-Multi-Winery-Support), you’ll also need to [configure the user for multiple facilities](#Multiple_Facilities).

3. Click Save.

## Available Roles

The following [roles](https://support.vintrace.com/hc/en-us/articles/32303349421588-Roles-and-Permissions/#Roles) can be assigned to system users.

| Role | Description |
| --- | --- |
| Lab technician | Includes the system user in dropdowns and searches for lab work. |
| Operator | Includes the system user in dropdowns and searches for operational work. |
| Wine Maker | Includes the system user in dropdowns and searches for winemaker operations. |

## Configuring a User for Multiple Facilities

If you [use vintrace across multiple facilities](https://support.vintrace.com/hc/en-us/articles/32301304791316-Configuration-for-Multi-Winery-Support), you’ll also need to specify the following settings:

- All Winery Access (user permission) — If you want to allow the user to switch between different facilities, be sure to select the All Winery Access user permission.
- Default Winery — A user’s default winery is the facility that displays when they log into vintrace. Select the user’s default winery from the list.

![Multi Winery User 20240401.png](https://support.vintrace.com/hc/article_attachments/32329125056404)

## Removing System Users

For information on removing a system user, refer to our [Removing a Contact](https://support.vintrace.com/hc/en-us/articles/32303349216916) article.
