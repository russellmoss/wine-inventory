# Setting Up PMS Rates

Available starting with [vintrace 9.4.2](https://support.vintrace.com/hc/en-us/articles/32303276924308).

The ability to set up and specify PMS rates for fruit bookings is not enabled by default. If you would like to use this functionality, please contact our support team.

To add a PMS rate:

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32329206500372) Set Up in the sidebar.
2. Search for *PMS Rate*, or select Fruit Sources.
3. From the PMS Rate tile, click Configure.
4. Click New PMS Rate. The PMS Rate window displays.

![PMS_Rate_20230105.png](https://support.vintrace.com/hc/article_attachments/32329227042708)

5. Specify the PMS rate.
6. Click Save.

You can also add a PMS rate while [recording a fruit booking](https://support.vintrace.com/hc/en-us/articles/32303268370324) by clicking the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329239342996) add icon beside the PMS Rate field.

![Scale_Booking_-_PMS_Rate_Add_Icon_20230106.png](https://support.vintrace.com/hc/article_attachments/32329211350804)

After setting up PMS rates, you can specify the rate for a fruit booking.
