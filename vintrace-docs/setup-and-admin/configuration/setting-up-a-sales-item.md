# Setting Up a Sales Item

To set up a sales item:

1. Click ![Setup Icon 20200318.png](https://support.vintrace.com/hc/article_attachments/32328750522516) Set Up in the sidebar.
2. Click Sales, or search for *Sales Item*.
3. From the Sales Item tile, click Configure.
4. Click New Item. The Sales Item window displays.

![Create Sales Item 20240820.png](https://support.vintrace.com/hc/article_attachments/32328750484756)

5. Specify the details for the sales item.
6. Click Save.
