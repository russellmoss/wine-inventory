# Setting Up a Tank Map

The vintrace tank map provides a bird’s eye view for displaying and planning the layout of your [tanks](https://support.vintrace.com/hc/en-us/articles/32301359425428).

![Tank_Farm_20201030.png](https://support.vintrace.com/hc/article_attachments/32329154259348)
When you [set up a tank map](#h_01EP7H0SCJ4A1ZYEG3RF05WC43), vintrace displays a grid with the number of columns and rows that you specify.

![Create_Tank_Map_-_Map_20201029.png](https://support.vintrace.com/hc/article_attachments/32329131434004)

You’ll also be able to specify what information you want displayed for the tanks that you add to the map.

![Tank_Details_Customizable_20201030.png](https://support.vintrace.com/hc/article_attachments/32329163026068)

You can set up multiple tank maps. You could even include the same tank in different tank maps so that you can display different information throughout the year or season. For example, you could create a tank map for fermentation that includes the tank’s brix, temperature, and pH.

For details on viewing and using a tank map, refer to our [Viewing a Tank Map](https://support.vintrace.com/hc/en-us/articles/32301346088340) article.

## Setting Up a Tank Map

To add a tank map:

1. Click ![Tank_Map_20201029.png](https://support.vintrace.com/hc/article_attachments/32329144960404) Tank Map from the sidebar.
2. Click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329154310804) add icon.
3. Select New Tank Map.

![Add_-_New_Tank_Map_20201103.png](https://support.vintrace.com/hc/article_attachments/32329177071380)

The Tank Map window displays.

![Create_Tank_Map_20201029.png](https://support.vintrace.com/hc/article_attachments/32329145008788)

4. Specify the following details for the tank map.

- What name do you want to give to this view? — The name of the tank map.
- How many tanks do you want to be able to see horizontally? — The number of columns to display in the tank map.
- How many tanks do you want to be able to see vertically? — The number of rows to display in the tank map.
- What starting zoom level do you want when viewing this tank map? — The default zoom level used to display the tank map.

5. Select up to 5 details and 5 metrics that you want displayed for the tanks in this map.

![Create_Tank_Map_-_Selected_Details_and_Metrics_20201030.png](https://support.vintrace.com/hc/article_attachments/32329168080788)

Only measured metrics are available for display. Calculated and indeterminate metrics are not available.

6. Click Next. A blank tank map displays a grid with the number of columns and rows that you specified.
7. Search for the tanks that you want to include in the tank map by applying filters to find the vessels, and/or entering the vessel’s name.

![Create_Tank_Map_-_Finding_Vessels_20201029.png](https://support.vintrace.com/hc/article_attachments/32329131598484)

- To find vessels by filtering their attributes (e.g., name, winery, location), click From Search. This displays the Vessel Search window. To include vessels in the tank map, select their rows, then click Use. To include all vessels from a search, click All.

![Vessel_Search_20201029.png](https://support.vintrace.com/hc/article_attachments/32329163154708)

- If you know the name of your vessel, enter its name in the Add Equipment field, then select it from the list.

![Create_Tank_Map_-_Enter_Name_20201029.png](https://support.vintrace.com/hc/article_attachments/32329168154900)

The vessels that you selected display in the tank card deck that’s on the left side of the Tank Map window. A card displays for each vessel that you selected.

![Create_Tank_Map_-_Tank_Card_Deck_20201029.png](https://support.vintrace.com/hc/article_attachments/32329145128468)

To remove a vessel from the tank card deck, click the X in the card’s upper right.

8. Drag the vessel’s card from the tank card deck to the grid.

![Dragging_Tanks_to_Map_20201103.gif](https://support.vintrace.com/hc/article_attachments/32329154538260)

9. Click Save.
