# Setting Up a Loss Reason

To set up a loss reason:

1. Click ![Setup](https://support.vintrace.com/hc/article_attachments/32328817148180) Set Up in the sidebar.
2. Click Winemaking, or search for *Reason for Loss*.
3. From the Reason for Loss tile, click Configure.
4. Click New Loss Reason. The Loss Reason window displays.

![Loss](https://support.vintrace.com/hc/article_attachments/32328817116436)

5. Specify the details for the loss reason.
6. Click Save.
