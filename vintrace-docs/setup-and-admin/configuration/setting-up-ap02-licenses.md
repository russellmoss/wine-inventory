# Setting Up AP02 Licenses

Under US law, clients who make wine at your facility under bond can be set up as an Alternating Proprietorship (AP02). Because your client will be operating as a winery within a winery, the AP02 bond will always take precedence over your bond.

To add a client's AP02 bond number to vintrace:

1. [Add the client to your address book](https://support.vintrace.com/hc/en-us/articles/32301367488788) as an organization.

![Address_Book_-_Add_Menu_-_Organization_20200805.png](https://support.vintrace.com/hc/article_attachments/32329169473940)

2. When setting up the client, you may want to prefix the name of Alternating Proprietorships with *AP* so that you can distinguish it from other owners.
3. Select the Owner role, then click Edit beside it.

![Address_Book_-_Adding_Alternating_Proprietorship_20201103.png](https://support.vintrace.com/hc/article_attachments/32329146388628)

The Configure Owner Settings window displays.

4. Select the AP Owner checkbox.
5. Click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329183572756) add icon displayed beside the Bond field.

![Configure_Owner_Settings_-_AP_Owner_20201103.png](https://support.vintrace.com/hc/article_attachments/32329164663060)
The TTB Bond Info window displays.

![Create_TTB_Bond_Info_20201103.png](https://support.vintrace.com/hc/article_attachments/32329164680596)

6. Enter the details for the bond, including its registered name and bond number, then click Save.
7. Click OK to close the Configure Owner Settings window.
8. Click Save to save the client’s information.

After you’ve saved the client’s bond information, you can start using the AP bond.

If your client has bulk wine that needs to be transferred to a new AP02, refer to our [Transferring Wines Between Bonds (US) article](https://support.vintrace.com/hc/en-us/articles/32303307244308) for details.
