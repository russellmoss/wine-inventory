# Specifying a Winery-Specific Prefix

You can specify a winery-specific prefix that will be used for new work orders, fruit dockets, and bills of lading. For example, if a winery’s prefix is J, the work order numbers, docket numbers, and bills of lading will include that prefix.

![Prefix_20220316.png](https://support.vintrace.com/hc/article_attachments/32329198998676)

vintrace will use numbers without a prefix if any of the following are true:

- A prefix is not specified for a winery
- A user is not switched to a specific winery and/or vintrace cannot determine which winery a work order, docket, or bill of lading is for.

To specify a winery’s prefix:

1. Click ![Setup_Icon_20200318.png](https://vinx2-old.zendesk.com/hc/article_attachments/4413466786319/Setup_Icon_20200318.png) Set Up in the sidebar.
2. Search for Winery.
3. From the Winery tile, click Configure.
4. Select the winery.
5. In the Prefix field, enter the prefix that you want to use for the winery’s new work orders, fruit dockets, and bills of lading.

![Winery_Prefix_15032022.jpg](https://vinx2-old.zendesk.com/hc/article_attachments/4508697302159/Winery_Prefix_15032022.jpg)

6. Click Save.
