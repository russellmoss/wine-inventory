# vintrace Vineyard Management (eVineyard) Integration: Specifying Vineyard Block IDs

If you plan to integrate vintrace with the standalone vintrace Vineyard Management product (formerly 'eVineyard'), you'll need to ensure that the Vineyard Block ID in Vineyard Management is specified for the relevant block in vintrace.

To link the Vineyard Block ID within vintrace:

1. Click ![Blocks 20200803.png](https://support.vintrace.com/hc/article_attachments/32329238563604) Blocks in the sidebar.
2. Select the block you want to update.
3. Select the Advanced Details tab.
4. Click the ![Pencil Gray Icon 20200414.png](https://support.vintrace.com/hc/article_attachments/32329205862932) pencil icon beside Admin.

![Block Overview - Advanced Details Tab 20231030.png](https://support.vintrace.com/hc/article_attachments/32329226312212)

The Block window displays the selected block's information.

5. Be sure that the Block Details tab is selected.
6. In the Auto-Code field, enter the block's Vineyard Block ID from Vineyard Management/eVineyard.

![Block Update - Block Details 20231030.png](https://support.vintrace.com/hc/article_attachments/45805938599828)

6. Click Save.
