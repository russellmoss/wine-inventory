# Managing API tokens

API tokens linked to a user account can now be generated in vintrace for use with the [vintrace API](https://support.vintrace.com/hc/en-us/articles/32301322539668).

The [Local vintrace Admin permission](https://support.vintrace.com/hc/en-us/articles/32303349421588) is required to create and delete API Tokens.

## Creating an API Token

To create a new API Token:

1. Click ![Sidebar - Setup 20241118.png](https://support.vintrace.com/hc/article_attachments/34316321630996) Set Up in the sidebar.
2. Click API.
3. From the API tokens tile, click Configure.

![API_Set_Up_Page_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329207703060)

4. Click the ![Plus in Green Circle 20200319.png](https://support.vintrace.com/hc/article_attachments/34316329482772) Plus icon.

API Tokens previously generated by support may appear in the list without a Name. Please contact support before making any changes to these existing tokens.

![API_Tokens_Page_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329228327700)

5. Enter the Token Name.

![Create_New_API_Token_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329240642196)

6. Select a User from the list.

![Select_User_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329207594260)

7. Click Save.

![Save_API_Token_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329207619220)

8. Click Copy to clipboard to copy the API Token.

The token will only be displayed once after it is created and will need to be copied at this step.

![Success_API_Token_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329222578836)

## Deleting Existing API Tokens

To delete an existing API Token:

1. Click ![Sidebar - Setup 20241118.png](https://support.vintrace.com/hc/article_attachments/34316321630996) Set Up in the sidebar.
2. Click API.
3. From the API tokens tile, click Configure.
4. Click ![Delete_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329240521236)for the API Token that will be deleted.
5. Click Delete.

![Delete_API_Token_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329212350356)

![Delete_API_Token_Confirmation_2021-10-06.jpg](https://support.vintrace.com/hc/article_attachments/32329199313044)
