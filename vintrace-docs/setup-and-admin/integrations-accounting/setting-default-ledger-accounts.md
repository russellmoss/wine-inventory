# Setting Default Ledger Accounts

vintrace includes a number of default ledger accounts. You may wish to update these to link with specific accounts of your choosing within vintrace. These vintrace accounts can then link to specific accounts in Xero if you have it connected. To review and update these accounts, edit the defaults from your Winery Set Up (Set up > Accounts > configure Accounts tile > Defaults)

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/41051001294100) Set up in the sidebar.
2. Click Accounts
3. Click Configure on the Accounts tile.
4. Click Defaults
5. Assign (or create and assign) the accounts as necessary.

![default_ledgers.png](https://support.vintrace.com/hc/article_attachments/41051380179476)

6. Click OK

NOTE: Point the default account to the appropriate vintrace account. The vintrace account will provide the link to its connected Xero account if you have a Xero integration, as per its [account set up](https://support.vintrace.com/hc/en-us/articles/32301385387668-Setting-Up-an-Account).
