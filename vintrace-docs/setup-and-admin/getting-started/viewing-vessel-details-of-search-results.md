# Viewing Vessel Details of Search Results

After you apply [filters](https://support.vintrace.com/hc/en-us/articles/32301344204308) and sorting to the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924), you can use the controls in the upper right of the breadcrumb bar to [scroll through the Vessel Details of the search results](#Scrolling_Search_Results), or [return to the Vessels page](#Returning_to_Vessels_Page).

## Scrolling Through Search Results

When you’re scrolling through the search results, you’ll be prompted to return to the Vessels page if any vessels have been emptied.

To scroll through the [Vessel page’s search results](https://support.vintrace.com/hc/en-us/articles/32301344204308):

1. Sort the search results in the order that you’d like to view them.
2. View a batch.
3. Click the ![Previous_Chevron.jpg](https://support.vintrace.com/hc/article_attachments/32329276144276) and ![Next_Chevron.jpg](https://support.vintrace.com/hc/article_attachments/32329242964756) arrow icons in the upper right of the breadcrumb bar to view the previous and next result.

![Navigation.jpg](https://support.vintrace.com/hc/article_attachments/32329251449364)

## Returning to the Vessels Page

When [scrolling through the Vessel page’s search results](#Scrolling_Search_Results), you can return to the Vessels page by clicking the link between the ![Previous_Chevron.jpg](https://support.vintrace.com/hc/article_attachments/32329276144276) and ![Next_Chevron.jpg](https://support.vintrace.com/hc/article_attachments/32329242964756) arrow icons in the upper right.

The Vessels page displays with the [filters](https://support.vintrace.com/hc/en-us/articles/32301344204308) and sorting that you last applied. The list of search results on the Vessels page will reflect any new vessels that were filled, emptied, created, or changed.

![Navigation_Return.jpg](https://support.vintrace.com/hc/article_attachments/32329224209684)
