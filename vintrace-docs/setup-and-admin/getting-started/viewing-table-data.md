# Viewing Table Data

NOTE: The following is applicable to the tables in the [Vessels](https://support.vintrace.com/hc/en-us/articles/32301344827924), [Batch Explorer](https://support.vintrace.com/hc/en-us/articles/32301345941268), [Vineyards & Blocks](https://support.vintrace.com/hc/en-us/articles/32301313410196-Vineyards-Blocks-Page), and [Product Allocation](https://support.vintrace.com/hc/en-us/articles/32301319185940-Product-Allocations-Page) pages.

When the number of columns in a table exceeds your screen size, a horizontal scroll bar displays at the bottom of the table. The horizontal scrollbar will only display when needed such as when you include additional columns in a table, or when a table contains a lengthy value such as a description, vessel, or batch.

If a table’s column header or a value exceeds the column’s width, ellipses (i.e., …) display at the end of the value to indicate that the entire value is not displayed. You can hover over the displayed value to view the entire value.

![Horizontal_Scrollbar_and_Long_Value_20210401.png](https://support.vintrace.com/hc/article_attachments/32328923808916)

The tables on the Vessels, Batch Explorer, Vineyards & Blocks, and Product Allocation pages will automatically scroll back to the first column when you add a new column or change the order of the columns. Changing how the table’s data is sorted on the Vessels and Vineyards & Blocks pages will also automatically scroll to the first column of the table.
