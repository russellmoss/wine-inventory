# Column Filters

## Applying a Column Filter

You can filter what’s displayed in a table by applying a filter to any of the columns displayed.

To apply a column filter:

1. Click the ![Filter_Gray_20210621.png](https://support.vintrace.com/hc/article_attachments/32328878632212)filter icon displayed beside the column that you would like to filter.

![Filter_columns.png](https://support.vintrace.com/hc/article_attachments/41120721706260)

2. Select the value (or values) that you want to include in the table by checking the respective checkbox.
3. Click the ![Checkmark_20200901.png](https://support.vintrace.com/hc/article_attachments/32328878645396) checkmark icon to apply the filter(s).

![Apply_filters.png](https://support.vintrace.com/hc/article_attachments/41120721706900)

The table displays the rows that meet your criteria. The column header's text where you applied the filter displays in a different colour.

![Filtered_view.png](https://support.vintrace.com/hc/article_attachments/41120693639828)

NOTE: You can have multiple column filters active at once.

## Removing a Column Filter

To remove a column filter.

1. Click the ![Filter_Green_20210621.png](https://support.vintrace.com/hc/article_attachments/32328878739860)filter icon displayed beside the column.
2. Click the x icon beside the values that you want to remove from the filter.

![Column_Filter_-_Removing_20210621.png](https://support.vintrace.com/hc/article_attachments/32328900878484)

3. Click the ![Checkmark_20200901.png](https://support.vintrace.com/hc/article_attachments/32328878645396) checkmark icon.

Alternatively, navigating to another page within vintrace will reset the filters.
