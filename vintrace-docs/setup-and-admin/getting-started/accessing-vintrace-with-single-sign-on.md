# Accessing vintrace with Single Sign-On

Please contact support for more details on the availability for Single sign-on (SSO).

Single sign-on (SSO) is technology that lets you use your login credentials to access multiple applications, including vintrace. For example, instead of using separate usernames and passwords to access your email and vintrace, you can log in once to access both.

In addition to the usual login and password fields, the vintrace login page on the web application and mobile app also has options to log in with Apple, Google, Microsoft and Okta.

## Existing vintrace System Users and Owner Logins

If you have an existing vintrace system user or owner login, your username is your email address. You will need to reset your password the first time you log in with Single Sign-On.

## New vintrace System Users and Owner Logins

If you’re a new vintrace user or owner login, you should have received an email invitation to access your vintrace account. This email invitation is only valid for 24 hours and may only be used once.

1. From the email invitation, click Accept Invitation.

![Email_-_Accept_Invitation_20220301.png](https://support.vintrace.com/hc/article_attachments/32328603278228)

The vintrace login page displays options for logging in with Google, Apple, Microsoft, Okta, or your username and password.

![Login_Page_Carafe.jpg](https://support.vintrace.com/hc/article_attachments/32328635410068)

2. Log in using the option you’d like to use. This is typically where the invitation was sent and accepted from. For example, Google if the email was sent to Gmail, Microsoft if the email was sent to Outlook.

The option you select will continue to be used to log you into vintrace.

IMPORTANT: Be sure to select an appropriate SSO option once you receive your SSO login invitation. Attempting to log in using the standard username/password combination can cause errors in your vintrace database once SSO has been enabled for your user account.
