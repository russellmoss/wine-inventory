# Navigating vintrace

vintrace provides a number of ways for you to navigate its pages.

## The Sidebar

The sidebar displays along the left side and is used to access vintrace’s pages and features.

![Sidebar_Location_Annotated_20200917.png](https://support.vintrace.com/hc/article_attachments/32328809700884)

The currently selected sidebar option displays in green. You can hover over each icon to view a tooltip that describes the page. You can also click the arrow beside the ![vintrace_Icon_20200917.png](https://support.vintrace.com/hc/article_attachments/32328832225428) vintrace icon to expand or minimize the sidebar.

![Sidebar_Annotated_20200917.png](https://support.vintrace.com/hc/article_attachments/32328801784212)

At the bottom of the sidebar are icons for opening vintrace Classic, opening another vintrace tab, and accessing online help or requesting support.

![Sidebar_Bottom_Icons_Annotated_20200917.png](https://support.vintrace.com/hc/article_attachments/32328796088340)

## Breadcrumbs

Breadcrumbs provide a visual indicator as to where you are within vintrace and provide shortcuts for getting back to a previous page. Breadcrumbs are available on a number of pages including:

- Blocks page
- Calendar page
- Jobs page
- Products page
- Setup page
- [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924)

In the example below, we can tell from the breadcrumb that we’re looking at the product page of a bulk white wine. You could click “Vessels” in the breadcrumb to display the Vessels page, or click ![Home_20200917.png](https://support.vintrace.com/hc/article_attachments/32328826135700) Home to display the Dashboard page.

![Breadcrumb_Annotated_20200917.png](https://support.vintrace.com/hc/article_attachments/32328832123156)

## Quick Search

The Quick Search feature lets you quickly find wines, vessels, stock items, and blocks in vintrace. Simply enter a name in the Quick Search field and vintrace displays a list of wines, vessels, stock items, and blocks that match.

![Quick_Search_Annotated_20200917.png](https://support.vintrace.com/hc/article_attachments/32328801860884)
