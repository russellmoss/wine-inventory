# Setting Your Default Start Page

By default, vintrace displays the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924-The-Vessels-Page) when you log in.

You can update your default start page to be one of the following pages:

- Dashboard
- Vessels
- Batch Explorer
- Calendar
- Jobs
- Vineyards & Blocks
- Products
- More Options

TIP: This default applies to your individual login only, and won't affect other users.

![Vessels_Page_Unfilled_Star_Icon.jpg](https://support.vintrace.com/hc/article_attachments/32329224235796)

## Setting Your Default Start Page

1. From the sidebar, open the page that you want to set as your default start page.
2. Click the ![Star_Icon_Unfilled.jpg](https://support.vintrace.com/hc/article_attachments/32329230550036) star icon in the breadcrumb bar. Your start page is now configured and will display by default the next time you log into vintrace, or open a new vintrace tab.

## Removing Your Default Start Page

1. From the sidebar, open your default start page if it's not already displayed.
2. Click the ![Star_Icon_Filled.jpg](https://support.vintrace.com/hc/article_attachments/32329214021268) star icon in the breadcrumb bar. Your default start page is now reset to the [Vessels page](https://support.vintrace.com/hc/en-us/articles/32301344827924).
