# System Requirements

## Internet

Because vintrace is a web-based application, an internet connection is required.

## Web Browser

vintrace is supported on the following browsers. We recommend using the most recent version of your web browser to improve security, performance, and stability. The browser that you use must have both cookies and JavaScript enabled.

| Browser | Supported Version |
| --- | --- |
| Chrome | Latest and the last prior version |
| Firefox | Latest and the last prior version |
| Safari | Latest and the last prior version |

Although vintrace my work on other browsers or older versions, it does not officially support them.

## Files and Reports

vintrace produces reports and other files using the following file formats:

- CSV
- Adobe PDF
- Microsoft Word
- Microsoft Excel

We recommend the following applications and versions for viewing these files:

- Adobe Reader 2023
- Microsoft Word version 16 or higher
- Microsoft Excel version 16 or higher

Although there are many other applications that can used to read files produced by vintrace, the applications listed above are the ones officially supported.

## Mobile Apps

We regularly release updates to our [mobile apps](https://support.vintrace.com/hc/en-us/categories/32300676287124-Mobile-App). Please keep the app up to date to ensure it has the latest features and improvements.

Our mobile apps are compatible with the following mobile operating systems:

- iOS version 15 or higher
- Android version 7 or higher

## Barcode Scanner

The [vintrace mobile app](https://support.vintrace.com/hc/en-us/categories/32300676287124-Mobile-App) can be run on dedicated barcode scanners. Refer to our [Recommended Barcode Scanner article](https://support.vintrace.com/hc/en-us/articles/32303293958036) for a list of supported devices.

## Printers

Refer to our [Recommended Printers and Labels](https://support.vintrace.com/hc/en-us/articles/32303330395284) article for details.

## Additional Recommendations

- When using vintrace, a second screen can be useful.
- Optimize your screen resolution when running the application.
- Over time we review our minimum supported versions for our web browsers and mobile operating systems. These may change as these platforms evolve and mature. If there are any significant changes to supported versions, notifications will be sent ahead of time.
