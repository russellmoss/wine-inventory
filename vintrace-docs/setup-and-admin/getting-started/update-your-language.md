# Update your language

To change your language:

1. Click the ![Profile_20200402.png](https://support.vintrace.com/hc/article_attachments/32328634300308) Profile icon, then select the current language next to the ![Language_Icon 20231201.jpg](https://support.vintrace.com/hc/article_attachments/32328627689748)Language icon.

![Change Language 20231201.jpg](https://support.vintrace.com/hc/article_attachments/32328602339092)

2. Select the required language.

![Change Language_2 20231201.jpg](https://support.vintrace.com/hc/article_attachments/32328627720468)

3. Click Update.

TIP: This only changes the language setting for your individual login. Other system users will be unaffected.
