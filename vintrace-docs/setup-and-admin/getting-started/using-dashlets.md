# Using Dashlets

Dashlets and [insights](https://support.vintrace.com/hc/en-us/articles/32303307942292) display data that you’re interested in tracking on your vintrace Dashboard.

What you add to your dashboard is unique to your login - so arrange it to best suit your needs.

![Insights_and_Dashlets_20201029.png](https://support.vintrace.com/hc/article_attachments/32329095820820)

A dashlet displays data in either a list or graph. For example, there’s a dashlet that displays a list of your saved searches, and a dashlet that lists your bulk wine batches. There are other dashlets that display your wines by variety as a bar graph, or your available barrels by oak type in a pie chart.

You can add any dashlet you need, then control how it displays on your Dashboard. For example, you can [change its title](#Renaming), [resize the dashlet](#Resizing), or [change its position](#Moving).

## Types of Dashlets

There are two types of dashlets in vintrace: standard and custom.

Standard dashlets are pre-defined queries included with vintrace. You can access these by following [the instructions below](#Adding) to pick from the standard list.

Below is a list of the available standard dashlets.

|  |  |
| --- | --- |
| - My Searches - Additives Expiring in < 30 days - Empty Barrel Capacity by Forest - Empty Barrel Capacity by Oak Type - Empty Barrel Capacity by Year - Bulk Wine Batches - Bulk Wine by Customer/Vintage - Bulk Wine by Location - Bulk Wine by Winery/Vintage - Bulk Wine by Owner - Bulk Wine by Owner - Bar Chart - Empty Tanks - Recently Created Batches - Ullaged Vessels - Variety Volume on Hand - Vessel Capacity In Use - Volume by Container Type - Volume by Vintage/Program - Wine Product States - Wine by Product State - Invoiced Service Revenue by Customer - Service Orders - Cost by Program/Vintage | - Cost by Program/Vintage $ - Cost by Vintage/Program - Cost by Vintage/Program $ - Total Costing Breakdown - Expected vs. Received Fruit Bar Chart - Expected vs. Received Fruit - Default Vintage - Ferment Dates - Received Fruit by Vintage/Variety - Received Fruit by Date Stack Chart - Dispatched Wine - Line Graph - Latest Bulk Intakes - Open Work Orders - Latest Analysis - Outstanding Lab Work - Batch/Notes - New Purchase Orders - Purchase Orders - Invoiced Product Revenue by Customer - New Sales Orders - Sales Orders - Sales Not Dispatched - Expected vs Received - Bar Chart - Lot Explorer |

Custom dashlets are special request items that vintrace may be able to create for you in certain circumstances. In order to complete this custom work, you’ll need to provide vintrace with specific information, including:

- The data that should be used
- How the data should be filtered and sorted
- How you’d like the custom dashlet to look

To request a custom dashlet, contact [support@vintrace.com](mailto:support@vintrace.com). We’ll evaluate the request to determine if the dashlet can be developed as requested. Because this is custom work, vintrace cannot make any guarantees regarding turnaround time and may incur a fee for development.

## Adding a Dashlet to Your Dashboard

To add a dashlet to your Dashboard:

1. Click ![Dashboard_20200402.png](https://support.vintrace.com/hc/article_attachments/32329095835028) Dashboard in the sidebar.
2. Click the ![Plus_in_Green_Circle_20200319.png](https://support.vintrace.com/hc/article_attachments/32329121334804) plus icon.
3. Select New dashlet.

![Add_-_New_Dashlet_20201028.png](https://support.vintrace.com/hc/article_attachments/32329121398420)

4. Select the dashlet that you want to add to your dashboard from the window that appears.

If the name of a dashlet is too long to be displayed in its entirety, you can hover over the title to see the full name.

The icon next to the dashlet indicates the type of dashlet:

|  |  |  |  |
| --- | --- | --- | --- |
| list.icon.png List | bar.icon.png Bar Chart | pie.icon.png Pie Chart | line.icon.png Line Chart |

![Available_Dashlets_-_Hover_Over_Name_20201028.png](https://support.vintrace.com/hc/article_attachments/32329135018132)

Once the dashlet is added to your Dashboard, you can hover over the dashlet's title bar to access options to [rename](#Renaming), [resize](#Resizing), [move](#Moving), [refresh](#Refreshing), and [remove](#Removing) it. Some dashlets also have an option for [downloading its data](#Downloading) to an XLS file.

![Dashlet_Icons_20201029.png](https://support.vintrace.com/hc/article_attachments/32329095955220)

## Changing a Dashlet’s Title

The dashlet’s title is above the data it displays.

![Dashlet_Title_20201029.png](https://support.vintrace.com/hc/article_attachments/32329095915284)

To change a dashlet’s title:

1. Hover over the dashlet. Several icons display in the upper right of the dashlet.
2. Click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329135045140)icon.
3. Select Configure.

![Dashlet_-_More_Settings_-_Configure_20201029.png](https://support.vintrace.com/hc/article_attachments/32329095967636)
The Configure window displays.

![Configure_20201029.png](https://support.vintrace.com/hc/article_attachments/32329110483476)

4. Enter the title that you want to display for the dashlet.
5. Click Save.

## Resizing a Dashlet

There are three sizes available for dashlets: small, medium, and large.

![Dashlet_Sizes_20201029.png](https://support.vintrace.com/hc/article_attachments/32329121474964)

To resize a dashlet:

1. Hover over the dashlet. Several icons display in the upper right of the dashlet.
2. Click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329135045140) icon.
3. Select Configure.

![Dashlet_-_More_Settings_-_Configure_20201029.png](https://support.vintrace.com/hc/article_attachments/32329095967636)
The Configure window displays.

4. Select the dashlet’s display size.

![Configure_-_Size_Options_20201029.png](https://support.vintrace.com/hc/article_attachments/32329087619348)

5. Click Save.

## Moving a Dashlet

You can change a dashlet’s position on your Dashboard.

![Moving_Dashlet_20201029.gif](https://support.vintrace.com/hc/article_attachments/32329110618516)
To move a dashlet:

1. Hover over the dashlet’s title bar.

![Dashlet_Title_Bar_20201029.png](https://support.vintrace.com/hc/article_attachments/32329121641364)

2. When the ![Drag_Cursor_20201029.png](https://support.vintrace.com/hc/article_attachments/32329096041748)cursor displays, click and drag the dashlet to its new position.

## Refreshing a Dashlet

Refreshing a dashlet updates the information that’s displayed in it. To refresh a dashlet, hover over the dashlet’s title bar, then click the ![Refresh_20201029.png](https://support.vintrace.com/hc/article_attachments/32329135149204) refresh icon.

![Dashlet_Refresh_20201029.png](https://support.vintrace.com/hc/article_attachments/32329135211284)

NOTE: You can also set an automatic refresh timer for all elements on your dashboard by clicking the ![Gear_Green_Icon_20200417.png](https://support.vintrace.com/hc/article_attachments/49846082024596)gear icon in the top right of the Dashboard. The default setting is for 30 minutes, but it can be set as low as 5 minutes between auto-refreshes.

## Downloading a Dashlet’s Data

You can download an XLS file of the data displayed in dashlets that contain a list.

To download a dashlet’s data to an XLS file:

1. Hover over the dashlet. Several icons display in the upper right of the dashlet.
2. Click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329135045140) icon.
3. Select Download.

![Dashlet_-_More_Settings_-_Download_20201029.png](https://support.vintrace.com/hc/article_attachments/32329135198356)

4. Save the file.

## Removing a Dashlet

If you no longer need a dashlet, you can remove it from your Dashboard.

Removing a dashlet simply removes it from your current display, but does not delete it. You can [re-add the dashlet](#Adding) if needed.

To remove a dashlet from your Dashboard:

1. Hover over the dashlet. Several icons display in the upper right of the dashlet.
2. Click the ![Three_Vertical_Dots_20200623.png](https://support.vintrace.com/hc/article_attachments/32329135045140) icon.
3. Select Remove.

![Dashlet_-_More_Settings_-_Remove_20201029.png](https://support.vintrace.com/hc/article_attachments/32329106511764)
