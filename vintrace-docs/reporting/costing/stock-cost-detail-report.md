# Stock Cost Detail Report

The Stock Cost Detail Report shows the cost breakdown by category and unit cost. The report also includes a history of all cost events that impacted the wine or stock item.

![Stock_Cost_Detail_Report_PDF_-_19Rose_Harvest_Demo_20210105.png](https://support.vintrace.com/hc/article_attachments/32328924346132)

The report’s output displays in a PDF file.

## Running the Stock Cost Detail Report

To run the Stock Cost Detail Report:

1. View the wine or stock item in the [Stock Item Overview window](https://support.vintrace.com/hc/en-us/articles/32303270808468).
2. From the Reports menu in the lower left, select Lot/cost report.
