# VQA Support in vintrace

## Enabling VQA Support in vintrace

To enable VQA in vintrace, you’ll need to do the following:

- Ensure that you have a [metric](https://support.vintrace.com/hc/en-us/articles/32301345260948) named *VQA Brix*.

![Metric_Update_-_VQA_Brix_20200831.png](https://support.vintrace.com/hc/article_attachments/32328951558036)

- In your defaults, ensure that the Composition Metric default is set to *VQA Brix*.

![Default_Composition_Metric_20200831.png](https://support.vintrace.com/hc/article_attachments/32328927179028)

- Update your Fruit Intake [analysis template](https://support.vintrace.com/hc/en-us/articles/32301372281748) so that it’s the default for fruit receival and includes the VQA Brix metric.

![Analysis_Template_-_Fruit_Intake_20200831.png](https://support.vintrace.com/hc/article_attachments/32328932858516)

## Receiving Fruit

When you [receive fruit](https://support.vintrace.com/hc/en-us/articles/32303268370324) in vintrace, be sure to use the analysis template that includes the VQA brix metric.

![Intake_Details_-_Analysis_Template_with_VQA_Brix_20200831.png](https://support.vintrace.com/hc/article_attachments/32328923391252)

If you don’t have the VQA brix measurement when you receive the fruit, you’ll want to [correct the fruit intake](https://support.vintrace.com/hc/en-us/articles/32303331960980) at a later time to input the VQA brix.

## Finding Your VQA Wines

In vintrace, tagging is a way for you to add keywords to wines so that you can easily view wines. To make it easier for you to view all your VQA wines, you can [add a tag](https://support.vintrace.com/hc/en-us/articles/32303337168148) (e.g., VQA wine, VQA) and assign that tag to your VQA wines.

![20HillsBlend_Tag_20200831.png](https://support.vintrace.com/hc/article_attachments/32328906143508)

Tagging is NOT a validation process. You determine which wines you want to associate with the tag.

You can then create a search filter to find wines with the tag and add a [saved search](https://support.vintrace.com/hc/en-us/articles/32301344204308) that lists the wines that you’ve associated with your VQA tag.

![Search_Filters_-_Tags_VQA_Wines_20200831.png](https://support.vintrace.com/hc/article_attachments/32328926831636)

You can then perform operations on all VQA wines.

![Vessels_-_Tags_VQA_Wines_-_Actions_Menu_20200831.png](https://support.vintrace.com/hc/article_attachments/32328906285972)

## Viewing VQA Brix by Weigh Tag

Although you can customize the [product page](https://support.vintrace.com/hc/en-us/articles/32303310460948) to display the wine’s average VQA brix, you can view the VQA brix for a weigh tag:

1. From the wine’s product page, select the Fruit tab.
2. Click the docket number.

![20HillsBlend_-_Fruit_Tab_20200901.png](https://support.vintrace.com/hc/article_attachments/32328951755540)

3. Select the intake and click View.

![Intake_Delivery_-_Viewing_Intake_20200901.png](https://support.vintrace.com/hc/article_attachments/32328951791252)

4. From the Intake Details window, select the QA/Cost/Analysis tab. The VQA brix and other metrics display.

![Intake_Details_-_VQA_Brix_20200901.png](https://support.vintrace.com/hc/article_attachments/32328914388756)

## Transferring Wines

When you transfer wine, you’ll be able to specify whether it's a VQA or non-VQA transfer. This is especially useful when you’re transferring from one VQA to another VQA facility.

![Racking_-_VQA_Xfe_20200908.png](https://support.vintrace.com/hc/article_attachments/32328933100564)

When you transfer a VQA wine, vintrace generates a [VQA Letter of Audited Origin report](#h_01EHQNZAY4P22M27T56GGC0BP2).

## Grape Delivery Report

The Grape Delivery Report lets you view the VQA brix for weigh tag numbers.

![Grape_Delivery_Report_-_VQA_Brix_20200908.png](https://support.vintrace.com/hc/article_attachments/32328951683092)

This report is located in the Vintage/Harvest report category.

![Winery_Reports_-_Grape_Delivery_Report_20200908.png](https://support.vintrace.com/hc/article_attachments/32328926858772)

You can output the report to a CSV or PDF. When you select the PDF format, the report includes the average VQA brix.

![Grape_Delivery_Report_-_Avg_VQA_Brix_20200908.png](https://support.vintrace.com/hc/article_attachments/32328940309012)

## Label Integrity Statement

The [Label Integrity Statement](https://support.vintrace.com/hc/en-us/articles/32301330173972) report enables you to include the average VQA brix in the report.

![Label_Integrity_Statement_20200831.png](https://support.vintrace.com/hc/article_attachments/32328926961812)

The Label Integrity Statement is located in the Label Integrity report category.

![Winery_Reports_-_Label_Integrity_Statement_20200831.png](https://support.vintrace.com/hc/article_attachments/32328927003156)

In addition to specifying the wine, winemaker, and customer, be sure to select the Average VQA Brix by Vintage/Variety checkbox when you run the report.

![Label_Integrity_Statement_-_Average_VQA_Brix_Checkbox_20200831.png](https://support.vintrace.com/hc/article_attachments/32328927081364)

## VQA Letter of Audited Origin Report

The VQA Letter of Audited Origin report displays the VQA brix for each varietal in a wine.

![VQA_Letter_of_Audited_Origin_20200831.png](https://support.vintrace.com/hc/article_attachments/32328927060372)

The VQA Letter of Audited Origin report is located in the Label Integrity report category.

![Winery_Report_-_VQA_Letter_of_Audited_Origin_20200831.png](https://support.vintrace.com/hc/article_attachments/32328927115284)

When you select the wine and the region for the VQA Letter of Audited Origin report, vintrace automatically includes its varietals.

![Winery_Reports_-_VQA_Letter_of_Audited_Origin_-_Varietals_20200831.png](https://support.vintrace.com/hc/article_attachments/32328952023316)

## VQA Transfer Report

The VQA Transfer Report enables you to track the transfers, losses, and costs of VQA wines.

![VQA_Transfer_Report_20200831.png](https://support.vintrace.com/hc/article_attachments/32328951976596)

The VQA Transfer Report is located in the Label Integrity report category.

![Winery_Reports_-_VQA_Transfer_Report_20200831.png](https://support.vintrace.com/hc/article_attachments/32328952043412)

## VQA Reports and Trial Blends

For trial blends, you can run all the VQA reports as if the blend exists. This helps you ensure that the blend meets the VQA requirements before you create the blend.
