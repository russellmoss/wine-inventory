# Accessing Key Reports

To access key reports in vintrace, click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328991128596) Reports in the sidebar.

![Winery_Reports_-_Loss_20210615.png](https://support.vintrace.com/hc/article_attachments/32328978632724)

Along the left are different report categories that you can select. Each report category contains a number of reports within it.
