# Bulk Inventory by Allocation Report

The Bulk Inventory by Allocation Report is a companion to the [Bulk Stock Report](https://support.vintrace.com/hc/en-us/articles/360000814095-Composition-of-Bulk-Wine#h_01ET5Z7FNBF9ESXG8FT32EMB7K) and [allocations table](https://support.vintrace.com/hc/en-us/articles/32301328118932). The report details allocated gallons and costs by product. You can sort, filter, and analyze the report’s output in the CSV file to determine whether you need to edit batches or products in the allocations table, or examine the batches or products further in vintrace.

In addition to viewing the allocation status of your wines, the Bulk Inventory by Allocation Report is the only place in vintrace where you can determine the cost of a particular product across all batches.

![Bulk_Inventory_by_Allocation_Report_20220425.png](https://support.vintrace.com/hc/article_attachments/32328855652244)

- The Volume (gal) column displays the amount of wine in the vessel.
- The value displayed in the Vessel and the Volume (gal) columns can contain the same value across multiple rows if the vessel is allocated to more than one product.

## Running the Bulk Inventory by Allocation Report

To run the Bulk Inventory by Allocation Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328868146196) Reports in the sidebar.
2. Select the Allocations category.

![Winery_Reports_-_Bulk_Inventory_by_Allocations_20220425.png](https://support.vintrace.com/hc/article_attachments/32328855633300)

3. Specify the report’s filters.
4. Click Email.

Although you can click Generate, this report may take some time to run so we recommend emailing the output.

5. Specify the email addresses where you want to send the report.
6. Click Send.
