# Bulk Stock Receipt Report

The Bulk Stock Receipt Report shows all [bulk intakes](https://support.vintrace.com/hc/en-us/articles/32303303281428) for the specified date range.

![Bulk_Stock_Receipt_Report_CSV_20201216.png](https://support.vintrace.com/hc/article_attachments/32328921963156)

You can save the report’s output to a CSV file, or email it.

## Running the Bulk Stock Receipt Report

To run the Stock Receipt Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328913015316) Reports in the sidebar.
2. Click Inventory.

![Winery_Reports_-_Inventory_-_Bulk_Stock_Receipt_Report_20201216.png](https://support.vintrace.com/hc/article_attachments/32328913054484)

3. Specify the filters for the report:

- From — The starting date received. This defaults to the first day of the current month.
- To — The ending date received. This defaults to the last day of the current month.
- Owner
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to include in the report. If blank, the report includes all wineries in your setup.
- Ext. P.O. Ref — An external P.O. reference.

4. Click Generate or Email.
