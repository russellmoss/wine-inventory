# Packaged/Manufactured Stock Report

The Packaged/Manufactured Stock Report shows all [packaging, bottling,](https://support.vintrace.com/hc/en-us/articles/32303327186836) and [manufacturing](https://support.vintrace.com/hc/en-us/articles/32303341990548) operations during the specified date range.

![Packaged_Manufactured_Stock_Report_PDF_20201217.png](https://support.vintrace.com/hc/article_attachments/32328921230228)

You can save the report’s output to a PDF or CSV file, or email the output.

## Running the Packaged/Manufactured Stock Report

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328903869204) Reports in the sidebar.
2. Click Inventory.

![Winery_Reports_-_Inventory_-_Packaged_Manufactured_Stock_Report_20201217.png](https://support.vintrace.com/hc/article_attachments/32328880861716)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

The following filters and options are available for the Packaged/Manufactured Stock Report:

- From — The first date to include in the report. This defaults to the first day of the current month.
- To — The last date to include in the report. This defaults to the last day of the current month.
- Format — The report’s output format (i.e., CSV or PDF).
- Stock Item — The stock item to include in the report.
- Lot/Batch No. — The lot or batch number to include in the report.
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to run the report for. If blank, the report includes all wineries in your setup.
- Owner — The stock item’s owner to include in the report.
- Work Order — The work order number to include in the report.
- Ext Work Order Ref — The external work order reference to include in the report.
- Bottling/Manufactures/Show All — The operations to include in the report.
- Report Wine In — The unit of measure to use for the report.
