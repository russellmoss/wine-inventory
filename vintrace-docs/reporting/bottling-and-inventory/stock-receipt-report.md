# Stock Receipt Report

The Stock Receipt report shows all [inventory](https://support.vintrace.com/hc/en-us/sections/32300847422868-Bottling-and-Inventory) that you’ve [received](https://support.vintrace.com/hc/en-us/articles/32303350382356) during the specified date range.

![Stock_Receipt_CSV_20201216.png](https://support.vintrace.com/hc/article_attachments/32328551989396)

You can save the report’s output to a CSV file, or email it.

## Running the Stock Receipt Report

To access the Stock Receipt Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328571848212) Reports in the sidebar.
2. Click Inventory.

![Winery_Reports_-_Stock_Receipt_Report_20201216.png](https://support.vintrace.com/hc/article_attachments/32328571862676)

3. Specify the filters for the report:

- From — The starting date received. This defaults to the first day of the current month.
- To — The ending date received. This defaults to the last day of the current month.
- Owner
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to include in the report. If blank, the report includes all wineries in your setup.

4. Click Generate or Email.
