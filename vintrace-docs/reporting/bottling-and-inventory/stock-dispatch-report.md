# Stock Dispatch Report

The Stock Dispatch Report shows the [dispatch](https://support.vintrace.com/hc/en-us/articles/32303319075988) history for your stock during the specified date range. You can save the report’s output to a PDF or CSV file, or email the output.

![Stock_Dispatch_Report_PDF_and_CSV_20221129.png](https://support.vintrace.com/hc/article_attachments/32328921887124)

## Running the Stock Dispatch Report

To run the Stock Dispatch Report:

1. Click Reports in the sidebar.
2. Click Inventory.

![Winery_Reports_-_Inventory_-_Stock_Dispatch_Report_20201216.png](https://support.vintrace.com/hc/article_attachments/32328925539988)

3. Specify the [filters and options](#h_01ESPBS2Z8S61TFK2Y4P1Z5TQ8) for the report.
4. Click Generate or Email.

## Filters and Options

The following filters and options are available for the Stock Dispatch Report:

- From — The first date to include in the report. This defaults to the first day of the current month.
- To — The last date to include in the report. This defaults to the last day of the current month.
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to include in the report. If blank, the report includes all wineries in your setup.
- Sent To — The contact to who the stock was dispatched.
- Show Equivalent — Includes dispatches for the selected volume equivalent.
- Ext Work Order Ref — The external work order reference.
- Owner — The stock item’s owner.
- Format — The report’s output format (i.e., CSV or PDF).
- Dispatch Type — The type of stock dispatched.
