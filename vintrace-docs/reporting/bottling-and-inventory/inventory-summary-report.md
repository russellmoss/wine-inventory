# Inventory Summary Report

The Inventory Summary Report shows a summary of the selected stock category during the specified date range.

![Inventory_Summary_Report_CSV_20201216.png](https://support.vintrace.com/hc/article_attachments/32328925165460)

You can save the report’s output to a CSV file, or email it.

## Running the Inventory Summary Report

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328925134484) Reports in the sidebar.
2. Click Inventory.

![Winery_Reports_-_Inventory_-_Inventory_Summary_Report_20201216.png](https://support.vintrace.com/hc/article_attachments/32328880926484)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

- From — The first date to include in the report. This defaults to the first day of the current month.
- To — The last date to include in the report. This defaults to the last day of the current month.
- Owner — The stock item’s owner.
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to include in the report. If blank, the report includes all wineries in your setup.
- Category — The stock category to include in the report.

You must select at least one category.
