# Wine Addition Impact Report

The Wine Addition Impact Report is an on-screen report that enables you to view the additions that you've made to your current bulk wine, dispatched bulk wine, and packaged wine.

![Wine_Addition_Impact_Report_with_Results_20230316.png](https://support.vintrace.com/hc/article_attachments/32328818008596)

The results of the report display on-screen, but can be exported to a CSV or PDF file.

## Running the Wine Addition Impact Report

To run the Wine Addition Impact Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328832300692) Reports in the sidebar.
2. Select Product History.

![Winery_Reports_-_Product_History_-_Wine_Addition_Impact_Report_20230316.png](https://support.vintrace.com/hc/article_attachments/32328817921812)

3. Specify the filters and options for the report.

You must specify at least one filter for additive, lot/batch, or wine/batch. You must also select at least one checkbox in the the Show Results For section.

![Wine_Addition_Impact_Report_-_Filters_20230316.png](https://support.vintrace.com/hc/article_attachments/32328809795604)

4. Click View. The Wine Addition Impact Report window displays the results. If needed, you can specify additional filters or change the results displayed.

If you selected more than one checkbox in the Show Results For section, you will need to click the arrow beside each to view the results.

![Wine_Addition_Impact_Report_-_Expanding_Results_20230316.png](https://support.vintrace.com/hc/article_attachments/32328802040340)

## Exporting the Wine Addition Impact Report

You can export the results of the Wine Addition Impact Report to a CSV or PDF file by selecting the file type below the results.

![Wine_Addition_Impact_Report_-_Exporting_20230316.png](https://support.vintrace.com/hc/article_attachments/32328796208020)
