# Stock Summary Report

The Stock Summary Report shows the history of a wine and when it was active.

![Stock_Summary_Report_PDF_20201222.png](https://support.vintrace.com/hc/article_attachments/32328765158164)

You can save the report’s output to a PDF file, or email it.

## Running the Stock Summary Report

To run the Stock Summary Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328720782868) Reports in the sidebar.
2. Click Bulk Wine.

![Winery_Reports_-_Bulk_Wine_-_Stock_Summary_Report_20201229.png](https://support.vintrace.com/hc/article_attachments/32328740875668)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

The Stock Summary Report has the following filters and options:

- From — The first date to include in the report.
- To — The last date to include in the report. This defaults to the current date.
- Winery — The bulk wine's winery.
- Owned By — The bulk wine’s owner. You must select an owner.
- Batch
- Order
- Page Size
- Landscape — Selected by default. De-select to change the report’s orientation to portrait.
- Show Reversed/Admin — Select to include operations that were reversed.
- Show Op History — Select to include all operations performed on the wine.

![Stock_Summary_Report_-_Show_Op_History_Enabled_20201229.png](https://support.vintrace.com/hc/article_attachments/32328782280340)

- Show Op Balance — Select to show the balance after each operation.
