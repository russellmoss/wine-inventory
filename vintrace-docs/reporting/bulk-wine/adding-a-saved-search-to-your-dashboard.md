# Adding a Saved Search to Your Dashboard

You can add a [saved search that you created from the Bulk Wine Search window](https://support.vintrace.com/hc/en-us/articles/32303332410516) to your dashboard. The saved search will be listed in the My Searches dashlet.

![My_Searches_Dashlet_20210615.png](https://support.vintrace.com/hc/article_attachments/32328977663380)

To add a saved search that you created in the Bulk Wine Search window to your dashboard, click the ![Pin_20210615.png](https://support.vintrace.com/hc/article_attachments/32328973231508).

![Bulk_Wine_Search_-_Pin_20210615.png](https://support.vintrace.com/hc/article_attachments/32328948450324)
