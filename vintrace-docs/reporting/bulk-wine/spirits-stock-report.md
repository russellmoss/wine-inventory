# Spirits Stock Report

This functionality is available starting with [vintrace 9.4.3](https://support.vintrace.com/hc/en-us/articles/32303276816020), but not enabled by default. If you would like to use this functionality, please contact our support team.

The Spirits Stock Report includes the proof gallons for liquids in a Spirits tax class. This report can be used to reconcile the TTB report for proof gallons.

![Spirits_Stock_Report_CSV_20230306.png](https://support.vintrace.com/hc/article_attachments/32329221461012)

To run the Spirits Stock Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329238989076) Reports in the sidebar.
2. Click Bulk Wine.

![Winery_Reports_-_Bulk_Wine_-_Spirits_Stock_Report_20230306.png](https://support.vintrace.com/hc/article_attachments/32329211158164)

3. Specify the filters for the report:

- Show Stock as of - Includes spirits/liquids that are in stock as of the selected date and time.
- Owned By - Filters on the spirits in the vessel with the selected owner.
- Winery - Filters on all spirits in the selected winery.
- Bond - Filters on all spirits in the selected bond.

4. Click Generate or Email.
