# Blended, Moved or Used Wines

There are several reports that you can run to report on blended, moved, and used wines.

- [Stock Summary Report](https://support.vintrace.com/hc/en-us/articles/32301367122708) - The Stock Summary Report can include a summary of all operations on the wine. To include all operations, be sure to select the Show Op History checkbox when you run the report.

![Stock_Summary_Report_-_Including_Wine_History_20201229.png](https://support.vintrace.com/hc/article_attachments/32329016255508)

- [Operation Throughput Report](https://support.vintrace.com/hc/en-us/articles/32301321300756) - The Operation Throughput Report shows the operations performed on a specific batch. To include the summary for each operation, select the Include Operation Summaries checkbox when you run the report.

You can also filter the wines displayed in the [Batch Explorer](https://support.vintrace.com/hc/en-us/articles/32301345941268) to only include depleted wines.
