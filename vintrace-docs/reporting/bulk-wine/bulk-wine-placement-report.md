# Bulk Wine Placement Report

The Bulk Wine Placement report is a report showing the origin of your bulk wine, similar to the origin of fruit in the [Fruit Placement report.](https://support.vintrace.com/hc/en-us/articles/32301312850196-Fruit-Placement-Report)

This report can be run as a PDF or CSV.

## Running the Bulk Wine Placement Report

To run the Bulk Wine Placement Report:

3. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32727362106644) Reports in the sidebar.
4. Click Bulk Wine

   ![Bulk Wine Placement Report](https://support.vintrace.com/hc/article_attachments/32727362111508)
5. Specify the filters for the report.
6. Click Generate or Email.
