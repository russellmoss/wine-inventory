# Vessel Forecasting Report

This functionality is available starting with version 9.20.1, and is available in our **Winery Enterprise** plan. If you would like to use this functionality, please contact our support team.

The Vessel Forecasting Report details the operations that resulted in a vessel’s forecasted batch and volume.

![Vessel Forecasting Report CSV 20240624.png](https://support.vintrace.com/hc/article_attachments/32329196510612)

You can save the report’s output to a CSV file, or email the output.

## Running the Vessel Forecasting Report

To run the Vessel Forecasting Report:

1. Click ![Reports Menu Option 20200406.png](https://support.vintrace.com/hc/article_attachments/32329191825812) Reports in the sidebar.
2. Select Bulk Wine.

![Winery Reports - Bulk Wine - Vessel Forecasting 20240528.png](https://support.vintrace.com/hc/article_attachments/32329191892244)

3. Specify the filters and options for the report.
4. Click Generate or Email.
