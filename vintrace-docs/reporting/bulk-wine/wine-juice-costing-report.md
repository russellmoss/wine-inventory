# Wine/Juice Costing Report

The Wine/Juice Costing report shows costing events and their impact on a bulk wine’s unit cost.

![Wine_Juice_Costing_CSV_20201230.png](https://support.vintrace.com/hc/article_attachments/32329276588180)

You can save the report’s output to a CSV file, or email it.

## Running the Wine/Juice Costing Report

To run the Wine/Juice Costing Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329276563220) Reports in the sidebar.
2. Select Bulk Wine.

![Winery_Reports_-_Bulk_Wine_-_Wine_Juice_Costing_20210105.png](https://support.vintrace.com/hc/article_attachments/32329230743060)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

You can apply the following filters to the Wine/Juice Costing Report:

- From — The first date to include in the report.
- To — The last date to include in the report.
- Batch — The batch to include in the report. You must select a batch.
