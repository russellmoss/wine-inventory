# Wine Barrel History

The Equipment Usage Report (Product History) shows the barrel’s history including what’s been transferred in and out of a vessel.

![Equipment_Usage_Report_PDF_BJG0901_20210104.png](https://support.vintrace.com/hc/article_attachments/32329016306324)

You can save the report’s output to a PDF file, or email the output.

## Running the Equipment Usage Report

To run the Equipment Usage Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328977834388) Reports in the sidebar.
2. Select Equipment.

![Winery_Reports_-_Equipment_-_Equipment_Usage_Report_20210104.png](https://support.vintrace.com/hc/article_attachments/32328973409172)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

You can apply the following filters to the Equipment Usage Report:

- Equipment — The barrel you want to run the report for. You must select a vessel.
- From — The first date to include in the report.
- To — The last date to include in the report.
