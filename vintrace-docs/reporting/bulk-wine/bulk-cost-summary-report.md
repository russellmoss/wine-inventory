# Bulk Cost Summary Report

The Bulk Cost Summary Report shows the opening and closing balance, and all cost differentials for the specified date range. This report helps reconcile how costing was impacted by cost adjustments and winery operations. It can help provide you with information to journal COGS adjustments in your [accounting package](https://support.vintrace.com/hc/en-us/articles/32303315132180).

![Bulk_Cost_Summary_CSV_-_19Rose_Harvest_Demo_20200105.png](https://support.vintrace.com/hc/article_attachments/32328924355732)

You can save the report’s output to a CSV file, or email the output.

## Running the Bulk Cost Summary Report

To run the Bulk Cost Summary Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328897625876) Reports in the sidebar.
2. Select Bulk Wine.

![Winery_Reports_-_Bulk_Wine_-_Bulk_Cost_Summary_20221020.png](https://support.vintrace.com/hc/article_attachments/32328880287508)

3. Specify the [filters and options](#Filters) for the report.
4. Click Email.

Although you can click Generate, this report may take some time to run so we recommend emailing the output.

5. Specify the email addresses where you want to send the report.
6. Click Send.

## Filters and Options

You can apply the following filters to the Bulk Cost Summary Report:

- From — The first date to include in the report. You must specify a date.
- To — The last date to include in the report. You must specify a date.
- Batch — The batch to include in the report.
- Owner — The owner to include in the report.
- Winery — The winery to include in the report.
- Cost Rate — Select the cost rate you’d like to use for the report (i.e., Total $ or $/gal).
- Summarize By — Select how you’d like to group the output (i.e., program or product).

If you’d like the ability to track and summarize product information, contact support to have the feature enabled.

- Group By Stock Item Type – If checked, the columns for consumed stock items and additives are summarized.

![Bulk_Cost_Summary_CSV_-_Summarized_by_Program_-_Grouped_by_Stock_Item_Type_20221020.png](https://support.vintrace.com/hc/article_attachments/32328880238484)

If unchecked, each consumed stock item and additive is displayed in its own column.

![Bulk_Cost_Summary_CSV_-_Summarized_by_Program_-_Not_Grouped_20221020.png](https://support.vintrace.com/hc/article_attachments/32328897715988)

- Show Active Only — Select to only include active wines.
