# Composition of Bulk Wine

There are two reports that enable you to view the composition of a wine: the [Label Integrity Report](#h_01ET5Z78J8NZQ16PSE8557AARS) and the [Bulk Stock Report](#h_01ET5Z7FNBF9ESXG8FT32EMB7K).

## Label Integrity Statement

The [Label Integrity Statement](https://support.vintrace.com/hc/en-us/articles/32301330173972) shows you the components of a wine by vintage, variety, and region/sub AVA.

![Label_Integrity_Statement_-_2019_Hills_Red_20201222.png](https://support.vintrace.com/hc/article_attachments/32328977400212)

To include the composition details on the Label Integrity Report, be sure to select the Include Composition Breakdown checkbox.

![Winery_Reports_-_Label_Integrity_-_Include_Composition_Selected_20201222.png](https://support.vintrace.com/hc/article_attachments/32328948297876)

## Bulk Stock Report

The Bulk Stock Report is useful when you want to report on multiple bulk wines and their composition.

![Bulk_Stock_Report_PDF_20201222.png](https://support.vintrace.com/hc/article_attachments/32328984047892)

Be sure to select the Composition Details checkbox.

By default, the output is grouped by Origin Breakdown. However, you can select a different option from the list. The most commonly used grouping options are *Grouped Origin* or *Origin Breakdown*. We recommend trying the different options to determine which works best for your needs.

![Winery_Reports_-_Bulk_WIne_-_Bulk_Stock_Report_-_Composition_Details_Selected_20201222.png](https://support.vintrace.com/hc/article_attachments/32328948311188)

The Bulk Stock Report’s output can be sent to either a PDF or CSV file. If you output the report to a CSV file, you can select the Breakdown Comp. checkbox. This adds a unique line to the CSV file for each composition element so that the file is easier to read. For example, a 50/50 blend of 2018 Yarra Valley Chardonnay and 2019 Yarra Valley Marsanne would display two lines in the CSV file, each with a 50% composition.

![Bulk_Stock_Report_-_Output_Options_and_Breakdown_Comp_Checkbox_20220322.png](https://support.vintrace.com/hc/article_attachments/32328973159572)

As with all vintrace reports, you specify filters for the report. For example, if you process any custom crush/contract wine, you can specify the customer in the Owned By filter.
