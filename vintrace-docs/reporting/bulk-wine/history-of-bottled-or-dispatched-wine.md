# History of Bottled or Dispatched Wine

There are several ways to view the details of a wine that’s no longer in a vessel. The method you use will depend on whether the wine has been [bottled](#Bottled), or [dispatched](#Dispatched).

## History of Bottled Wine

To view the history of wine that was bottled and still in stock:

1. [Search for the product in the Stock Item Overview window](https://support.vintrace.com/hc/en-us/articles/32303350682388).
2. Select the Components tab. The list of components includes a link to the bulk wine.

![Stock_Item_Overview_-_Components_20201223.png](https://support.vintrace.com/hc/article_attachments/32329253315348)

3. Click the link for the bulk wine. The wine’s Product page displays.
4. Select the Jobs tab.

![CS19Pro_Group_8_-_Jobs_20201223.png](https://support.vintrace.com/hc/article_attachments/32329215516052)

## History of Dispatched Wine

To view a dispatched wine’s history:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329278220692) More Options in the sidebar.
2. From the Tools tile, click Dispatch Search.
3. Use the filters to find the wine.

![Dispatch_Search_20201220.png](https://support.vintrace.com/hc/article_attachments/32329259429524)

4. Click the wine.
5. Click View Wine Details.

![View_Wine_Details_Button_20201223.png](https://support.vintrace.com/hc/article_attachments/32329278298644)

6. If there are multiple packaging operations for the wine, select the operation.
7. Select the History tab. You can also run reports such as the Label Integrity Report or Status Report from the Reports menu in the lower left.

![11CHBLRR_-_History_and_Reports_Menu_20201223.png](https://support.vintrace.com/hc/article_attachments/32329259455764)

If you need more details, you can also run the [Stock Dispatch Report](https://support.vintrace.com/hc/en-us/articles/32301330369684).
