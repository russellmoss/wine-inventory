# Bulk Cost Movement by Posted Date Report

This functionality is available starting with [vintrace 9.4.3](https://support.vintrace.com/hc/en-us/articles/32303276816020)

The Bulk Cost Movement by Posted Date report lets you report on costs for a specified date range and include all changes to the costs within that time frame. That is, the report includes backdated fruit costs as an input; any costs that affect the cost in the reporting period would be shown as an adjustment.

For example, suppose the following events occurred:

- September: Fruit from block A was received.
- October: Fruit was crushed and blended into batch X, Y, and Z.
- November: Some of the wine was sold.

If we run the Bulk Cost Movement by Posted Date report in December, the report will include the backdated fruit costs as input.

You can save the report’s output to a CSV file, or email it.

## Running the Bulk Cost Movement by Posted Date Report

To run the Wine/Juice Costing Report:

1. Click ![Reports Menu Option 20200406.png](https://support.vintrace.com/hc/article_attachments/32329210947476) Reports in the sidebar.
2. Select Bulk Wine.

![Winery Reports - Bulk Wine - Bulk Cost Movement by Posted Date 20230608.png](https://support.vintrace.com/hc/article_attachments/32329221239956)

3. Specify the date range for the report.
4. Click Generate or Email.
