# Label Integrity Statement

The Label Integrity Statement shows you a wine’s composition by vintage, variety, and region. If the wine contains any [allergens](https://support.vintrace.com/hc/en-us/articles/360000812775-Setting-Up-an-Additive), these are included in the report.

![Label_Integrity_Statement_with_Allergen_19CH-B2.1_20200917.png](https://support.vintrace.com/hc/article_attachments/32328764828564)

If you’ve [enabled VQA support](https://support.vintrace.com/hc/en-us/articles/32301321710868) in vintrace, you can [include the average VQA brix of your wine in the Label Integrity Statement](https://support.vintrace.com/hc/en-us/articles/360001853155#LabelIntegrityStatement).

![Label_Integrity_Statement_20200831.png](https://support.vintrace.com/hc/article_attachments/32328738832788)

You can save the report’s output to a PDF file, or email it.

## Running the Label Integrity Statement

To run the Label Integrity Statement:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328764938900) Reports in the sidebar.
2. Click Label Integrity.

![Winery_Reports_-_Label_Integrity_Statement_20201230.png](https://support.vintrace.com/hc/article_attachments/32328720569236)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

The Label Integrity Statement has the following filters and options:

- Juice/Wine
- Winemaker
- Date
- Reference Code
- Customer
- Order No.
- Owner
- Format
- Override Amount
- Include Composition Breakdown
- Show Full Composition
- Average VQA Brix by Vintage/Variety — This checkbox is only available if VQA is enabled.
- Include Govt. Reg. # — This checkbox is only available if VQA is enabled.
- Override Allergen Text
