# Causes of Loss Report

The Causes of Loss report charts the reasons for loss during the specified date range. You can hover over a segment to show its count and percentage.

![Causes_of_Loss_-_Volume_Chart_Hover_20221114.png](https://support.vintrace.com/hc/article_attachments/32328829497364)

## Running the Causes of Loss Report

To run the Causes of Loss report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328805243668) Reports in the sidebar.
2. Click Loss.

![Winery_Reports_-_Loss_-_Causes_of_Loss_20221114.png](https://support.vintrace.com/hc/article_attachments/32328821592724)

3. Specify the filters and options for the report.
4. Click Show Chart.

## Downloading the Report's CSV File

After running the report, you can download a CSV of the report's data.

![Causes_of_Loss_CSV_20221114.png](https://support.vintrace.com/hc/article_attachments/32328812984468)

To download a CSV file of the report's data:

1. [Run the report](#h_01GHVSZQ0H2PVWYGEKH7G0X9KF).
2. Click the ![Three_Horizontal_Lines_20201030.png](https://support.vintrace.com/hc/article_attachments/32328829525140) icon.
3. Select Download CSV.

![Causes_of_Loss_-_Volume_-_Download_CSV_20221114.png](https://support.vintrace.com/hc/article_attachments/32328821558804)

4. Specify where you would like to save the file.
