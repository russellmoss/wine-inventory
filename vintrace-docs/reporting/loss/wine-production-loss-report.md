# Wine Production Loss Report

The Wine Production Loss report provides insight into your winery’s production costs.

![Wine Production Loss as CSV 20231023.png](https://support.vintrace.com/hc/article_attachments/32329221959060)

The ability to track a wine batch's product information is not enabled by default. If you're interested in using this feature, contact support to have it enabled.

To run the report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329239576980) Reports in the sidebar.
2. Click Loss.

![Winery Reports - Loss - Wine Production Loss 20231023.png](https://support.vintrace.com/hc/article_attachments/32329221899924)

3. Specify the [filters and options](#filters_options) for the report.
4. Click Generate or Email.

## Filters and Options

You can apply the following filters to the Wine Production Loss as CSV report:

- Batch — The batch to include in the report.
- From — The first date to include in the report. You must specify a date.
- To — The last date to include in the report. You must specify a date.
- Products Owned By — The owner to include in the report.
- Winery — The winery to include in the report.
- Show Each Loss Event — If checked, the output will include a row for each loss event and a column for the work order number.

![Wine Production Loss as CSV - Show Each Loss Event 20231023.png](https://support.vintrace.com/hc/article_attachments/32329227427092)

 If unchecked, the output includes a single row for each Batch and columns for each loss reason grouping.

![Wine Production Loss as CSV 20231023.png](https://support.vintrace.com/hc/article_attachments/32329227440148)

- Show Physical Loss Only - If checked, only physical loss will be included in the output.
