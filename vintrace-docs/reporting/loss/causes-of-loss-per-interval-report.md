# Causes of Loss Per Interval Report

The Causes of Loss Per Interval report displays a bar chart of the different loss reasons during the specified date range. The counts for each loss reason are grouped together using the specified time interval (i.e., days, weeks, months, years). You can hover a bar to display the count for the interval.

![Causes_of_Loss_by_Month_-_Hover_20221114.png](https://support.vintrace.com/hc/article_attachments/32329211377172)

## Running the Causes of Loss Per Interval Report

To run the Causes of Loss Per Interval report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329221640212) Reports in the sidebar.
2. Click Loss.

![Winery_Reports_-_Loss_-_Causes_of_Loss_per_Interval_20221114.png](https://support.vintrace.com/hc/article_attachments/32329198161556)

3. Specify the filters and options for the report.
4. Click Show Chart.

## Downloading the Report's CSV File

After running the report, you can download a CSV of the report's data.

![Causes_of_Loss_by_Month_CSV_20221114.png](https://support.vintrace.com/hc/article_attachments/32329206629908)

To download a CSV file of the report's data:

1. [Run the report](#h_01GHVTCMNP4YPJDPDQQ3AKCHG6).
2. Click the ![Three_Horizontal_Lines_20201030.png](https://support.vintrace.com/hc/article_attachments/32329206640916) icon.
3. Select Download CSV.

![Causes_of_Loss_by_Month_-_Download_CSV_20221114.png](https://support.vintrace.com/hc/article_attachments/32329221708436)

4. Save the file.
