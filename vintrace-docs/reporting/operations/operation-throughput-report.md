# Operation Throughput Report

The Operation Throughput Report shows the operations performed during the specified date range.

![Operation_Throughput_Report_PDF_20201222.png](https://support.vintrace.com/hc/article_attachments/32328949059348)

You can save the report’s output to a CSV or PDF file, or email it.

## Running the Operation Throughput Report

To run the Operation Throughput Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328921137172) Reports in the sidebar.
2. Click Operations.

![Winery_Reports_-_Operations_-_Operation_Throughput_20201229.png](https://support.vintrace.com/hc/article_attachments/32328903768596)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

The Operations Throughput Report has the following filters and options:

- From — The first date to include in the report. You must specify a From date.
- To — The last date to include in the report. You must specify a To date.
- Owner
- Include Operation Summaries — Select to include the operation summary.
- Operation
- Treatment
- Winery
- Format
- Large Page Size
- Landscape — Selected by default. De-select to change the report’s orientation to portrait.
- Batch
