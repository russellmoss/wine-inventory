# Annual California Grape Crush Report (US)

If you’ve entered analysis data after [receiving fruit](https://support.vintrace.com/hc/en-us/articles/32301356626324), you can run the Crush/Extraction Report.

![Crush_Extraction_Report_PDF_20210105.png](https://support.vintrace.com/hc/article_attachments/32329007389716)

The Crush/Extraction Report shows the average analysis information, total tonnes received, and volume extracted for the selected Group By options. For example, you could group the information by grower and varietal.

![Crush_Extraction_Report_-_Group_By_Growner_Variety_20210105.png](https://support.vintrace.com/hc/article_attachments/32328978504340)

You can save the report’s output to a PDF file, or email the output.

## Running the Crush/Extraction Report

To run the Crush/Extraction Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329016931860) Reports in the sidebar.
2. Select Vintage/Harvest.

![Winery_Reports_-_Vintage_Harvest_-_Crush_Extraction_Report_20210105.png](https://support.vintrace.com/hc/article_attachments/32328991083540)

3. Specify the filters and options for the report.
4. Click Generate or Email.
