# Out-of-Range Metrics in the Grape Delivery Report

This functionality uses your specified [metric thresholds](https://support.vintrace.com/hc/en-us/articles/32301340717972-Metric-Thresholds-and-Metric-Action-Policies) and only applies if [grower contracts](https://support.vintrace.com/hc/en-us/articles/4404515723919) are enabled.

Metrics in the [Grape Delivery report](https://support.vintrace.com/hc/en-us/articles/9108027177743) can be marked with a symbol (^) to indicate that they’re out-of-range.

To mark out-of-range metrics, be sure to do the following when running the Grape Delivery report:

- Set the report’s format to CSV.
- Select the Show Analysis checkbox, click Options, then select the Highlight Out of Range Metrics checkbox.
