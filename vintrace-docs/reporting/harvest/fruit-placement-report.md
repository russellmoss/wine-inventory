# Fruit Placement Report

The Fruit Placement Report shows the bulk dispatches, packaging runs, and bulk wine where fruit has been used.

![Fruit_Placement_Report_-_Fruit_Origin_PDF_20230427.png](https://support.vintrace.com/hc/article_attachments/32329221311764)

You can save the report’s output to a PDF, or email it as a CSV.

The report will also include costs if you have the [Can View Costs permission](https://support.vintrace.com/hc/en-us/articles/360000813755-Roles-and-Permissions#Permissions). Specifically:

- For bulk dispatches, the cost for each row will be the total cost of the wine up to the date/time the wine was dispatched, proportioned to the volume of wine dispatched.
- For packaging runs, the cost for each row will be the total cost of the wine up to the date/time the wine was packaged, proportioned to the volume of wine dispatched.
- For bulk wine on hand, the cost for each row will be the total cost of the wine up to the date/time the report was run. That is, the "Show Stock as of" date/time, less any dispatched and/or packaged values.

This is available starting with vintrace 9.4.3. This functionality is not enabled by default. If you would like to use this functionality, please contact our support team.

## Running the Fruit Placement Report

To run the Fruit Placement Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329221271188) Reports in the sidebar.
2. Click Vintage/Harvest.

![Winery_Reports_-_Vintage_Harvest_-_Fruit_Placement_Report_20230427.png](https://support.vintrace.com/hc/article_attachments/32329197530516)

3. Specify the filters for the report.
4. Click Generate or Email.

You can also run the Fruit Placement Report from the from Block Overview window.

![Fruit_Placement_Report_from_Block_Overview_20230427.png](https://support.vintrace.com/hc/article_attachments/32329206062612)
