# Fruit Bookings Tank Report

The Fruit Bookings Tank Report shows vessel details for scheduled and completed extraction jobs. You can generate the report ahead of time to view the vessels scheduled for the selected booking’s extraction work orders and crush/press loads.

![Fruit_Bookings_Tank_Report_20210105.png](https://support.vintrace.com/hc/article_attachments/32328905845140)

The report’s output displays in a PDF file.

If the functionality to filter the Fruit Bookings Report by crusher/press is enabled for your account, the name of the crusher or press that was used to filter the report displays at the top.

## Running the Fruit Bookings Tank Report

You can run the Fruit Bookings Tank Report from the Harvest Calendar, or from the Fruit Intake Console.

### From the Harvest Calendar

To run the report from the Harvest Calendar:

1. Click ![Calendar_Icon_20200423.png](https://support.vintrace.com/hc/article_attachments/32328951387028) Calendar in the sidebar.
2. Select Harvest.
3. Click the ![Reports_Icon_20200403.png](https://support.vintrace.com/hc/article_attachments/32328905678740) reports icon.
4. Select Bookings with Tanks.

![Report_-_Bookings_with_Tanks_20210105.png](https://support.vintrace.com/hc/article_attachments/32328914161172)

### From the Fruit Intake Console

To run the report from the [Fruit Intake Console](https://support.vintrace.com/hc/en-us/articles/32303330881044), click Reports, then select Bookings with Tanks.

![Fruit_Intake_Console_-_Reports_-_Bookings_with_Tanks_20210105.png](https://support.vintrace.com/hc/article_attachments/32328905918100)
