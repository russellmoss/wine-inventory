# Grower Contract Remittance Report

The Grower Contract Remittance Report shows the paid-to-date and previously paid [installment](https://support.vintrace.com/hc/en-us/articles/360000826076-Managing-Grower-Contract-Payments-and-Fruit-Costs#MakinganInstallment) amounts, including any [bonus costs](https://support.vintrace.com/hc/en-us/articles/360000826076-Managing-Grower-Contract-Payments-and-Fruit-Costs#MetricValueRuleBonuses) per fruit receival for each grower contract.

You can generate the report and send it to the grower as a proof of payment after each installment is paid.

![Report_3_20210531.jpg](https://support.vintrace.com/hc/article_attachments/32329241354772)

![Report_4_20210531.jpg](https://support.vintrace.com/hc/article_attachments/32329212902164)

If you run the report for multiple growers, vintrace generates a ZIP file that contains a PDF file for each grower.

## Running the Grower Contract Remittance Report

To generate the report, a grower must be setup on the Grower Contract, with at least one installment paid. Refer to our [Managing Grower Contract Payments and Fruit Costs article](https://support.vintrace.com/hc/en-us/articles/360000826076-Managing-Grower-Contract-Payments-and-Fruit-Costs) to learn more.

To run the Grower Contract Remittance Report:

1. Click ![More_Options_20200323.png](https://support.vintrace.com/hc/article_attachments/32329199746836) in the sidebar. The More Options page displays.
2. From the Accounts tile, click Manage Grower Contracts.
3. Specify the filters.
4. Select the Deferred Payments checkbox to search for grower contracts with installments enabled.

![Report_1_20210531.jpg](https://support.vintrace.com/hc/article_attachments/32329208200340)

If accounting integration isn't set up, contact vintrace support to enable a setting that will allow you to select Defer Payments on a contract and make installments.

5. Click Search.
6. Select the grower contracts that you want to run the report for. To select all grower contracts, select the Select All/None checkbox.
7. From the Reports list that’s located in the lower left, select Remittance.

![Report_2_20210531.jpg](https://support.vintrace.com/hc/article_attachments/32329223080468)
