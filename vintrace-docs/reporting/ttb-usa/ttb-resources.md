# TTB Resources

The [TTB website](https://www.ttb.gov/) provides a number of resources that cover not only wine, but all aspects of regulating alcohol. These resources include:

- [Wine FAQs](https://www.ttb.gov/wine/wine-faqs), including the rules covering the tasting and sales of wine (see question W5)
- [Guide to Form 5120.17](https://www.ttb.gov/ttb-form-512017)
- [TTB 5120.17 Color Coded Sample Report of Wine Premises Operations](https://www.ttb.gov/images/pdfs/ttb-p-512017.pdf)
- [Report of Wine Premises Operations Form 5120.17 Reminder](https://www.ttb.gov/regulated-commodities/beverage-alcohol/wine/report-of-wine-premises-operations)
- [Conversion tables](https://www.ttb.gov/regulated-commodities/beverage-alcohol/distilled-spirits/conversion-tables), including proof gallons if you’re dealing with spirits
