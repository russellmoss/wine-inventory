# Setting Up a Tax Class

You can set up a tax class from the Winery Setup window (Setup Options > Policy > Tax Class):

1. Click ![Setup_Icon_20200318.png](https://support.vintrace.com/hc/article_attachments/32328879856148) Set Up in the sidebar.
2. Click TTB.
3. From the Tax Classes tile, click Configure.
4. Click New Tax Class. The Tax Class window displays.

![Tax_Class_Create_20210113.png](https://support.vintrace.com/hc/article_attachments/32328879883156)

5. Specify the details for the tax class.
6. Click Save.
