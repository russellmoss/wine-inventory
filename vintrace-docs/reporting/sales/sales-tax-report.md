# Sales Tax Report

The Sales Tax Report shows each tax type and amount for each sales order within the specified date range. The report only includes approved sales orders.

![Sales_Tax_Report_PDF_20210107.png](https://support.vintrace.com/hc/article_attachments/32328897345556)

You can save the report’s output to a PDF file, or email it.

## Running the Sales Tax Report

To run the Sales Tax Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32328897305492) Reports in the sidebar.
2. Click Sales.

![Winery_Reports_-_Sales_-_Sales_Tax_Report_20210107.png](https://support.vintrace.com/hc/article_attachments/32328902453780)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

The Sales Tax Report has the following filters and options:

- From — The first date to include in the report. You must specify a From date.
- To — The last date to include in the report. You must specify a To date.
- Customer
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to include in the report. If blank, the report includes all wineries in your setup.
- Sale Type
