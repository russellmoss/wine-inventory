# Sales Order Reporting

There are several reports that you can run for your [sales orders](https://support.vintrace.com/hc/en-us/articles/32303318150164):

- [Sales Report](https://support.vintrace.com/hc/en-us/articles/32301306265620) — The Sales Report summarizes sales orders during the specified date range. The default report details the sales orders with the customer, invoice date, type of sale, status and total. You have the option to run a detailed version of the report that includes each item in the sales order along with the quantity.

![Sales_Report_PDF_-_Regular_and_Detailed_20210107.png](https://support.vintrace.com/hc/article_attachments/32328991429140)

- [Sales Summary Report](https://support.vintrace.com/hc/en-us/articles/32301316392340) — The Sales Summary Report summarizes the sales amounts, taxes, costs, and profits for each of your sale items. The report enables you to select how you want to group the output.

![Sales_Summary_Report_-_Group_by_Item_20201230.png](https://support.vintrace.com/hc/article_attachments/32328985138964)

- [Sales Tax Report](https://support.vintrace.com/hc/en-us/articles/32301329865620) — The Sales Tax Report shows each tax type and amount for each sales order within the specified date range. The report only includes approved sales orders.

![Sales_Tax_Report_PDF_20210107.png](https://support.vintrace.com/hc/article_attachments/32329017513364)
