# Sales Report

The Sales Report summarizes sales orders during the specified date range. The default report details the sales orders with the customer, invoice date, type of sale, status and total.

![Sales_Report_PDF_20210107.png](https://support.vintrace.com/hc/article_attachments/32329251619604)

You have the option to run a detailed version of the report that includes each item in the sales order along with the quantity.

![Sales_Report_PDF_-_Detailed_20210107.png](https://support.vintrace.com/hc/article_attachments/32329243295636)

You can save the report’s output to a PDF file, or email it.

## Running the Sales Report

To run the Sales Report:

1. Click ![Reports_Menu_Option_20200406.png](https://support.vintrace.com/hc/article_attachments/32329230639508)Reports in the sidebar.
2. Click Sales.

![Winery_Reports_-_Sales_-_Sales_Report_20210107.png](https://support.vintrace.com/hc/article_attachments/32329276553492)

3. Specify the [filters and options](#Filters) for the report.
4. Click Generate or Email.

## Filters and Options

The Sales Report has the following filters and options:

- From — The first date to include in the report. You must specify a From date.
- To — The last date to include in the report. You must specify a To date.
- Use Invoice Date — Select to apply the specified date range to the invoice date instead of the sales order date.
- Customer
- Winery — This filter is only available if you’re in a [multi-winery setup](https://support.vintrace.com/hc/en-us/articles/32303328608660). The winery to include in the report. If blank, the report includes all wineries in your setup.
- Sale Type
- Sale State
- Detailed — Select to include each item in the sales order along with the quantity.
