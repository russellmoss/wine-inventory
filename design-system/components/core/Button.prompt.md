Savvy's primary action control — calm and premium; gold solid for the main action, ink outline or quiet ghost for the rest. Labels are sentence case.

```jsx
<Button variant="primary" size="md" onClick={...}>Book a call</Button>
<Button variant="secondary">Learn more</Button>
<Button variant="ghost" iconRight={<Arrow />}>See the plan</Button>
```

Variants: `primary` (Savvy Gold), `secondary` (ecru outline on white), `ghost` (gold text on soft tint), `inverse` (white on dark sections), `link` (inline gold link). Sizes: `sm` / `md` / `lg`. Supports `iconLeft` / `iconRight`, `fullWidth`, `disabled`.
