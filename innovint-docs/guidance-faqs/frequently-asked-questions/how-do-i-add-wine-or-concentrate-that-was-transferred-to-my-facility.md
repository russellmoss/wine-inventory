# How do I add wine or concentrate that was transferred to my facility?

This article details how to receive wine or concentrate into your InnoVint account from a non-InnoVint facility. To receive wine or concentrate from another facility using InnoVint, we recommend using the [Bond to Bond to another InnoVint winery option.](https://support.innovint.us/hc/en-us/articles/360018542692-bond-to-bond-transfers?hsLang=en#B2B_IV)

If you're receiving bulk wine or concentrate into inventory, it's a two-step process:

### **Step 1: Create a new lot**

From the Lot Explorer, click the **+ Add Lot** button in the upper right. Complete the Create Lot form for the wine coming in, and be sure to set the tax class correctly.

Note that you will need to set the composition for this bulk wine at Lot Creation. If the composition details are unknown you have two options:

#### Option 1. **Use Vineyard Components.**

Go to Vineyard Explorer / Add Vineyard

Add a vineyard called "Bulk Wine", "Bulk Juice", or "Bulk Concentrate" for each appellation (can be broad like *California*, or specific like *Coombsville*) and add blocks for each varietal component.

If the varietal is not known you can select from several options (e.g. Red or White Wine, Other Red/White, Unknown Red/White).

#### Option 2. **Use Bulk Components.**

By checking the "Bulk Component" box in the Create Lot screen, you can create a lot using only vintage, varietal and appellation.  InnoVint will **not** require a vineyard and block component to create a Juice/Wine lot with this method.

### **Step 2: Fill the new lot with volume**

From the new lot detail page go to Record Action and select the "**B2B Transfer In"** action. Select the tank getting filled, and set the ending fill to the new volume. You can leave a note with any BOL (Bill of Lading) information you want to capture (e.g. source winery/bond and address, fruit/juice/wine details, etc). After submitting the action, you can even attach a phto or file attachment as a [note](https://support.innovint.us/hc/en-us/articles/360009672091-how-to-add-a-photo-note?hsLang=en).

This will record the Bond to Bond Transfer under the lot's tax class (Received in Bond, Page 1, Line 7) on your 5120.17 report.

**Note:** The Bond to Bond transfer action can be backdated to any date, even prior to lot creation.

For specific instructions on how to add *juice*, see our Support Center article here: [Receive Juice](//innovint-6865708.hs-sites.com/hc/en-us/articles/360006901751-receive-juice?hsLang=en)
