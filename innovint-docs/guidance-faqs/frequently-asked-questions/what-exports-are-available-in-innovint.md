# What exports are available in InnoVint?

There are many places where data from InnoVint can be exported to .csv or .xlsx format. Look for the blue 'Export' button, usually located near the top right corner of the page. Upon clicking the link, the download will automatically begin.

**Note**: Exports function best through the Google Chrome browser. There are known limitations with downloading exports from InnoVint in other browsers.

This article provides details of each available export listed below:

- [Lots](#Lots)
  - [Lot Explorer](#Lot_Explorer)
  - [Individual Lot Details](#Individual_Lot) (such as analysis or history)
- [Vessels](#Vessels)
  - [Vessel Explorer](#Vessel_Explorer)
  - [Individual Vessel Details](#Individual_Vessel)
- [Samples Explorer](#Samples_Explorer)
- [Dry Goods](#Additives)
  - [Dry Goods Explorer](#Additive_Explorer)
  - [Individual Batch Details](#Individual_Additive)

- [Vineyards](#Vineyards)

  - [Individual Block Details Lot Tab](#indivblock)
  - [Vineyard Events](#vyrdevents)

- [Report Explorer](#reportexplo)

### Lots

#### Lot Explorer

- *Simple Export includes*:
  - Lot Code
  - Lot Name
  - URL
  - Type
  - Volume
  - Weight
  - Fruit Weight
  - Tax Class
  - Bond
  - Stage
  - Tanks
  - Bins
  - Barrels
  - Kegs
  - Carboys
  - Steel Drums
  - Vintage
  - Top Vintage %
  - Varietal
  - Top Varietal %
  - Appellation
  - Top Appellation %
  - Vineyard
  - Block
  - Color
  - Archived: True or False
  - Owners (if activated)
  - Tags
  - Alc. %
  - RS
  - Free SO2
  - Total SO2
  - TA
  - VA
  - pH

- *Export Lot Components includes*:
  - Lot Code & Name
  - By individual component:
    - Percentage
    - Volume
    - Vintage
    - Varietal
    - Appellation
    - Vineyard
    - Block
    - Clone
    - Vineyard tags
    - Block tags

#### Individual Lot Details

- *Analysis > All analysis in a list export includes*:
  - Recorded at date
  - Analysis type
  - Value & Unit
  - Performed on: Lot Comp or Indiv. Vessel

- *Lot History export includes*:
  - Effective at date
  - Deleted at date
  - Created at date
  - Action
  - Action URL
  - Involved lots
  - Tax Classes at Time
  - Bond
  - Lot Owners
  - Details
  - Notes
  - Performed by
  - Status
  - Deleted On
  - Deleted By
  - Lot #1
  - Lot #1 Tax Class at Time
  - Lot #1 Change
  - Lot #2
  - Lot #2 Tax Class at Time
  - Lot #2 Change
  - Lot #3...

- *Lot Composition export includes*:
  - By individual component:
    - Percentage
    - Volume
    - Vintage
    - Varietal
    - Appellation
    - Vineyard
    - Block
    - Clone

- *Lot Cost export (requires the Costing add-on):*
  - Action effective at date
  - Action
  - New fill
  - Fill change
  - New cost total
  - New cost per unit
  - Delta
  - Additional Costs
  - Other
  - Other Delta
  - Storage
  - Storage Delta

### Samples Explorer

The Samples Explorer is only available for accounts with ETS Laboratories Integration activated.

- *Export includes:*
  - Status: Pulled or Not Pulled
  - Sample ID
  - Date pulled
  - Lot pulled from
  - Lab Source
  - Client
  - Analyses requested

### Dry Goods

The Dry Goods Explorer is only available for accounts with the Dry Goods Tracking.

#### Dry Goods Explorer

- *Export includes:*
  - Product name
  - Type of additive
  - Category
  - Amount on hand
  - On Hand Unit
  - Cost of Hand (if Costing is activated)
  - Cost per Unit (if Costing is activated)
  - Cost total (if Costing is activated)
  - Archived status
  - Tags
  - Owners

#### Individual Batch Details

- *Export includes:*
  - Addition action *effective at* date
  - Addition action *created at* date
  - Action: Addition
  - Type of additive
  - Product name
  - Involved Lot Code
  - Involved Lot Name
  - Lot owner
  - Amount used
  - Amount on Hand
  - Units
  - Batch #
  - Additive owner
  - Performed by
  - Action notes
