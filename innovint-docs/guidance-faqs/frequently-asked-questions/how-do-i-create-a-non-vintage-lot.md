# How do I create a Non-Vintage lot?

InnoVint requires a 4-digit code to be input into the "Vintage" box under **New Lot Composition**. If you are creating a non-vintage lot, we recommend using an old vintage that couldn't possibly be real or match any current vintage, such as 1999 or even 1950. Be sure to always use this same year to indicate non-vintage lots in your account, then you can easily search for all non-vintage wines under this year.

![How do I create a Non-Vintage lot-new lot vintage](https://support.innovint.us/hs-fs/hubfs/How%20do%20I%20create%20a%20Non-Vintage%20lot-new%20lot%20vintage.webp?width=688&height=219&name=How%20do%20I%20create%20a%20Non-Vintage%20lot-new%20lot%20vintage.webp)

Additionally, you can tag these wine as "Non-Vintage" and search by tags to easily pull up these lots.
