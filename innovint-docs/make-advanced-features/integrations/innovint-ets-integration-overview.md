# InnoVint + ETS Integration Overview

InnoVint and ETS have established a partnership that allows for analysis results to be posted directly into InnoVint's platform. Unique sample codes are generated in InnoVint, and communicated in return by ETS, allowing for a seamless integration without any manual data entry.

This feature requires activation. If you wish to activate it please contact Customer Success at [support@innovint.us](mailto:support@innovint.us).

### **How does it work?**

The ETS Integration feature is as easy as **1-2-3**!

1. Authorize ETS to share your analysis results with InnoVint.
   1. From an Admin on the ETS account, send an email to support@etslabs.com and copy support@innovint.us with the following content for each ETS account:

      *Dear ETS,*

      *Please release our ETS account data to InnoVint.*

      *InnoVint Account Name(s): \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_*

      *ETS Account Name: \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_*

      *ETS Client ID (if you have it): \_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_*

      ETS will respond to all copied parties once the data has been released. InnoVint will complete the integration and follow up with an email confirmation.
2. Pull a sample in InnoVint, either by work order or via your new Samples Explorer page, and adhere the InnoVint-generated sample label (and code!) to your container.
3. Send your samples to ETS and then sit back and relax! As soon as ETS posts the results your analysis data will populate your InnoVint account. You can check the status of your sample results from the Samples Explorer and once posted they will automatically populate under the Analysis tab in your Lot Details page.

For **step-by-step instructions** on using the ETS Integration feature, see [here.](https://support.innovint.us/hc/en-us/articles/115001540726-ets-integration-step-by-step-instructions-?hsLang=en)

### **FAQ**

**Q. What size/kind of labels do I need?**

*ETS analysis sheet sample labels should be printed on 2-1/3" x 3-3/8" size labels. We recommend Avery labels - product code #[8395](http://www.avery.com/avery/en_us/Products/Name-Badges/Name-Badges/Self-Adhesive-Removable-Name-Badge-Labels-PVC-Free_08395.htm?N=0&Ns=&refchannel=c042fd03ab30a110VgnVCM1000002118140aRCRD) (160 labels) or #[5395](http://www.avery.com/avery/en_us/Products/Name-Badges/Name-Badges/Self-Adhesive-Removable-Name-Badge-Labels-PVC-Free_05395.htm?N=0&Ns=&refchannel=c042fd03ab30a110VgnVCM1000002118140aRCRD) (400 labels).*

*If you use ribbon labels that accept pdf files (such as a Dymo or Zebra printer), we support 4" x 2-1/8" and 2-3/4" x 2-1/8" size labels.*

**Q. Can I import historical ETS analyses?**

*Historical ETS results cannot be automatically imported via the ETS Integration feature. This is because InnoVint generates a unique code for each sample that must be scanned by ETS to link the results to your account. However, you can import analyses via csv file using our [Analysis Import feature](//innovint-6865708.hs-sites.com/hc/en-us/articles/115002687291-how-to-import-analyses-via-csv?hsLang=en). This will allow you to upload and backdate results with the ETS logo.*

**Q. Can I participate and share my variety and appellation data with ETS for their Vintage Portal?**

*Yes! We support the ETS [Vintage Portal](https://www.etslabs.com/our-services/vintage-portal) by automatically printing the lot's variety and appellation on the ETS label. To learn more or opt out, contact info@ETSlabs.com.*
