# Riddling

### How to record a Riddling action

1. Select the Riddling action
   ![Riddling-select action](https://support.innovint.us/hs-fs/hubfs/Riddling-select%20action.webp?width=209&height=272&name=Riddling-select%20action.webp)![Riddling-record](https://support.innovint.us/hs-fs/hubfs/Riddling-record.webp?width=688&height=272&name=Riddling-record.webp)
2. (optional) Click on **+ Add note** to add a note to the action
3. Leave details or instructions in the **Details** text box
4. Select the vessels to apply the Riddling to
5. (optional) Backdate the action as needed
6. Click on  ![Riddling-record button](https://support.innovint.us/hs-fs/hubfs/Riddling-record%20button.webp?width=90&height=27&name=Riddling-record%20button.webp)
7. Update the Stage to Riddling if desired
   ![Riddling-stage](https://support.innovint.us/hs-fs/hubfs/Riddling-stage.webp?width=275&height=422&name=Riddling-stage.webp)
   ![Riddling-success](https://support.innovint.us/hs-fs/hubfs/Riddling-success.webp?width=404&height=91&name=Riddling-success.webp)
