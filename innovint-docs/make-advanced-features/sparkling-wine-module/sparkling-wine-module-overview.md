# Sparkling Wine Module Overview

InnoVint's Sparkling module allows you to track all steps of the Traditional Method sparkling wine production process.  This feature allows you to record additional actions after the base wine is made, including [bottling en tirage](https://support.innovint.us/hc/en-us/articles/360051230671-bottling-en-tirage-?hsLang=en), [riddling](https://support.innovint.us/hc/en-us/articles/360051383611-riddling-?hsLang=en), and [disgorge, dosage and package](https://support.innovint.us/hc/en-us/articles/360051230691-disgorge-dosage-packaging-?hsLang=en).

Check out step by step workflows [here](https://support.innovint.us/hc/en-us/articles/360050744032-sparkling-wine-production-feature-overview?hsLang=en).

In this article, read about more features to make lifer easier for our sparkling producers!

- [Sparkling Lot Properties](#properties)
- [Tirage Bins](#bins)
- [Sparkling Actions](#Actions)

This feature requires activation. If you wish to activate it please contact our Customer Success Team at [support@innovint.us](//innovint-6865708.hs-sites.com/hc/en-us/kb-tickets/new?hsLang=en).

### Sparkling Lot Properties

#### Designate lots as Sparkling

To designate a lot as sparkling, simply click the **Sparkling wine** radio button by Lot Style during lot creation. This will update the lot dot to include bubbles in it. 🥂

![Sparkling OVerview_lot list](https://support.innovint.us/hs-fs/hubfs/Sparkling%20OVerview_lot%20list.webp?width=250&height=154&name=Sparkling%20OVerview_lot%20list.webp)

![Sparkling OVerview_lot attributes](https://support.innovint.us/hs-fs/hubfs/Sparkling%20OVerview_lot%20attributes.webp?width=250&height=374&name=Sparkling%20OVerview_lot%20attributes.webp)

This can also be done when changing lot properties from the **More**drop-down on the Lot Details Page.

#### Set and report on sparkling tax class

There are now two tax classes for sparkling wines: Sparkling - Bottle Fermented and Sparkling - Bulk Processed. To edit the tax class, reference our article [here](https://support.innovint.us/hc/en-us/articles/207936576-declare-or-edit-tax-class?hsLang=en).

![Sparkling overview_tax class](https://support.innovint.us/hs-fs/hubfs/Sparkling%20overview_tax%20class.webp?width=468&height=366&name=Sparkling%20overview_tax%20class.webp)

#### Set sparkling lot stages

The new sparkling lot stages - En Tirage or Riddling - can be updated at any time from the Lot details page. See the article [here](https://support.innovint.us/hc/en-us/articles/204339859-how-to-change-lot-stage?hsLang=en) for additional information. Setting lot stages will allow you to [customize](https://support.innovint.us/hc/en-us/how-to-set-your-lot-stage-analyses?hsLang=en) your lot's dashboard analysis!

![Sparkling overview_stage](https://support.innovint.us/hs-fs/hubfs/Sparkling%20overview_stage.webp?width=250&height=388&name=Sparkling%20overview_stage.webp)

These properties can all be seen in the Lot Attributes tile on the Lot Details Page.

### Tirage Bins

Track your bulk sparkling wine while it is aging en tirage via an additional vessel type that is bottle format specific and describes capacity in bottles.

Add your tirage bins to your Vessel Explorer via the +Add vessel button.

### Sparkling Actions

Find step by step instructions for each action in the articles linked below.

#### Bottling en Tirage

A standalone movement, supported as an action or task (available for completion and submission via InnoApp).  **[Bottling en Tirage](https://support.innovint.us/hc/en-us/articles/360051230671-bottling-en-tirage-?hsLang=en)** allows you to move your base wine into bottles for their secondary fermentation, and track that lot in bottles contained within tirage bins as part of your bulk inventory.

This action supports consumption of packaging used in the bottling (such as bidules, caps and bottles) and allows you to track dry goods inventory and packaging costs on your lot.

#### Riddling

A standalone custom action or task (available for completion and submission via InnoApp).

**[Riddling](https://support.innovint.us/hc/en-us/articles/360051383611-riddling-?hsLang=en)** allows you to record, track or request riddling for your lots en tirage.

#### Disgorge, dosage and package

Use the flexibility of custom actions (or tasks) in conjunction with bottling actions (or tasks) in order to build your ideal disgorge, dosage and final packaging workflow. Tasks built for this workflow are supported by InnoApp.

In certain cases, our sparkling wineries may activate a special standalone "Disgorge, Dosage and Package" action; this action is not compatible with all InnoVint features or winery workflows. This action is available only as a direct action.

Find out about disgorge, dosage and package actions and workflows **[here](https://support.innovint.us/hc/en-us/articles/360051230691-disgorge-dosage-packaging-?hsLang=en)**!
