# Harvest Launchpad

This feature is available only with specific subscription levels (currently included with our "Grow" module.  If you'd like to find out more, please reach out to InnoVint Support at [support@innovint.us](mailto:support@innovint.us).

The Harvest Launchpad is a single interface for crushpad operations. It's a great tool for planning what equipment you will need ready that week based on your incoming fruit.

Access the Harvest Launchpad from the Vineyard Dashboard (click on Launchpad from the Quick Links at the top) or by selecting Harvest Launchpad from the left navigation bar (Vineyards > Harvest Launchpad).

![Harvest Launchpad-menu](https://support.innovint.us/hs-fs/hubfs/Harvest%20Launchpad-menu.webp?width=688&height=397&name=Harvest%20Launchpad-menu.webp)

The Harvest Launchpad displays all open work orders that contain at least one Receive Fruit task, and any submitted work orders for the current day. Un-submitted work orders from previous days will also populate the launchpad.

*Ex. In the screenshot below, Saturday, July 6* *shows a completed and submitted work order from 6:00am, as well as two completed but unsubmitted work order tasks with a 7.30am timestamp.  The submitted task shows the actual weight received, while unsubmitted tasks will show the expected weight until submission.*

*There is another block scheduled to receive on Monday, July 8.*

![Harvest Launchpad-launchpad](https://support.innovint.us/hs-fs/hubfs/Harvest%20Launchpad-launchpad.webp?width=688&height=289&name=Harvest%20Launchpad-launchpad.webp)For each date, you can see the aggregated total expected or received weight for the expected day of arrival by color. As work orders are submitted, the total weight shown will update accordingly.

Under each date, each Receive Fruit task is shown with:

- The expected time of arrival
- Expected weight for the task
- Any notes on the Receive Fruit task
- Work order icon showing the status (i.e. not started, started, completed)
- "Open work order" button, or "Received" with a link to the action details
- Vineyard name
- Block Name
- Expected weight from the block, and the fruit lot code
- Owner tags (if activated)

![Harvest Launchpad-weight](https://support.innovint.us/hs-fs/hubfs/Harvest%20Launchpad-weight.webp?width=688&height=127&name=Harvest%20Launchpad-weight.webp)

If unscheduled fruit is received (i.e. without a work order), the button in the top right corner of the Launchpad ![Harvest Launchpad-rcv button](https://support.innovint.us/hs-fs/hubfs/Harvest%20Launchpad-rcv%20button.webp?width=129&height=22&name=Harvest%20Launchpad-rcv%20button.webp) will open a Receive Fruit direct action page.
