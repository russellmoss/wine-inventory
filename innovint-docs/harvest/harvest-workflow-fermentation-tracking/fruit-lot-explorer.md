# Fruit Lot Explorer

Want to find out more about what a Fruit Lot is and does? Check out our article on the difference between fruit lots and wine lots [here](https://support.innovint.us/hc/en-us/what-is-the-difference-between-a-fruit-lot-and-a-juice/wine-lot?hsLang=en).

#### Where to create Fruit Lots

Fruit lots can be created:

1. within a **Receive Fruit** action
2. from the Lot Explorer using the "**+ Add lot**" button or from the Fruit Lot Explorer using the "**+ Add Fruit Lot**" button
3. within a work order when creating a Receive Fruit task

Need more help creating a Fruit Lot? Find out more [here](https://support.innovint.us/hc/en-us/articles/360005034292-how-to-create-a-fruit-lot-in-innovint?hsLang=en).

#### How to find the Fruit Lot Explorer

Once you've created your Fruit Lots, you can find them in your Fruit Lot Explorer. To get there, go to your Vineyards tab > Vineyard Explorer.

![Fruit Lot Explorer-menu](https://support.innovint.us/hs-fs/hubfs/Fruit%20Lot%20Explorer-menu.webp?width=688&height=468&name=Fruit%20Lot%20Explorer-menu.webp)

At the top under the title bar, next to the Vineyard Explorer tab, you will see your Fruit Lot Explorer tab. This shows all *active* fruit lots whether they currently have contents or not. *Archived* fruit lots are automatically hidden.

**Please note that the default view is for the current calendar year vintage - if you don't see the lots you are looking for, try checking the Vintage filter first.**

![Fruit Lot Explorer-filter vint](https://support.innovint.us/hs-fs/hubfs/Fruit%20Lot%20Explorer-filter%20vint.webp?width=688&height=243&name=Fruit%20Lot%20Explorer-filter%20vint.webp)

You can filter by Archived status, vintage, varietal, vineyard, block (after vineyard is specified), etc. This Explorer is also exportable. To find your weight tags associated with fruit lots, visit the Fruit Intake Report in the Vineyards tab or the Report Explorer. Learn more about that report [here](https://support.innovint.us/hc/en-us/articles/205606335-fruit-intake-report?hsLang=en).

#### **FAQ**

**Q. I can't find my fruit lot.**

**![Fruit Lot Explorer-filters](https://support.innovint.us/hs-fs/hubfs/Fruit%20Lot%20Explorer-filters.webp?width=688&height=68&name=Fruit%20Lot%20Explorer-filters.webp)**

*A. The two most common ways to find a fruit lot involving checking the default filters:*

*1) Check the vintage filter. It always defaults to the current calendar year's vintage - if you are looking for a fruit lot from a different calendar year (such as looking for a 2024 fruit lot when it is January 2025), make sure to clear or reset the filter!*

*2) If you may have already processed the fruit lot (submitted a Process to Volume or Weight action involving the lot), be sure to include archived fruit lots. Fruit lots may have been auto-archived when they were emptied.*

**Q. I have weight remaining after harvest. How do I remove it?**

*A. If there is still weight populating your Fruit Lot Explorer that should not be there, we recommend either recording a [Fruit Weight Adjustment](//innovint-6865708.hs-sites.com/hc/en-us/articles/360006618412-weight-and-volume-adjustments-for-undeclared-fruit-or-juice?hsLang=en) to reduce extra weight (if this is extra unprocessed weight) or by deleting the Receive Fruit action (if the fruit was recorded in duplicate or error). Be sure to add any necessary notes. Be mindful that both of these actions do have implications for your TTB report so please have a look at [this section](https://support.innovint.us/hc/en-us/ttb-101?hsLang=en#part-iv-grape-material) of our Troubleshooting the TTB article, which discusses Part IV  "Summary of Materials Received and Used."*
