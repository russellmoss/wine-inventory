# Current Harvest Report

The Current Harvest Report dives deep into the details of harvest. This report focuses on your scheduled and received fruit (by block) status for the current harvest.

#### **How to find the Current Harvest Report**

Via the Vineyard menu:

![Current Harvest Report-vineyard menu](https://support.innovint.us/hs-fs/hubfs/Current%20Harvest%20Report-vineyard%20menu.webp?width=688&height=310&name=Current%20Harvest%20Report-vineyard%20menu.webp)

or, via the Reports header in the Vineyard Dashboard:

**![Current Harvest Report-vineyard dash](https://support.innovint.us/hs-fs/hubfs/Current%20Harvest%20Report-vineyard%20dash.webp?width=688&height=327&name=Current%20Harvest%20Report-vineyard%20dash.webp)**

#### **What is included in the Current Harvest Report?**

- Harvest status by block and the date associated with that status
  - **Received**: Fruit has been received on the block via a [Receive Fruit action or Work Order task](https://support.innovint.us/hc/en-us/articles/360005125552-receive-fruit?hsLang=en), and there are no open work orders to receive more. Status will be applied whether fruit is received through a work order or a direct action.  After the Receive Fruit work order is submitted, the block gets marked as "Received."
  - **Received - Partial**: Fruit has been received on the block and there is also an open work order to receive more. Status will be applied whether fruit is received through a work order or a direct action.
  - **Scheduled**: No fruit has been received on the block but there is an open work order to receive fruit. Fruit that is "Scheduled" (has an open Receive Fruit work order) will also appear on the [Harvest Calendar](https://support.innovint.us/hc/en-us/articles/115001093751-harvest-calendar?hsLang=en).
  - **Unplanned**: No fruit on the block has been received via direct action or work order, and no fruit on the block is planned on an open work order.
  - **Forecasted**: Shows if you enter a "Forecast a Harvest" event from the "Vintage Phenology and Forecasting Report" or from the Block Details Page. A Forecasted event is used a month or two in advance of harvest to give a general idea or when fruit may be coming in, based on phenology events. "Forecasted" fruit does not show on the Harvest Calendar.

- Average weight per block
  - The Average weight is a measure across all receive fruit actions for all harvests ever recorded on the block in InnoVint. Users can compare this average yield vs. expected yield vs. total received for the current vintage. This can help minimize mistakes in the vineyard.
- Crop estimates per block
  - Displays Crop Estimates recorded via a Record Crop estimate [action](https://support.innovint.us/hc/en-us/articles/360026806092-how-to-record-and-track-data-and-analyses-on-vineyard-blocks?hsLang=en#cropestimate)
- Weight received to date per block
  - The total of fruit received to date on the block via a Receive Fruit action or submitted Work Order task.

Additionally, filters across the top can be utilized to see when all fruit from a certain vineyard, block (if vineyard is selected), variety, appellation or grower has come in, or if it has been scheduled.

A 'forecasted' harvest event will not appear on the [Harvest Calendar](https://support.innovint.us/hc/en-us/articles/115001093751-harvest-calendar?hsLang=en). Only work order with open "Receive Fruit" tasks appear on the Harvest Calendar.

Forecasted events appear on the Vintage Phenology and Forecasting Report and in the Current Harvest Report.
