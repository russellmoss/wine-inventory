# Racking and Blending Workflows

In this video, we share some tips and tricks to optimize racking and blending actions in InnoVint. This session covers:

- When and how to use Topping versus Top-Off actions and tasks (3:06)
  - Adding breakdown vessels for the Top-Off task (or action) (9:08)
- When and how to use Racking versus Rack and Return actions and tasks (12:52)
- Recent changes to the Blend task:
  Requesting specific volumes "to remove" versus requesting "ending fills" (16:50)
