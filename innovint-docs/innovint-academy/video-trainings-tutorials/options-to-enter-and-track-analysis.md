# Options to Enter and Track Analysis

In this InnoVint Academy session, we review the multiple ways to record analyses into your account. This includes:

- ETS integrations for direct analysis import
- Reporting analysis data
- How to graph your data for easy viewing
- Importing analyses from a csv file
- Entering and viewing analyses by lot and by vessel
- Creating panels for frequently entered analyses

Find the full support article for entering analysis [here](https://support.innovint.us/hc/en-us/articles/360006665732-options-to-record-analysis-datachoose?hsLang=en).
