# Advanced Vineyard Tracking

### Vineyard Tracking 2023

In this session, we delve into some of the coolest, data-rich features of our Advanced Vineyard Tracking module, with a focus on pre-harvest activities. This session covers:

- A review of the Vineyard Dashboard *(3:55)*
- A deep-dive into the Vineyard and Block Details pages *(8:50)*
- Visualizing data with Vineyard Reports
  - Vintage Phenology and Forecasting Report *(20:50)*
  - Custom Vineyard Reports *(26:01)*
- Using the mobile app to record vineyard data in real-time *(29:02)*

### Vineyard Tracking 2021

This session offers a first tour of the Advanced Vineyard Tracking feature where we learn about the available functionality and reports. We will also cover vineyard contracts, how to record vineyard data, and how you can use that data as the season progresses in the vineyard. This session covers:

- Vineyard Dashboard & Navigation overview *(1:05)*
- Contracts
  - Adding contracts *(4:40)*
  - Contract Explorer display & data *(9:15)*
  - Contract, Fruit Cost Worksheet & Block Details - Cost Setting interactions (*10:17)*
- Recording Vineyard Block Data
  - Analysis *(13:16)*
  - Events *(15:31)*
  - Crop Estimate *(17:55)*
  - Forecast Harvest *(18:25)*
- Reports
  - Vintage Phenology & Forecasting Report *(19:50)*
  - Current Harvest Report *(24:48)*
  - Custom Analysis Report (Vineyard) *(26:16 & 30:05)*
  - Harvest Launchpad *(26:25)*
