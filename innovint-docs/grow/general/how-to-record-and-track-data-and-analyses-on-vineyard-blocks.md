# How to Record and Track Data and Analyses on Vineyard Blocks

With InnoVint's GROW module activated, you can record analysis, phenology, and vineyard events to track vineyard blocks throughout the growing season. Crop estimates can also be recorded to help with vintage forecasting and preparation for harvest. This article deals specifically with desktop functionality; you can also [record events and analysis using InnoApp](https://support.innovint.us/hc/en-us/mobile-app-how-to-enter-vineyard-events-and-crop-estimates?hsLang=en) - even offline!

Don't see the Vineyard Dashboard? The Vineyard Dashboard is a feature that requires the GROW module. If you wish to activate GROW, please contact our Customer Success Team at [support@innovint.us](mailto:support@innovint.us).

We support both Southern and Northern Hemisphere growing seasons: depending on your winery settings, this changes the current year "vintages" setting:

- Northern Hemisphere: current vintage is set for January 1 - December 31
- Southern Hemisphere: current vintage is set for July 1 - June 30th

### How to record analysis on a block

From the Record dropdown, select Analysis.

- **Vineyard** Select from the dropdown.
- **Block** Select from the dropdown.
- **Vintage** Defaults to the current year. Enter a different year if backdating historical data.
- **Source** Select from the dropdown.
- **Analyses** Use a panel or add analyses.
  - Enter your values
- **Notes**Enter any additional information.
- **Backdate** If entering historical data, remember to backdate your entry.
  - You will get an orange warning message if the backdated year does not match the vintage year.

### How to record a vineyard event

From the Record dropdown, select Event.

- **Vineyard** Select from the dropdown.
- **Block** Select one, multiple or all blocks from the dropdown.
- **Vintage** Defaults to the current year. Enter a different year if backdating historical data.
- **Type**Select a Phenology stage, Application tracking, or Visual Observation from the dropdown.
  - For Phenology, multiple dates may be entered for each stage. Enter in the % for each entry.
- **Details** Enter any additional information
- **Backdate** If entering historical data, remember to backdate your entry.
  - You will get an orange warning message if the backdated year does not match the vintage year.

![How to record and track vineyard events_record event](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20events_record%20event.webp?width=670&height=327&name=How%20to%20record%20and%20track%20vineyard%20events_record%20event.webp)

If you are recording multiple events, don't forget that you can import multiple vineyard events via CSV file. Find the import instructions [here.](https://support.innovint.us/hc/en-us/how-to-import-vineyard-events-via-csv-file?hsLang=en)

Vineyard analysis is not supported in this import.

### How to record a crop estimate

From the Record dropdown, select Crop estimate.

- **Vineyard** Select from the dropdown.
- **Block** Select one, multiple, or all blocks from the dropdown.
- **Vintage** Defaults to the current year. Enter a different year if backdating historical data.
- **Last Estimate**Will display the most recent crop estimate for the select block(s)
- **Estimate**Enter the crop estimate per block.
- **Details**Enter any additional information.
- **Backdate** If entering historical data, remember to backdate your entry.
  - You will get an orange warning message if the backdated year does not match the vintage year.

![How to record and track vineyard events_record crop estimate](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20events_record%20crop%20estimate.webp?width=670&height=328&name=How%20to%20record%20and%20track%20vineyard%20events_record%20crop%20estimate.webp)

If you are recording multiple crop estimates, don't forget that you can import multiple estimates via CSV file. Find the import instructions [here.](https://support.innovint.us/hc/en-us/how-to-import-vineyard-events-via-csv-file?hsLang=en)

### How to edit or delete recorded vineyard data and analysis

#### Edit or delete a vineyard event or crop estimate

Go to the Events tab in the Vineyard Details page.

![How to record and track vineyard event_event tab](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20event_event%20tab.webp?width=670&height=99&name=How%20to%20record%20and%20track%20vineyard%20event_event%20tab.webp)

Use the filters to refine the list. Select the event or crop estimate you would like to edit or delete.

![How to record and track vineyard event_edit event tab](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20event_edit%20event%20tab.webp?width=670&height=193&name=How%20to%20record%20and%20track%20vineyard%20event_edit%20event%20tab.webp)

Click on *Edit event* or *Delete event*. If you are editing a vineyard event or crop estimate, make the necessary changes and select *Record*.

#### Edit or delete recorded analysis

InnoVint allows you to edit the value for a recorded analysis. You cannot edit the effective date or analysis type. To change the effective date or analysis type for a recorded analysis, we recommend deleting the entry and re-recording a new analysis.

Go to the Analysis tab in the Vineyard Details page. Then go to *All analyses in a list*.

![How to record and track vineyard events_edit-delete analysis](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20events_edit-delete%20analysis.webp?width=670&height=185&name=How%20to%20record%20and%20track%20vineyard%20events_edit-delete%20analysis.webp)

Find the analysis in the list, then click on *Edit* or *Delete* in the far right of the row.

### How to view and track data and analysis on vineyard blocks

#### From the Vineyard Details page

Go to the Analysis tab of the Vineyard Details page.

![How to record and track vineyard events_view analysis](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20events_view%20analysis.webp?width=670&height=398&name=How%20to%20record%20and%20track%20vineyard%20events_view%20analysis.webp)

You can select different types of graphs from the sub-tabs.

- **Multi-block graph** allows you to filter for multiple blocks to view a single analysis type per vintage.
- **Multi analysis-type graph** can display multiple analysis types for a single block per vintage.
- **Multi-vintage graph** allows you to view a single analysis type for a block over multiple vintages.

You can also view analysis:

- **Grouped by date**. Use the filters at the top to narrow your search to specific analysis types for one or all of you blocks.
- **In a list.** Use the filter to narrow your search by analysis types. This report can also be exported.

#### Vineyard Reports

Go to *Reports* in the Vineyard Dashboard.

![How to record and track vineyard events_find vineyard reports](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20events_find%20vineyard%20reports.webp?width=670&height=57&name=How%20to%20record%20and%20track%20vineyard%20events_find%20vineyard%20reports.webp)

- **Vintage Phenology and Forecasting Report** displays all recorded phenology and harvest dates for the current vintage and historical averages.
- **Current Harvest Report** populates with block data to give you an overview of the current harvest status. Displays the harvest status, forecasted and scheduled fruit receival dates, and fruit weight (historical average, crop estimates, and received tonnage).
- **Fruit Intake Report** displays all the fruit received during harvest via Receive Fruit direct action or work order task. This report is also accessible via the Report Explorer and you can see more information on it [here](https://support.innovint.us/hc/en-us/articles/205606335-fruit-intake-report?hsLang=en).
- **Custom Vineyard Reports** allow you to create a custom report to view and export block analyses.
  - Go to the Reports card on the Vineyard Dashboard to create a new Custom Vineyard Report.
    ![How to record and track vineyard events_vineyard dashboard_phenologywebp](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20and%20track%20vineyard%20events_vineyard%20dashboard_phenologywebp.webp?width=573&height=244&name=How%20to%20record%20and%20track%20vineyard%20events_vineyard%20dashboard_phenologywebp.webp)

- **Export vineyard events**provides a csv export of all Vineyard Events, including Phenology, Application, Visual Observations and Crop Estimates. This report can also be exported from the Vintage Phenology Forecasting report page.
