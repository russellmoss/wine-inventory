# How to manage vineyard contracts

Vineyard contracts provide a great deal of added value but are not available at all subscription levels. For more information, please contact Customer Success at [support@innovint.us](mailto:support@innovint.us).

With this feature, InnoVint allows you to manage vineyard contracts for fruit to buy and sell. Contracts link with your block data, and receive & process fruit actions in order to calculate and distribute direct fruit costs.

This article covers:

- [How to create a vineyard contract](#create)
- [How to edit or delete a vineyard contract](#edit)
- [How and where to view active contracts](#view)
- [Vineyard contracts and the Fruit Cost Worksheet](#fruitcostWS)

### How to create a vineyard contract

Go to ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360026561991Screen_Shot_2019-04-17_at_6.16.10_PM.png?width=101&name=support.innovint.ushcarticle_attachments360026561991Screen_Shot_2019-04-17_at_6.16.10_PM.png) in the top right corner of the Contract Explorer.

![How to manage vineyard contracts-explorer](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-explorer.webp?width=670&height=204&name=How%20to%20manage%20vineyard%20contracts-explorer.webp)

#### 1. Create vineyard contract

- **Contract to**: either Buy or Sell
- **Contract type**: Written or Verbal
  - Tip! attach a written contract as a pdf in a note!
  ![How to manage vineyard contracts-create](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-create.webp?width=283&height=132&name=How%20to%20manage%20vineyard%20contracts-create.webp)

You can only create a single "buy" contract per vineyard, but you can also add multiple "sell" contracts.

**A vineyard with multiple contracts can only support a single buy/sell price.**

#### 2. Select vineyard, blocks, and vintages

Select the vineyard, contracted blocks, and contracted vintages. Contract provisions, such as penalties, apply to all selected blocks and vintages of the contract:

- **Vineyard** Select from the dropdown of current vineyards. Each contract is limited to one (1) vineyard.
- **Block** Select all, one, or multiple blocks from the dropdown.
  ![How to manage vineyard contracts-block](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-block.webp?width=488&height=204&name=How%20to%20manage%20vineyard%20contracts-block.webp)
- **Vintages** Select one or more vintages that apply to this contract. Type in a vintage if it doesn't already appear in the list, and select it in the dropdown.
  ![How to manage vineyard contracts-vintage](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-vintage.webp?width=670&height=148&name=How%20to%20manage%20vineyard%20contracts-vintage.webp)
- **Price** For each block and vintage, enter the cost and select the unit (per ton/tonne/kg, acre/hectare, or block). If you enter a price *per weight*, also enter the contracted weight.
   -> If the cost and unit remain consistent across vintages, click on *Apply to all vintages.*
  ![How to manage vineyard contracts-price](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-price.webp?width=670&height=162&name=How%20to%20manage%20vineyard%20contracts-price.webp)

#### 3. Evergreen Clause

- Select *Yes* or *No.* If *Yes,* you will need to record:
  - **Effective Date**: Enter the date that the Evergreen Clause goes into effect
  - **Reminder Date**: This date will appear in the Contract Explorer list to help you remember the deadline to renegotiate or cancel the contract.![How to manage vineyard contracts-evergreen clause](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-evergreen%20clause.webp?width=670&height=91&name=How%20to%20manage%20vineyard%20contracts-evergreen%20clause.webp)

#### 4. Provisions

Select *Yes* or *No. Provisions apply to all selected blocks and vintages of the contract.*

If *Yes,* you can enter some or all of the following provisions:

- **Practices:** Enter vineyard practices outlined in the contract
- **Penalties/Bonuses:** Select *Yes* or *No.* If *Yes,*
  - Sugar: Select *Yes* or *No.* If *Yes,*
    - Select the Sugar analysis type and unit
    - Enter the Under/Over limits for the penalty or bonus
    - (optional) Enter the penalty/bonus terms
  - Acid: Select *Yes* or *No.* If *Yes,*
    - Select the Acid analysis type and unit
    - Enter the Under/Over limits for the penalty or bonus
    - (optional) Enter the penalty/bonus terms
  - Defect: Select *Yes* or *No.* If *Yes,*
    - Enter the penalty/bonus terms (required)
  - MOG: Select *Yes* or *No.* If *Yes,*
    - Enter the penalty/bonus terms (required)![How to manage vineyard contracts-provisions](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-provisions.webp?width=670&height=347&name=How%20to%20manage%20vineyard%20contracts-provisions.webp)

#### 5. Viticulture, harvest, and delivery specifications

- (optional) Enter details. Applies to entire contract.

#### 6. Payment Terms

- (optional) Enter details. Applies to entire contract.

#### 7. Misc.

- (optional) Create or select a Tag to apply to the contract.

Click ![How to manage vineyard contracts-record another](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-record%20another.webp?width=188&height=19&name=How%20to%20manage%20vineyard%20contracts-record%20another.webp) or ![How to manage vineyard contracts-record](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-record.webp?width=115&height=19&name=How%20to%20manage%20vineyard%20contracts-record.webp) to create the contract.

If you want to add your contract pdf as a note to the contract, **first** create the contract. Next, reselect the contract and open it for editing in order to attach a file to a new note.

Note attachments have a file name character limit, and a file size limit:

- You can only have 100 characters in the file name, and having too many characters will cause the note submission to fail
- There is a file size limitation of 2.5MB - a file that is too large will fail

### How to edit or delete a vineyard contract

Go to the Contract Explorer and select a contract to edit from the list.

![How to manage vineyard contracts-edit](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-edit.webp?width=670&height=148&name=How%20to%20manage%20vineyard%20contracts-edit.webp)

- Click on the blue pencil next to *Contract* to edit
- Click on *Delete contract* to delete

Note: You must open a contract in edit mode in order to add a note after the contract is created, or to attach a file to a note.

### How and where to view contracts

#### Contract Explorer

A list of contracts can be found in the Contract Explorer. Select a vintage from the filter to display active contracts for that year.

Calculated costs are determined by the set cost per acre or block. If fruit costs are set per weight unit, the calculated costs are estimated based on the contracted tonnage (*Est.*) or if available, the historic, average yield (*Avg*). This will update based on Crop Estimates (*CE*) if entered, or else once fruit is received for that vintage.

### Vineyard Contracts and the Fruit Cost Worksheet

Your vineyard contracts interact with the Fruit Cost worksheet in the COGS Tracking module.

Updating costs in the Fruit Cost Worksheet will update your contract data, and conversely, when you enter costs in vineyard contracts, it will update your Fruit Cost Worksheet data.

![How to manage vineyard contracts-worksheet](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-worksheet.webp?width=670&height=193&name=How%20to%20manage%20vineyard%20contracts-worksheet.webp)

Vintage direct fruit costs can also be updated in a third location - in the Block details page, on the Vintages tab.
![How to manage vineyard contracts-vintage tab](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-vintage%20tab.webp?width=670&height=163&name=How%20to%20manage%20vineyard%20contracts-vintage%20tab.webp)

![How to manage vineyard contracts-edit cost](https://support.innovint.us/hs-fs/hubfs/How%20to%20manage%20vineyard%20contracts-edit%20cost.webp?width=670&height=280&name=How%20to%20manage%20vineyard%20contracts-edit%20cost.webp)

Editing cost in any of the three locations will also update it in the other two.
