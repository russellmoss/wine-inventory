# Vineyard Block Analysis Reports

Vineyard block analysis reports tracking are only available with activation of the GROW module. Reach out to [InnoVint Support](mailto:support@innovint.us) for more information.

A **Vineyard Block Analysis Report** gives you the ability to create analytical reports per block, tailored to your needs. You can create multiple reports with custom analyses and specific filters. Save these reports - including your filters - and come back to them at any time for reference. The filters and analysis values will update in real-time from your current data in the system.

This article covers:

- [How to create a vineyard block analysis report](#create)
- [How to modify or delete a vineyard block analysis report](#modify)
- [Where to find vineyard block analysis reports](#find)

### **How to create a vineyard block analysis report:**

1. Navigate to the **Vineyard Dashboard**
2. Under reports, click on **+ Create New**
   ![Vineyard Block Analysis Reports-create new](https://support.innovint.us/hs-fs/hubfs/Vineyard%20Block%20Analysis%20Reports-create%20new.webp?width=507&height=219&name=Vineyard%20Block%20Analysis%20Reports-create%20new.webp)
3. Begin by adding filters (eg vintage and tags) and then select desired analyses
   ![Vineyard Block Analysis Reports-add filters](https://support.innovint.us/hs-fs/hubfs/Vineyard%20Block%20Analysis%20Reports-add%20filters.webp?width=498&height=417&name=Vineyard%20Block%20Analysis%20Reports-add%20filters.webp)
4. Click on **Update Report**. This will update the report on the screen
5. Save your report

   - If this is a new report, you will only see the option to **Save as new report**
     ![Vineyard Block Analysis Reports-save new](https://support.innovint.us/hs-fs/hubfs/Vineyard%20Block%20Analysis%20Reports-save%20new.webp?width=688&height=104&name=Vineyard%20Block%20Analysis%20Reports-save%20new.webp)

1. - If you want to modify an established report, and save it as a new separate report, then save your changes by clicking on **Save as new report**. This will prompt you for a new name and keep each report separate.
   - If you want to modify an established report, click on **Save report**. This will save over the existing report.
     ![Vineyard Block Analysis Reports-save over existing report](https://support.innovint.us/hs-fs/hubfs/Vineyard%20Block%20Analysis%20Reports-save%20over%20existing%20report.webp?width=688&height=111&name=Vineyard%20Block%20Analysis%20Reports-save%20over%20existing%20report.webp)

### **How to modify or delete a vineyard block analysis report**

1. Go to the report that you would like to modify or delete
   - To modify - you can add or remove analyses and change filters within the report. You can choose to Save report to override your previous version, or click on Save as new report to create a separate, modified version
   - To delete - click on Delete report to remove the report from your list

### Where to find vineyard block analysis reports

In your Report Explorer: find existing reports, or create a new report.

![Vineyard Block Analysis Reports-find report](https://support.innovint.us/hs-fs/hubfs/Vineyard%20Block%20Analysis%20Reports-find%20report.webp?width=688&height=325&name=Vineyard%20Block%20Analysis%20Reports-find%20report.webp)

On your Vineyard Dashboard: link to existing reports or create a new report from the Report tile, or find a list of your reports in the Report meu.

![Vineyard Block Analysis Reports-access dashboard](https://support.innovint.us/hs-fs/hubfs/Vineyard%20Block%20Analysis%20Reports-access%20dashboard.webp?width=688&height=191&name=Vineyard%20Block%20Analysis%20Reports-access%20dashboard.webp)
