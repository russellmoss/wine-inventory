# Overview: User Permissions and Capability Levels

Winery members and users can be granted 1 of 4 different capability levels.

- [Admin](#admin)
- [Team Member](#teammember)
- [Team Member - Cannot Submit Work Orders](#tmcswo)
- [Read Only](#ro)

For accounts with the Costing feature activated, winery members can also be granted 1 of 3 access levels:

- [Full access](#full)
- [Read Only](#ro-c)
- [No access](#na)

If your account has Owner-based Permissions, user capabilities can be further limited by Owner tags. See our Support Center overview of [Owner-based Permissions and User Capabilities.](https://support.innovint.us/hc/en-us/articles/218236223-owner-based-permissions-and-member-capabilities-overview-article-?hsLang=en)

Capability levels (eg Admin, Team Member, Read Only) affect a winery member's ability to view and manage winery inventory and activities. Capability levels do not affect access to the Costing feature. For example, a winery member with Read Only capability might also have Full Access to Costing. This member would not be able to record winery activities or edit inventory, but will be able to record cost items and view costing reports.

Admins can add, remove, and edit winery member permissions by going to Settings > Member Management:

### Member Permissions: Capability

### Admin

Admins have unrestricted access within the account. Admins can:

- View, add, and edit all inventory and activity
- Perform all operations, including creating and submitting work orders and recording actions
- Add, remove, and edit users and their capability levels
- Define work order settings:
  - Set default work order submission & edit options
  - Edit and delete work order templates and analysis panels
  - Set visibility of print options
  - Set InnoApp work order and vessel check off parameters
- Create and edit Custom Attributes
- Lock winery & cost backdating
- Add, edit, and delete dip charts
- Manage harvest settings (i.e. set up Advanced Receive Fruit system and set expected yields)
- Create new and edit Shipping Locations
- Set Vessel Location dimensions
- Update templated & default legal language on the BOL
- Create, edit and save default Lot Explorer layouts
- Create, edit and save Custom Reports
- Set Custom Stage Dependent Analysis  displays
- View, run, and export files and reports
- *If custom crush permissions is activated* - Set up new owners and edit owner names

### Team Member

Team Members have access to any part of the platform related to operations. Team members can:

- View, add, and edit all inventory and activity
- Perform all operations, including creating and submitting work orders and recording actions
- Edit and delete work order templates and analysis panels
- Add, edit, and delete dip charts
- Create new and edit Shipping Locations
- Set Vessel Location dimensions
- Update templated & default legal language on the BOL
- Create, edit and save default Lot Explorer layouts
- Create, edit and save Custom Reports
- Set Custom Stage Dependent Analysis  displays
- View, run, and export files and reports

### Team Member - Cannot Submit Work Orders

This capability level is the same as a ***Team Member*** (above) with one further restriction: they cannot submit work orders. Users are still able to start and complete work orders, but a Team Member or Admin would have to submit after completion.

### Read Only

Read Only users have unrestricted access to view inventory and reports, but cannot make any changes or perform actions within the system. Read Only users can:

- View all inventory and activity
- View, run, and export files and reports (incl. inventory reports and TTB 5120.17)
- Access specific Settings such as "[Display Preferences](https://support.innovint.us/hc/en-us/preferences-in-settings?hsLang=en)"

### Member Permissions: Costing access

### Full Access

This access level allows the winery member to view, record, and edit direct costs and indirect costs in InnoVint, including all costing reports and cost history data. This access level also allows the winery member to view, record, and edit vineyard contracts.

### Read Only

Read Only grants the winery member access to view and export cost data, cost reports, and vineyard contracts. This access level does not allow the winery member to record or edit any cost data or contracts.

### No access

A winery member with No Access to Costing will not be able to view, record, or edit cost data or vineyard contracts in InnoVint.
