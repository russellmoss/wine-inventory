#### InnoApp: Lot Explorer

The mobile Lot Explorer is the entry point for accessing any lots you have in the system. The entire lot list displays in alpha/numeric order for scrolling. This explorer includes juice/wine lots, fruit lots and case good lots.

The Lot Explorer shows each lot (including the lot dot color) and its fill.

From the Lot Explorer, lot are searchable by entering text into the search filter. You can search by lot type (especially useful for looking for CG- and FL- lots) and lot code.

![InnoApp Lot -explorer](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-explorer.webp?width=209&height=379&name=InnoApp%20Lot%20-explorer.webp)

You will also see a status flag on a lot in the Lot Explorer if the lot is archived (archived lots are still available and searchable in the Explorer, but sorted at the bottom).

![InnoApp Lot -archived](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-archived.webp?width=217&height=48&name=InnoApp%20Lot%20-archived.webp)

#### InnoApp: Lot Details

By clicking on any of the lots in the Lot Explorer, you are taken to the Lot Details Page, which gives an even more in-depth view of that lot.  From this page you can view: Lot Contents (volume), Vintage, Varietal, Lot Stage, Lot Tax Class, the number and type of filled vessels, tags, owner tags, open work order icons (tap to view a list and access any open work order), notes (tap the link to view All Notes), the most recent analyses (with a link to All Analysis), and the history associated with the lot (with a link to View all History).

You can record actions here, too, as well as add and view notes.

![InnoApp Lot -details](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-details.webp?width=192&height=377&name=InnoApp%20Lot%20-details.webp)  ![InnoApp Lot -history notes](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-history%20notes.webp?width=266&height=386&name=InnoApp%20Lot%20-history%20notes.webp)

Here are some additional details on the data that you have available within this screen:

- **Vintage/Varietal:** Tap on Vintage or Varietal, and see the detailed Lot Composition

![InnoApp Lot -vint var](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-vint%20var.webp?width=193&height=253&name=InnoApp%20Lot%20-vint%20var.webp)   ![InnoApp Lot -vint var details](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-vint%20var%20details.webp?width=236&height=256&name=InnoApp%20Lot%20-vint%20var%20details.webp)

- **Vessel icon(s):** Tap on a Vessel icon in the Lot Details Page to find a list of all the vessels in the lot. Tapping into an individual vessel here will take you to that [vessel details](https://support.innovint.us/hc/en-us/vessel-explorer-page?hsLang=en#Details) page.

![InnoApp Lot -vessel](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-vessel.webp?width=201&height=252&name=InnoApp%20Lot%20-vessel.webp)   ![InnoApp Lot -vessel details](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-vessel%20details.webp?width=294&height=249&name=InnoApp%20Lot%20-vessel%20details.webp)

- **Work Order icon(s):** Tap into Work Order icons to see the list of open work orders related to the lot. Tap into an individual work order to see details, and complete or submit the work order.

![InnoApp Lot -wo icon](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-wo%20icon.webp?width=160&height=201&name=InnoApp%20Lot%20-wo%20icon.webp) ![InnoApp Lot -open wo](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-open%20wo.webp?width=247&height=201&name=InnoApp%20Lot%20-open%20wo.webp) ![InnoApp Lot -transfer](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-transfer.webp?width=228&height=202&name=InnoApp%20Lot%20-transfer.webp)

- **Lot Analyses:** InnoApp will display the most recent Lot Stage analysis on the Lot details page. See more information about which analysis populates the dashboard by Lot Stage [here](https://support.innovint.us/hc/en-us/articles/205001715-analysis-reporting?hsLang=en#dashboardsnap).Tap **All Analyses** to access a full list of analyses recorded on the lot. This is only available online unless you have the lot favorited. If the lot is a favorite, then you will be able to access the first 30 results while offline.
  **![InnoApp Lot -all analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-all%20analysis.webp?width=174&height=351&name=InnoApp%20Lot%20-all%20analysis.webp)  ![InnoApp Lot -all anlaysis detail](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-all%20anlaysis%20detail.webp?width=219&height=354&name=InnoApp%20Lot%20-all%20anlaysis%20detail.webp)**
- **Lot History:** View a history of actions recorded on the lot.

**![InnoApp Lot -history](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-history.webp?width=231&height=334&name=InnoApp%20Lot%20-history.webp)**

- **Lot Notes:** View previously recorded Notes and add Notes here. Tap **All Notes** to see all notes recorded on the lot.

#### InnoApp: Lot Actions

Currently, a limited number of direct actions are available on InnoApp:

![InnoApp Lot -actions](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-actions.webp?width=223&height=343&name=InnoApp%20Lot%20-actions.webp)

Don't forget! All of this data can be recorded while offline, and it will update in the cloud once you are back in range. Full lot details are available offline for [recently accessed or favorited items](https://support.innovint.us/hc/en-us/mobile-app-navigation-tips?hsLang=en#Favs). Remember to favorite Lots you want to access while offline!

- **Note**: record a written or photo note for your lot using this Record Action menu, or use the "Add Note" button in the Lot Details Notes section (bottom of the page).
- **Analysis**
  - Choose between Lot Composite or Individual Vessel analysis types
  - Use any pre-existing panel that you have in your account, or select specific types
  - Change the result units by tapping on them and toggling through your options
  - Use "Edit Analyses" to add or remove the selected Analysis types
    ![InnoApp Lot -analysis record](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-analysis%20record.webp?width=280&height=302&name=InnoApp%20Lot%20-analysis%20record.webp)  ![InnoApp Lot -record data analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-record%20data%20analysis.webp?width=216&height=311&name=InnoApp%20Lot%20-record%20data%20analysis.webp)
- **Sugars & Temp**: This is a shortcut option to record your harvest analyses. The sugar unit (Brix/Baumé) will default per your account settings, and all you need to do is enter the result!
  ![InnoApp Lot -brix temp](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-brix%20temp.webp?width=221&height=391&name=InnoApp%20Lot%20-brix%20temp.webp)
- **Custom Action**: Use this like you would a regular [custom action](https://support.innovint.us/hc/en-us/articles/204848455-using-a-custom-action-or-custom-task?hsLang=en) - a flexible, editable action that will be applied to the lot. Custom Actions can be used to apply actions or give instructions for activities that are not covered in InnoVint's current list of actions. This can include things like sanitizing tanks, laying out barrels, scheduling staff meetings, etc. The possibilities are endless!
  ![InnoApp Lot custom](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20custom.webp?width=222&height=293&name=InnoApp%20Lot%20custom.webp)
- **Pumpover/Punchdown/Delestage/Pulsair/Dry Ice/Stir:** All your fermentation management and harvest activities are here for you to record on the lot.
  ![InnoApp Lot -cap mgmt](https://support.innovint.us/hs-fs/hubfs/InnoApp%20Lot%20-cap%20mgmt.webp?width=227&height=251&name=InnoApp%20Lot%20-cap%20mgmt.webp)
