# InnoApp: How to Add, Remove or Replace Vessel Tags

This article covers how to use Vessel tags in InnoApp:

- [Via the Quick Action menu](#quick)
- [Via Vessel Details as an Action](#vessel)
- [In Bulk via work order tasks](#WO)
- [Demo Video (30 sec)](#demo)

InnApp doesn't yet support adding tags to Vineyards or Lots

#### Via the Quick Action menu

- Find and select the **Tag Vessels** icon in your **Quick Actions** menu at the bottom of your homescreen.
  ![InnoApp How to Add Remove or Replace Vessel Tags-quick action menu](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-quick%20action%20menu.webp?width=176&height=310&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-quick%20action%20menu.webp)
- Choose to either *Add*, *Remove*, or *Remove & Replace* tags on vessels. Then click on Select Tags to select an existing tag, or create a new one.
  - Use the **Search** function at the top right to find an existing tag,  if required.
    ![InnoApp How to Add Remove or Replace Vessel Tags-record select tags](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-record%20select%20tags.webp?width=190&height=304&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-record%20select%20tags.webp)  ![InnoApp How to Add Remove or Replace Vessel Tags-select tags](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-select%20tags.webp?width=174&height=308&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-select%20tags.webp)

- - To create a new tag, type the desired name, and click on **Create Tag**, then **Select**.
    ![InnoApp How to Add Remove or Replace Vessel Tags-create new tag](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-create%20new%20tag.webp?width=187&height=336&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-create%20new%20tag.webp)  ![InnoApp How to Add Remove or Replace Vessel Tags-select](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-select.webp?width=190&height=341&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-select.webp)
- Add vessels by scanning QR codes.  InnoVint's mobile app features continuous scanning functionality, so you can scan as many QR codes as you want. Click on **Done** when you have completed scanning.
  ![InnoApp How to Add Remove or Replace Vessel Tags-scan vessels](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-scan%20vessels.webp?width=187&height=302&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-scan%20vessels.webp)  ![InnoApp How to Add Remove or Replace Vessel Tags-complete scan](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-complete%20scan.webp?width=190&height=305&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-complete%20scan.webp)
- Confirm the total number of selected vessels, vessel codes and contents, and then **Select Vessels**.
  ![InnoApp How to Add Remove or Replace Vessel Tags-confirm select vessels](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-confirm%20select%20vessels.webp?width=190&height=305&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-confirm%20select%20vessels.webp)
- You will see a final confirmation screen and a summary of the tag changes and total vessel count before you click **Record**. Your tags will update on all vessels.

![InnoApp How to Add Remove or Replace Vessel Tags-final confirm record](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-final%20confirm%20record.webp?width=186&height=303&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-final%20confirm%20record.webp)

#### Via Vessel Details as an Action

- Access the Vessel Details page via the Vessel Explorer, or by scanning the vessel's QR code.
- Tap the **Record Action** menu and select **Tag**.
  ![InnoApp How to Add Remove or Replace Vessel Tags-record vessel details](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-record%20vessel%20details.webp?width=186&height=316&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-record%20vessel%20details.webp)  ![InnoApp How to Add Remove or Replace Vessel Tags-select tag](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-select%20tag.webp?width=180&height=318&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-select%20tag.webp)
- You will see the same tagging interface as the Quick Actions "Tag Vessels" button, but with the vessel already preselected:
  ![InnoApp How to Add Remove or Replace Vessel Tags-quick action tag](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-quick%20action%20tag.webp?width=186&height=299&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-quick%20action%20tag.webp)
  - You may scan in additional vessels using the **Add More** option, if needed.
    ![InnoApp How to Add Remove or Replace Vessel Tags-scan add more](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-scan%20add%20more.webp?width=185&height=297&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-scan%20add%20more.webp)
- Confirm tag and vessel details, and then click on **Record**. Your tags will update.
  ![InnoApp How to Add Remove or Replace Vessel Tags-confirm tag record](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-confirm%20tag%20record.webp?width=188&height=308&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-confirm%20tag%20record.webp)

Don't forget! All this data saves while you are off-line and out of range!

#### In Bulk via work order tasks

If you'd like to update vessel tags for specific vessels utilized within a specific task on a work order, you can easily find and update the tags via the lot/vessels input screen.  Tap the three dots to access the **Quick Menu** on the task.

All vessels previously scanned in or added to the task will automatically populate the Tag vessels screen.  You can edit these as needed (this updates the vessels included for the add tag action, not the work order task itself!) by tapping the blue pencil next to the vessel icons.

![InnoApp How to Add Remove or Replace Vessel Tags-via bulk work order](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-via%20bulk%20work%20order.webp?width=688&height=363&name=InnoApp%20How%20to%20Add%20Remove%20or%20Replace%20Vessel%20Tags-via%20bulk%20work%20order.webp)

These tag changes will be updated as soon as the "Record" button is tapped, regardless of task submission status.
