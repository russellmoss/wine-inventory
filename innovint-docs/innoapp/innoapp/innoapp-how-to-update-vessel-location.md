# InnoApp: How to update Vessel Location

InnoApp's quick, continuous scanning function makes it a great tool for updating or clearing your vessel locations.

Need to find out more about how InnoVint helps you manage Vessel Locations? Check out this [article](https://support.innovint.us/hc/en-us/vessel-location-tracking?hsLang=en) first to get started.

### How to update a vessel location

1. Navigate to the Vessel Details page
   ![InnoApp How to update Vessel location-details pg](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-details%20pg.webp?width=252&height=231&name=InnoApp%20How%20to%20update%20Vessel%20location-details%20pg.webp)
2. Click on the blue pencil next to the location field:
   ![InnoApp How to update Vessel location-blue pencil](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-blue%20pencil.webp?width=252&height=444&name=InnoApp%20How%20to%20update%20Vessel%20location-blue%20pencil.webp)
3. Select the location
   ![InnoApp How to update Vessel location-select location](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-select%20location.webp?width=259&height=281&name=InnoApp%20How%20to%20update%20Vessel%20location-select%20location.webp)
4. Record location
   ![InnoApp How to update Vessel location-record location](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-record%20location.webp?width=254&height=244&name=InnoApp%20How%20to%20update%20Vessel%20location-record%20location.webp)

NOTE: InnoApp does not yet support setting a vessel *position*.

### How to update vessel locations via work orders

If you'd like to update the vessel locations for specific vessels utilized within a specific task on a work order, you can easily find and update the locations via the lot/vessels input screen.  Tap the three dots to access the **Quick Menu** on the task.

![InnoApp How to update Vessel location-update via wo](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-update%20via%20wo.webp?width=688&height=315&name=InnoApp%20How%20to%20update%20Vessel%20location-update%20via%20wo.webp)

These location changes will be updated as soon as the "Record" button is tapped, regardless of task submission status.

### How to update vessel locations in bulk

1. Access this function via your Quick Actions menu (scroll towards the bottom of the home page):
   ![InnoApp How to update Vessel location-bulk](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-bulk.webp?width=203&height=269&name=InnoApp%20How%20to%20update%20Vessel%20location-bulk.webp)
2. Select "**Scan Vessels**" to begin scanning your vessels:
   ![InnoApp How to update Vessel location-scan](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-scan.webp?width=204&height=317&name=InnoApp%20How%20to%20update%20Vessel%20location-scan.webp)
3. You can scan one QR Code after another, continuously. InnoVint will never record a duplicate QR code - you will see "Already Selected" if you scan the same code twice. Click "**Done**" after scanning all barrels:
   ![InnoApp How to update Vessel location-cont scan](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-cont%20scan.webp?width=208&height=288&name=InnoApp%20How%20to%20update%20Vessel%20location-cont%20scan.webp)
4. Confirm the total number of vessels and their contents.
   - You can continue scanning additional vessels by clicking on "Add More," to the upper right (in blue).
   - If you need to start over, click on "Deselect All", at the top right of the screen.
   - When you're happy, click on "**Select Vessels**":
     ![InnoApp How to update Vessel location-select](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-select.webp?width=204&height=246&name=InnoApp%20How%20to%20update%20Vessel%20location-select.webp)
5. Select the new vessel location in a few clicks:
    a. ![InnoApp How to update Vessel location-select location2](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-select%20location2.webp?width=202&height=253&name=InnoApp%20How%20to%20update%20Vessel%20location-select%20location2.webp)  b. ![InnoApp How to update Vessel location-b](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-b.webp?width=167&height=251&name=InnoApp%20How%20to%20update%20Vessel%20location-b.webp)   c. ![InnoApp How to update Vessel location-c](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-c.webp?width=144&height=254&name=InnoApp%20How%20to%20update%20Vessel%20location-c.webp)
6. Confirm your selections and record!
   ![InnoApp How to update Vessel location-record](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-record.webp?width=202&height=271&name=InnoApp%20How%20to%20update%20Vessel%20location-record.webp)
7. View your new vessel locations throughout InnoVint:
   *Via Lot Details / Vessels*
   ![InnoApp How to update Vessel location-dash vessels](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-dash%20vessels.webp?width=688&height=179&name=InnoApp%20How%20to%20update%20Vessel%20location-dash%20vessels.webp)
   *In the Vessel Explorer*
   ![InnoApp How to update Vessel location-explorer](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-explorer.webp?width=688&height=134&name=InnoApp%20How%20to%20update%20Vessel%20location-explorer.webp)
   *See the location on an individual vessel in the app*
   ![InnoApp How to update Vessel location-app](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-app.webp?width=206&height=249&name=InnoApp%20How%20to%20update%20Vessel%20location-app.webp)

### How to clear vessel locations in bulk

Follow the steps above through 5b. Instead of selecting a location, click on "Clear Location", and then "Record" to save the update.

![InnoApp How to update Vessel location-clear location](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-clear%20location.webp?width=185&height=305&name=InnoApp%20How%20to%20update%20Vessel%20location-clear%20location.webp)  ![InnoApp How to update Vessel location-clear record](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-clear%20record.webp?width=211&height=305&name=InnoApp%20How%20to%20update%20Vessel%20location-clear%20record.webp)

All selected vessels will clear their current locations.

If, for any reason, you want to see the past location of a vessel, you can do that in the Vessel History / Vessel location history.

![InnoApp How to update Vessel location-location history](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20update%20Vessel%20location-location%20history.webp?width=688&height=185&name=InnoApp%20How%20to%20update%20Vessel%20location-location%20history.webp)Read more [here](https://support.innovint.us/hc/en-us/vessel-location-tracking?hsLang=en#history).

### FAQ

**Q: Can I set the vessel position on the mobile app?**

*A: No, InnoApp currently only allows you to set the location, not a specific position (what you would most likely use to track individual barrels and/or racks).  You can set a vessel position in the main desktop browser interface via the individual vessel's details page, or the bulk Manage Vessels menu. Find out how [here](https://support.innovint.us/hc/en-us/vessel-location-tracking?hsLang=en#trackandupdate).*

**Q: Can I update locations offline?**

*A: Yes! InnoApp stores information offline until you are back in range of a signal. Then location data will be uploaded to the cloud, and throughout your InnoVint apps.*
