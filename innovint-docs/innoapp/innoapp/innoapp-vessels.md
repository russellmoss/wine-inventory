# InnoApp: Vessels

This article covers:

[InnoApp: Vessel Explorer](#explorer)

[InnoApp: Vessel Details](#Details)

#### InnoApp: Vessel Explorer

- InnoApp's Vessel Explorer is the entry point for finding any vessels you have in the system. The entire vessel list displays numerically for scrolling, and displays the vessel code, lot and lot fill (if it has contents), and capacity (if empty).
  ![innoapp vessels-explorer](https://support.innovint.us/hs-fs/hubfs/innoapp%20vessels-explorer.webp?width=187&height=341&name=innoapp%20vessels-explorer.webp)
- You will also see status flags in the InnoApp Vessel Explorer if a barrel is partial or a vessel is archived (archived vessels are still available in the explorer, sorted at the bottom).
  ![innoapp vessels-status flag](https://support.innovint.us/hs-fs/hubfs/innoapp%20vessels-status%20flag.webp?width=192&height=88&name=innoapp%20vessels-status%20flag.webp)
- From the InnoApp Vessel Explorer, vessels are searchable by entering any text into the search field. You can search by vessel type (you must type in the entire type, i.e. "tirage" not "tir", or "tank" not "tan"), vessel code, or even lot code.
- Find your empty vessels using our "magic" search, which will pull up empty vessels by typing "MT" "Empty" or a combination of either in conjunction with the vessel type ("MT keg" "Empty tank").
  ![innoapp vessels-vessel fill](https://support.innovint.us/hs-fs/hubfs/innoapp%20vessels-vessel%20fill.webp?width=197&height=167&name=innoapp%20vessels-vessel%20fill.webp)

#### InnoApp: Vessel Details

By clicking on one of the vessels on the InnoApp Vessel Explorer, you are taken to the **Vessel Details Page**, which gives more detailed information on that particular vessel.

From this page, you can see:  the vessel code, the lot contained in the vessel, the volume in that vessel, vessel attributes such as Cooper and Toast and Capacity, vessel location (if one has been set), tags, owner tags (if relevant), any current analyses (with a link to All Analyses), and the history associated with the vessel.

If you have a [dip chart](/hc/en-us/articles/360050058652-managing-and-using-dip-charts-in-innovint?hsLang=en) associated with the vessel, the current dip will also display next to the volume.

You can update the vessel location, record actions, and add & view notes here.

![innoapp vessels-vessel detail](https://support.innovint.us/hs-fs/hubfs/innoapp%20vessels-vessel%20detail.webp?width=192&height=336&name=innoapp%20vessels-vessel%20detail.webp)

![innoapp vessels-history](https://support.innovint.us/hs-fs/hubfs/innoapp%20vessels-history.webp?width=194&height=306&name=innoapp%20vessels-history.webp)

TIP: Use the Record Action button to add or remove a tag on a single vessel without using the scanner. You can also add notes either via Record Action button, or in the bottom Notes section.

Don't forget! All of this data can be recorded while offline, and it will update in the cloud once you are back in range. Full vessels details are available offline for [recently accessed or favorited items](https://support.innovint.us/hc/en-us/mobile-app-navigation-tips?hsLang=en#Favs). Remember to favorite any Vessels you want to access while offline!
