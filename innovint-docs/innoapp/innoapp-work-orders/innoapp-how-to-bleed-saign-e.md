# InnoApp: How to Bleed/Saignée

This article explains how to complete and submit a Bleed/Saignée tasks in InnoApp:

1. Select your work order and tap into the Task details:
   ![InnoApp How to Bleed Saignee-task](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bleed%20Saignee-task.webp?width=251&height=402&name=InnoApp%20How%20to%20Bleed%20Saignee-task.webp)
2. From the Task details page, you will be able to view the lot summaries for the "**From**" and "**To**" lots by tapping anywhere on the lots. The Task details page will show the lots, the number of selected involved vessels (or the vessel code if it is a single vessel), the requested Bleed Amount, as well as total requested "**To**" lot volume change.
   ![InnoApp How to Bleed Saignee-from to lots](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bleed%20Saignee-from%20to%20lots.webp?width=245&height=396&name=InnoApp%20How%20to%20Bleed%20Saignee-from%20to%20lots.webp)
3. The "**From"** Lot details will display the current weight of the bleed lot, and offer the ability to check off the bleed vessels via tapping or scanning. You may not edit lots, lot weights, or vessels for the "**From**" lot.
   ![InnoApp How to Bleed Saignee-vessels via scan](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bleed%20Saignee-vessels%20via%20scan.webp?width=253&height=410&name=InnoApp%20How%20to%20Bleed%20Saignee-vessels%20via%20scan.webp)
4. The "**To**" Lot details will allow you to edit volumes by tapping on the "Ending fill" fields in the Lot details screen. You can also scan to check off, or add more vessels here, using the Quick menu at the bottom right:
     ![InnoApp How to Bleed Saignee-ending fill](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bleed%20Saignee-ending%20fill.webp?width=254&height=416&name=InnoApp%20How%20to%20Bleed%20Saignee-ending%20fill.webp)
   1. White highlighted fields are available for edit, or tap the blue "Ending Fill" to find your bulk update options for multiple vessels.
   2. Tap to the left of the vessel icons to check off vessels as they are filled.
   3. Tap on the three dots to see your **Quick Menu** options to add additional vessels via search or scanning - or to scan to check off vessels.
   4. When you are done updating lots, hit the back arrow to return to the Task details page.
5. After updating all lot and vessel volumes, go back to the Task details screen to tap **Complete Task.**
   **![InnoApp How to Bleed Saignee-submit task](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bleed%20Saignee-submit%20task.webp?width=248&height=405&name=InnoApp%20How%20to%20Bleed%20Saignee-submit%20task.webp)**
6. If permissions are available, and the task is Complete, you will be able to Submit the task (or work order) as of the desired time.

#### FAQ

**Q. I don't see open fields to update the weight or volume - how do I update the weight or volume on a vessel?**

*A. Make sure you have clicked into the Lot details screen.  If you do not have any white highlighted vessel volume fields to update, please check to see if the task is already completed.  You will not be able to update volumes after the task is completed.*

*If you are trying to edit the weight on the bleed from lot, you cannot edit the weight on a Bleed/Saignée task.*

**Q. How can I unstart or uncomplete a task?**

*A. Use the "**More**" menu, found on the **Task details** screen. You will be able to SKIP a task if it is unstarted, to UNSTART a task if it is started, or to UNCOMPLETE a task, if it is completed. You will not see the **More** menu here after task submission.*

**Q. Can I edit the Requested Bleed amount?**

*A. InnoApp does not support editing the bleed amount at this time. To edit this volume, please edit the work order via the desktop app.*
