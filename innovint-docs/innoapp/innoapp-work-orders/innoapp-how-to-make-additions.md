# InnoApp: How to make Additions

This article explains how to complete and submit Addition tasks in InnoApp:

1. Select your work order and tap into the Task details:
   ![How to make Additions-select wo](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-select%20wo.webp?width=170&height=308&name=How%20to%20make%20Additions-select%20wo.webp)
2. From the Task details page, you will be able to view involved additives and lots, and start the task:
   ![How to make Additions-task detail pg](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-task%20detail%20pg.webp?width=160&height=300&name=How%20to%20make%20Additions-task%20detail%20pg.webp)
3. Tap into each product to:
   1. View the Additive product(s), addition rate and instructions. You *cannot* edit products, add rates, or instructions in InnoApp.
   2. View, add or edit the Additive batches:
      - Use "**Add Batch**" to select available batches.
      - Use the "**More**" menu at the bottom left to remove unwanted batches.
      - Edit the amount removed from the batch or choose to fully deplete the batch.
      ![How to make Additions-add batch](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-add%20batch.webp?width=186&height=346&name=How%20to%20make%20Additions-add%20batch.webp)  ![How to make Additions-remove batch](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-remove%20batch.webp?width=187&height=348&name=How%20to%20make%20Additions-remove%20batch.webp)  ![How to make Additions-additive](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-additive.webp?width=187&height=347&name=How%20to%20make%20Additions-additive.webp)
   3. Tap into a lot to view the requested vessels and add amounts:
      - Edit the quantity added per vessel.
      - Check off vessels as adds are made.
      ![How to make Additions-edit amount add](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-edit%20amount%20add.webp?width=180&height=334&name=How%20to%20make%20Additions-edit%20amount%20add.webp)  ![How to make Additions-check vessels](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-check%20vessels.webp?width=179&height=331&name=How%20to%20make%20Additions-check%20vessels.webp)

The displayed addition values in InnoApp are equal to what is populated in desktop "Added" field, and can be edited per vessel in the app.

![How to make Additions-add per vessel](https://support.innovint.us/hs-fs/hubfs/How%20to%20make%20Additions-add%20per%20vessel.webp?width=468&height=130&name=How%20to%20make%20Additions-add%20per%20vessel.webp)

#### FAQ

**Q. I don't see open fields to update the quantity - how do I update quantity added?**

*A. Make sure you have clicked into the Lot details screen.  If you do not have any white highlighted fields to update, please check to see if the task is already completed.  You will not be able to update volumes after the task is completed.*

**Q. How can I unstart or uncomplete a task?**

*A. Use the "**More**" menu, found on the **Task details** screen. You will be able to SKIP a task if it is unstarted, to UNSTART a task if it is started, or to UNCOMPLETE a task, if it is completed. You will not see the **More** menu here after task submission.*

**Q. Can I add another lot or another additive to the task?**

*A. InnoApp does not support adding additional lots or additives at this time. To add lots, or additional additives, please edit the work order via the desktop app.*
