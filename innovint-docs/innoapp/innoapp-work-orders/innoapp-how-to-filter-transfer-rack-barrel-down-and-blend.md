# InnoApp: How to Filter, Transfer, Rack, Barrel Down, and Blend

This article explains how to complete and submit common movement tasks in InnoApp:

- [Filter, Transfer, Rack and Barrel Down](#transfer)
- [Blend](#blend)
- [FAQ](#FAQ)

#### How to Filter, Transfer, Rack and Barrel Down (movements from one lot ➡️ one or many lots)

1. Select your work order and tap into the Task details
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-select wo bbl down](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-select%20wo%20bbl%20down.webp?width=198&height=362&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-select%20wo%20bbl%20down.webp)
2. From the Task details page, you will be able to edit lot vessels and volumes on the "**From**" and "**To**" lots by tapping anywhere on the lots. The lot summary will show the total requested lot volume change and the number of selected involved vessels (or the vessel code if it is a single vessel):
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-from to](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-from%20to.webp?width=197&height=359&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-from%20to.webp)
3. Edit volumes by tapping on the "Ending fill" fields in the Lot details screen. You can also scan to check off, or add more vessels here, using the **Quick menu** at the bottom right:
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-edit volumes](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-edit%20volumes.webp?width=436&height=406&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-edit%20volumes.webp)
   1. White highlighted fields are available for edit, or tap the blue "Ending Fill" to find your bulk update options for multiple vessels.
   2. Tap to the left of the vessel icons to check off vessels as they are filled.
   3. Tap on the three dots to see your **Quick Menu** options to add additional vessels via search or scanning - or to scan to check off vessels.
   4. When you are done updating lots, hit the back arrow to return to the Task details page.
4. After updating all lot and vessel volumes, go back to the Task details screen to tap **Complete Task.**
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-complete bbl down](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-complete%20bbl%20down.webp?width=196&height=361&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-complete%20bbl%20down.webp)
5. If permissions are available, and the task is Complete, you will be able to Submit the task (or work order) as of the desired time.

#### Blend tasks (movements from one or many lots ➡️ one lot)

1. Select your work order and tap into the Task details:
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-blend task](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-blend%20task.webp?width=196&height=360&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-blend%20task.webp)
2. From the Task details page, you will be able to edit lot vessels and volumes on the "From" and "To" lots by tapping anywhere on the lots. The lot summary will show the total requested lot volume change and the number of selected involved vessels (or the vessel code if it is a single vessel):
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-blend from to](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-blend%20from%20to.webp?width=192&height=356&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-blend%20from%20to.webp)
3. Edit volumes by either tapping on the white highlighted fields to individually edit volumes, or by using the blue "Remove"/"Add"/"Ending Fill" column headers to bulk update the vessel volumes.

   Blend tasks will open either "Remove"/"Add" fields OR the "Ending Fill" fields, based on the original work order request. Find out more about this option to select "Remove" or "Ending Fill" volumes [here](https://support.innovint.us/hc/en-us/articles/204742525-how-to-record-a-blend-action?hsLang=en#relative).
   ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-from bbls](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-from%20bbls.webp?width=191&height=353&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-from%20bbls.webp)  ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-from tk](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-from%20tk.webp?width=190&height=354&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-from%20tk.webp)  ![InnoApp How to Filter, Transfer, Rack, Barrel Down, and Blend-to tk](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-to%20tk.webp?width=189&height=348&name=InnoApp%20How%20to%20Filter%2c%20Transfer%2c%20Rack%2c%20Barrel%20Down%2c%20and%20Blend-to%20tk.webp)
4. In order to check off vessels, tap to the left of the vessel icons to check off vessels as they are emptied or filled, or click on the three dots to access the **Quick Menu** and find the "scan to check off" option.
5. If you need to add additional vessels to the task, click on the three dots to access the **Quick Menu** in order to either search for, or scan, additional vessels.
6. Update all lot and vessel volumes, and hit the back arrow to return to the Task details page. Then tap **Complete Task.**
7. If permissions are available, and the task is Complete, you will be able to Submit the task as of the desired time, either from the Task details page, or from the Work Order page (if all tasks on the work order are complete).

#### FAQ

**Q. I don't see open fields to update the volume - how do I update the volume on a vessel?**

*A. Make sure you have clicked into the Lot details screen.  If you do not have any white highlighted vessel volume fields to update, please check to see if the task is already completed.  You will not be able to update volumes after the task is completed.*

**Q. How can I unstart or uncomplete a task?**

*A. Use the "**More**" menu, found on the **Task details** screen. You will be able to SKIP a task if it is unstarted, to UNSTART a task if it is started, or to UNCOMPLETE a task, if it is completed. You will not see the **More** menu here after task submission.*

**Q. Can I add another lot to the blend or transfer tasks?**

*A. InnoApp does not support adding additional lots at this time. To add lots, including Lees lots, please edit the work order via the desktop app.*
