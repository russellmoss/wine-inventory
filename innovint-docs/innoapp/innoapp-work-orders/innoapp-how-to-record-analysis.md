1. [How to record Lot Composite or Individual Vessel Analysis](#lot)
   1. [Via Quick Actions](#Quick_action_lot) ([Record Analysis](#recordanalysis) or [Sugar/Temps](#Bs_Ts))
   2. [Via Lot or Vessel Details page](#Action_lot)

InnoVint's mobile InnoApp is a powerful tool for recording analysis on the go: in the cellar, in the cave, and in the vineyard.

The app allows analysis data to be recorded and saved offline for upload to the cloud when you are back in range of a signal.

### How to record Lot Composite or Individual Vessel Analysis

#### Via Quick Actions

Jump straight into analysis entry from the Quick Actions menu with either the "Record Analysis" or the "Record Sugar (Brix) & Temp" option.

![InnoApp How to Record Analysis-brix and temp](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-brix%20and%20temp.webp?width=217&height=256&name=InnoApp%20How%20to%20Record%20Analysis-brix%20and%20temp.webp)

**Option 1. Record Analysis**

This option allows you to record analysis against Lots (Lot Composite or Individual vessel) or Vineyard blocks.

1. Select subject of analysis: Lot.
   ![InnoApp How to Record Analysis-subject](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-subject.webp?width=208&height=332&name=InnoApp%20How%20to%20Record%20Analysis-subject.webp)
2. To record lot analysis, select your lot and choose whether to record the analysis as Lot Composite or Individual vessel type analysis.
   The Individual vessel type analysis selection will prompt a vessel selection if the lot is contained in more than one vessel.  Otherwise, it will default to a single vessel.
   ![InnoApp How to Record Analysis-comp vs indv](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-comp%20vs%20indv.webp?width=201&height=300&name=InnoApp%20How%20to%20Record%20Analysis-comp%20vs%20indv.webp)  ![InnoApp How to Record Analysis-lot comp](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-lot%20comp.webp?width=201&height=301&name=InnoApp%20How%20to%20Record%20Analysis-lot%20comp.webp)  ![InnoApp How to Record Analysis-indv vessel](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-indv%20vessel.webp?width=167&height=299&name=InnoApp%20How%20to%20Record%20Analysis-indv%20vessel.webp)
3. Select any existing analysis panel "+ Add Panel", or "Select Analysis."
   If you choose "Select Analysis," commonly used analyses pre-populate the screen, but use the search icon to find any existing analysis type in InnoVint.
   ![InnoApp How to Record Analysis-add panel](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-add%20panel.webp?width=176&height=314&name=InnoApp%20How%20to%20Record%20Analysis-add%20panel.webp)  ![InnoApp How to Record Analysis-search](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-search.webp?width=176&height=314&name=InnoApp%20How%20to%20Record%20Analysis-search.webp)
4. Input analysis. Tap on the units to toggle through and change to any available units for the analysis type.
   ![InnoApp How to Record Analysis-units](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-units.webp?width=185&height=298&name=InnoApp%20How%20to%20Record%20Analysis-units.webp)  ![InnoApp How to Record Analysis-units ppm](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-units%20ppm.webp?width=166&height=298&name=InnoApp%20How%20to%20Record%20Analysis-units%20ppm.webp)
5. Click **Record** to save analysis in the system.

Like everything else in this app, this works offline! Analysis results will be queued and saved until they can be uploaded to the cloud for access anywhere on the InnoVint platform.

**Option 2. Record Sugar (Brix/Baume) & Temp**

This option pre-populates the analysis type with your default sugar unit and temperature analysis.

1. Select your lot and choose whether to record the analysis as Lot Composite or Individual vessel type analysis. The Individual vessel type analysis selection will prompt a vessel selection if the lot is contained in more than one vessel.
   ![InnoApp How to Record Analysis-brix lot comp](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-brix%20lot%20comp.webp?width=167&height=298&name=InnoApp%20How%20to%20Record%20Analysis-brix%20lot%20comp.webp)
2. Input your analysis results. Use the +/- icon to generate a negative result, and click on the units to toggle through and change available units on an analysis type.
   ![InnoApp How to Record Analysis-input brix and temp](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-input%20brix%20and%20temp.webp?width=170&height=305&name=InnoApp%20How%20to%20Record%20Analysis-input%20brix%20and%20temp.webp)  ![InnoApp How to Record Analysis-pos neg brix](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-pos%20neg%20brix.webp?width=172&height=306&name=InnoApp%20How%20to%20Record%20Analysis-pos%20neg%20brix.webp)
3. If you want to add an additional analysis type to your sugar/temps, just click on "Edit Analyses" to update the action.
4. Click **Record** to save analysis in the system.

#### Record Analysis Via Lot Details or Vessel Details

Use the Record Action button from within either a Lot or Vessel Details page.

![InnoApp How to Record Analysis-record button](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20button.webp?width=346&height=251&name=InnoApp%20How%20to%20Record%20Analysis-record%20button.webp)

1. Select Analysis from the Record Action menu.
   ![InnoApp How to Record Analysis-menu analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-menu%20analysis.webp?width=170&height=303&name=InnoApp%20How%20to%20Record%20Analysis-menu%20analysis.webp)
2. Select either Lot Composite or Individual vessel type analysis.
   ![InnoApp How to Record Analysis-select run on](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-select%20run%20on.webp?width=369&height=304&name=InnoApp%20How%20to%20Record%20Analysis-select%20run%20on.webp)
    If you access the Analysis action via the Lot Details page, it will default to Lot Composite type analysis.

   If you access the action via the Vessel Details page, it will default to Individual vessel type analysis, and default to the selected vessel.

   In either case, you can choose to update selection as desired.
3. Select any existing analysis panel "+ Add Panel", or "Select Analysis."
   If you choose "Select Analysis," some commonly used analyses pre-populate the screen, but use the search icon to find any existing analysis type in InnoVint.
   ![InnoApp How to Record Analysis-add panel2](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-add%20panel2.webp?width=170&height=303&name=InnoApp%20How%20to%20Record%20Analysis-add%20panel2.webp)
4. Input your analysis results. Use the +/- icon to generate a negative result, and click on the units to toggle through and change available units on an analysis type.
5. Click **Record** to save analysis in the system.

#### Record Analysis Via Work Order Tasks

1. Tap into the analysis task details. You will see the analysis type (Lot Composite or Individual Vessel) and number of involved lots summarized at the top.
   ![InnoApp How to Record Analysis-record via wo](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20via%20wo.webp?width=178&height=332&name=InnoApp%20How%20to%20Record%20Analysis-record%20via%20wo.webp)
2. Enter the results in the available fields.  Tap on the units to toggle between available units.
    ![InnoApp How to Record Analysis-fso2 unit](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-fso2%20unit.webp?width=180&height=273&name=InnoApp%20How%20to%20Record%20Analysis-fso2%20unit.webp)
3. Complete task, and submit.
   ![InnoApp How to Record Analysis-complete task](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-complete%20task.webp?width=166&height=306&name=InnoApp%20How%20to%20Record%20Analysis-complete%20task.webp)  ![InnoApp How to Record Analysis-submit](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-submit.webp?width=162&height=303&name=InnoApp%20How%20to%20Record%20Analysis-submit.webp)

You cannot add analysis or lots via InnoApp.

### How to record Vineyard Block Analysis

#### Via Quick Actions

Jump straight into analysis entry from the Quick Actions menu using "Record Analysis".

![InnoApp How to Record Analysis-quick action](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-quick%20action.webp?width=203&height=259&name=InnoApp%20How%20to%20Record%20Analysis-quick%20action.webp)

This option allows you to record analysis against either Vineyard blocks or Lots.

1. Select subject of analysis: Vineyard
   ![InnoApp How to Record Analysis-vineyard](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-vineyard.webp?width=172&height=275&name=InnoApp%20How%20to%20Record%20Analysis-vineyard.webp)
2. Select your vineyard and block. The block selection is dependent on the vineyard selection.
   ![InnoApp How to Record Analysis-record vineyard](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20vineyard.webp?width=198&height=281&name=InnoApp%20How%20to%20Record%20Analysis-record%20vineyard.webp)  ![InnoApp How to Record Analysis-record block](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20block.webp?width=199&height=282&name=InnoApp%20How%20to%20Record%20Analysis-record%20block.webp)
3. Select any existing analysis panel, or ad hoc analysis. Commonly used analyses populate the screen if you choose "Select Analyses", but use the search icon to find any existing analysis type in InnoVint.
   ![InnoApp How to Record Analysis-common analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-common%20analysis.webp?width=160&height=283&name=InnoApp%20How%20to%20Record%20Analysis-common%20analysis.webp)  ![InnoApp How to Record Analysis-vineyard analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-vineyard%20analysis.webp?width=160&height=284&name=InnoApp%20How%20to%20Record%20Analysis-vineyard%20analysis.webp)
4. Input analysis. Tap on the units to toggle through and change to any available units on an analysis type.
   ![InnoApp How to Record Analysis-input vineyard analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-input%20vineyard%20analysis.webp?width=171&height=304&name=InnoApp%20How%20to%20Record%20Analysis-input%20vineyard%20analysis.webp)
5. Click **Record** to save analysis in the system.

#### Record Analysis Via Vineyard Details or Block Details

1. Use the Record Action button from within either the Vineyard Details page, or the Block Details page.
   ![InnoApp How to Record Analysis-record via vineyard or block details](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20via%20vineyard%20or%20block%20details.webp?width=498&height=250&name=InnoApp%20How%20to%20Record%20Analysis-record%20via%20vineyard%20or%20block%20details.webp)
2. Select Analysis from the Record Action menu.
   ![InnoApp How to Record Analysis-record action menu](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20action%20menu.webp?width=206&height=226&name=InnoApp%20How%20to%20Record%20Analysis-record%20action%20menu.webp)
   If you access the Analysis action via the Vineyard Details page, it will default to that Vineyard, and you still need to specify the block.
   If you access the action via the Block Details page, it will automatically populate both Vineyard and Block.
   In either case, you can choose to edit the selection with the blue pencil.
   ![InnoApp How to Record Analysis-vineyard2](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-vineyard2.webp?width=164&height=233&name=InnoApp%20How%20to%20Record%20Analysis-vineyard2.webp)  or ![InnoApp How to Record Analysis-record block2](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record%20block2.webp?width=165&height=235&name=InnoApp%20How%20to%20Record%20Analysis-record%20block2.webp)
3. Select any existing analysis panel, or ad hoc analysis. Commonly used analyses populate the screen if you choose "Select Analyses", but use the search icon to find any existing analysis type in InnoVint.
   ![InnoApp How to Record Analysis-select common analysis](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-select%20common%20analysis.webp?width=162&height=287&name=InnoApp%20How%20to%20Record%20Analysis-select%20common%20analysis.webp)  ![InnoApp How to Record Analysis-record](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Record%20Analysis-record.webp?width=162&height=287&name=InnoApp%20How%20to%20Record%20Analysis-record.webp)
4. Click **Record** to save analysis in the system.

Like everything else in this app, all of this data entry works offline! Analysis results will be queued and saved until they can be uploaded to the cloud for access anywhere on the InnoVint platform.
