# InnoApp: Bottle en Tirage

This article explains how to complete and submit a Bottle en Tirage task in InnoApp.

#### How to Bottle en Tirage

1. Select your work order and tap into the Task details.
   ![InnoApp_BeT task](https://support.innovint.us/hs-fs/hubfs/InnoApp_BeT%20task.webp?width=250&height=513&name=InnoApp_BeT%20task.webp)
2. From the Task details screen, you will be able to Start the task, and edit lot vessels and volumes on the "**From**" and "**To**" lots by tapping anywhere on the lots.
   ![InnoApp_BeT details](https://support.innovint.us/hs-fs/hubfs/InnoApp_BeT%20details.webp?width=250&height=515&name=InnoApp_BeT%20details.webp)
3. Edit volumes by tapping on the "**Ending fill**" field in the desired Lot details screen. The "From" lot will deplete in volume; the "To" lot shows its fill in bottles.
   You can also scan to update vessel tags/locations, check off vessels, or add more vessels here, using the **Quick menu** at the bottom right:
     ![InnoApp_BeT_Complete WO](https://support.innovint.us/hs-fs/hubfs/InnoApp_BeT_Complete%20WO.webp?width=250&height=509&name=InnoApp_BeT_Complete%20WO.webp)
4. Going back to the task details screen will convert the bottles to gallons to help you see gains and losses.
5. If you have entered packaging at Work Order creation, **you are not currently able to update packaging used on InnoApp.**
   ![InnoApp_BeT Packaging](https://support.innovint.us/hs-fs/hubfs/InnoApp_BeT%20Packaging.png?width=250&height=506&name=InnoApp_BeT%20Packaging.png)
6. If packaging is requested and outstanding, you will need to update packaging and submit the final task *in the desktop app*. ![InnoApp_BeT desktop app packaging](https://support.innovint.us/hs-fs/hubfs/InnoApp_BeT%20desktop%20app%20packaging.webp?width=688&height=229&name=InnoApp_BeT%20desktop%20app%20packaging.webp)
7. If all packaging has been updated (quantities entered for all used and scrap fields) on the task via the desktop app, or if no packaging has been requested, then you will be able to submit the task on InnoApp.

   If packaging was requested and has quantities outstanding, then you will receive an error message if you try to submit.

   ![InnoApp_BeT error](https://support.innovint.us/hs-fs/hubfs/InnoApp_BeT%20error.webp?width=250&height=478&name=InnoApp_BeT%20error.webp)
