# InnoApp: How to Bottle

This article explains how to complete and submit a Bottle task in InnoApp.

- [How to Bottle](#bottle)
- [FAQ](#faq)

#### How to Bottle

1. Select your work order and tap into the Task details.
   ![InnoApp How to Bottle-select task](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-select%20task.webp?width=184&height=342&name=InnoApp%20How%20to%20Bottle-select%20task.webp)
2. From the Task details screen, you will be able to Start the task, and edit lot vessels and volumes on the "**From**" lot by tapping anywhere on the lot.

   Edit volumes by tapping on the "**Ending fill**" fields in the Lot details screen. You can also scan to check off, or add more vessels here, using the **Quick menu** at the bottom right:
   ![InnoApp How to Bottle-from lot](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-from%20lot.webp?width=178&height=331&name=InnoApp%20How%20to%20Bottle-from%20lot.webp)  ![InnoApp How to Bottle-vessels select end fill](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-vessels%20select%20end%20fill.webp?width=182&height=337&name=InnoApp%20How%20to%20Bottle-vessels%20select%20end%20fill.webp)

   **CASE GOODS:** For those with the Case Goods module, from the Task details page, you will be able to see the bottling format and update the bottling outputs on the "**To**" lot by tapping anywhere on the case goods lot.
   ![InnoApp How to Bottle-case goods](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-case%20goods.webp?width=176&height=330&name=InnoApp%20How%20to%20Bottle-case%20goods.webp)  ![InnoApp How to Bottle-case good details](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-case%20good%20details.webp?width=196&height=329&name=InnoApp%20How%20to%20Bottle-case%20good%20details.webp)
   If you have entered packaging at Work Order creation, **you are not currently able to update packaging used on InnoApp.**  If packaging is requested, you will need to update packaging and submit the final task *in the desktop app*.
   ![InnoApp How to Bottle-use desktop for cg](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-use%20desktop%20for%20cg.webp?width=688&height=154&name=InnoApp%20How%20to%20Bottle-use%20desktop%20for%20cg.webp)
   You will see this message alerting you in the InnoApp Lot details screen:
   ![InnoApp How to Bottle-alert](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-alert.webp?width=194&height=336&name=InnoApp%20How%20to%20Bottle-alert.webp)
   **NO CASE GOODS:** For those without the Case Goods module, you will see a slightly different layout, with the formats specified in the task details page. Tap into each format to adjust the number of pallets/cases and bottles and finalize the task.
   ![InnoApp How to Bottle-bottle without cg](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-bottle%20without%20cg.webp?width=209&height=328&name=InnoApp%20How%20to%20Bottle-bottle%20without%20cg.webp)  ![InnoApp How to Bottle-format](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-format.webp?width=214&height=331&name=InnoApp%20How%20to%20Bottle-format.webp)

   If you have entered packaging at Work Order creation, **you are not currently able to update packaging used on InnoApp.**  If packaging is requested, you will need to update packaging and submit the final task *in the desktop app*.

   After you update the bottled output, hit the back arrow to return to the Task details page.
3. Ensure you have updated all lot and vessel volumes, and the bottled output, and then tap **Complete Task.**
   **![InnoApp How to Bottle-complete task](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-complete%20task.webp?width=178&height=330&name=InnoApp%20How%20to%20Bottle-complete%20task.webp)**
4. If all packaging has been updated (quantities entered for all used and scrap fields) on the task via the desktop app, or if no packaging is requested, then you will be able to submit the task on InnoApp. You will see a message like this one:
   ![InnoApp How to Bottle-message about submitting](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-message%20about%20submitting.webp?width=226&height=352&name=InnoApp%20How%20to%20Bottle-message%20about%20submitting.webp)

   If packaging was requested and has quantities outstanding:
   ![InnoApp How to Bottle-quantities outstanding](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-quantities%20outstanding.webp?width=688&height=69&name=InnoApp%20How%20to%20Bottle-quantities%20outstanding.webp) Then, you will see a message that packaging must be entered on the desktop app *prior to submission*, and also receive an error message if you try to submit.
   ![InnoApp How to Bottle-enter on desktop prior to submit](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-enter%20on%20desktop%20prior%20to%20submit.webp?width=218&height=350&name=InnoApp%20How%20to%20Bottle-enter%20on%20desktop%20prior%20to%20submit.webp)  ![InnoApp How to Bottle-error message](https://support.innovint.us/hs-fs/hubfs/InnoApp%20How%20to%20Bottle-error%20message.webp?width=250&height=352&name=InnoApp%20How%20to%20Bottle-error%20message.webp)

#### FAQ

**Q. I don't see open fields to update the volume or case numbers - how do I update the volume on a vessel or update my cases?**

*A. Make sure you have clicked into the Lot details screen.  If you do not have any white highlighted fields to update, please check to see if the task is already completed.  You will not be able to update volumes after the task is completed.*

**Q. How can I unstart or uncomplete a task?**

*A. Use the "**More**" menu, found on the **Task details** screen. You will be able to SKIP a task if it is unstarted, to UNSTART a task if it is started, or to UNCOMPLETE a task, if it is completed. You will not see the "**More"** menu here after task submission.*

**Q. Can I add another bottling lot or bottle format?**

*A. InnoApp does not support adding additional lots or formats at this time.*
