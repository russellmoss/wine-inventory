# 3/31/2026 Product Updates: MAKE & SUPPLY

## MAKE gets a new Lot Explorer & Custom Lot Attributes, InnoApp work order improvements, and the SUPPLY-C7 integration is even better!

### MAKE

#### New Lot Explorer

![image-png-Mar-16-2026-11-12-12-8278-PM](https://support.innovint.us/hs-fs/hubfs/image-png-Mar-16-2026-11-12-12-8278-PM.png?width=670&height=266&name=image-png-Mar-16-2026-11-12-12-8278-PM.png)
We've updated the original Lot Explorer to do more and support you better. It now has a fresh look and more powerful functionality. Our goal was to make the table easier to scan, reduce visual clutter, and enable more personalized workflows when working with your lots.

- Choose exactly which columns appear, reorder them, resize them, and pin important columns to the left
- Control which filters appear in the explorer, and their default settings
- Each user can save multiple table layouts so you don’t have to reconfigure columns and filters

Get the nitty gritty in [this article](/hc/en-us/lot-explorer?hsLang=en).

#### Custom Lot Attributes

Custom Lot Attributes allow you to create and manage new custom data fields on your juice/wine lots while also being able to reference these attributes within the Lot details page, Lot Explorer, Custom Reports & more locations. This enables you to incorporate your unique attributes directly into workflows and reporting. Find out more [here](/hc/en-us/custom-attributes-lots?hsLang=en)!

![image-png-Mar-16-2026-10-59-53-5836-PM](https://support.innovint.us/hs-fs/hubfs/image-png-Mar-16-2026-10-59-53-5836-PM.png?width=453&height=357&name=image-png-Mar-16-2026-10-59-53-5836-PM.png)

Custom Lot Attributes are available for MAKE Plus subscriptions only.

#### Appellation Rollup improvements!

Not bottling a vineyard designate?  Our [Appellation Rollup](/hc/en-us/appellation-roll-up?hsLang=en) now includes all California, Oregon and Washington State appellations so you can see the aggregated lot composition across smaller but overlapping geographic groupings! InnoVint will now show the total percentage that "rolls up" - into a larger appellation.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Mar-30-2026-06-39-44-1017-PM.png?width=670&height=129&name=image-png-Mar-30-2026-06-39-44-1017-PM.png)

#### Analysis Import Changes

- Our "Modified" template now supports the same date/time format as the "Original" template, better supporting you when recording analyses performed at different times in the same day. Time stamps are supported in DD/MM/YY 00:00 format (24 hr time).
  ![](https://support.innovint.us/hs-fs/hubfs/image-png-Mar-27-2026-11-31-11-4342-PM.png?width=670&height=102&name=image-png-Mar-27-2026-11-31-11-4342-PM.png)
- Analysis Import (regular and AI) now allows you to edit or entirely remove lots & vessels after uploading a file for analysis.

#### More options for easier use!

- For metric accounts

  - We've added two more metric addition rates: ml/tonne and g/tonne
  - We now fully support our [Transfer Volume to Weight action](/hc/en-us/transfer-volume-to-weight?hsLang=en) for metric units (Bleed/Saignee actions also now update your expected yield properly!)
- Looking for something? We've made it easier to filter for your received fruit in the Fruit Intake Report - date filters are now available!![](https://support.innovint.us/hs-fs/hubfs/image-png-Mar-31-2026-08-50-35-9147-PM.png?width=670&height=368&name=image-png-Mar-31-2026-08-50-35-9147-PM.png)
- **Transfer (Inter-facility)** has been renamed **B2B Transfer (Inter-facility)** to help clarify the intended and best function of this action.  Learn more about [it here!](/hc/en-us/bond-to-bond-b2b?hsLang=en)

### FINANCE

We added the lot volume total to the cost item action lot picker, and the bottom of the lot list! You need to know that you are spreading cost across the right volume. You can now see the total gallons of lots selected in the Lot Picker, and when reviewing your added costs, letting you know if you have the right lots selected or if you are reconciling against the wrong volume/lots.

![Lot Cost Picker_Total](https://support.innovint.us/hs-fs/hubfs/Lot%20Cost%20Picker_Total.png?width=436&height=357&name=Lot%20Cost%20Picker_Total.png)

### InnoApp

- It's always harvest for someone!  We've added support for the [Transfer Volume to Weight](/hc/en-us/transfer-volume-to-weight?hsLang=en) task to InnoApp.
- More quality control:

  - You can now **require** that vessels are checked off in InnoApp prior to starting work order tasks, or prior to work order task completion.
  - You can require that vessels are scanned to check off (if you have our QR codes) or leave the option to tap or scan.

    Find these settings in your desktop app Settings / InnoApp work orders. Learn more [here](/hc/en-us/wo-overview#settings)!

  ![Settings_InnoApp Checkoff](https://support.innovint.us/hs-fs/hubfs/Settings_InnoApp%20Checkoff.png?width=670&height=355&name=Settings_InnoApp%20Checkoff.png)

### SUPPLY

#### More flexibility!

You can now Archive or Unarchive SKUs, and even delete them! Find out when you can delete, and when you can archive [in this article](/hc/en-us/how-to-archive-or-delete-skus?hsLang=en).

#### Increased functionality for the Commerce7 integration

- SUPPLY now displays the quantity of your C7 SKU's allocated inventory.

  ![SUPPLY-C7_on-order_C7 allocated](https://support.innovint.us/hs-fs/hubfs/SUPPLY-C7_on-order_C7%20allocated.png?width=670&height=122&name=SUPPLY-C7_on-order_C7%20allocated.png)

  You'll see this called out and rolled up into your SKU's Inventory Availability widget, and visible in all of the other "on order" columns in your explorers and pickers.

  ![SUPPLY_C7 on-order](https://support.innovint.us/hs-fs/hubfs/SUPPLY_C7%20on-order.png?width=670&height=242&name=SUPPLY_C7%20on-order.png)
- We now have C7 Order number visibility in SUPPLY!  Orders fulfilled in Commerce7 will now surface their C7 order number on the corresponding SUPPLY Depletion action. This order number will be visible for all synched transactions following the release:it is not retroactive. Click onto the order number to link directly to the order in Commerce7!

  ![SUPPLY_C7 order number](https://support.innovint.us/hs-fs/hubfs/SUPPLY_C7%20order%20number.png?width=670&height=293&name=SUPPLY_C7%20order%20number.png)
- Commerce7 inventory locations can now link to SUPPLY "in-bond" locations.  For in-bond inventory, depending on the transaction type that occurs in C7, you may need to manually update the compliance reason on the action that is generated in SUPPLY.
