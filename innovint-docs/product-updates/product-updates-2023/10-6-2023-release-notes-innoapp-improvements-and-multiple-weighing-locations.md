# 10/6/2023 Release Notes: InnoApp Improvements and Multiple Weighing Locations

Release Notes through October 6, 2023 include:

### Features

#### InnoVint's original iOS work order app has been officially retired!

We've said it before, and we'll say it again: this app has been removed from the Apple Store, and is no longer supported.  Check out [InnoApp](https://support.innovint.us/hc/en-us/highlights-of-the-new-mobile-experience?hsLang=en) today!

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-05-2023-11-29-07-9903-PM.png?width=345&height=325&name=image-png-Oct-05-2023-11-29-07-9903-PM.png)

#### Multiple weighing locations are now available!

If you have multiple scales onsite, or run more than one facility via a single InnoVint winery account, InnoVint now supports multiple weighing locations!

Manage multiple scales with respective sequential weigh tags easily via your Weigh tag settings and Receive Fruit actions or tasks. More details on available settings are found [here](https://support.innovint.us/hc/en-us/articles/115003779846-activating-the-advanced-receive-fruit-option-to-generate-weigh-tags?hsLang=en#MWL). Weighing locations are also included on Fruit Cost Report and Fruit Intake Report.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-05-2023-11-34-13-0531-PM.png?width=329&height=200&name=image-png-Oct-05-2023-11-34-13-0531-PM.png)

This capability requires prior activation of [Advanced Receive Fruit](https://support.innovint.us/hc/en-us/articles/115003779846-activating-the-advanced-receive-fruit-option-to-generate-weigh-tags?hsLang=en) and an appropriate subscription level. Reach out to [InnoVint Support](mailto:support@innovint.us) to check your subscription and get started!

### Improvements

**We've been working on InnoApp! You can now do and see more while in app. ![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-09-2023-08-06-19-7406-PM.png?width=166&height=300&name=image-png-Oct-09-2023-08-06-19-7406-PM.png)**

- We've made it easier to [add vessels to a lot](https://support.innovint.us/hc/en-us/beta?hsLang=en#Choosevessels): tapping on **Add vessels** now defaults to your scanner for quicker vessel input.
- You can now easily [reassign work orders](https://support.innovint.us/hc/en-us/beta?hsLang=en#navigate) from the Work Order details page.
- InnoApp now supports viewing All Analysis for both lots and vessels (while online).
- Need help with the scanner? Check out our troubleshooting page [here](https://support.innovint.us/hc/en-us/innoapp-troubleshooting-your-scanner?hsLang=en).
- Don't forget to keep your phone's browser up to date!

**Improved TankNet Integration data efficiency**

Our TankNet integration records temperature analysis data if it has been >24 hours since InnoVint last recorded a reading for the tank (ensuring a minimum of one reading per day), or if there is a 1+ degree difference from the last reading.

To prevent information overload on your Analysis tab, analysis is now received at a maximum of  every 15 min.

Find out more about our TankNet integration [here.](https://support.innovint.us/hc/en-us/overview-tanknet-integration?hsLang=en)

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-09-2023-08-26-38-6834-PM.png?width=402&height=250&name=image-png-Oct-09-2023-08-26-38-6834-PM.png)

**ETS barcode improvement**

We've increased the resolution for bar code generation on your ETS labels to improve scan ability and reduce issues with printer capabilities.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-09-2023-08-29-28-8567-PM.png?width=280&height=150&name=image-png-Oct-09-2023-08-29-28-8567-PM.png)

**We added two more Lab Sources!**

For our Mexican cohort, we've added Enologia Integral.  And for our phenolics geeks, you can now clearly mark your WineXRay results and imports.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-09-2023-08-37-49-9710-PM.png?width=229&height=100&name=image-png-Oct-09-2023-08-37-49-9710-PM.png)   ![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-09-2023-08-50-10-4124-PM.png?width=429&height=127&name=image-png-Oct-09-2023-08-50-10-4124-PM.png)

### Bugs

- Did you get stuck needing to insert an extra column to do the math? The Cost Audit Report export had been reporting volumes in equivalent gallons for our metric wineries. Volumes are now correctly reporting in Liters or Hectoliters.
- You can now create a lot without composition - most helpful for our wineries who prefer to set up lot codes ahead of time for work orders!
- Did you have troubles exporting a Lot Cost Report using multiple filters? You can now filter and export at will!
