# 6/2/2023 Release Notes: Save "camera" angles on Tank Maps & more metric support!

### Improvements

#### Save your favorite Tank Map views with Camera settings!

We know that zooming and rotating to the same vantage point every time you login isn't very convenient.

Now, you can use the **Camera settings** menu to save multiple preferred views (the rotation aspect and Zoom level are saved), and to set a default that will load every time you login or refresh the Tank Map browser. Find out more [here](https://support.innovint.us/hc/en-us/tank-maps?hsLang=en#Save_View).

![](https://support.innovint.us/hs-fs/hubfs/image-png-Jun-07-2023-11-05-12-0544-PM.png?width=261&height=178&name=image-png-Jun-07-2023-11-05-12-0544-PM.png)

#### Expected yields are now accurate for accounts in metric

Previously, all volume and weight units defaulted to the same yield: 150 [volume unit]/weight unit, but InnoVint now defaults to yields that are based on your winery specified volume and weight units. Find out more about yield settings [here](https://support.innovint.us/hc/en-us/articles/115001094151-harvest-settings-receive-fruit-options-and-expected-yield-?hsLang=en#settingexpyield).

Complete list of supported default yield settings:

- - - 150 gal/ton
    - 165 gal/tonne
    - 0.165 gal/kg
    - 570 liters/ton
    - 628 liters/tonne
    - 0.628 liters/kg
    - 5.7 hl/ton
    - 6.28 hl/tonne
    - 0.00628 hl/kg

### Bug Fixes

- Lots created using Bulk Components will now work in Rack and Return tasks and actions.
- The **All Analyses in a List** export is working again.
