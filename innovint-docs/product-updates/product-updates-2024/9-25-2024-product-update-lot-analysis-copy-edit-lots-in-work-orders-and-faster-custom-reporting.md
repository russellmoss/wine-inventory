# 9/25/2024 Product Update: Lot analysis copy, edit lots in work orders, and faster custom reporting

A few mighty improvements have recently been released! Product updates through September 25, 2024 include:

#### Analyses will follow volume into empty lots!

Juice and wine move around, and it is important that your analysis does too!

Now, when a supported movement action is performed and the filled lot is either a new lot or a lot with no contents, the most recent lot composite analysis value for each unique analysis on the "parent" lot will automatically flow through to the fill lot(s).
![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-25-2024-08-05-41-7584-PM.png?width=688&height=468&name=image-png-Sep-25-2024-08-05-41-7584-PM.png)

There are some nuances around editing and deleting movements and analyses, so be sure to learn all the details of how this works in the [support article](https://support.innovint.us/hc/en-us/lot-analysis-copy?hsLang=en)!

#### You can now change lots in open work order tasks!

We know that things change and mistakes get made. Now, after work order creation (but before work order submission!), you can change the lots included in most work order tasks.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-19-2024-09-00-11-5793-PM.png?width=688&height=292&name=image-png-Sep-19-2024-09-00-11-5793-PM.png)

![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-19-2024-09-02-55-7185-PM.png?width=688&height=295&name=image-png-Sep-19-2024-09-02-55-7185-PM.png)![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-19-2024-09-04-52-6579-PM.png?width=688&height=294&name=image-png-Sep-19-2024-09-04-52-6579-PM.png)

Updated tasks are:  B2B Transfer In/Out, Transfer (Inter Facility), Custom tasks (which include fermentation actions, stir and riddling),  Bleed/saignée, Transfer Volume to weight and Drain & Press, Blend, Transfer/Rack/Filter/Barrel Down, Rack & Return, Top Off, Bottling en Tirage, Bottle (including now adding additional formats and case good lots).

Find out more about how to edit work orders [here](https://support.innovint.us/hc/en-us/articles/206371326-how-to-edit-a-work-order?hsLang=en), and how to edit submitted work orders and actions [here](https://support.innovint.us/hc/en-us/articles/208141233-how-to-edit-or-delete-recorded-actions?hsLang=en)!

#### Custom Reports data updates faster!

Just finished entering your morning brix & temp? Running monthly analysis, and want to see the new trendlines? All of our custom reports now update data in near real time.  Most data will update on a row within 30 seconds of entry.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-20-2024-06-47-20-2767-PM.png?width=688&height=134&name=image-png-Sep-20-2024-06-47-20-2767-PM.png)

- There is a new *Refreshed At* column that you can add to your reports that shows when each row was last updated
- Updates typically occur at 30 seconds, but may take a bit longer
- You will need to refresh the page to see these updates
- This also applies to our State Compliance by Bond, ML Fermentation and Stability & Aging reports!

Learn about how to use Custom Reports [here](https://support.innovint.us/hc/en-us/custom-reports?hsLang=en).

#### Reducing compliance errors!

When you are busy, the tax class field can be an easy thing to miss, and the wrong tax class can cause trouble down the road! Now, when creating a new lot via the in-line slideover within actions or tasks, the tax class field on the new lot will default to your primary drained lot on the action.
![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-25-2024-09-01-56-4723-PM.png?width=688&height=353&name=image-png-Sep-25-2024-09-01-56-4723-PM.png)

#### Contract Explorer: Ability to Filter by Block

We added another one! Too many contracts to scroll through? Once you populate the vineyard filter, you will be able to select (and multi-select) specific blocks for your view.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Sep-20-2024-06-54-25-4578-PM.png?width=688&height=139&name=image-png-Sep-20-2024-06-54-25-4578-PM.png)

#### Improvements to User Log-in Challenges

As you know, at the end of July, we released some security improvements to how users log-in and are authenticated by the system.  In some cases, depending on browsers, devices and usage patterns, this has caused issues for some users.

We have released several changes intended to fix these edge cases, and are currently monitoring the situation.  If you are experiencing any trouble at log-in, please check the following:

1. Ensure you're using a Google Chrome browser (or Safari on an iPad)
2. Check that your device's operating system, and the browser, are up to date
3. Empty your browser cache; clear the Google Chrome Cache by following the steps [here](https://www.loom.com/share/caed8d898ba64f3f89d168343b016b17?sid=ccf4a2b3-a6ee-4ff1-a37c-d470e476b04f)
4. If none of these help, please reach out to [support@innovint.us](mailto:support@innovint.us) with your device and browser details, and a screenshot of any error message, if possible.
