# 2/23/2024 Product Update: InnoApp dip charts, VinWizard and more!

Release Notes through February 23, 2024 include:

### Features

#### VinWizard integration now available!

Stop updating two systems! Easily map your InnoVint vessel codes to VinWizard tanks, and push your InnoVint lot contents and stage or tags into VinWizard.  Check out the details [here](https://support.innovint.us/hc/en-us/overview-vinwizard-integration?hsLang=en)!

![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-12-35-49-3653-AM.png?width=650&height=226&name=image-png-Feb-22-2024-12-35-49-3653-AM.png)

### Improvements

#### New lot contents filter in Add/Remove Cost Lot Picker

For our COGS Tracking users, we've added a new lot contents filter in the Add/Remove Cost lot picker to enable you to remove cost from empty lots.  Previously, you could not access these empty lots in order to remove the cost.

- The default selection is '*Hide empty lots*', where the picker will only show lots with contents at your selected point in time. Now you can also: *Include empty lots*, or *Show only empty lots*.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-08-55-14-2268-PM.png?width=650&height=166&name=image-png-Feb-22-2024-08-55-14-2268-PM.png)

Find out more about adding and removing cost [here](https://support.innovint.us/hc/en-us/allocate-costs-indirect-direct?hsLang=en#costitem).

#### We've released more improvements to InnoApp!

- Dip Charts are now available! Use the Ending Fill header in each task in order to display dip chart fields for any vessels that have dip charts. Find out more about using dip charts [here](https://support.innovint.us/hc/en-us/wo-overview?hsLang=en#dips).
  ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-07-10-29-3173-PM.png?width=215&height=400&name=image-png-Feb-22-2024-07-10-29-3173-PM.png)   ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-07-12-36-8755-PM.png?width=225&height=400&name=image-png-Feb-22-2024-07-12-36-8755-PM.png)
- It's here! You can now remove vessels from most InnoApp tasks. Use the Quick Menu at the bottom right of the screen ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-07-05-02-1599-PM.png?width=21&height=19&name=image-png-Feb-22-2024-07-05-02-1599-PM.png)  to find your remove vessels option.![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-07-00-23-2599-PM.png?width=217&height=400&name=image-png-Feb-22-2024-07-00-23-2599-PM.png)

- For user with this feature, Transfer (Inter-Facility) task completion and submission is now supported in InnoApp. Find out more about Transfer (Inter-Facility) [here](https://support.innovint.us/hc/en-us/bond-to-bond-b2b?hsLang=en#B2B_IV).
  ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-08-48-43-9979-PM.png?width=215&height=248&name=image-png-Feb-22-2024-08-48-43-9979-PM.png)
- Offline improvement: view **All analysis** for both favorited and recently viewed lot & vessel analyses. The most recent 30 results will display offline, although you will need internet access to see additional results.
  ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-07-15-06-3733-PM.png?width=217&height=400&name=image-png-Feb-22-2024-07-15-06-3733-PM.png)
- Brazilian Portuguese is now a supported language - just scroll to the bottom of the InnoApp home screen to find your translation options. This selection is saved by user and device, so all your available wineries will update with your chosen language.
  ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-22-2024-07-07-48-1001-PM.png?width=215&height=275&name=image-png-Feb-22-2024-07-07-48-1001-PM.png)

#### The Report Explorer search function is improved:

You can now search all your reports by report description, as well as report name. Happy hunting!

![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-23-2024-11-19-47-4277-PM.png?width=600&height=212&name=image-png-Feb-23-2024-11-19-47-4277-PM.png)

### Bugs

- Not all Vineyard Events were displaying on the Phenology Report as expected.. but they will now record as expected in time for the growing season!
- An oldie but a goodie - some users may be used to working around the edit capabilities for the Topping action (the ending fill input field would not display for edit), but we've got that fixed up for you!
  ![](https://support.innovint.us/hs-fs/hubfs/image-png-Feb-24-2024-12-16-14-4602-AM.png?width=600&height=176&name=image-png-Feb-24-2024-12-16-14-4602-AM.png)
- For our metric friends - the Production Losses Across All Wineries Report was displaying gallons instead of liters - but now no more math is required. All volumes display in the correct units, with units in a separate column for any users working with both gallons and liters (or litres or hectolitres, etc).
