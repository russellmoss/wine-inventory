# 9/17/19 - Barrel Fermentation Worksheet and other improvements!

The Software Release on 9/17/19 includes:

### New Features:

- ### [Small Vessel Fermentation Worksheet](//innovint-6865708.hs-sites.com/hc/en-us/articles/360015385351-using-the-fermentation-worksheets-aka-ferm-gen-?hsLang=en)

We've released a new companion worksheet to the OG Ferm Gen! Now you can manage your barrel fermentations from a single interface. Quickly view the current status of your barrel lots and assign analysis, addition, and fermentation management tasks to your crew.

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360038837971Screen_Shot_2019-09-18_at_11.40.02_AM.png)

### Improvements:

- Create work orders from the Vineyard Explorer and Harvest Launchpad

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360038913451Screen_Shot_2019-09-19_at_8.36.35_AM.png?width=287&name=support.innovint.ushcarticle_attachments360038913451Screen_Shot_2019-09-19_at_8.36.35_AM.png)

- New addition rate for additives: *lb/ton*
- Note icon added in Analysis tabs
- Quicker loading when switching between Analysis tabs
- Owner filter added to Custom Winemaking Reports (for accounts with Custom Crush permissions activated)

### Bug fixes:

- Vessel capacity rounding error in Topping actions has been fixed
- Fix for filters and sorting in Fruit Intake and Analysis reports
