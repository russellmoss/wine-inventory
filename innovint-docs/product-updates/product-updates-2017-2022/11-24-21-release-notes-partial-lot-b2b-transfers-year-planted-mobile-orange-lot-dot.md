# 11/24/21 Release Notes: Partial Lot B2B Transfers, Year Planted, Mobile Orange Lot Dot

The software releases for November 24, 2021 include:

### Improvements

#### Partial Lot B2B Transfer to Another InnoVint Winery

Users can now perform a B2B transfer of part of a vessel in a single lot to another InnoVint winery. Just be sure to leave volume in your "Ending Fill" field, select the winery and bond you are transferring to, and backdate if needed.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Nov-17-2021-05-01-08-56-PM.png?width=516&name=image-png-Nov-17-2021-05-01-08-56-PM.png)

![](https://support.innovint.us/hs-fs/hubfs/image-png-Nov-17-2021-05-01-20-51-PM.png?width=517&name=image-png-Nov-17-2021-05-01-20-51-PM.png)

#### "Year Planted" Field in Vineyard Block Creation

Users can now input a "year planted" when creating a new block in a vineyard. This is an optional field.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Nov-17-2021-04-40-45-29-PM.png?width=600&name=image-png-Nov-17-2021-04-40-45-29-PM.png)

#### Orange Lot Dot Color in New Mobile Experience

The orange lot dot color is now available on the new mobile experience. All colors for both still and sparkling wines are now available.

![](https://support.innovint.us/hs-fs/hubfs/Screen%20Shot%202021-11-17%20at%208-52-02%20AM-png.png?width=232&name=Screen%20Shot%202021-11-17%20at%208-52-02%20AM-png.png)

- Multi-Winery Lot Overview Report
