# 06/09/20 Release Notes - Ability to Edit Topping Wine, Lot Cost Report Date Filter, Archived Lot Dot Color, and More!

The software release on June 09, 2020 includes:

### Improvements:

- The lot dot changes to a muted color when the lot is Archived, making it easy to spot archived lots in your *Lot Explorer,* *Winery Activity Feed,* and other reports!
  ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360073370651Screen_Shot_2020-10-18_at_9.55.58_PM.png?width=251&name=support.innovint.ushcarticle_attachments360073370651Screen_Shot_2020-10-18_at_9.55.58_PM.png)
- You can now edit a Topping action to change the topping wine
- We added a date filter to the *Lot Cost Report*

### Bug fixes:

- Lot "Archived" tag re-instated ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360059543592Screen_Shot_2020-06-15_at_12.48.00_AM.png?width=67&name=support.innovint.ushcarticle_attachments360059543592Screen_Shot_2020-06-15_at_12.48.00_AM.png)
- Additive batch code now displays correctly in Addition actions
