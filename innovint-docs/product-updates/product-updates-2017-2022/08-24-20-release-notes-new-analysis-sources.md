# 08/24/20 Release Notes - New Analysis Sources

The software release on August 24, 2020 includes:

### Improvements:

- Analysis Source
  BevTrac and Genesys have been added to the analysis source dropdown. 🔬
  ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360072865211Screen_Shot_2020-10-14_at_2.55.41_PM.png?width=120&name=support.innovint.ushcarticle_attachments360072865211Screen_Shot_2020-10-14_at_2.55.41_PM.png)

### Bug fixes:

- Searching for Vineyards by Owner is now operating correctly (for accounts with *Custom Crush Permissions*)
- Lot label printing for sparkling lots is fixed 📠
- Vessels that are filled above capacity are now considered full
