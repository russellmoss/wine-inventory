# 11/02/20 Release Notes - Improved Custom Vineyard Report Filter

The software releases for the week of November 2, 2020 include:

### Improvements:

### Custom Vineyard Report

The Block Name filter now searches for Vineyard or Block (for accounts with *Advanced Vineyard Tracking*).

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360075602092Screen_Shot_2020-11-06_at_10_09_23_AM.png?width=639&name=support.innovint.ushcarticle_attachments360075602092Screen_Shot_2020-11-06_at_10_09_23_AM.png)

### Bug Fixes:

- Lot stage change no longer required to create a Bottling work order
- Fix to Dip Charts radio button
- B2B Transfers respect clones more accurately
- Fix for cost change column in the Cost Audit Report
- *ML Fermentation* and *Stability & Aging* reports display the last 3 analyses
- *+Add Lot* button returned to Direct Actions
