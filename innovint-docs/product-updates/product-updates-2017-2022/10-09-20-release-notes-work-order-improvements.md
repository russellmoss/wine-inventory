# 10/09/20 Release Notes - Work Order Improvements

The software release on October 9, 2020 includes:

### Improvements:

#### Work Order Templates

In accounts with *Custom Crush Permissions* enabled, Work Order Templates now display this confirmation message if no owner is selected upon template creation.

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360072860611Screen_Shot_2020-10-14_at_1.45.58_PM.png?width=450&name=support.innovint.ushcarticle_attachments360072860611Screen_Shot_2020-10-14_at_1.45.58_PM.png)

### Bug fixes:

- New vineyards and blocks now auto-populate in the dropdown during lot creation 🍇
- The ability to *skip* an Analysis task in a Work Order when the lot has changed vessels
- Rack and Return task in work orders
- Sorting for Additions in the Winery Activity Feed now only displays *Additions*
- On-hand amount displays correctly after fully depleting an additive
