# 10/06/20 Release Notes - New Highlight Colors in Fermentation Worksheets

### Improvements:

#### Fermentation Worksheets

- Each user now has their own highlight color in Fermentation Worksheets (aka the *Ferm Gen*) !![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360072862271Screen_Shot_2020-10-14_at_2.12.20_PM.png)
- The ability to *show* or *hide* Lot name has been added to Fermentation Worksheets

### Bug fixes:

- Ability to remove Addition tasks from the Ferm Gen
- Sparkling actions removed from Fruit lots
- Fruit cost worksheet now auto-refreshes (for clients with *Costing* activated) 🍇
- Stage updates upon submission of Bottling work order
