# 9/25/19 Release Notes - Bottling report improvements and bug fixes

The software release on 9/25/19 includes:

### New Feature

- New Bottling Report export option: Export by formats

If multiple format types are bottled in the same Bottling action, this new export will separate each format type into individual rows. This export also includes the optional text fields for glass, closure, etc.

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360039407211Screen_Shot_2019-09-25_at_5_27_37_PM.png)

### Improvements

- Bottling exports: Lot code and lot name are now listed in separate columns

### Bug Fixes

- Vessel filter in Primary Fermentation Report is fixed
- Fix for partial lot code search in Winery Activity Feed
