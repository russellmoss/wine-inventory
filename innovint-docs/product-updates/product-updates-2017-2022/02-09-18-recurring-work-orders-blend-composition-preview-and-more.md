# 02/09/18 - Recurring Work Orders, Blend Composition Preview, and more

The software release on 2/9/18 includes:

- New Features
  - [Recurring work orders](#recurring-work-orders)
  - [Blend Composition Preview](#blend-composition)
  - [Partial vessel filter](#partial-vessel-filter)
  - [Rosé lot dot color](#rose-lot-dot)
- [Improvements](#improvements)
- [Bug Fixes](#bug-fixes)

**New Features:**

- **Recurring work orders -** InnoVint now allows you to create multiple work orders to recur in the future at intervals of your choosing. [Click here](//innovint-6865708.hs-sites.com/hc/en-us/articles/360000515492-how-to-create-recurring-work-orders?hsLang=en) for more information on how to utilize this feature.

  ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360000892731Screen_Shot_2018-02-08_at_8.12.32_PM.png?width=516&name=support.innovint.ushcarticle_attachments360000892731Screen_Shot_2018-02-08_at_8.12.32_PM.png)

- **Blend Composition Preview -**The composition of a new blend can now be previewed now on any movement action prior to submission.

  ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360000892751Screen_Shot_2018-02-08_at_8.09.51_PM.png?width=439&name=support.innovint.ushcarticle_attachments360000892751Screen_Shot_2018-02-08_at_8.09.51_PM.png)

- **Partial vessel filter -**We've improved a filter on the Vessel Explorer by renaming it from 'LOT' to 'CONTENTS' and added **Partial** as an option. Selecting Partial allows you to view all of your vessels that are not filled to capacity.

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360000889172Screen_Shot_2018-02-08_at_8.26.40_PM.png?width=429&name=support.innovint.ushcarticle_attachments360000889172Screen_Shot_2018-02-08_at_8.26.40_PM.png)

- **Rosé lot dot color** - We've finally embraced that rosé is neither a red nor a white wine by introducing a pink lot dot color! As long as the [lot name](//innovint-6865708.hs-sites.com/hc/en-us/articles/207711473-changing-lot-code-and-lot-name?hsLang=en) contains the word rosé (including the accent) the lot dot color will be pink.

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360000889212Screen_Shot_2018-02-08_at_8.36.31_PM.png?width=290&name=support.innovint.ushcarticle_attachments360000889212Screen_Shot_2018-02-08_at_8.36.31_PM.png)

**Improvements**:

- Added 'Source' to Analysis Action
- Changed column header that previously read 'Contents' to 'Fill' across the App. For clarity, this indicates the amount of volume or weight in that vessel.
- Added 'ETS Labs' as a Source in the analysis import tool
- Added 'Owners' column to Lot Inventory at Point in Time report export
- Custom reports now include Tags
- Hide work orders details on Vessel Detail Page

**Bug Fixes**

- Notes save on addition and analysis tasks
- Timestamp duplication in tax class change window
