# 06/28/20 Release Notes - New Bottled Costs Report, Improvements to the Winery Activity Feed, and More!

The software release on June 28, 2020 includes:

### New Feature:

#### Bottled Costs Report

See all bottlings and associated costs across cost categories! (for accounts with *Costing* activated)

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360073102612Screen_Shot_2020-10-18_at_10_14_58_PM.png?width=282&name=support.innovint.ushcarticle_attachments360073102612Screen_Shot_2020-10-18_at_10_14_58_PM.png)

### Improvements:

- The *Winery Activity Feed* now clarifies between Bond to Bond transfers *in* vs *out*
- Tank codes added to the *Lot Explorer* export

### Bug fixes:

- The InnoVint grape icon is back in working order. It spins and changes the header color again 🍇
- The *Bottled* stage has been returned to the Lot details page and Bottling actions
- Vineyard Explorer name filter fixed
- Add lot with more than 10 components fixed
