# 10/22/21 Release Notes: Owner tags in Dry Goods Selector & Locations in B2Bs

The software releases for October 22, 2021 include:

### New Features

#### Owner Visibility in Dry Goods Selector

Wineries with Custom Crush Permissions can now see which additives and packaging are assigned to a specific owner, are global, or have no owner tags.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-22-2021-06-03-12-56-PM.png?width=494&name=image-png-Oct-22-2021-06-03-12-56-PM.png)

### Improvements

#### **Locations on Bond to Bond Transfers**

When recording a B2B Transfer In or B2B Transfer Out, you can now add a "From" or "To"  location.  Learn more about adding and managing Locations [here](https://support.innovint.us/hc/en-us/locations?hsLang=en). Additionally, if you select a location, this location information will pre-populate your Bill of Lading (if generated).

![](https://support.innovint.us/hubfs/image-png-Oct-22-2021-06-04-18-14-PM.png)

#### Vineyard Explorer Tag Filter

The Vineyard Explorer now contains a Tag Filter, where you can search for Tags on both Vineyards and Blocks. This is helpful to classify vineyards that are Organic, need a certain style of farming, or have shown signs of disease.

![](https://support.innovint.us/hubfs/image-png-Oct-21-2021-05-51-08-01-PM.png)

If tags are applied only to blocks, they will appear on the Block Details Page, but not on the Vineyard Explorer.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Oct-21-2021-05-51-31-33-PM.png?width=238&name=image-png-Oct-21-2021-05-51-31-33-PM.png)
