# 01/24/20 Release Notes - Improvements to Addition action, editing barrel attributes, and reporting

The software release on January 24 2020 includes:

### Improvements:

- [Addition action and task improvements](//innovint-6865708.hs-sites.com/hc/en-us/articles/204321939-how-to-record-an-addition-additive-batch-tracker-add-on-?hsLang=en)

Some changes include multi-select lots, bulk select vessels across all lots, bulk apply additive amount across all selected vessels, and work order task editing. Click [HERE](https://www.loom.com/share/065202fdce93445cab88952bd64b0f24) to learn more.

- [Bulk edit year first used for existing barrels](//innovint-6865708.hs-sites.com/hc/en-us/articles/206240653-change-or-edit-vessel-details?hsLang=en#bulk)

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360048314991Screen_Shot_2020-01-27_at_3.26.21_PM.png)

- Fruit cost worksheet added for previous *and* current year

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360048188752Screen_Shot_2020-01-27_at_3.17.31_PM.png)

- ML Fermentation report

Analysis values for Malic Acid and Malic Acid - Chromatography are both included under recent malic readings. Lot tags have also been included.

- Exports

Analysis date added to the csv export for the Primary Fermentation and Bottling reports.
