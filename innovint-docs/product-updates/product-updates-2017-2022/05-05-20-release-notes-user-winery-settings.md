# 05/05/20 Release Notes - User Winery Settings

The software release on May 05, 2020 includes:

### New Feature:

- Now users with access to multiple InnoVint wineries can sort their winery dropdown menu *Chronologically by date added* or *Alphabetically.*Go to **Settings**, then **Preferences** to update.

### Bug fixes:

- *B2B to another InnoVint winery* correctly displays destination bond number 🚚
