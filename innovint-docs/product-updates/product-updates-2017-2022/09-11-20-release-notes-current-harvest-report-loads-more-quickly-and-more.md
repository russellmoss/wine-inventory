# 09/11/20 Release Notes - Current Harvest Report Loads More Quickly and More

The software release on September 11, 2020 includes:

### Bug fixes:

- Additives Explorer filters by Owner again (for accounts with *Custom Crush Permissions*)
- For accounts with *Sparkling* activated, the Ferm Gen now displays the "Sparkling Lot Dot" icon
  ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360072631732Screen_Shot_2020-10-14_at_2.42.39_PM.png?width=282&name=support.innovint.ushcarticle_attachments360072631732Screen_Shot_2020-10-14_at_2.42.39_PM.png)
- The Additives export now includes costs (for accounts with *Costing* enabled and additive costs entered) 💰
- Current Harvest Report is loading much faster (for accounts with *Advanced Vineyard Tracking* enabled)!
- The Current Harvest Report can now be *Exported 🌱*
