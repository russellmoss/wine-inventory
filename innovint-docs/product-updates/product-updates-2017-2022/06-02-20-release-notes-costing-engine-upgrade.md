# 06/02/20 Release Notes - Costing Engine Upgrade!

The software release on June 02, 2020 includes:

### Improvements:

- **Costing Engine Upgrade** **💲**This release includes a major upgrade to how the system distributes fruit costs from weight to volume as lots are split and blended through processing, juice bleeds and pressing. The system changes are thoroughly documented [here](//innovint-6865708.hs-sites.com/hc/en-us/articles/360044146111-system-rules-for-cost-distribution-in-innovint?hsLang=en).
