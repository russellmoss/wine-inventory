# 11/13/17 - Vintage Variability Report, Expanded Analysis Import, and more

The software release on 11/13/17 includes:

**New Features:**

- **Vintage Variability Report**: We've added a new report to your Winemaking Reports section. This report compares your estimated tonnage per block to actual tonnage received per block and reports on the % over/under for each vintage. It also predicts how many barrels you'll need and the estimated case count.

  ![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115003446532mceclip1.png)
- **Analysis Import**: We've taken the existing "FOSS analysis import" feature and overhauled it to accept a generic analysis upload as well as uploads branded for FOSS, myEnologist and Core Enology. There's also far more analyses and unit options to utilize! More information on the feature is located [here](//innovint-6865708.hs-sites.com/hc/en-us/articles/115002684812-analysis-import-format-guidelines-for-csv-file?hsLang=en).

  ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115003446552mceclip2.png?width=367&name=support.innovint.ushcarticle_attachments115003446552mceclip2.png)
- **Addition Export:**There is a new 'Export' link on your Additive History feed in the upper right hand corner. This export specifies the additive breakdown by lot and by batch.

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115003428351mceclip5.png)

- **Password Reset**- Forgot your password? You can reset it anytime by clicking the 'Forgot Password' link on the login page.

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115003428291mceclip4.png?width=454&name=support.innovint.ushcarticle_attachments115003428291mceclip4.png)

**Improvements**:

- Load current vessels for each lot being topped in an active topping task
- Custom Report selection is now a drop down and no longer runs across the page horizontally
- Zero values now populate on the Ferm Work Order Generator
- Lot selector field in actions and tasks refreshes every time a new page is loaded

**Bug Fixes**

- Recognizes system-set expected yield when calculating additive amounts for vessel contents in weight
