# 10/11/19 Release Notes - Receive Fruit task editing improvement, lot component volume, and bug fixes to the Ferm Gen

The software release on October 11, 2019 includes:

### Improvements:

- Exports of Lot Components now include the volume per component.

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360040588571Screen_Shot_2019-10-11_at_12_38_43_PM.png)

- Language in the Cost Over Time report has been changed to more accurately describe the lot status.

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360040589131Screen_Shot_2019-10-11_at_12.43.26_PM.png)

- Editing the arrival date in an open Receive Fruit work order will now automatically update the work order due date, and vice versa.

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360040589451Screen_Shot_2019-10-11_at_12_45_13_PM.png?width=594&name=support.innovint.ushcarticle_attachments360040589451Screen_Shot_2019-10-11_at_12_45_13_PM.png)![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360040585672Screen_Shot_2019-10-11_at_12.46.56_PM.png?width=487&name=support.innovint.ushcarticle_attachments360040585672Screen_Shot_2019-10-11_at_12.46.56_PM.png)

### Bug Fixes:

- Issue with 'freezing' in the Fermentation Worksheet for some accounts has been resolved
- Misspelling in error message has been fixed... although we kinda like our made-up word, *conflictioning* 😉
- The Barrel Fermentation Worksheet now displays the *number* of each vessel type on the far right, instead of the vessel code. This should now match the vessel count shown on the far left fo the worksheet.
