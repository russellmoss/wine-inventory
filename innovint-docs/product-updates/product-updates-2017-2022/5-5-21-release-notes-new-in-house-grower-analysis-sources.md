# 5/5/21 Release Notes - New In-House & Grower Analysis Sources!

The software releases for May 5, 2021 include:

### Improvements:

#### New Analysis Sources: In-House & Grower

Analysis source options of In-House and Grower are now available for both Analysis and Analysis Import actions. Additionally, we now default the analysis source to In-House.

![Screen_Shot_2021-05-05_at_8.25.49_AM.png](https://innovint.zendesk.com/hc/article_attachments/360094589372/Screen_Shot_2021-05-05_at_8.25.49_AM.png).     ![Screen_Shot_2021-05-05_at_8.31.35_AM.png](https://innovint.zendesk.com/hc/article_attachments/360094513511/Screen_Shot_2021-05-05_at_8.31.35_AM.png)

#### Vessel Explorer

You now have the ability to filter for *Full* vessels using the contents filter in the Vessel Explorer

![Screen_Shot_2021-05-05_at_8.44.52_AM.png](https://innovint.zendesk.com/hc/article_attachments/360094514711/Screen_Shot_2021-05-05_at_8.44.52_AM.png)

### Bug Fixes

- Vessel Explorer export
- Full note functionality restored
