# 7/01/21 Release Notes - New Permission Options for Editing Requested Volumes in a Work Order

### Improvements:

#### Work Orders

- New default: Admins and users that created a work order are the only users allowed to edit the *requested volumes* in an open work order.
  - To change this default go to the new options for Work Order Edit under *Settings*
- See more in our article [How to Edit a Work Order](//innovint-6865708.hs-sites.com/hc/en-us/articles/206371326-how-to-edit-a-work-order?hsLang=en). ![](https://support.innovint.us/hubfs/image-png-Jul-08-2021-08-40-37-15-PM.png)
