# 08/19/20 Release Notes - Winery Dashboard

The software release on August 19, 2020 includes:

### New Feature:

#### Winery Dashboard

The InnoVint Winery Dashboard enables winery members to understand the status of winery activity and inventory at a quick glance. This can help your team to plan and manage your day-to-day activities effectively. Check out more about the dashboard **[here](//innovint-6865708.hs-sites.com/hc/en-us/articles/360048244331-understanding-the-winery-dashboard?hsLang=en)**.
