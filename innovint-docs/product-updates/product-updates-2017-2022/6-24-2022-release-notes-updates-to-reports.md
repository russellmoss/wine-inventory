# 6/24/2022 Release Notes: Updates to Reports

The software releases for June 24, 2022 include:

### Some reports were getting out of hand. We fixed these up for you!

- ML Fermentation Report - ML Data not in reverse chronological order
- Number of open work orders too large for calendar display
- Custom Analysis Reports: analyses results that are 0 showing "-” instead of "0"
- Winery Activity Feed: Column Headers disappear after 9 Lots
- Multi-Winery Activity Feed not showing individual lot details and volumes
- Vineyard Explorer display showing excessive decimals for acreage
