# 10/10/18 - New Analysis Source Logos, Ferm Gen Printing, Editing Work Order Improvements, and more!

The software release on 10/10/18 includes:

### New features:

- New Analysis Source logos for [Baker Wine & Grape Analysis](https://www.bwga.net/) and Anton Paar!

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360013293471New_logo.png)

- Printable Ferm Gen!

Now you can print the Fermentation Work Order Generator in a single document (with or without selected analyses, additions, tasks, and/or assignees) to distribute in the cellar. And also create separate printouts for each assignee if needed!![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360013303031print_and_go__1_.png?width=631&name=support.innovint.ushcarticle_attachments360013303031print_and_go__1_.png)

- Other NEW! Ferm Gen features
  - [**Warning icon**](//innovint-6865708.hs-sites.com/hc/en-us/articles/360015385351-using-the-fermentation-worksheets-aka-ferm-gen-?hsLang=en#hoverable) : Hover over the icon for details of when the last fermentation action was recorded.
    ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360013587792warning_icon.png?width=501&name=support.innovint.ushcarticle_attachments360013587792warning_icon.png)
  - View most recent analysis data by value. Click "Show recent analysis values" to display the 3 most recent Brix and Temps readings with date and time, instead of showing the Brix and Temps sparklines.

### Improvements:

- Improvements to Harvest Work Order editing
  - **Receive Fruit task**: add and remove Fruit Lots from the work order
  - **Process Fruit task**: add and remove Fruit Lots to process and also lots to fill
  - **Addition task**: add and remove Additives from an addition (must have the [Additive Batch Tracker and Additions Calculator](//innovint-6865708.hs-sites.com/hc/en-us/articles/360000607792-feature-option-simple-additions-vs-additive-batch-tracking-calculator?hsLang=en) add-on activated)
- Now you can **Filter by Due date** in the Work Order Explorer![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360013307691work_order_due_date.png)
- Printed Weigh Tags (part of the [Advanced Receive Fruit option](//innovint-6865708.hs-sites.com/hc/en-us/articles/115003779846-activating-the-advanced-receive-fruit-option-to-generate-weigh-tags?hsLang=en)) are reconfigured to make more room for multiple fruit sources.
- Chose the wrong reason for a Volume Adjustment or Bond to Bond transfer? Now you can [edit the reason](//innovint-6865708.hs-sites.com/hc/en-us/articles/204178489-volume-adjustment?hsLang=en#faq) from the action details page.

### Bug fixes:

- Fruit lots are now visible in the lot selector for work order tasks. Previously hidden unless the filter was switched.
- Work Order icon display fix in the Vessel Explorer
- Starting costs in the Cost Over Time report now display as $0 if the start date of the report pre-dates lot creation and/or recorded costs.
