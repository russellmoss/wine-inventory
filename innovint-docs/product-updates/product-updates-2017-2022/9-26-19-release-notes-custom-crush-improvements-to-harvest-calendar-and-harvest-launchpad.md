# 9/26/19 Release Notes - Custom Crush Improvements to Harvest Calendar and Harvest Launchpad

The software release on 9/26/19 includes:

### Improvements

- Harvest calendar now hides vineyard name and ownership tag for users without access (*for accounts with Custom Crush permissions activated*)

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360039481911Screen_Shot_2019-09-26_at_3.45.22_PM.png?width=225&name=support.innovint.ushcarticle_attachments360039481911Screen_Shot_2019-09-26_at_3.45.22_PM.png)

- Harvest Launchpad only displays scheduled fruit that the user has access to via ownership tag (*for accounts with Custom Crush permissions activated*)

### Bug Fixes

- Fix for lot code search in Cost Item Report
