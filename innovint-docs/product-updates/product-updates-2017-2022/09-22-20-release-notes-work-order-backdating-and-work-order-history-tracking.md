# 09/22/20 Release Notes - Work Order Backdating and Work Order History Tracking

The software release on September 22, 2020 includes:

### New Features:

Overview Video:

#### Work Order Submission Options

Submit work order tasks with **3 new submission options**!

![](https://support.innovint.us/hubfs/image-png-Apr-30-2021-06-39-00-18-PM.png)
Account admins can go to ***Settings > Work order submission*** to **set the default behavior for the account**.

Learn more about the new work order submission options **[HERE](//innovint-6865708.hs-sites.com/hc/en-us/articles/360050001811-using-work-orders-in-innovint?hsLang=en#submission)**.

#### Work Order History + Edit Task Completion Date

**Work order and task history** allows you to view historical activity and updates to your work orders!

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360069908831work_order_history.png?width=450&name=support.innovint.ushcarticle_attachments360069908831work_order_history.png)

Once a task is completed, you also have the ability to **edit the completion date and time**.

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360069908891edit_completed_date.png?width=450&name=support.innovint.ushcarticle_attachments360069908891edit_completed_date.png)

Learn more about **[work order history tracking](//innovint-6865708.hs-sites.com/hc/en-us/articles/360050001811-using-work-orders-in-innovint?hsLang=en#history)** and [**editing task complete date**](//innovint-6865708.hs-sites.com/hc/en-us/articles/360050001811-using-work-orders-in-innovint?hsLang=en#task_history) in InnoVint's Online Support Center.

### Improvements:

#### Work Order ID

A numerical work order ID is now displayed in the header of each work order. You can also search for work orders by this number in the *Name* filter in the Work Order Explorer.

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360069956291WO_ID.png)

### Bug fixes:

- The ability to *edit* a Vineyard Event on multiple blocks is restored 🚜
- Custom Winemaking Report volumes fixed
