# 11/29/18 - Adding cost items to lots in weight, Lock Backdating for cost items, and more!

The Software Release on 11/29/18 includes:

### New Features

- Cost Items can now be added and applied to Fruit Lots and lots tracked in weight

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360016176012New_Release_graphics.png)

- [Lock backdating](//innovint-6865708.hs-sites.com/hc/en-us/articles/360020136992-cost-item-lock-backdating?hsLang=en) function added for Costing

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360016176372New_Release_graphics__2_.png?width=602&name=support.innovint.ushcarticle_attachments360016176372New_Release_graphics__2_.png)

- New [Bond to Bond transfer options](//innovint-6865708.hs-sites.com/hc/en-us/articles/360018542692-bond-to-bond-transfers?hsLang=en) (released 11/5/18)

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360016176892New_Release_graphics__3_.png?width=374&name=support.innovint.ushcarticle_attachments360016176892New_Release_graphics__3_.png)

### Improvements

- New Cost Item categories added: Consulting, Custom Crush, Debt Interest, Equipment Depreciation, and Lab Analysis

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360016277151B2B__simple_copy__6_.png?width=366&name=support.innovint.ushcarticle_attachments360016277151B2B__simple_copy__6_.png)

- Winery Activity Feed and export show change in weight for Fruit Weight Adjustment actions.
- *Partial* option added to the Contents filter dropdown in the Vessels tab of the Lot Details page to show vessels on ullage. This option was previously only available in the Vessel Explorer.

### Bug Fixes

- Fix for filter dropdowns in Cost Over Time report
- Lot and Vessel pickers now allow selection of archived inventory in bulk
- Fix for cost distribution on saignée volume
- We fixed our typo in the Tax Class "*Decleration*" tab header! 🤪
