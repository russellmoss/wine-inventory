# 7/29/21 Release Notes - Locations, File Attachment to Notes and ETS Ribbon-labels

The software releases for July 29, 2021 include:

### New Feature Release

#### Locations

- You can now add, edit and remove locations in your account to use when creating a Bill of Lading.

![Locations in BOLs](https://support.innovint.us/hs-fs/hubfs/Locations%20in%20BOLs.png?width=688&name=Locations%20in%20BOLs.png)

- Admins and Team Members can manage these locations from **Settings > Locations**
  ![Create Location](https://support.innovint.us/hs-fs/hubfs/Create%20Location.png?width=426&name=Create%20Location.png)

- Learn more in [this](https://support.innovint.us/hc/en-us/locations?hsLang=en) article.

### Improvements

#### ETS Ribbon-Labels

ETS labels now can be printed on ribbon-label printers. If you use ribbon-labels that accept pdf files (such as a Dymo or Zebra printer), we support 4" x 2-1/8" and 2-3/4" x 2-1/8" size labels. Learn more in [this](https://support.innovint.us/hc/en-us/articles/115001540726-ets-integration-step-by-step-instructions-?hsLang=en) article.

![](https://support.innovint.us/hubfs/image-png-Jul-23-2021-09-56-11-05-PM.png)

#### File Attachment to Notes

- Files (ie PDFs) can now be attached to notes on your inventory items (lots, vessels, dry goods, and vineyards) or on actions (direct actions or work orders).
- Click on the file link to download it to your computer.
- Learn more in [this](https://support.innovint.us/hc/en-us/articles/360009672091-how-to-add-a-photo-note?hsLang=en) article.

  ![](https://support.innovint.us/hubfs/image-png-Jul-26-2021-10-42-00-27-PM.png)

#### New Setting for Work Orders

- Admins can now decide whether any team member is permitted to edit work orders, or if they prefer the default behavior, that only Admins or the work order creator can edit the work order.

![](https://support.innovint.us/hubfs/undefined-3.png)

### Bug Fixes

- Bottled volumes
- Vineyard spacing
- Work order templates saving additives
- Analysis recorded at date
- QR codes for tirage vessels
