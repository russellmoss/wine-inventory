# 1/20/21 Release Notes - Bill of Lading Feature

The software release for January 19, 2021 includes:

### New Feature:

### Bills of Lading

InnoVint's new **Bill of Lading** feature allows you to generate and print a BOL document from within the platform. BOLs can be created from any Bond to Bond Transfer action or directly from the Report Explorer.

Learn more about how to create BOLs in InnoVint **[HERE](//innovint-6865708.hs-sites.com/hc/en-us/articles/360055473151-how-to-create-a-bill-of-lading-bol-?hsLang=en)**.
