# 5/5/2022 Release Notes: Export Updates

The software releases for May 5, 2022 include:

### We wrangled these mis-behaving exports, and corralled a few other things along the way:

- Custom Analysis Reports not exporting correct number of results
- Vintage Crush Cost data is missing from the Lot Cost Report export
- Add Packaging work order task not displaying or saving product or batch
- Bottling report > Export by bottling action button not downloading
- "Add all vessels" button only taking vessel on page and forward
- Edit Volume in Mock Blend to 1000 gal changes to 1 gal upon saving
- B2B Transfer In work order task has a broken link
- Cost Over Time and Cost Roll Forward Report showing gallons instead of liters for account with International setting
