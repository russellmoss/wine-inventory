# 4/22/2022 Release Notes: We Resolved a Few Things

The software releases for 4/22/2022 include:

- Harvest Forecast showing wrong expected arrival date
- Fruit Intake Report and Fruit Cost Report: Separate block and clone

- Vineyard spacing units reverting to "ft" in edit field
- Display current lot on vessel analysis task
- Increase maximum character allowance for contract fields
- COGS to aggregate costs across certain volume adjustment actions
- Ability to clear location from vessel
