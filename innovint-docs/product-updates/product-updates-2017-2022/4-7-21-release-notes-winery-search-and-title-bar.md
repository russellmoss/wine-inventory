# 4/7/21 Release Notes - Winery Search and Title Bar

## New Feature: Winery Search and Title Bar

![title_bar.png](https://innovint.zendesk.com/hc/article_attachments/360091812392/title_bar.png)

The title bar now includes a **winery search** feature, so you can quickly find and access that lot or work order you need to view.

We've also introduced the **Record Action** and **Create Work Order** buttons to allow you to submit data or assign work from any page in InnoVint.

Learn more about the winery search and title bar [here](//innovint-6865708.hs-sites.com/hc/en-us/articles/360059069692-winery-top-navigation-barh_01f2mmmwbg1gd2c0ae20yxfqfg?hsLang=en).
