# 5/21/21 Release Notes - New Forest Column in Vessel Explorer & Other Improvements

The software releases for May 21, 2021 include:

### Improvements:

#### Vessels

- Vessel Explorer and vessel pickers now have a forest column
  ![Screen_Shot_2021-05-21_at_1_24_28_PM.png](https://innovint.zendesk.com/hc/article_attachments/360096215791/Screen_Shot_2021-05-21_at_1_24_28_PM.png)
- Vessel pickers now how an option to *Select all vessels on page*
  ![Screen_Shot_2021-05-21_at_9_46_52_AM.png](https://innovint.zendesk.com/hc/article_attachments/360096159431/Screen_Shot_2021-05-21_at_9_46_52_AM.png)

#### New Analysis Source: Lodi Wine Labs

- Analysis source of Lodi Wine Labs is now available for both Analysis and Analysis Import actions.
  ![Screen_Shot_2021-05-21_at_9.51.25_AM.png](https://innovint.zendesk.com/hc/article_attachments/360096159591/Screen_Shot_2021-05-21_at_9.51.25_AM.png)

#### Vineyards

- Notes field now available in vineyard analysis
  ![Screen_Shot_2021-05-21_at_9_31_22_AM.png](https://innovint.zendesk.com/hc/article_attachments/360096205832/Screen_Shot_2021-05-21_at_9_31_22_AM.png)

#### Lot Explorer

- Vineyard filter now in alphabetical order
- Export now displaying top percentage of varietal, appellation and vintage instead of Blend

### Bug Fixes

- New per unit in the cost tab correct for metric volumes (for accounts with Costing activated)
- Total Bottles correct in Bottlings report export by format
