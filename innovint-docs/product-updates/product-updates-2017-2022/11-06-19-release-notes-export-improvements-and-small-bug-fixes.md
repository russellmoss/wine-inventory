# 11/06/19 Release Notes - Export improvements and small bug fixes

The software release on November 6, 2019 includes:

### Improvements:

- Lot code and Lot name are now separated into different columns in report exports.

  ![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360042391431Lot_Code_Lot_Name.png)

### Bug Fixes:

- Additive Explorer now sorts A-Z by default.
- Date filter for actions recorded after a certain time in Bottling report is fixed.
- Fix for column headers in Lot Cost Report export.
