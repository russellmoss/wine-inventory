# 07/06/17 - New Gift Box feature, Inventory Report at Point in Time, and more

The software release on 7/6/17 includes:

**New Features:**

- New Gift Box icon in the upper right of the web interface which links to the Online Help Center and details of all software releases to date to include new features, improvements and bug fixes

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115011366383mceclip6.png?width=339&name=support.innovint.ushcarticle_attachments115011366383mceclip6.png)

- New Report: "Inventory at Point in Time" which shows the lot volume and tax class as of a particular date in the past

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115011223666mceclip7.png)

- Pulled out Bond to Bond Transfer as a separate action from Volume Adjustment. It now appears separately in the action drop down

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115011222686mceclip2.png?width=283&name=support.innovint.ushcarticle_attachments115011222686mceclip2.png)

- Backdate icon implemented in History Feeds to indicate if an action has been backdated. Hover over the icon to show Effective At time and Submitted At time

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115011365863mceclip1.png?width=432&name=support.innovint.ushcarticle_attachments115011365863mceclip1.png)

- New "Blended" lot stage added

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments115011366323mceclip4.png?width=302&name=support.innovint.ushcarticle_attachments115011366323mceclip4.png)

**Improvements**:

- Volume Adjustment actions (including Bond to Bond Transfers) now appear on the Calendar
- Varietals are now displayed in the Vineyard Explorer
- For those with custom crush permissions enabled, the Owner filter is now in the Winery Activity Feed.
- Custom report exports now show the date and time stamp of historical analyses
- New addition rates added: lb/gal, g/100L (g/dL)
- Submitted work orders in All Submitted work order tab now sort from most recent to least recent
- Improved Lot Detail Page load time by limited History widget to 3 most recent activities. To view all history, go to History sub-tab.

**Bug Fixes:**

- Notes now display correctly on vessels when scrolling through pagination in Vessel Explorer
- Calendar date search now respects start date rather than date after start date
- Blend action summary now shows correct vessel count rather than (1) of each type
- Rack and Return action now prevents new lees lots to be created and submitted with no contents increase
- In the Lot Explorer, Archive and Fruit Lots filter selections are now saved if a lot is clicked then the user navigates back to Explorer
