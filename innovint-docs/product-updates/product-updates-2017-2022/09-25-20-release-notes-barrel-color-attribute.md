# 09/25/20 Release Notes - Barrel Color Attribute

The software release on September 25, 2020 includes:

### Improvements:

### Barrel color attribute

Track your barrel inventory by designated wine color!
Select from Red, White, or Rosé!

![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360069964412color.png?width=521&name=support.innovint.ushcarticle_attachments360069964412color.png)

All existing inventory is currently set to N/A.

To update your current barrel inventory in bulk -
1. Go to **Vessel Explorer > Manage Vessels**.
2. Select **Edit barrel attribute on vessels.**3. Select your **vessels**, then **Barrel Attribute: Color**

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360069969052edit_barrel_color.gif)

### Bug fixes:

- Winery Dashboard now displays all widgets in the correct units
- The ability to add multiple vessels at one time is restored
- Custom Vineyard Reports *Block* and *Vintage* filters fixed 🍇
