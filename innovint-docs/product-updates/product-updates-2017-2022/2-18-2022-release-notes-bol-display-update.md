# 2/18/2022 Release Notes: BOL display update

The software releases for February 18, 2022 include:

### Improvements

#### Print Lot Composition on a separate page of the BOL

Users can now choose to print the full lot composition on a separate page. This will generate a second page that displays your entire lot composition, with the phrase: "Full composition attached" on the front of the BOL.

![](https://support.innovint.us/hubfs/image-png-Feb-18-2022-06-45-31-95-PM.png)

### Bug Fixes

- Vessels removed from B2B Transfer In/Out work order tasks
- Vessels removed from a work order still showing the work order action in their history
- Specific Gravity unit in analysis import
- Lot import extra spaces on csv error
