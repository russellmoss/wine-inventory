# 2/24/21 Release Notes - New TTB Audit Report!

The software release for February 24, 2021 includes:

### New Feature:

#### TTB Audit Report

The TTB Audit Report is now in beta!

![](https://support.innovint.us/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360087501611audit_report.png)

This report provides a detailed list of the particular lots and actions that contribute to the reported values in the TTB 5120.17. You can learn more in our Support Center [here](//innovint-6865708.hs-sites.com/hc/en-us/understanding-the-ttb-audit-report?hsLang=en).

Please share any feedback about this beta release to [support@innovint.us](//innovint-6865708.hs-sites.com/hc/en-us/kb-tickets/new?hsLang=en).

### Improvement:

#### Costing: Cost Over Time Report

Lots that have zero fill, but carry costs, now show in the Cost Over Time report.

### Bug Fixes:

- Vintages now listed in descending order in the *Vintages* tab under Block details
- Disgorge, Dosage & Package actions report as bottled in the TTB report
- Fruit Intake Report sortable by Grower
