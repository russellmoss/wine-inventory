# 11/16/20 Release Notes - Faster Page Loading, Ability to Add & Remove Lees Lots!

The software releases for the week of November 16, 2020 include:

### Improvements:

#### Faster Page Loading

Enjoy faster load times in the *Lot Explorer*, individual *Lot details* pages, the *Winery Activity Feed*, and more!

#### Add & Remove Lees Lots

The ability to edit an open work order and *add* or *remove* a lees lot is now possible with the following tasks:

- Blend
- Rack
- Transfer
- Filter
- Barrel Down
- Top Off

### Bug Fixes:

- Fix to create new juice/wine lot titles
- Member permissions headers updated
- Lot component text updated
- Fix to analysis display in the *Vessel details* page
