# 12/03/21 Release Notes: B2B Location edits, New BOL fields

The software releases for December 3, 2021 include:

### Improvements

#### Locations are editable on submitted B2B Transfer In and Out actions

User can now change the 'location' of a submitted B2B Transfer In or B2B Transfer Out action. User can select a new location from their drop-down menu, or clear the field entirely. Click the blue pencil next to the the 'To' location to edit.

![](https://support.innovint.us/hubfs/image-png-Dec-03-2021-07-35-13-67-PM.png)

Click "Set new location" and refresh the page to see the new destination and download a new BOL.

![](https://support.innovint.us/hubfs/image-png-Dec-03-2021-07-35-21-25-PM.png)

Learn more about location in InnoVint [here](https://support.innovint.us/hc/en-us/locations?hsLang=en).

#### Bill of Lading #

Bills of Lading now feature a BOL # field users can fill in if desired. Enter in the number before downloading the BOL to have it populate in the pdf. The number will appear in the upper right-hand corner of the BOL next to the date.

![](https://support.innovint.us/hubfs/image-png-Dec-02-2021-08-16-29-90-PM.png)

![](https://support.innovint.us/hubfs/image-png-Dec-02-2021-08-18-31-97-PM.png)

Check out this article for more information on [how to create a BOL](https://support.innovint.us/hc/en-us/articles/360055473151-how-to-create-a-bill-of-lading-bol-?hsLang=en).

**Scale Ticket # and Seal #**

Users also have the option to enter in a Scale ticket # and Seal #s if desired. These must be entered after BOL generation, but before downloading the BOL pdf.

![](https://support.innovint.us/hubfs/image-png-Dec-02-2021-08-23-29-51-PM.png)
