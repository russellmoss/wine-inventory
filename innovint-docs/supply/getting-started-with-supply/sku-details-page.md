# SKU Details Page

The **SKU Details Page** is the home base for all details regarding a specific SKU.

**![SKU Details Page](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page.jpg?width=688&height=318&name=SKU%20Details%20Page.jpg)**

The top of the SKU details page displays the SKU code and action menus.

- Click the '**+ Add Inventory button**' to add inventory to this specific SKU.
- Use the 'More' dropdown in the upper right to manage the SKU. **![SKU Details Page_More dropdown menu](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page_More%20dropdown%20menu.jpg?width=190&height=122&name=SKU%20Details%20Page_More%20dropdown%20menu.jpg)**
  - Select Edit SKU to edit SKU code, name, color, or style. ![SKU Details page_Edit SKU](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20page_Edit%20SKU.webp?width=306&height=309&name=SKU%20Details%20page_Edit%20SKU.webp)
  - Select Edit C7 product to link, change, or unlink the SKU from a product in Commerce 7.
    ![SKU Details Page_Edit C7](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page_Edit%20C7.png?width=299&height=125&name=SKU%20Details%20Page_Edit%20C7.png)

Select [Onboard Inventory](https://support.innovint.us/hc/en-us/how-to-onboard-inventory-in-supply?hsLang=en) to bring your current inventory into the system at the time you set up your SUPPLY account.

#### Your SKU details page also includes:

- [SKU Attributes](#sku-attributes)
- [Total Inventory](#total-inventory)
- [Inventory by Location](#inventory-by-location)
- [History](#history)

**SKU Attributes**

- SKU code
- SKU Name
- Stage - This can be edited using the blue edit pencil.
- Format (container size and type)
- Formats/Group (eg 12 bottles/case)
- Tax Class
- C7 Product - This links directly to the C7 product to which this SKU is associated.
  **![SKU Details Page_SKU Attributes](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page_SKU%20Attributes.jpg?width=287&height=362&name=SKU%20Details%20Page_SKU%20Attributes.jpg)**

**Total Inventory**

This widget displays the total amount of inventory (across all locations) currently on the SKU, broken out by group total (eg cases) plus the number of individual units (eg bottles). Inventory quantities for In-bond and Taxpaid are also broken out to the right of the total inventory.

This widget will update when Add, Move, Deplete or Reconcile Inventory actions are recorded on the SKU.

**![SKU Details Page_Total Inventory](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page_Total%20Inventory.jpg?width=688&height=50&name=SKU%20Details%20Page_Total%20Inventory.jpg)**

**Inventory by Location**

This widget displays the inventory available in each location that holds this SKU, including the Tax Status of the location, and inventory count broken out by group total plus the number of individual units for each location.

On the far right is the Action column, which is used to record an inventory action for the specific inventory in that location.

For Commerce 7 users, locations that display a C7 icon are linked to your Commerce 7 account's Inventory locations.
**![SKU Details Page_Inventory by Location](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page_Inventory%20by%20Location.jpg?width=688&height=118&name=SKU%20Details%20Page_Inventory%20by%20Location.jpg)**

**History**

The SKU History shows the entire action history and SKU attribute changes for the SKU in reverse chronological order. Click on an action to view the Action details page. **![SKU Details Page_Action History](https://support.innovint.us/hs-fs/hubfs/SKU%20Details%20Page_Action%20History.jpg?width=688&height=178&name=SKU%20Details%20Page_Action%20History.jpg)**
