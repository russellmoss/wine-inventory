# How to Change Lot Stage

Lot stages can be changed at any point in time from the Lot Details page for a bulk wine, or for a bottled case good. Lot stages should be changed whenever the lot is complete with one phase of its life, such as finishing primary fermentation and moving to malolactic fermentation. You can also use these for Case Good lots to help manage shiners or wines ready for order.

Stages are primarily helpful for reporting. When you are working with lots or viewing lots in a report or Explorer, you can always filter the lots by Stage.  Stages also impact which analyses display on the Lot Dashboard. Find out more [here](https://support.innovint.us/hc/en-us/how-to-set-your-lot-stage-analyses?hsLang=en).

Changing the lot stage does not impact your TTB Reporting or update a lot's tax class. Find out how to update tax class on a lot [here](https://support.innovint.us/hc/en-us/articles/207936576-declare-or-edit-tax-class?hsLang=en).

This article covers how to change the stage of a lot:

- [from the Lot Details page](#lot_details)
- [from within an action or task](#action_WO_task)
- [in bulk](#bulk)

### From the Lot Details page

Click on the blue pencil next to your current stage to select a new stage.

![How to Change Lot Stage-from lot details pg](https://support.innovint.us/hs-fs/hubfs/How%20to%20Change%20Lot%20Stage-from%20lot%20details%20pg.webp?width=377&height=536&name=How%20to%20Change%20Lot%20Stage-from%20lot%20details%20pg.webp)

### From within an action or work order task

Stages can be selected within some actions and tasks. For example, when going through the **Process Fruit to Weight or Volume** tasks, or the **Drain and Press** task, you can select to change the current Lot Stage at the conclusion of that task.

![How to Change Lot Stage-within an action](https://support.innovint.us/hs-fs/hubfs/How%20to%20Change%20Lot%20Stage-within%20an%20action.webp?width=548&height=180&name=How%20to%20Change%20Lot%20Stage-within%20an%20action.webp)

The **Bottle** *action* also allows you to set the lot stage within the action (and defaults that stage to "Bottled").

The **Addition** action and task allows the user to change the stage of all lots involved at one time. This increases efficiency and is very useful tool when adding yeast to change lot stages from Processed to Fermenting, or when adding SO2 to change the lot stage from ML to Aging.

![How to Change Lot Stage-addition](https://support.innovint.us/hs-fs/hubfs/How%20to%20Change%20Lot%20Stage-addition.webp?width=550&height=300&name=How%20to%20Change%20Lot%20Stage-addition.webp)

### In bulk

Go to the Lot Explorer and click on the Manage Lots button in the top right corner. Select *Set stage across lots*.

![How to Change Lot Stage-in bulk](https://support.innovint.us/hs-fs/hubfs/How%20to%20Change%20Lot%20Stage-in%20bulk.webp?width=353&height=320&name=How%20to%20Change%20Lot%20Stage-in%20bulk.webp)

Choose your lots from the lot picker, then select a stage from the Stage dropdown. Click on Set Stage to change the stage for the selected lots.

![How to Change Lot Stage-bulk edit stage](https://support.innovint.us/hs-fs/hubfs/How%20to%20Change%20Lot%20Stage-bulk%20edit%20stage.webp?width=688&height=442&name=How%20to%20Change%20Lot%20Stage-bulk%20edit%20stage.webp)
