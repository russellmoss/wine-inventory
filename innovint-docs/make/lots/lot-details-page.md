# Lot Details Page

The **Lot Details Page** is the home base for all details regarding a specific lot and there is a huge amount of data to explore. In this article, we cover:

- [What's on the Lot details page?](#details)
- [What's on the Dashboard?](#dashboard)
- [Beyond the Dashboard: Exploring your tabs](#tabs)

![Lot details page-additives](https://support.innovint.us/hs-fs/hubfs/Lot%20details%20page-additives.png?width=688&height=376&name=Lot%20details%20page-additives.png)

#### What's on the Lot details page?

- Across the top: Lot code, lot name and action menus (record actions and change lot properties via "more" menu)
  - Use the 'Record action' dropdown to select an action to perform on the lot.
    - In most cases, the selected action will auto-select the lot.
    - Some actions are *only* available in this action menu (B2B within InnoVint Winery and B2B to another InnoVint Winery).
  - Use the 'More' dropdown in the upper right to manage the lot.
    ![Lot Details Page-more dropdown](https://support.innovint.us/hs-fs/hubfs/Lot%20Details%20Page-more%20dropdown.webp?width=161&height=159&name=Lot%20Details%20Page-more%20dropdown.webp)
    - Select "Change lot properties" to edit the Lot code, Lot name, Lot color or Lot style (if the  Sparkling module is activated).
      ![Lot details page-change lot properties](https://support.innovint.us/hs-fs/hubfs/Lot%20details%20page-change%20lot%20properties.webp?width=251&height=189&name=Lot%20details%20page-change%20lot%20properties.webp)
    - Delete a lot. Lots cannot be deleted if they've ever had any activity recorded on them - they must be archived instead.
    - Archive your lot. Hide this lot from most default views across the platform. All lot history is saved ad available for recall. Lots cannot be archived with contents - the volume must be depleted first.

- - - Open a standalone page (applicable to the open lot details tab, including Additives and Composition) for easy printing
- Your **Dashboard & other lot detail tabs**

#### What's on the Dashboard?

- Attributes tile: quickly view and edit those properties with an asterisk\*
  - Expected yield in volume/weight (during harvest, for lots in weight)\*
  - Expected volume (during harvest, for lots in weight)
  - Bond
  - Stage\*
  - Fill volume
  - Approximate cases
  - Tax Class\*
  - Intended Use (if activated)
  - Sparkling designation
  - Tags\*
  - Owners\* (if activated)
  - Notes\*

    \* The blue pencil opens up edit functionality for that attribute
- **Analysis**: This dashboard widget displays the most recent set of analyses according to the Lot Stage. Find out how to customize your display [here](https://support.innovint.us/hc/en-us/how-to-set-your-lot-stage-analyses?hsLang=en).
- **Vessels**: this widget provides a summary of all vessels containing your lot, grouped by vessel type and capacity. Click on the arrow or the Vessels tab to see a full list by vessel.
- **Open Work Orders**: displays any open (un-submitted) work order containing the lot. Click straight into a work order from here.
- **Recent history**: shows the last three actions on the lot. Click on "See all" or the History tab to find the complete activity history.
- **Composition**: a simple varietal breakdown is shown on the dashboard. Click See all or the Composition tab to see all lot components.
- **Notes**: save text, photos or file attachments to your lot.

#### Beyond the Dashboard: Exploring your tabs

Use the **tabs across the top** of the Dashboard to dive deeper into each aspect of the lot.

- Analysis
  - Access multiple views of results such as analysis grouped by date or vessel, graphed results, a built in sugar/temperature graph for harvest, or view all analysis in a list (access analyses for export, edits or deletion here).
    ![Lot details page-analysis](https://support.innovint.us/hs-fs/hubfs/Lot%20details%20page-analysis.webp?width=688&height=159&name=Lot%20details%20page-analysis.webp)
- Additives
  - Access [Calculated Additives](https://support.innovint.us/hc/en-us/calculated-additives?hsLang=en) for the lot here.
    ![Lot details - Additives](https://support.innovint.us/hs-fs/hubfs/Lot%20details%20-%20Additives.png?width=688&height=190&name=Lot%20details%20-%20Additives.png)
- Vessels
  - See and interact with a complete vessel list and also opt to print it as a pdf list
    ![Lot Details Page-vessels](https://support.innovint.us/hs-fs/hubfs/Lot%20Details%20Page-vessels.webp?width=688&height=125&name=Lot%20Details%20Page-vessels.webp)
- History
  - View the lot's full activity history, and also find the lot properties history (Lot code, Lot name, Stage, Lot style and Archived status changes can be viewed here)
    ![Lot Details Page-history](https://support.innovint.us/hs-fs/hubfs/Lot%20Details%20Page-history.webp?width=688&height=160&name=Lot%20Details%20Page-history.webp)
- Timeline
  - Check out a visual overview of actions that have occurred over the life of your lot
    ![Lot Details Page-timeline](https://support.innovint.us/hs-fs/hubfs/Lot%20Details%20Page-timeline.webp?width=688&height=334&name=Lot%20Details%20Page-timeline.webp)
- Composition
  - Find your vintage, varietal, appellation, appellation-varietal, appellation roll-up totals and vineyard breakdowns here. You can export the composition percentages in the Percentages section, or choose to export all components in the lower Components widget.  Please note that in the Percentages section, individual components <0.001 will be grouped as "Other."
    ![Lot Details_Coomposition](https://support.innovint.us/hs-fs/hubfs/Lot%20Details_Coomposition.png?width=688&height=340&name=Lot%20Details_Coomposition.png)
- Cost (if you have FINANCE with COGS Tracking activated)
  - See the cost history on a lot including cost changes due to movement actions and added or removed indirect costs
    ![Lot Details Page-cogs](https://support.innovint.us/hs-fs/hubfs/Lot%20Details%20Page-cogs.webp?width=688&height=260&name=Lot%20Details%20Page-cogs.webp)
  - Get to know the cost tab better [here](https://support.innovint.us/hc/en-us/exploring-the-cost-tab?hsLang=en)!
