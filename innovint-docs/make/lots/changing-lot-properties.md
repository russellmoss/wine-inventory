# Changing Lot Properties

You can change the lot code, lot name, lot color or lot style at any point in time, whether your lot has contents or not, as long as your lot is not archived. These lot properties will change to display the new code, name, color or style *throughout the entire history* of the lot.

InnoVint does not track when and how lot codes are changed in the Lot history (where you see the action history). These changes are not tracked as actions (like a Tax Class change). We recommend adding a note to the lot or recording a custom action to keep a visual record of when a lot code has been changed in the Lot history.

However, you can find the record of these changes in the [**Lot Properties History**](#history) (see more below).

#### How to edit Lot Properties

To edit the lot code, name, color or style, select "Change lot properties" from the "More" dropdown menu on the lot details page.

![Changing Lot Properties-more dropdown](https://support.innovint.us/hs-fs/hubfs/Changing%20Lot%20Properties-more%20dropdown.webp?width=670&height=121&name=Changing%20Lot%20Properties-more%20dropdown.webp)

Enter the details that you want to change to and press "Save lot properties."

![Changing Lot Properties-save](https://support.innovint.us/hs-fs/hubfs/Changing%20Lot%20Properties-save.webp?width=302&height=226&name=Changing%20Lot%20Properties-save.webp)

You're all set with your new lot details!

#### Lot Properties History

Find the **Lot properties history**, on the History tab of the Lot details page.

![Changing Lot Properties-history](https://support.innovint.us/hs-fs/hubfs/Changing%20Lot%20Properties-history.webp?width=670&height=86&name=Changing%20Lot%20Properties-history.webp)

Here, you can find all lot properties and attribute changes that InnoVint tracks as of a point in time (you will not find a record of Owner or Tag changes, since that time-stamped information is not recorded).

![Changing Lot Properties-lot history](https://support.innovint.us/hs-fs/hubfs/Changing%20Lot%20Properties-lot%20history.webp?width=670&height=322&name=Changing%20Lot%20Properties-lot%20history.webp)

You can find a winery level [Lot Properties History Report](/hc/en-us/lot-properties-history-report?hsLang=en) in the Report Explorer, under Activity Reports.

![052126_Report Explorer_Lot Properties History Report_Updated Screenshot_Annotated](https://support.innovint.us/hs-fs/hubfs/052126_Report%20Explorer_Lot%20Properties%20History%20Report_Updated%20Screenshot_Annotated.png?width=670&height=319&name=052126_Report%20Explorer_Lot%20Properties%20History%20Report_Updated%20Screenshot_Annotated.png)
