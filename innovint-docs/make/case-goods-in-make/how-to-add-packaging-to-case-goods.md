# How to Add Packaging to Case Goods

Need to add labels to your shiners in MAKE after bottling? Or re-pack your case goods with new packaging items? Record an 'Add packaging' action or task to apply packaging items to your case goods!

This article covers:

- [How to record an Add Packaging action](#how)
- [Where does added packaging report?](#where)

### How to record an Add Packaging action

Go to the 'Add packaging' action from the Record Action dropdown in the Case Good lot details page. Alternatively, an 'Add packaging' task can be added to a work order.

![How to Add Packaging to Case Goods-action](https://support.innovint.us/hs-fs/hubfs/How%20to%20Add%20Packaging%20to%20Case%20Goods-action.webp?width=688&height=344&name=How%20to%20Add%20Packaging%20to%20Case%20Goods-action.webp)

**Lot** - Select your Case Goods lot requiring packaging from the lot dropdown or lot picker

**Add Packaging to** - Check the box to add packaging to the entire lot, or else enter the number of pallets, cases, and bottles affected.

**Transfer to** - Select to 'Retain Case Goods lot', 'Combine with an existing Case Goods lot' (both lots must have the exact same composition), or 'Create a new Case Goods lot'.

- - 'Retain Case Goods lot' means that you are adding packaging to the same lot code
  - 'Combine with an existing Case Goods lot' means that you are adding packaging to the specified number of cases, and moving those cases into an existing lot code
  - 'Create a new Case Goods lot' means that you will moving the specified number of cases into a new, empty lot code, with the requested packaging.

**Packaging** - Click on 'Edit packaging' to select and enter your packaging products, batches, used and scrap amounts.

If you are adding packaging to an entire lot, consider updating the [Lot Stage](https://support.innovint.us/hc/en-us/articles/204339859-how-to-change-lot-stage?hsLang=en) from "Shiner" to the next appropriate lot stage.

### Where does added packaging report?

The packaging added in this action applies to the selected case good. This means that you find evidence of the added packaging/cost:

- On the Case Good lot details page in the Packaging tab
  ![Add Packaging_Packaging tab](https://support.innovint.us/hs-fs/hubfs/Add%20Packaging_Packaging%20tab.png?width=688&height=203&name=Add%20Packaging_Packaging%20tab.png)
- In the [Packaging History Report](https://support.innovint.us/hc/en-us/cogs-and-dry-goods-tracking?hsLang=en#additive-history-report)
- In the Lot Cost Report (filtered for the lot type Case Good) you will find any added direct cost from the applied packaging

You will not find any added packaging in the Bottled Cost Report. Find out more about what populates that report [here](https://support.innovint.us/hc/en-us/cost-reports?hsLang=en#bottled).
