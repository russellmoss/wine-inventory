# How to Edit a Work Order

Open work orders can be edited to change the title, due date, and assigned to, as well as instructions, and vessel & lot selections within the tasks.

Please note that work orders cannot be edited once a work order is completed or submitted, and you cannot "unsubmit" a work order.

### How to edit a work order

To open any editable fields in the work order, click the 'Edit work order' button in the header of the work order. You can edit work order attributes or tasks in open work orders that are not completed. You may not edit submitted work orders.

![How to edit a work order_header](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_header.webp?width=670&height=56&name=How%20to%20edit%20a%20work%20order_header.webp)

If the edit work order button is greyed out, then you have either completed or submitted the work order. Completed work orders can have tasks un-completed, and then edited. Learn about how to edit actions after work orders are submitted [here](https://support.innovint.us/hc/en-us/articles/208141233-how-to-edit-or-delete-recorded-actions?hsLang=en).

![How to edit a work order_uncomplete](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_uncomplete.webp?width=670&height=139&name=How%20to%20edit%20a%20work%20order_uncomplete.webp)

#### How to edit the work order attributes

With the work order in edit mode, you will see blue pencil icons appear next to the title, "Assigned to", "Due", and "Owners" fields (with owner permissions activated). You will also see the "+ Add task" button appear.

Click the pencil icon next to the header attribute that you would like to change.

![How to edit a work order_edit header](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_edit%20header.webp?width=670&height=71&name=How%20to%20edit%20a%20work%20order_edit%20header.webp)

Make any required changes and then click the blue button to save your update on the relevant box or slideover screen.

![How to edit a work order_reassign](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_reassign.webp?width=670&height=181&name=How%20to%20edit%20a%20work%20order_reassign.webp)

When you have completed your changes, be sure to **Save work order edits**! This ensures that the changes are reflected in the work order for all users.

![How to edit a work order_save edit](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_save%20edit.webp?width=670&height=67&name=How%20to%20edit%20a%20work%20order_save%20edit.webp)

#### How to edit, add or remove lots in the work order

While the work order is uncompleted and in edit mode, you can change, add or remove lots within a task as needed.

**Change or edit a lot**

On the "single lot" side of the task, you will see the lot dropdown menu and lot picker available. Select the corrected lot, and save your work order edit in the header:

![How to edit a work order_edit lot](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_edit%20lot.webp?width=670&height=285&name=How%20to%20edit%20a%20work%20order_edit%20lot.webp)

Need to change lots on the other side (the "multi-lot" side) of the task? If you need to change one or many lots involved in a blend, or being filled by a transfer or rack action, then you must perform a two step correction to swap the lot(s). You will first add your corrected lot(s), and then remove the unwanted lot(s).  Keep reading to learn more!

**Add additional lots**

With the work order in edit mode, you will see the "+ Add lot" button appear in the relevant task location, depending on whether you can add multiple lots on the drained or filled side of the movement. You can also add in a lees lot. Enter the updated lot and vessel information in the slideover:

![How to edit a work order_transfer to lees edit](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_transfer%20to%20lees%20edit.webp?width=155&height=87&name=How%20to%20edit%20a%20work%20order_transfer%20to%20lees%20edit.webp)

![How to edit a work order_add lot](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_add%20lot.webp?width=670&height=209&name=How%20to%20edit%20a%20work%20order_add%20lot.webp)

![How to edit a work order_add blend lot](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_add%20blend%20lot.webp?width=155&height=59&name=How%20to%20edit%20a%20work%20order_add%20blend%20lot.webp)

![How to edit a work order_add blend lot2](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_add%20blend%20lot2.webp?width=670&height=168&name=How%20to%20edit%20a%20work%20order_add%20blend%20lot2.webp)

![How to edit a work order_edit blend to](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_edit%20blend%20to.webp?width=155&height=64&name=How%20to%20edit%20a%20work%20order_edit%20blend%20to.webp)

![How to edit a work order_edit blend to2](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_edit%20blend%20to2.webp?width=670&height=214&name=How%20to%20edit%20a%20work%20order_edit%20blend%20to2.webp)

Please check whether you are requesting "remove" "add" or "ending fill" on your vessels when adding lots as you will not be able to change this selection after adding the new lot.

![How to edit a work order_show-hide vessels](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_show-hide%20vessels.webp?width=670&height=320&name=How%20to%20edit%20a%20work%20order_show-hide%20vessels.webp)

**Remove unwanted or incorrect lots**

If you have more than one lot being drained or filled, you will see the ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcen-usarticle_attachments202432116Screen_Shot_2016-01-10_at_12.18.44_PM.png?width=92&height=18&name=support.innovint.ushcen-usarticle_attachments202432116Screen_Shot_2016-01-10_at_12.18.44_PM.png) button to the right of the lot. You must have more than one lot available in order to remove one. If you only have a single lot being drained or filled, you will instead see the lot picker option re-open, and you can only update the selected lot (see above on how to change or edit a lot).

To remove the lot, simply press the Remove lot button to delete that particular lot from the work order.

![How to edit a work order_remove lot](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_remove%20lot.webp?width=670&height=307&name=How%20to%20edit%20a%20work%20order_remove%20lot.webp)

Don't forget! When edits are complete, press the 'Save work order edits' button at the top of the work order in order to continue with the task.

#### How to edit requested volumes

With an open, uncompleted work order task open in edit mode, the "Requested" fields will open for editing, and the "Actual" fields are closed:

![How to edit a work order_edit removed vol](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_edit%20removed%20vol.webp?width=670&height=344&name=How%20to%20edit%20a%20work%20order_edit%20removed%20vol.webp)

After you save the edits, the "Requested" fields will close, and users can again record the final volumes in the "Actual" fields:

![How to edit a work order_edit removed vol2](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_edit%20removed%20vol2.webp?width=670&height=297&name=How%20to%20edit%20a%20work%20order_edit%20removed%20vol2.webp)

Please note that you cannot change whether to request an Remove/Add vs Ending Fill volume after the task is created.

Many edits in open work orders can only be made by Admins and users that created the work order.

To change this default behavior go to *Settings > Work Order Edit* to allow Team Members and Team Members cannot Submit to re-assign work orders, or edit lots and requested volumes.

![How to edit a work order_permissions](https://support.innovint.us/hs-fs/hubfs/How%20to%20edit%20a%20work%20order_permissions.webp?width=670&height=168&name=How%20to%20edit%20a%20work%20order_permissions.webp)
