# Deleting a Work Order

This article covers:

- [How to delete a work order ***before*** it is submitted](#before)
- [How to delete a work order ***after*** it is submitted](#after)
- [Frequently Asked Questions (FAQ)](#faq)

### Deleting a Work Order *before* it is submitted:

Work orders can be deleted at any time before they are submitted, but can only be deleted from the web interface/desktop app, and not from the mobile app.

To delete an active work order:

![Deleting a Work Order-1](https://support.innovint.us/hs-fs/hubfs/Deleting%20a%20Work%20Order-1.webp?width=688&height=98&name=Deleting%20a%20Work%20Order-1.webp)![Deleting a Work Order-2](https://support.innovint.us/hs-fs/hubfs/Deleting%20a%20Work%20Order-2.webp?width=688&height=95&name=Deleting%20a%20Work%20Order-2.webp)![Deleting a Work Order-3](https://support.innovint.us/hs-fs/hubfs/Deleting%20a%20Work%20Order-3.webp?width=471&height=132&name=Deleting%20a%20Work%20Order-3.webp)

**Note:** At least one task in a work order must be uncompleted to edit or delete a work order.

### Deleting a Work Order *after* it is submitted:

After a work order is submitted, you can delete each task's action. If there were multiple tasks within a work order, each one can be deleted individually. You can delete one, some, or all.

- From the Work order details page:
  ![Deleting a Work Order-delete from view](https://support.innovint.us/hs-fs/hubfs/Deleting%20a%20Work%20Order-delete%20from%20view.webp?width=444&height=336&name=Deleting%20a%20Work%20Order-delete%20from%20view.webp)
- From the Lot details page > History tab:
  ![Deleting a Work Order-history tab](https://support.innovint.us/hs-fs/hubfs/Deleting%20a%20Work%20Order-history%20tab.webp?width=688&height=177&name=Deleting%20a%20Work%20Order-history%20tab.webp)

### FAQ

**Q. How do I delete more than one work order at a time?**

*A: Currently work orders can only be deleted one at a time.*

**Q. If I create a recurring work order, can I delete them all later?**

*A: After a recurring work order has been created, each work order is independent of the others. To delete multiple recurring work orders each one will need to be deleted or edited one-by-one.*
