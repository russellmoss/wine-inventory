# Notifications in InnoVint

Notifications are enabled in InnoVint's MAKE desktop functionality as helpful reminders of important work order activity. The Notification Center is located in the desktop application for MAKE in the upper right corner (the white bell).   Popup notifications will link to the referenced work order directly if clicked.

![Notifications_Notifications Bell](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Notifications_Notifications%20Bell.webp?width=96&height=80&name=Notifications_Notifications%20Bell.webp)

### Receive Notifications

When a notification is sent, that user will receive it in the desktop web interface on cellar.innovint.us.

Notifications are issued from the system when any of the following take place:

- A work order is assigned to you
- A work order you created is completed
- A note is added to a task in a work order that you have assigned
- At 8:00 a.m. PT you will receive a notification to submit any open work orders
- At 5:00 p.m. PT you will receive a notification to submit any completed work orders

Note that you will *not* receive a notification if you are both the work order creator and assignee (you are writing and assigning work orders to yourself).
