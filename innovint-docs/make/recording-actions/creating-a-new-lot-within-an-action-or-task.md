# Creating a New Lot within an Action or Task

You can create a new lot via the [+Add lot button](https://support.innovint.us/hc/en-us/articles/204106579-step-3-enter-your-current-wine-lots-into-the-system?hsLang=en) on the Lot Explorer.  If you haven't created a new lot ahead of time, you can also create a new lot within most actions and tasks.

- Select "Create new lot" within the action and press the ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcen-usarticle_attachments202086155Screen_Shot_2015-04-06_at_10.37.39_AM.png?width=65&name=support.innovint.ushcen-usarticle_attachments202086155Screen_Shot_2015-04-06_at_10.37.39_AM.png) button
- Select the bond, choose a lot code, lot name (optional), tax class, and stage. There is the option to make it a sparkling lot by checking the box as well as to add tags
- Click the green "Add new lot" button

![Creating a New Lot within an Action or Task-add new lot button](https://support.innovint.us/hs-fs/hubfs/Creating%20a%20New%20Lot%20within%20an%20Action%20or%20Task-add%20new%20lot%20button.webp?width=688&height=266&name=Creating%20a%20New%20Lot%20within%20an%20Action%20or%20Task-add%20new%20lot%20button.webp)

- Select the receiving vessels for the new lot
- Both new wine lots and new lees lots can be created

![Creating a New Lot within an Action or Task-create new lot and lees lot](https://support.innovint.us/hs-fs/hubfs/Creating%20a%20New%20Lot%20within%20an%20Action%20or%20Task-create%20new%20lot%20and%20lees%20lot.webp?width=688&height=197&name=Creating%20a%20New%20Lot%20within%20an%20Action%20or%20Task-create%20new%20lot%20and%20lees%20lot.webp)
