# How to backdate an action

Backdating is an available feature in direct actions and work orders. This article contains:

- [Backdating actions](#actions)
- [Backdating work orders](#WO)
- [When can I backdate?](#when)

#### Backdating actions

You will see the Backdate action checkbox in the bottom left of the action page.

#### Backdating work orders

You can backdate the submission of work order tasks by choosing the option to Submit tasks as of a specific date and time.

![Backdate work order](https://support.innovint.us/hs-fs/hubfs/Backdate%20work%20order.png?width=491&height=180&name=Backdate%20work%20order.png)

#### When can I backdate?

The system will only allow backdating if it does not overlap with another movement action.

**Note:** Analysis and Custom Actions are not considered movement actions, and therefore can be backdated to any time in the history of a lot. Addition actions can only be backdated within the last 14 months (430 days). Movement actions can also be backdated to before those actions, but not before other movement actions.
