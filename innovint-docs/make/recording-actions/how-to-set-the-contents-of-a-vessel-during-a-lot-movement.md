# How to set the contents of a vessel during a lot movement

When completing a movement in an action or work order task, you need to specify the change of contents on each vessel involved in the action.

![How to set ending fill_select vessels](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20ending%20fill_select%20vessels.png?width=688&height=281&name=How%20to%20set%20ending%20fill_select%20vessels.png)

After you select your vessels, you will see the fields to set ending contents of vessels.

Choose from the drop-down menu to auto-fill

**OR**

Type value into the volume fields individually

![How to set ending fill_fill fields](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20ending%20fill_fill%20fields.png?width=688&height=473&name=How%20to%20set%20ending%20fill_fill%20fields.png)

Auto/bulk-fill options include:

- "Empty" all the vessels to zero contents![How to set ending fill_bulk options](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20ending%20fill_bulk%20options.png?width=150&height=175&name=How%20to%20set%20ending%20fill_bulk%20options.png)
- Keeping the contents as they currently are with "Current fill"
- Filling all vessels to they capacity assigned to them with "Vessel capacity"
- Enter a your own value to be applied across the entire selection of vessels (if you are low filling barrels, for example)

Some actions and tasks allow you set either the Ending Fill OR a Removed or Added volume:
