# Using a Custom Action or Custom Task

A Custom Action/Task is a flexible, editable action that can be applied to the entire winery, a single lot, or one or more vessels.

Custom Actions/Tasks are useful for activities not covered by InnoVint’s standard action list, such as sanitizing tanks, laying out barrels, scheduling staff meetings, and more.

This article covers the Custom Action/Task:

- [via Direct Action](#direct)
- [via Work Order](#workorder)
- [Reporting](#Report)

### via Direct Action

Find the Custom Action via the Lot details Record Action menu, or in the top navigation bar's global Record Action menu:

1. Click the pencil icon next to **Custom Action** to rename the action.
2. (Optional) Click **+ Add note** to include a note.
3. Enter details or instructions in the **Details** field.
4. Select whether to apply the action to:
   - The entire winery
   - A single lot
   - One or more vessels
5. (Optional) Backdate the action as needed.
6. Click **Create Action**.

![Custom Action_markup](https://support.innovint.us/hs-fs/hubfs/Custom%20Action_markup.png?width=670&height=299&name=Custom%20Action_markup.png)

### via Work Order

Find and add Custom Tasks when you [create a new work order](/hc/en-us/articles/360050001811-using-work-orders-in-innovint?hsLang=en):

1. Enter a title for the work order.
2. Click **+ Add task** and select **Custom task**.
3. Click the pencil icon next to **Custom task** to rename the task.
4. (Optional) Click **+ Add note** to include a note.
5. Enter details or instructions in the **Details** field.
6. Select whether to apply the task to:
   - The entire winery
   - A single lot
   - One or more vessels
7. Save the work order.

### Reporting on Custom Actions/Tasks

A custom action/task will show up in various reports.

- **The Winery Activity Feed**

  If you apply the action/task to a lot, vessels or the winery, it will show in the Winery Activity Feed. You can search the Winery Activity Feed by action *type*, and can filter to see only "Custom task".   If you export the Winery Activity Feed, you will see the edited action name and be able to filter for any specific Custom action names that you have used.
- **Custom Reports**
  If you apply the action/task to a lot, Custom Reports will show the name of the action/task, and allow you to filter for it in the action columns.
  ![Custom Action_Custom Report](https://support.innovint.us/hs-fs/hubfs/Custom%20Action_Custom%20Report.png?width=670&height=185&name=Custom%20Action_Custom%20Report.png)
- **Lot details History**

  An action or task applied to a lot will display in the Lot details > History widget or tab.
- **Vessel details History**

  An action or task applied to vessels will show up on the Vessel details > History.
