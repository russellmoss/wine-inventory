### Recording an Addition via Direct Action

If you don't see any dropdowns for additive inventory or batches, you may not have the Dry Goods module activated for your account. If you want to activate it, please contact Support@innovint.us for more information.  Or, check out how to record a Simple Addition [here](https://support.innovint.us/hc/en-us/simple-additions?hsLang=en).

#### 1. Select an additive and batch

![How to Record an Addition-select additive](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-select%20additive.webp?width=670&height=298&name=How%20to%20Record%20an%20Addition-select%20additive.webp)

- Select an additive from the dropdown list. This list is populated by your [Additive inventory](//innovint-6865708.hs-sites.com/hc/en-us/articles/115000825066-how-to-create-additives-and-additive-batches?hsLang=en). Click on ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360048181512Screen_Shot_2020-01-27_at_1.33.46_PM.png?width=119&name=support.innovint.ushcarticle_attachments360048181512Screen_Shot_2020-01-27_at_1.33.46_PM.png) (under the **Additives** header) to add multiple additives in the same action.
- Select a batch from the Batch # dropdown. This list is populated by the active (non-depleted) batches available for the selected additive. Click on ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360048181752Screen_Shot_2020-01-27_at_1.35.44_PM.png?width=99&name=support.innovint.ushcarticle_attachments360048181752Screen_Shot_2020-01-27_at_1.35.44_PM.png) (under the **Batches used** sub-header) to pull from multiple batches of the same additive.
- Adjust the default addition rate and instructions as needed per additive. Editing the addition rate will update the calculated values based on the new rate and fill of each vessel.
- The amount removed from each batch will automatically update to the sum of the amount added to each vessel in the action. Click on the blue pencil to manually adjust the amount removed if it is different than the total added amount. This is the amount that is removed from your batch in inventory.
- If a batch is now empty as a result of the action, check the 'Fully depleted' box to remove the batch from your active inventory for future additions.
- Note: For SO2 additions, the *Addition Rate* becomes the *Rate of SO2 to add*.
  ![How to Record an Addition-rate of so2](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-rate%20of%20so2.webp?width=670&height=217&name=How%20to%20Record%20an%20Addition-rate%20of%20so2.webp)

#### 2. Option: Add one or more lots within the same action

- Add a single lot and select from the dropdown or picker.
  ![How to Record an Addition-add single lot](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-add%20single%20lot.webp?width=670&height=180&name=How%20to%20Record%20an%20Addition-add%20single%20lot.webp)
- Select multiple lots in the picker. This prompts a slideover to filter, search, and select multiple lots.
  ![How to Record an Addition-mult lots](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-mult%20lots.webp?width=670&height=115&name=How%20to%20Record%20an%20Addition-mult%20lots.webp)

#### 3. Select vessels within each lot

- Bulk select all vessels across all lots

  ![How to Record an Addition-bulk vessels](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-bulk%20vessels.webp?width=670&height=183&name=How%20to%20Record%20an%20Addition-bulk%20vessels.webp)
- Select all vessels per lot
  ![How to Record an Addition-select all vessels](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-select%20all%20vessels.webp?width=670&height=268&name=How%20to%20Record%20an%20Addition-select%20all%20vessels.webp)
- Select one or more individual vessels per lot
  ![How to Record an Addition-select indv vessel](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-select%20indv%20vessel.webp?width=670&height=268&name=How%20to%20Record%20an%20Addition-select%20indv%20vessel.webp)

#### 4. Enter the amount added per additive

- Bulk apply an additive across all selected vessels in all lots. You can apply the InnoVint-calculated amount or enter the same amount for all vessels across all lots in the action.
  ![How to Record an Addition-enter amount per](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-enter%20amount%20per.webp?width=670&height=428&name=How%20to%20Record%20an%20Addition-enter%20amount%20per.webp)
- Bulk apply an additive across all selected vessels per individual lot. You can apply the InnoVint-calculated amount or enter the same amount for all vessels in a single lot.
  ![How to Record an Addition-bulk apply](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-bulk%20apply.webp?width=670&height=430&name=How%20to%20Record%20an%20Addition-bulk%20apply.webp)
- Enter a unique value per vessel
  ![How to Record an Addition-unique amount](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-unique%20amount.webp?width=670&height=293&name=How%20to%20Record%20an%20Addition-unique%20amount.webp)

**Tip**: To use a different addition rate per lot - If different from the default addition rate, update the additional rate for your first lot, then enter the actual amount added for each vessel. Next, scroll to the top of the page to adjust the addition rate, then add a second lot new lot to the action. The InnoVint-calculated values will update with the new addition rate, but any values previously entered for any lot will not change. For each new lot, you'll want to update the addition rate at the top of the page for InnoVint to recalculate the values.

### Editing an addition action

Submitted Addition actions are not subject to any dependent action restrictions, and can always be edited. You can also edit the date they were recorded using the [date edit.](https://support.innovint.us/hc/en-us/articles/208141233-how-to-edit-or-delete-recorded-actions?hsLang=en#edit-date)

![](https://support.innovint.us/hs-fs/hubfs/image-png-Jun-24-2026-09-58-49-2679-PM.png?width=3276&height=1634&name=image-png-Jun-24-2026-09-58-49-2679-PM.png)

If you are editing the quantity added to a lot, be sure to also ensure that the quantity removed from the additive is correct as well. This will update the remaining on-hand quantity of your additive.

![](https://support.innovint.us/hs-fs/hubfs/image-png-Jun-24-2026-10-00-20-5410-PM.png?width=3278&height=1642&name=image-png-Jun-24-2026-10-00-20-5410-PM.png)

### Creating an Addition Task in a Work Order

When assigning an addition task in a work order, the interface is nearly identical as the above. However there are some variations to consider.

- Batches are selected by the work order assignee when completing the task. To specify the use of a particular additive batch, we recommend entering those instructions in the task notes.
  ![How to Record an Addition-batch used](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-batch%20used.webp?width=670&height=311&name=How%20to%20Record%20an%20Addition-batch%20used.webp)
- Vessel selection includes an option to let cellar staff choose vessels, either as a bulk selection across all lots or per individual lot.
  ![How to Record an Addition-vessel selection](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-vessel%20selection.webp?width=670&height=108&name=How%20to%20Record%20an%20Addition-vessel%20selection.webp)

**Tip**: To use a different addition rate per lot - If different from the default addition rate, update the additional rate for your first lot, then enter the requested amount added for each vessel. Next, scroll to the top of the page to adjust the addition rate, then add a second lot new lot to the action. The InnoVint-calculated values will update with the new addition rate, but any values previously entered for any lot will not change. For each new lot, you'll want to update the addition rate at the top of the page for InnoVint to recalculate the values.

### Editing an Addition Task in a Work Order

To edit a created work order, click on 'Edit work order' at the top.

![How to Record an Addition-edit wo](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-edit%20wo.webp?width=670&height=61&name=How%20to%20Record%20an%20Addition-edit%20wo.webp)

Edit mode allows you to add or remove additives, lots, and vessels within the work order task.

![How to Record an Addition-edit options](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20an%20Addition-edit%20options.webp?width=670&height=388&name=How%20to%20Record%20an%20Addition-edit%20options.webp)
