# Primary Fermentation Report

The Primary Fermentation reportis a great way to track all of your vessels and lots going through Primary Fermentation. The default Lot Stage Filter shows all lots in the Settling, Cold Soak, and Fermenting stages but these stages can also be edited using the filter.

![Primary Fermentation Report-filter](https://support.innovint.us/hs-fs/hubfs/Primary%20Fermentation%20Report-filter.webp?width=688&height=387&name=Primary%20Fermentation%20Report-filter.webp)

Vessels are tracked individually and display the analysis data recorded on either the individual vessel or lot composite, whichever is the most recent.

Also shown are the 3 most recent dates for Additions and the last action recorded on the vessel.

Each column in the report can also be sorted by clicking on the header.

Click on the vessel code/icon or the lot code to open a new tab with the vessel or lot details page, respectively.
