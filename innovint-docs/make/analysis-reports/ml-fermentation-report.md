# ML Fermentation Report

This report structure is built on our custom reporting technology (find out how Custom Reports work [here](https://support.innovint.us/hc/en-us/custom-reports?hsLang=en)), and is throughly flexible to arrange the data you need to see when evaluating your wines.

By default, the ML Fermentation Report displays:

- all lots in the winery with the lot stages **ML** and **Aging**
- the first Barrel Down & last Topping action dates
- the 3 most recent readings for Sugar, Malic Acid, and Volatile Acidity per lot
- vessels & tags

These custom columns and filters are truly flexible and can also be altered to include or exclude different stages, additional data columns, and even analysis trendlines.

Use the Column and Filter selectors on the righthand of the screen to add more data points, analysis types or desired lot attributes.

![ML Ferm Report_overview](https://support.innovint.us/hs-fs/hubfs/ML%20Ferm%20Report_overview.webp?width=688&height=260&name=ML%20Ferm%20Report_overview.webp)

- Results can sorted by clicking on any of the headers within the report.
- Search for lots within the text filter to narrow down search results.
- Use filters to search for certain results (i.e. above or below a specific result threshold)
- If you edit the default columns and filters on this report, the change will not be saved when you leave the page.
- This report does not instantaneously refresh, but you can see the time stamp (approximately every hour) in the top right hand corner of the report.

Click on *Export* at the top to download the report as an excel file.

- Recent Sugar Readings include both Brix & Baumé and Residual Sugar & Glucose/Fructose analysis types.
- Recent Malic Readings include values for Malic Acid and Malic Acid - Chromatography.
- Recent Volatile Acidity Readings include values for Volatile Acidity (ie it does not display Acetic Acid results).
