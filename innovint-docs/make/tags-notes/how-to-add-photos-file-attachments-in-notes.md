# How to add Photos & File Attachments in Notes

Want to attach a BOL to a bond transfer in, or take a picture of the fruit quality when receiving fruit? You can do this using Notes on actions, tasks, lots, products, batches and more!

![Add photos and attachments to notes_Attach file button](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Add%20photos%20and%20attachments%20to%20notes_Attach%20file%20button.webp?width=450&height=217&name=Add%20photos%20and%20attachments%20to%20notes_Attach%20file%20button.webp)

This article covers how to add a Photo or File Attachment Note:

### Adding a Photo or File Attachment Note to Inventory Items

You can add a photo or file attachment note to any active or archived inventory items that support saving notes.

Available inventory items include Lots, Vessels, Dry Goods Products and Batches, Vineyards, Vineyard Contracts, and Vineyard Blocks.

Notes with photo or file attachments must be saved after the point of creation, i.e. you cannot attach your dry goods packing slip pdf to your new yeast product at the same time as you create this new yeast product in InnoVint. Just save your new item, and then click back into it to create a note with an attached file.

1. First, navigate to the inventory details page.
2. Scroll to the bottom left of the page, to the Notes section![Add photos and attachments to notes__lot details page_add note](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Add%20photos%20and%20attachments%20to%20notes__lot%20details%20page_add%20note.webp?width=655&height=364&name=Add%20photos%20and%20attachments%20to%20notes__lot%20details%20page_add%20note.webp)
3. Click on *+Add note*
4. Enter your text in the pop-up dialog box (optional)
5. Click on *Attach file* and choose your file (optional)
6. Click on *Add note* to submit

Voila!

**Note:**Photos will display as a thumbnail and can be clicked to view larger. All photos and files are added as an attachment (blue link text). Click on the attachment link to download the file to your computer; PDF files will open in a new browser tab.

### Add a Photo & File Attachment Note to Direct Actions and Work Order Tasks

***Photo and file attachment* notes can be added to work order tasks after work order creation and after recording a direct action.**

Only text notes are supported before the work order task is created or a direct action is recorded.

Add your desired photo or file attachment notes after the task/action has been submitted.

1. In the created work order task, or action details page, click on *+Add note.*
2. Enter your text in the pop-up dialog box (optional).
3. Click on *Attach file* and choose your file. You can only attach files or photos after a work order is created, and after the direct action is submitted.
4. Click on *Add note* to submit.

### Add a Photo or File Attachment Note via InnoApp

Photo and file attachment notes can be added to [lots, vessels, vineyards, blocks](#Lot-vessel-blocks), and [work order tasks](#wo_tasks) from within the mobile app. These notes will save offline per the usual Mobile App [offline functionality](https://support.innovint.us/hc/en-us/highlights-of-the-new-mobile-experience?hsLang=en#offline).

#### Lots, Vessels, Vineyards and Blocks

1. Go to the Lot, Vessel, Vineyard, or Block details page and tap on **Record Action**:
   ![Add photos and attachments to notes_InnoApp_Record action](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Add%20photos%20and%20attachments%20to%20notes_InnoApp_Record%20action.webp?width=225&height=230&name=Add%20photos%20and%20attachments%20to%20notes_InnoApp_Record%20action.webp)
2. Select "**Note**" from the action list and enter your note text, or click on "**+ Photo"** to choose an attachment, or select your camera to take a photo. *Please note that attaching photo notes on your mobile device also requires some text to save.*
3. Tap on "**+ Add Note**"
   ![Add photos and attachments to notes_InnoApp_Add Note_1](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Add%20photos%20and%20attachments%20to%20notes_InnoApp_Add%20Note_1.webp?width=215&height=404&name=Add%20photos%20and%20attachments%20to%20notes_InnoApp_Add%20Note_1.webp) ![Add photos and attachments to notes_Add Note_2](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Add%20photos%20and%20attachments%20to%20notes_Add%20Note_2.webp?width=215&height=405&name=Add%20photos%20and%20attachments%20to%20notes_Add%20Note_2.webp) ![Add photos and attachments to notes_Add Note_3](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Add%20photos%20and%20attachments%20to%20notes_Add%20Note_3.webp?width=215&height=405&name=Add%20photos%20and%20attachments%20to%20notes_Add%20Note_3.webp)
4. Once the text and/or file is uploaded, the new note will display on the lot or vessel.

2. Tap on the desired task.

4. Tap on "**+ Add Note"** at the bottom of the screen

#### Troubleshooting Tips

*Why is my file attachment not saving?*

- You can only have 100 characters in the file name, and having too many characters will cause the note submission to fail
- There is a file size limitation of 2.5MB - a file that is too large will also fail
- At this time, only text notes are supported *before* a work order task is created or a direct action is recorded. You can only add photo or file attachment notes after the task/action has been submitted.
  - This is generally the same for when you add inventory items, such as Dry Goods.  New Dry Goods or Vineyard Contracts should be created, and then you can add notes.
