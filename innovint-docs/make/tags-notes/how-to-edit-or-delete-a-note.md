# How to edit or delete a note

Recorded notes on lots, vessels, dry goods, vineyards, and actions can be edited or deleted by the person who created the note. When viewing the note in the desktop application, click the blue pencil to make changes.

You may not add a new file attachment to an existing note.  Please add a new note with the file attachment.  Find out more about attaching files to notes [here](https://support.innovint.us/hc/en-us/articles/360009672091-how-to-add-a-photo-note?hsLang=en).

![How to edit or delete a note_blue edit pencil](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/How%20to%20edit%20or%20delete%20a%20note_blue%20edit%20pencil.webp?width=247&height=364&name=How%20to%20edit%20or%20delete%20a%20note_blue%20edit%20pencil.webp)

Only the person that recorded the note can edit or delete it.
