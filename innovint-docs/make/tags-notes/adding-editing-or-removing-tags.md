# Adding, Editing, or Removing Tags

Tags are simple to add and remove, and can be edited at any time. Want to know about what tags are in InnoVint? Check out our intro to tags [here](https://support.innovint.us/hc/en-us/articles/204503439-what-are-tags-used-for-in-innovint-?hsLang=en).

This article covers:

- [How to add a new or existing tag](#How)
- [Where to go to add, edit or remove tags](#where)
  - [Add, edit or remove in bulk](#bulk)
  - [Options to add, edit or remove tags on individual items](#Single)
- [Add, edit or remove tags via InnoApp](#innoapp)
- [How to remove a tag from your tag menu](#remove)

### How to add a new or existing tag

Add an *existing* Tag: Start typing in the tag field. A drop-down list of existing tags will appear. Choose your tag(s) by clicking on the highlighted tag, or type it all the way in (tags are case sensitive) and add a comma, or hit TAB on your keyboard.

![Add,edit,remove tags_add existing](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_add%20existing.webp?width=688&height=164&name=Add%2cedit%2cremove%20tags_add%20existing.webp)

Add a *new* Tag: Type in your new Tag(s), separated by comma(s), or click on the blue highlighted line to select the new tag.

![Add,edit,remove tags_add new](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_add%20new.webp?width=528&height=182&name=Add%2cedit%2cremove%20tags_add%20new.webp)

Newly added tags may not immediately populate in your explorer menus. Refresh your browser to see new tags!

### Where to go to add, edit, or remove tags

#### **Add, edit or remove in bulk**

Tags for **Lots** and **Vessels** can be managed in **bulk** from the Lot and Vessel Explorer pages, under Manage Lots or Manage Vessels.

![Add,edit,remove tags_bulk manage](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_bulk%20manage.webp?width=212&height=221&name=Add%2cedit%2cremove%20tags_bulk%20manage.webp)

![Add,edit,remove tags_bulk edit](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_bulk%20edit.png?width=688&height=341&name=Add%2cedit%2cremove%20tags_bulk%20edit.png)

- To add a tag, start typing the desired tag name - existing tags containing typed text will appear.
- Continue to enter your desired text: it will populate on a new line highlighted in blue and underlined. Click on this blue highlighted line - or enter a comma to add more tags (new or existing). Note! Tags are case sensitive and will be treated as unique depending on case.
  ![Add,edit,remove tags_add new](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_add%20new.png?width=688&height=337&name=Add%2cedit%2cremove%20tags_add%20new.png)
- After you click on a tag or enter a comma, the new tag will populate in grey in the Added tag field. Repeat to add additional new or existing tags.
  ![Add,edit,remove tags_added tag grey](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_added%20tag%20grey.png?width=688&height=333&name=Add%2cedit%2cremove%20tags_added%20tag%20grey.png)

Newly added tags may not immediately populate in your explorer menus. Refresh your browser to see new tags!

#### **Options to add, edit or remove tags on individual items**

Tags forindividual items  (**Lots**, **Vessels**, **Dry Goods**, **Vineyards** and **Blocks)** can be added, edited, and/or removed from the item details page.

- **Lots -** Try the bulk update option above, or else update tags on single lots using the Lot Details page.
  - Existing lot: click on the blue pencil next to Tags.
    ![Add,edit,remove tag_single lot](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_single%20lot.webp?width=248&height=316&name=Add%2cedit%2cremove%20tag_single%20lot.webp)

- - New lot: add your tags at Lot creation. Lot Explorer > Add lot. Add your desired Tag(s) in the Lot attributes Tag field. Or include your tags in the Lot import template.

- **Vessels -** Try the bulk update option above via the Vessel Explorer, or use the *Vessel Details* page.
  - Updating an existing lot: click on the blue pencil next to Tags
    ![Add,edit,remove tags_indvessel](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tags_indvessel.png?width=688&height=208&name=Add%2cedit%2cremove%20tags_indvessel.png)

- - New vessel: add tags at Vessel creation. Vessel Explorer > Add vessel. Add your desired Tag(s) in the Vessel attributes Tag field. Or include your tags in the Vessel import template.

- **Dry Goods**

- - Products: Go to the Additive or Packaging Product details page > click on the blue pencil next to Tags
    ![Add,edit,remove tag_Dry Good Product](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_Dry%20Good%20Product.webp?width=240&height=343&name=Add%2cedit%2cremove%20tag_Dry%20Good%20Product.webp)
  - Batches: Go to the Batch details page > click on the blue pencil next to Tags
    ![Add,edit,remove tag_Dry Good Batch](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_Dry%20Good%20Batch.webp?width=240&height=307&name=Add%2cedit%2cremove%20tag_Dry%20Good%20Batch.webp)

- **Vineyards & Blocks**
  - *Existing vineyard source:* Use the Vineyard Details page > click on the blue pencil next to Tags.
    ![Add,edit,remove tag_Vineyard](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_Vineyard.webp?width=240&height=321&name=Add%2cedit%2cremove%20tag_Vineyard.webp)
  - *New vineyard:* Add tags at Vineyard creation. Vineyard Explorer > Add vineyard. Add your desired Tag(s) in the new Vineyard attributes Tag field

- - Add tags to vineyard blocks when adding new blocks, or on the block details page.

#### Add, edit or remove tags via InnoApp

Scan vessel QR codes (online or offline) to add, edit or update tags in the mobile app. Find out more [here](https://support.innovint.us/hc/en-us/how-to-tag?hsLang=en).

![Add,edit,remove tag_InnoApp](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_InnoApp.webp?width=240&height=366&name=Add%2cedit%2cremove%20tag_InnoApp.webp)

InnoApp doesn't yet support adding tags to vineyards or lots.

### How to remove a tag from your tag menu

In order to remove a tag completely from your winery tag menus, you need to remove it from all items in the winery (including archived items). Once a tag is removed from all vessels/lots/vineyards/dry goods, it should auto-disappear from your drop-down list. Tag history is not tracked in InnoVint.

For example, if you want to remove a tag that is generally used on lots, start in your Lot Explorer > Manage lots > Edit tags to bulk edit tags on lots. Select the lots you want the tag to be removed from by filtering for the applicable tag(s). You can search for all the lots carrying this tag by using the tag filter at the top (***be sure to include Archived lots*)**.

![Add,edit,remove tag_Remove1](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_Remove1.webp?width=688&height=359&name=Add%2cedit%2cremove%20tag_Remove1.webp)

Select all lots holding the tag you want to remove from your dropdown list, opt to remove tags and specify the tag(s) to remove, and then click on Save tags on lots.

![Add,edit,remove tag_Remove2](https://support.innovint.us/hs-fs/hubfs/Add%2cedit%2cremove%20tag_Remove2.webp?width=688&height=330&name=Add%2cedit%2cremove%20tag_Remove2.webp)

Refresh your browser and check for the tag in the tag menu. If the tag still exists in the drop down, you will need to check other items where the tag may be saved in the winery account: it could also be attached to vessels, vineyards, blocks, or dry goods.

You should check those explorers, and remove the tag wherever you find it.  The Vessel Explorer supports the bulk update and removal of tags, but vineyards, blocks and dry goods will require individual removal via an item details page.

If you think you have removed the tag form all items, but are still seeing it in the drop-down, we suggest double checking for Archived items.
