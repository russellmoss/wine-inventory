# How to Pull ETS Samples for Analysis

This article covers instructions for using the ETS integration.   Need more information on the integration or how to set it up? Check out the overview article [here](https://support.innovint.us/hc/en-us/articles/115001540266-ets-integration-overview?hsLang=en).

- [Pulling a sample via the Samples Explorer](#samplesexp)
- [Pulling a sample via Work Order request](#workorderrequest)
- [How to Download and Print Sample Labels](#downloadprintsamplelabels)
- [Viewing Requested ETS Samples](#viewsamples)
- [Frequently Asked Questions](#faqs)

**IMPORTANT**: A sample ***must***be created in InnoVint for ETS results to link to the lot or vessel analysis results. The sample code received by ETS *must be generated* in InnoVint.

### From the Samples Explorer

Go to the Samples Explorer ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360032813912Screen_Shot_2019-07-08_at_12.26.33_PM.png?width=20&name=support.innovint.ushcarticle_attachments360032813912Screen_Shot_2019-07-08_at_12.26.33_PM.png) in the left navigation bar and select ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360032796371Screen_Shot_2019-07-08_at_12.26.12_PM.png?width=133&name=support.innovint.ushcarticle_attachments360032796371Screen_Shot_2019-07-08_at_12.26.12_PM.png) in the upper right corner

1. Select the ETS client ID. More than one may exist in the same InnoVint account
2. Add any sample instructions or container specifics
3. Select the sample type: wine or juice
4. Select the panel(s) and/or analytics requested. If you require any analytics that are not listed, simply write them in the "Other" field.
5. Choose to pull a lot composite or to pull from individual vessels, and select your lot(s)/vessel(s). Each selected lot/vessel will create a unique sample in InnoVint.

InnoVint generates a 6-digit link code for each sample, followed by your auto-filled lot code. You can edit the lot code for samples if desired.

InnoVint defaults to download the sample labels upon creating the samples. You can un-select the box if you'd prefer to print the labels at a later time.

Click on ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360016020812Screen_Shot_2018-11-27_at_1.35.29_PM.png?width=100&name=support.innovint.ushcarticle_attachments360016020812Screen_Shot_2018-11-27_at_1.35.29_PM.png) to complete the action.

### Work Order Request

![How to Pull ETS Samples for Analysis-order](https://support.innovint.us/hs-fs/hubfs/How%20to%20Pull%20ETS%20Samples%20for%20Analysis-order.webp?width=688&height=31&name=How%20to%20Pull%20ETS%20Samples%20for%20Analysis-order.webp)

When creating a work order, the task form will look almost identical to the Direct Action above. The option to download the sample label(s) and print is available once the work order has been created.

![How to Pull ETS Samples for Analysis-download print](https://support.innovint.us/hubfs/How%20to%20Pull%20ETS%20Samples%20for%20Analysis-download%20print.bmp)

The work order task **MUST BE SUBMITTED** for the results to populate in the Lot or Vessel Analysis History. If a work order is submitted after sample results are posted by ETS, the results will populate the Lot or Vessel Analysis History with the next automatic import of ETS data.

### Download and Print Sample Labels

When downloading sample labels, a dialog box will open to select the type of label (sheet or ribbon), along with label size options. If you select sheet labels, it will expand so that you can select the placement of the labels on your sheet. All labels will download to PDF for you to print.  [This article](https://support.innovint.us/hc/en-us/articles/115000605043-which-labels-are-recommended-and-where-can-i-purchase-them-?hsLang=en) outlines which labels we recommend and where to find them.

### Viewing Samples

If samples are pulled via Direct Action, they will show in the Sample Explorer list with the green, full-container icons.

If they are assigned in a work order to be pulled, they will show up with empty container icons. Once the work order is submitted, the icon will turn green.

**Note:** Work Orders **MUST BE SUBMITTED** for the results to populate in the Lot or Vessel Analysis History. If a work order is submitted after sample results are posted by ETS, the results will populate the Lot or Vessel Analysis History with the next automatic import of ETS data.

### FAQ

#### **Q. What size/kind of labels do I need?**

*ETS analysis sheet sample labels should be printed on 2-1/3" x 3-3/8" size labels. We recommend Avery labels - product code #[8395](http://www.avery.com/avery/en_us/Products/Name-Badges/Name-Badges/Self-Adhesive-Removable-Name-Badge-Labels-PVC-Free_08395.htm?N=0&Ns=&refchannel=c042fd03ab30a110VgnVCM1000002118140aRCRD) (160 labels) or #[5395](http://www.avery.com/avery/en_us/Products/Name-Badges/Name-Badges/Self-Adhesive-Removable-Name-Badge-Labels-PVC-Free_05395.htm?N=0&Ns=&refchannel=c042fd03ab30a110VgnVCM1000002118140aRCRD) (400 labels).*

*If you use ribbon labels that accept pdf files (such as a Dymo or Zebra printer), we support 4" x 2-1/8" and 2-3/4" x 2-1/8" size labels.  See more details [here](https://support.innovint.us/hc/en-us/articles/115000605043-which-labels-are-recommended-and-where-can-i-purchase-them-?hsLang=en).*

#### **Q. Can I import historical ETS analyses?**

*Historical ETS results cannot be automatically imported via the ETS Integration feature. This is because InnoVint generates a unique code for each sample that must be scanned by ETS to link the results to your account. However, you can import analyses via csv file using our [Analysis Import feature](//innovint-6865708.hs-sites.com/hc/en-us/articles/115002687291-how-to-import-analyses-via-csv?hsLang=en). This will allow you to upload and backdate results with the ETS logo.*

#### **Q. Does the ETS integration import my berry sample results into my block details page?**

*Currently, the ETS integration only works with Fruit and Juice/wine lots. We hope to bring this functionality in the future. For now, you will have to either manually input these analysis results on the [Block details page](//innovint-6865708.hs-sites.com/hc/en-us/articles/360026806092-how-to-record-and-track-data-and-analyses-on-vineyard-blocks?hsLang=en) or [record them on Fruit lots](//innovint-6865708.hs-sites.com/hc/en-us/articles/212998763-how-to-track-vineyard-data-in-fruit-lots?hsLang=en).*

#### **Q. Can I bulk archive my old samples in the Samples Explorer?**

*No, currently, we recommend archiving completed samples as you go.  Archiving samples does not impact the analyses recorded on the lot or vessel, and must be completed via the Sample details page:*

*![ETS archive sample](https://support.innovint.us/hs-fs/hubfs/ETS%20archive%20sample.png?width=688&height=137&name=ETS%20archive%20sample.png)*

#### **Q. Where do I go to print my sample labels?**

*To download and print your sample labels, go to the Sample Explorer and select the sample for which you'd like to print a label. Click on the link in the top right corner of the slide-over, Print ETS Label. If the sample was requested via work order, you can also download sample labels from within the work order task.*

#### **Q: My labels are not aligning correctly when I print from the PDF. How do I check my settings?**

*Make sure that the "Fit to Page" box is NOT selected. This will print true to the size of the page.*
