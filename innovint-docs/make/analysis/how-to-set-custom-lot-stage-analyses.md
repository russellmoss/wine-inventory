# How to set Custom Lot Stage Analyses

- [What is Custom Lot Stage Analyses?](#what)
- [How do I set my Custom Analyses?](#How)
- [FAQ](#faq)

#### What is Custom Lot Stage Analyses?

InnoVint automatically populates certain analysis displays with a set of Analysis Types based on the Lot Stage. Your InnoVint winery account has these preset with a selection of default analyses (see which ones [here](#Default)).

![How to set Custom Lot Stage Analyses-dashboard](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20Custom%20Lot%20Stage%20Analyses-dashboard.webp?width=688&height=152&name=How%20to%20set%20Custom%20Lot%20Stage%20Analyses-dashboard.webp)

With Custom Lot Stage Analysis, you can choose up to seven (7) analyses to display for each stage.

Once saved, these selected analyses will display for all users within your winery:

- on Lot and Vessel details in the main InnoVint browser application
- on Lot and Vessel details in the Mobile App
- in [Tank Maps](https://support.innovint.us/hc/en-us/tank-maps?hsLang=en) (on the hover labels, when "Stage Dependent Analyses" is toggled on)

This customization allows you to choose Glu/Fru instead of RS, Acetic Acid instead of VA, ethanol instead of alcohol, add VA to your Cold Soak stage, etc. You get to choose what best fits your panels and protocols!

#### How do I set my Custom Analyses?

Team member and Admin users can update the analysis info displayed via the Analysis display menu in Settings:

![How to set Custom Lot Stage Analyses-display menu](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20Custom%20Lot%20Stage%20Analyses-display%20menu.webp?width=688&height=356&name=How%20to%20set%20Custom%20Lot%20Stage%20Analyses-display%20menu.webp)

Once opened for editing, you can remove or re-order the analyses as desired:

![How to set Custom Lot Stage Analyses-remove reorder](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20Custom%20Lot%20Stage%20Analyses-remove%20reorder.webp?width=688&height=283&name=How%20to%20set%20Custom%20Lot%20Stage%20Analyses-remove%20reorder.webp)

Select any available analysis you want to display at any of the lot stages in InnoVint.

![How to set Custom Lot Stage Analyses-set display](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20Custom%20Lot%20Stage%20Analyses-set%20display.webp?width=688&height=362&name=How%20to%20set%20Custom%20Lot%20Stage%20Analyses-set%20display.webp)

Click on "Save analyses" and ... any stage dependent analysis displays will automatically update:

![How to set Custom Lot Stage Analyses-automatic update](https://support.innovint.us/hs-fs/hubfs/How%20to%20set%20Custom%20Lot%20Stage%20Analyses-automatic%20update.webp?width=688&height=153&name=How%20to%20set%20Custom%20Lot%20Stage%20Analyses-automatic%20update.webp)

**Q. How many analyses can I choose?**

*A. At least one analysis must be selected/displayed, but you can pick up to seven (7).*

**Q. Can I update the Case Good Lot Details Dashboard?**

*A. No, at this time, the custom Lot Stage analysis only applies for Juice and Wine lots - not Case Good Lots or Fruit Lots.*

**Q. Who can edit these displays?**

*A. Admins, Team Members and Team Member Cannot Submit all have access to update these displays.*

**Q. Who sees the updated analyses?**

*A. Everyone! If a user updates the default analyses, the change will apply to the entire winery and all its users.*
