# How to Record Analysis via Direct Action or Work Order Task

### Recording Analysis

Analysis data can be recorded on one or more fruit, juice/wine lots, case good lots, or on one or more individual vessels.

This article covers:

- [Recording analysis as a direct action](#direct_action)
- [Scheduling analysis in a work order](#work_order)
- [How to edit or delete analysis data](#edit)

For other options on how to record Analysis data in InnoVint (e.g. [Analysis Import](https://support.innovint.us/hc/en-us/articles/115002687291-how-to-import-analyses-via-csv?hsLang=en), our [ETS Integration](https://support.innovint.us/hc/en-us/articles/115001540726-ets-integration-step-by-step-instructions-?hsLang=en), or [InnoApp](https://support.innovint.us/hc/en-us/mobile-app-how-to-record-analysis?hsLang=en)), please see our overview article: [Options to Record Analysis Data](https://support.innovint.us/hc/en-us/articles/360006665732-options-to-record-analysis-datachoose?hsLang=en).

### How to edit or delete analysis

#### To edit or delete analysis on an individual Lot

Go to your Lot details page, the Analysis tab, and select All analyses in a list:

![How to Record Analysis via Direct Action or Work Order Task-delete order](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-delete%20order.webp?width=688&height=31&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-delete%20order.webp)

![How to Record Analysis via Direct Action or Work Order Task-edit delete](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-edit%20delete.webp?width=688&height=220&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-edit%20delete.webp)

The Edit Analysis window only allows you to change the analysis value. To change any other attributes (e.g. date, source, etc.), you will need to delete and re-enter the analysis data.

![How to Record Analysis via Direct Action or Work Order Task-edit window](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-edit%20window.webp?width=358&height=176&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-edit%20window.webp)

You will see a confirmation screen before InnoVint deletes your selected analysis.

![How to Record Analysis via Direct Action or Work Order Task-confirm delete](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-confirm%20delete.webp?width=360&height=177&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-confirm%20delete.webp)

To delete an entire Analysis action:

Find the Analysis action detail page, either via searching the Winery Activity Feed, or the Lot details History tab.

![How to Record Analysis via Direct Action or Work Order Task-delete order2](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-delete%20order2.webp?width=688&height=31&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-delete%20order2.webp)

![How to Record Analysis via Direct Action or Work Order Task-winery activity feed](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-winery%20activity%20feed.webp?width=688&height=178&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-winery%20activity%20feed.webp)

Within the Action details page, you can delete the entire action, edit the Lab source on the action, or edit the individual analyses on the action.

![How to Record Analysis via Direct Action or Work Order Task-delete entire action](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-delete%20entire%20action.webp?width=688&height=219&name=How%20to%20Record%20Analysis%20via%20Direct%20Action%20or%20Work%20Order%20Task-delete%20entire%20action.webp)
