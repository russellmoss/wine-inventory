# Edit and Delete Analysis Data

This article covers how to edit and delete individual analysis values as well as entire Analysis actions.

- [Edit individual analysis results](#edit_analysis)
- [Delete individual analysis results](#Delete_analysis)
- [Delete analysis actions](#Delete_action)

### Edit individual analysis results

Find the Edit analysis window via the Analysis action details page:

*![Edit and Delete Analysis Data-action details](https://support.innovint.us/hs-fs/hubfs/Edit%20and%20Delete%20Analysis%20Data-action%20details.webp?width=622&height=422&name=Edit%20and%20Delete%20Analysis%20Data-action%20details.webp)*

... or via the Lot details page Analysis tab using the All analyses in a list or the Sugar/temp screen:

*![Edit and Delete Analysis Data-tab](https://support.innovint.us/hs-fs/hubfs/Edit%20and%20Delete%20Analysis%20Data-tab.webp?width=623&height=308&name=Edit%20and%20Delete%20Analysis%20Data-tab.webp)*

You can edit analyses anywhere you find the edit button.

The Edit analysis window allows you to change the recorded date/time, analysis value and the analysis unit. To change any other attributes (e.g. source), you will need to delete and re-enter the analysis data.

![Edit and Delete Analysis Data-edit](https://support.innovint.us/hs-fs/hubfs/Edit%20and%20Delete%20Analysis%20Data-edit.webp?width=455&height=222&name=Edit%20and%20Delete%20Analysis%20Data-edit.webp)

### Delete individual analysis results

Access the Lot details page Analysis tab, and find All analyses in a list:
