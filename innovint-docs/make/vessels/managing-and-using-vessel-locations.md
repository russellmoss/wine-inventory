# Managing and Using Vessel Locations

This feature allows users to add location information to any vessel. Users can create an unlimited number of vessel locations or areas within InnoVint, then add location information to each vessel. Users can add multiple buildings (warehouse, barrel room, etc) and sub-areas within those buildings (Cold side, Bay 1, etc), and drill down locations to the exact vessel position (Aisle 15, Rack 4, Position 2). Alternatively, users can chose to track vessels by general location, such as "North Wall of Cave 1" or "Row 12".

### How to Create Vessel Position Settings

Users wishing to track individual vessel positions should set these parameters first. Think of a vessel position as the exact coordinates of where a single barrel sits in your cellar - perhaps it is in the first stack, two barrels high, on the left side of the rack. This setting allows you to customize the wording for these coordinates.

1. Go to the Settings tab ![](https://support.innovint.us/hs-fs/hubfs/Screen%20Shot%202022-03-20%20at%2012-31-00%20PM-png.png?width=25&name=Screen%20Shot%202022-03-20%20at%2012-31-00%20PM-png.png) at the bottom of your left-hand Navigation Bar
2. Click on Vessel locations
3. Select the best terms for describing exact dimensions for vessel positions in your winery

*Note*: You can't use the same term to describe more than one dimension (that would be confusing!). The system defaults to Aisle, Rack and Position, but you can change this at any time.

### Managing and Using Vessel Locations-settings

These terms will be used when you are setting the location on each vessel and select *Track position*.

### How to Create Vessel Locations and Areas

The next step is to create locations and areas where vessels are stored.

![Managing and Using Vessel Locations-vessel](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-vessel.webp?width=306&height=105&name=Managing%20and%20Using%20Vessel%20Locations-vessel.webp)

**Vessel Locations** are the spaces that contain your vessels. You may consider your entire cellar a single vessel location. Or, maybe you define the east side of the cellar as one vessel location, and the west side as another. The smallest 'group' of vessels is a vessel location.

**Areas** and sub-areas are different (and entirely optional) - they *contain* vessel locations. For example, you may create an area named *Warehouse 1*, a sub-area named *Bay 2,* which contains vessel locations of *Aisle 4, Aisle 5 and Aisle 6*.

Users build the location hierarchy best suited for their winery. Users can set these locations to be granular or a high-level overview.

#### To create vessel locations and areas:

- Go to the Vessel Explorer Page
- Click on the Manage vessels button and select **Manage vessel locations**
  **![Managing and Using Vessel Locations-manage tab](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-manage%20tab.webp?width=256&height=305&name=Managing%20and%20Using%20Vessel%20Locations-manage%20tab.webp)**
- Click **+ Add vessel location** to create new vessel locations, areas, and sub-areas
- Parent locations can be created first as 'Areas', then chosen from the drop-down when creating sub-areas or individual vessel locations

![Managing and Using Vessel Locations-areas](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-areas.webp?width=346&height=329&name=Managing%20and%20Using%20Vessel%20Locations-areas.webp)

Users can view Parent areas and utilize the caret to see sub-areas & locations.

![Managing and Using Vessel Locations-caret](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-caret.webp?width=432&height=403&name=Managing%20and%20Using%20Vessel%20Locations-caret.webp)

### Set and Update Vessel Locations

Users can set an individual vessel location, down to the position, in two places via the main desktop browser App. Find out more about updating vessel locations on the Mobile App [here](https://support.innovint.us/hc/en-us/mobile-app-how-to-update-vessel-location?hsLang=en).

#### Via the Vessel Details Page (update a single vessel location)

- On an individual Vessel Details page under Attributes, click on the blue pencil by *Location:*
  *![Managing and Using Vessel Locations-location](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-location.webp?width=262&height=194&name=Managing%20and%20Using%20Vessel%20Locations-location.webp)*
- Choose a new *Location* from the drop-down menu
  ![Managing and Using Vessel Locations-location drop down](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-location%20drop%20down.webp?width=688&height=228&name=Managing%20and%20Using%20Vessel%20Locations-location%20drop%20down.webp)
- Enter in a new location, and position (if applicable) and **Save**.

#### Via the Vessel Explorer page (update multiple vessel locations)

- Go to the Vessel Explorer > Manage Vessels > Edit Location on Vessels
  ![Managing and Using Vessel Locations-explorer pg](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-explorer%20pg.webp?width=688&height=132&name=Managing%20and%20Using%20Vessel%20Locations-explorer%20pg.webp)
- Select the Vessels to update
  ![Managing and Using Vessel Locations-vessels to update](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-vessels%20to%20update.webp?width=302&height=161&name=Managing%20and%20Using%20Vessel%20Locations-vessels%20to%20update.webp)
- Select a new Location from the drop-down menu to change or add.
  ![Managing and Using Vessel Locations-dropdown change add](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-dropdown%20change%20add.webp?width=688&height=228&name=Managing%20and%20Using%20Vessel%20Locations-dropdown%20change%20add.webp)
  - Utilize the "X" to remove or clear any location setting.
    ![Managing and Using Vessel Locations-x to remove](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-x%20to%20remove.webp?width=688&height=125&name=Managing%20and%20Using%20Vessel%20Locations-x%20to%20remove.webp)
- Click **Update Location**

### View Vessel Locations

- Locations can be viewed and exported from the Vessel Explorer. They can also be easily searched for using the *location* filter at the top.
- Vessel locations are included in the Vessel at Point in Time export (this reports on vessel location at the requested point in time).
- Vessel locations are available on any Vessel details page or on any Lot details page > Vessels tab. There is always an option to see specific positions by clicking on *Show position.*
- The location filter is available on any vessel picker page.
- Vessel locations are included on InnoApp, on the [Vessel Details page](https://support.innovint.us/hc/en-us/vessel-explorer-page?hsLang=en#Details).

### Delete Vessel Locations

To delete general vessel locations, go to the Vessel Explorer page > Manage Vessels > Manage vessel locations > Edit vessel location > Delete location.

![Managing and Using Vessel Locations-delete](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-delete.webp?width=319&height=416&name=Managing%20and%20Using%20Vessel%20Locations-delete.webp)

If there are vessels currently tagged with this location, users will see a pop-up indicating such.

![Managing and Using Vessel Locations-delete tags](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-delete%20tags.webp?width=418&height=249&name=Managing%20and%20Using%20Vessel%20Locations-delete%20tags.webp)

If a location or area has ever had vessels recorded on it, you will not be able to delete it, and will see an error message.

![Managing and Using Vessel Locations-error message](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-error%20message.webp?width=280&height=102&name=Managing%20and%20Using%20Vessel%20Locations-error%20message.webp)

We recommend either re-naming it and reusing it as an additional vessel location, or else renaming it to ZZ-Do not use, which will keep the location at the bottom of your Vessel Location manager, and difficult to accidentally utilize.

You will be able to delete areas and locations that were never utilized.

### Vessel Location History

Users can see the history of where their vessels have been within their production space.

From the Vessel Details page, click on the "Vessel location history" link under History:

![Managing and Using Vessel Locations-history](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-history.webp?width=449&height=435&name=Managing%20and%20Using%20Vessel%20Locations-history.webp)

Users will see a pop-up with a history of everywhere this vessel has been located:

![Managing and Using Vessel Locations-pop up history](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-pop%20up%20history.webp?width=688&height=295&name=Managing%20and%20Using%20Vessel%20Locations-pop%20up%20history.webp)

Vessel location history will be logged starting with the first location change submitted *after* 6/6/2022. If a vessel currently has location info on it, and you would like that recorded in the vessel location history, you can do a one-time mass-location-save:

- Go to the Vessel Explorer page
- Under the Manage Vessels drop-down, select Edit location on vessels
- In the vessel selector, select ALL vessels, or break up the vessel list into chunks if you have many vessels
- On the Bulk Update Vessel Locations screen, scroll to the bottom and click Update Location. Although no vessel locations were actually changed, InnoVint will save all selected vessel locations and will then populate the vessel location history.

Any vessel location tracking prior to 6/6/2022 will not be included in location history.

### FAQs

**Q: When do I use an Area vs Vessel Location?**

*A. Areas are more general and individual vessels cannot be assigned/located to them. This feature is useful if your winery has different 'layers' of specificity.*

*For example:*

***Area***

*Warehouse*

***Vessel Locations***

*1. Warehouse North*

*2. Warehouse South*

*Users cannot assign individual vessels to the ‘Warehouse’ as a location. It is a more general area to which Warehouse North and Warehouse South belong.*

![Managing and Using Vessel Locations-area1](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-area1.webp?width=468&height=148&name=Managing%20and%20Using%20Vessel%20Locations-area1.webp)

*Vessel Locations is the specific place that your vessel lives. You may consider your entire cellar a vessel location. Or, maybe you define the east side of the cellar as one vessel location, and the west side as another. The smallest 'group' of vessels is a vessel location.*

![Managing and Using Vessel Locations-location2](https://support.innovint.us/hs-fs/hubfs/Managing%20and%20Using%20Vessel%20Locations-location2.webp?width=468&height=127&name=Managing%20and%20Using%20Vessel%20Locations-location2.webp)

**Q: Which reports/exports feature Vessel Location?**

*A: Users can currently find vessel locations on the Vessel Explorer export, or on the Vessel at Point in Time export.*

**Q: Can I enter in a vessel position with a letter?**

*A: No, we currently support only numerical values. No letters or symbols may be used at this time.*

**Q: Can I give two different vessels the same position within the same vessel location?**

*A: Yes, in order to provide maximum flexibility, InnoVint allows users to assign two vessels with the same exact position (even though it’s probably not what’s physically happening).*

**Q: Can all member capabilities set and edit vessel locations?**

*A: Admins, Team Members, and Team Members Cannot Submit Work Orders can add and edit vessel locations, as well as change the Vessel Location descriptors in Settings. Read Only user cannot add or edit locations on vessels.*
