# Change or Edit Vessel Details

After creation of a vessel, if you find that one or more of the vessel characteristics are incorrect, have changed, or need to be updated, you can edit the vessel details to correct the vessel information.

This article covers how to:

- [Edit individual vessels](#individually)
- [Edit multiple vessels in bulk](#bulk)

### Edit individual vessels from the Vessel Details page

From the **Vessel Explorer** page, select the vessel you'd like to edit. Go to More in the top right corner.

![Change or Edit Vessel Details-explorer pg](https://support.innovint.us/hs-fs/hubfs/Change%20or%20Edit%20Vessel%20Details-explorer%20pg.webp?width=688&height=190&name=Change%20or%20Edit%20Vessel%20Details-explorer%20pg.webp)

Select your option from the dropdown.

- Edit vessel attributes

![Change or Edit Vessel Details-edit attributes](https://support.innovint.us/hs-fs/hubfs/Change%20or%20Edit%20Vessel%20Details-edit%20attributes.webp?width=604&height=520&name=Change%20or%20Edit%20Vessel%20Details-edit%20attributes.webp)

- Change vessel code

![Change or edit vessel details -change code](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-change%20code.webp?width=667&height=279&name=Change%20or%20edit%20vessel%20details%20-change%20code.webp)

- Change vessel capacity

![Change or edit vessel details -change capacity](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-change%20capacity.webp?width=552&height=243&name=Change%20or%20edit%20vessel%20details%20-change%20capacity.webp)

- Change vessel type

![Change or edit vessel details -change type](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-change%20type.webp?width=557&height=267&name=Change%20or%20edit%20vessel%20details%20-change%20type.webp)

### Edit multiple vessels in bulk

From the **Vessel Explorer**, click on the Manage Vessels button in the top right corner.

![Change or edit vessel details -mult vessel in bulk](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-mult%20vessel%20in%20bulk.webp?width=688&height=214&name=Change%20or%20edit%20vessel%20details%20-mult%20vessel%20in%20bulk.webp)

Select your option from the dropdown.

- Edit owners on vessels

![Change or edit vessel details -edit owners](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-edit%20owners.webp?width=688&height=423&name=Change%20or%20edit%20vessel%20details%20-edit%20owners.webp)

- Edit tags on vessels

![Change or edit vessel details -edit tags](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-edit%20tags.webp?width=688&height=387&name=Change%20or%20edit%20vessel%20details%20-edit%20tags.webp)*Note: "Remove all tags and replace with new tags" removes ALL tags on the vessel, while "Remove tags" only removes the tag selected.*

- Edit capacity on vessels

![Change or edit vessel details -edit capacity](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-edit%20capacity.webp?width=688&height=378&name=Change%20or%20edit%20vessel%20details%20-edit%20capacity.webp)

- Edit barrel attribute on vessels

![Change or edit vessel details -edit attr](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-edit%20attr.webp?width=688&height=467&name=Change%20or%20edit%20vessel%20details%20-edit%20attr.webp)

- Archive or unarchive vessels

![Change or edit vessel details -archive unarchive](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-archive%20unarchive.webp?width=688&height=336&name=Change%20or%20edit%20vessel%20details%20-archive%20unarchive.webp)

*Note: Vessels must be empty before being archived.*

- Print QR code labels for vessels

![Change or edit vessel details -print](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-print.webp?width=688&height=404&name=Change%20or%20edit%20vessel%20details%20-print.webp)

- Delete vessels

![Change or edit vessel details -delete](https://support.innovint.us/hs-fs/hubfs/Change%20or%20edit%20vessel%20details%20-delete.webp?width=688&height=322&name=Change%20or%20edit%20vessel%20details%20-delete.webp)

*Note: Only vessels with no vessel history are eligible for deletion.*
