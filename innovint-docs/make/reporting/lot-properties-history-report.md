# Lot Properties History Report

The Lot Properties History Report gives you a winery-level view of all lot property and attribute changes over time. Use this report to audit changes to lot properties including lot code, name, color, style, stage, tax class or archived status, as well as any Custom Attributes.

📌 Note: Neither orange Owner tag nor blue Tag changes are included in this report as they are not tracked as of a point in time.

**Accessing the Report**

You can access the Lot Properties History Report in two ways:

- **Lot level:** Navigate to the **History** tab on any Lot Details page to see property changes for that specific lot.
- **Winery level:** Find the full winery-level report in the **Report Explorer** under **Activity Reports**.

**Filtering the Report**

The Lot Properties History Report includes filters to help you narrow down results. Filters are always visible at the top of the report.

- **Lot** — Search by lot name or lot code to find changes for a specific lot.
- **Property** — Filter by the type of property that was changed. You can choose to match **any**, **all**, or **none** of your selected properties. Options include Lot Code, Lot Name, Lot Color, Lot Style, Stage, Expected Yield, Tax Class, Archived status, and any Custom Attributes configured for your winery. Only properties that appear in your report will be listed.
- **Date Range** — Filter changes to a specific time period.
- **By** — Filter by the user who made the change. You can choose to match **any** or **none** of your selected users. Only users who appear in your report will be listed.

💡 **Tip: How the Any vs. All filter affects your results**

When filtering by multiple properties, the match option you choose changes what results are returned:

- **Any** returns all recorded changes for your selected property types — for example, filtering for Lot Name and Stage will return every instance where either of those properties was changed. ![052126_Lot Properties History Report_ANY filter_Annotated](https://support.innovint.us/hs-fs/hubfs/052126_Lot%20Properties%20History%20Report_ANY%20filter_Annotated.png?width=670&height=310&name=052126_Lot%20Properties%20History%20Report_ANY%20filter_Annotated.png)
- **All** returns only results where all of your selected properties were changed at the same time — for example, filtering for Lot Name and Stage will only return instances where both were updated simultaneously. ![image-png-May-20-2026-03-27-54-7147-PM](https://support.innovint.us/hs-fs/hubfs/image-png-May-20-2026-03-27-54-7147-PM.png?width=670&height=232&name=image-png-May-20-2026-03-27-54-7147-PM.png)
