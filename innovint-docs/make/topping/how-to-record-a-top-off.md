# How to Record a Top Off

## The Top Off action/task is used to completely fill vessels that are only partially filled, and requires a volume change on all involved vessels. This is different from Topping, which does not change the fill level of topped lots.

This article includes:

[About the Top Off feature](#About)

[How to Record a Top Off action](#how-to)

[Tips and Video](#Tips)

### About the Top Off feature

- Top Off is both an action and a task
- Top Off can be accessed via the 'record action' dropdown in the top navigation bar or the lot details page, or added as a task to a work order
- Top Off is available to all accounts
- Top Off and Topping are different
  - **Use a Top Off** action/task when filling partial vessels to full capacity. For example, you may have barrels partially filled during a white fermentation. Once the fermentation finishes and you are ready to top up the barrels, you will want to use the Top Off action or task to fill them up to capacity.
  - **Use Topping** action/task instead if maintaining inventory volume (e.g., compensating for evaporation) when doing regular topping rounds. Here is a link to the [Topping action article](https://support.innovint.us/hc/en-us/articles/204177099-using-the-topping-feature?hsLang=en).

### How to Record a Top Off action

#### 1) Complete the "**Top Off from"** section

Select the vessel that you are removing volume from.

In this case, we are topping off a partial barrel from other kegs. Specify the amount that was removed from this vessel and used to top off the others.

![How to Record a Top Off-from](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20a%20Top%20Off-from.webp?width=688&height=223&name=How%20to%20Record%20a%20Top%20Off-from.webp)

#### 2) Complete the **Top Off to** section

Select the lot(s) and vessel(s) that were filled. Each vessel must show a change in volume.

You can manually fill in the cells under the Add or Ending Fill column, or you can use the Ending Fill dropdown to apply a volume to all vessels in bulk.

![How to Record a Top Off-to](https://support.innovint.us/hs-fs/hubfs/How%20to%20Record%20a%20Top%20Off-to.webp?width=688&height=260&name=How%20to%20Record%20a%20Top%20Off-to.webp)

#### 3) Review the summary for accuracy

Check your gains/losses.  Add a note if you wish, then record the Top Off action.

### Tips & Video

- The Top Off action only allows you to remove volume from one lot at a time. You can use that same lot to top off multiple lots and vessels within the same action. Click on ![](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/support.innovint.ushcarticle_attachments360049424031Screen_Shot_2020-02-10_at_9.25.02_AM.png?width=71&name=support.innovint.ushcarticle_attachments360049424031Screen_Shot_2020-02-10_at_9.25.02_AM.png) under the **Top Off to** header to add more lots to be topped off.
- The Top Off action is different to the Topping action in that there is no defined option to add a breakdown vessel. In order to add a breakdown vessel or lot, just use the option to +Add new lot, and select the option to "Retain lot code".  Specify your breakdown vessel and fill it as required.
