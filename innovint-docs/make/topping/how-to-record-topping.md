# How to record Topping

## Use the Topping feature to deplete your topping lot, but not record a volume change on the topped lots. This is different from a Top Off, which requires a volume change for topped vessels.

This article includes:

[About the Topping feature](#about)

[How to record a Topping action/task](#how-to)

### About the Topping feature

- Topping is an action and a task
- Topping can be accessed via the 'record action' dropdown in the top navigation bar or the lot details page, or added as a task to a work order
- Topping is available to all accounts
- Topping and Top off are different

- - **Use Topping** when maintaining inventory volume (e.g., compensating for evaporation) during regular topping rounds.
    - The volume of the vessel(s) being topped via a [Topping](https://support.innovint.us/hc/en-us/articles/204177099-using-the-topping-feature) action/task will NOT increase.
    - The Topping action/task tops complete Lots and not individual vessels, and assumes that all vessels in that lot are topped proportionally. This impacts [lot composition](https://support.innovint.us/hc/en-us/articles/205493245-how-are-topping-wines-blended-into-topped-lots-?hsLang=en).
    - InnoVint assumes that the volume used to top each vessel is replacing volume lost due to evaporation. Find out more about this [here](https://support.innovint.us/hc/en-us/articles/204847635-is-the-fill-increasing-in-topped-vessels-?hsLang=en).

- - **Use a Top Off** action/task instead when filling partial vessels to full capacity. For example, you may have barrels partially filled during a white fermentation. Once the fermentation finishes and you are ready to top up the barrels, you will want to use the [Top Off](https://support.innovint.us/hc/en-us/articles/115002951483-using-the-top-off-feature?hsLang=en) action or task to fill them up to capacity.

### How to Record a Topping action

An individual lot or multiple lots can be topped with the same wine within one task or action. The steps to topping are:

#### 1) Choose one or more lots to be topped

Choose one or more wines that will be topped with a single lot. All vessels in the selected lot(s) will be included in the action and topped proportionately.

Find out more about how fill level changes work [here](https://support.innovint.us/hc/en-us/articles/204847635-is-the-fill-increasing-in-topped-vessels-?hsLang=en) and how composition changes work [here](https://support.innovint.us/hc/en-us/articles/205493245-how-are-topping-wines-blended-into-topped-lots-?hsLang=en).

#### 2) Record topping details

Add any specific details or instructions that you may want to include with the action or task.

#### 3) Select your topping wine

You can top with the same lot you are topping, or an entirely different lot. Only one topping lot can be used per task or action.

Specify the vessels utilized for topping, and either the Ending Fill, or the amount Removed from the vessel.

![How to record Topping-action](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20Topping-action.webp?width=688&height=462&name=How%20to%20record%20Topping-action.webp)

To use a second topping lot, add a new Topping task to the work order, or complete a second Topping action.

#### 4) Add breakdown vessels if used

This is optional, and breakdown vessels can be partial or filled to capacity.

![How to record Topping-breakdown vessel](https://support.innovint.us/hs-fs/hubfs/How%20to%20record%20Topping-breakdown%20vessel.webp?width=688&height=434&name=How%20to%20record%20Topping-breakdown%20vessel.webp)

#### 5) Review the Summary for accuracy

Make sure all the information looks correct.

In the Topping action (but not the task), you can also preview the lot composition to make sure you are not going outside compliance parameters for vintage, variety, etc.

Don't forget to backdate the action if you need.

Once everything is completed, press "Record Topping" to record the action.
