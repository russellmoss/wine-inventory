# How to print QR Code Vessel Labels

QR codes can be printed in bulk for many vessels at once or one at a time.

Looking for lot labels? Find out about printing lot labels [here](https://support.innovint.us/hc/en-us/articles/360002989051-how-do-i-generate-lot-labels-to-print-?hsLang=en).

This article covers:

- [How to print QR codes labels in bulk](#bulk)
  - [for sheet labels](#sheet)
  - [for ribbon label printers](#ribbon)
- [How to print QR code labels for individual vessels](#individual)

### How to print QR code labels in bulk

In the **Vessel Explorer**, from the **Manage Vessels** button in the upper right, select **Print vessel QR codes**.

![How to print QR Code Labels_Bulk manage](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20QR%20Code%20Labels_Bulk%20manage.png?width=255&height=245&name=How%20to%20print%20QR%20Code%20Labels_Bulk%20manage.png)

You will be taken to an interface to select the vessels. Use the selector and the filters to select all the vessels you will be printing QR codes for. You can select multiple vessel types if desired.

Once you have your vessels selected, choose your desired label type and size. After the size is selected, you will be able to specify fields to print on the label (the number of allowable fields depends on the label size):

![How to print QR Code Labels_Print screen](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20QR%20Code%20Labels_Print%20screen.png?width=688&height=382&name=How%20to%20print%20QR%20Code%20Labels_Print%20screen.png)

#### Sheet Label Printing

If you are starting with a partial page of blank labels you can select the label locations for the first page. Lastly, rename the file if you choose to, and click the blue button at the bottom of the page to generate the labels in a PDF document to print.

This will generate one or more pages populating all your QR codes per vessel. You are then ready to print from these PDFs.

![How to print QR Code Labels_pdf](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20QR%20Code%20Labels_pdf.png?width=322&height=404&name=How%20to%20print%20QR%20Code%20Labels_pdf.png)

#### Ribbon Label Printing

InnoVint will generate a pdf of labels - 1 page per label. This pdf can be downloaded to your computer, then sent to your ribbon label printer to print.

![How to print QR Code labels_pdf roll](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20QR%20Code%20labels_pdf%20roll.png?width=221&height=476&name=How%20to%20print%20QR%20Code%20labels_pdf%20roll.png)

### How to print QR codes for individual vessels

From that vessel's detail page, in the more column in the upper right, select **Print QR code label for vessel**.

![How to print QR code labels_indvessel](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20QR%20code%20labels_indvessel.png?width=688&height=249&name=How%20to%20print%20QR%20code%20labels_indvessel.png)

This will take you to the same PDF print screen as above. Select your desired size and attributes to print. Then you are ready to print!
