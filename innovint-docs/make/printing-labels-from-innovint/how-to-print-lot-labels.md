# How to print Lot Labels

Lot labels can be printed in bulk for many lots at once or [one lot](#lot) at a time. You can choose how many labels to print per each vessel, or per each lot.

Are you trying to print QR Codes to scan your vessels? Check out the article on QR Code labels [here](https://support.innovint.us/hc/en-us/articles/115000609563-how-do-i-generate-qr-code-vessel-labels-to-print-?hsLang=en).

### To print lot labels in bulk

From the **Lot Explorer**, go to the **Manage Lots**button in the upper right, select **Print lot labels for vessels.**

**![Ho to print lot labels_bulk manage](https://support.innovint.us/hs-fs/hubfs/Ho%20to%20print%20lot%20labels_bulk%20manage.png?width=235&height=193&name=Ho%20to%20print%20lot%20labels_bulk%20manage.png)**

You will be taken to an interface to select the lots and determine the label layout.

- Use the selector and the filters to select all the lots you will be printing labels for.
- Choose the number of labels you need, either by lot or by vessel.
- Choose your desired label type and size. For more information on which labels we support, check out this [article](https://support.innovint.us/hc/en-us/articles/115000605043-which-labels-are-recommended-and-where-can-i-purchase-them-?hsLang=en).
- Based on the label size, you will be able to select specific data to print on the labels

![How to print lot labels_print screen](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20lot%20labels_print%20screen.png?width=688&height=350&name=How%20to%20print%20lot%20labels_print%20screen.png)

#### Sheet labels

If you are starting with a partial page of blank sheet labels you can select the label locations for the first page.

- **4" x 2" Labels:**
  If you are printing 4" x 2" labels, choose up to 5 attributes to include on each label (total of 6; Lot code is required).  Owners and Intended Use are available for accounts with those features enabled.
  ![How to print lot labels_large](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20lot%20labels_large.webp?width=688&height=91&name=How%20to%20print%20lot%20labels_large.webp)
- **2 5/8" x 1" Labels:**
  Choose up to 3 attributes if printing 2 5/8" x 1" labels (total of 4; Lot code is required). Owners and Intended Use are available for accounts with those features enabled.
  ![How to print lot labels_small](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20lot%20labels_small.webp?width=688&height=92&name=How%20to%20print%20lot%20labels_small.webp)

Rename the file if you choose to, and click the blue button at the bottom of the page to generate the labels in PDF.

This will generate one or more pages of labels, with the labels uncollated (all labels for the same vessel or lot will be next to each other). You can then send the PDF to your printer.

![lot label pdf](https://support.innovint.us/hs-fs/hubfs/lot%20label%20pdf.png?width=300&height=387&name=lot%20label%20pdf.png)

#### Label printer (ribbon labels)

- **2 1/4" x 1 1/4" Labels:**
  If you are printing 2 1/4" x 1 1/4" labels, choose up to 4 attributes to include on each label (total of 5; Lot code is required). Owners and Intended Use are available for accounts with those features enabled.
  ![How to print lot labels_small(5)](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20lot%20labels_small(5).webp?width=688&height=113&name=How%20to%20print%20lot%20labels_small(5).webp)
- **4" x 2 1/8" Labels:**
  Choose up to 5 attributes if printing 4" x 2 1/8" labels (total of 6; Lot code is required). Owners and Intended Use are available for accounts with those features enabled.
  ![How to print lot labels_large](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20lot%20labels_large.webp?width=688&height=91&name=How%20to%20print%20lot%20labels_large.webp)

Rename the file if you choose to, and click the blue button at the bottom of the page to generate the labels in PDF.

This will generate one or more pages of labels, with the labels uncollated (all labels for the same vessel or lot will be next to each other) You can then send the PDF to your printer.

![Lot label pdf(roll)](https://support.innovint.us/hs-fs/hubfs/Lot%20label%20pdf(roll).png?width=300&height=665&name=Lot%20label%20pdf(roll).png)

### To print lot labels for individual lots

From that lot details page, click on "More" in the top right corner and select **Print lot labels for vessels**.

![How to print lot labels_indvessel-1](https://support.innovint.us/hs-fs/hubfs/How%20to%20print%20lot%20labels_indvessel-1.png?width=688&height=147&name=How%20to%20print%20lot%20labels_indvessel-1.png)

This will take you to the PDF print screen where you can follow the same instructions as [above](#bulk).
