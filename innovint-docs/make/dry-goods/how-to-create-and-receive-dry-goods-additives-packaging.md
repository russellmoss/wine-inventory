# How to Create and Receive Dry Goods - Additives & Packaging

InnoVint manages all of the Dry Goods and Liquid Solutions that you add to your wine from fruit to bottle. This article will cover how to add additives and/or packaging into inventory via the Dry Goods Explorer:

- [Step 1: Select or create a new product](#NewProduct)
- [Step 2: Select or create a new batch](#NewBatch)
- [Step 3: Enter amount received and costs](#AmountReceived)
- [Step 4: Enter purchase and delivery details](#PurchaseDetails)
- [FAQ](#FAQ)

You may also follow along with the InnoVint Academy (a recording from our live webinar):

### Where to create Additives and Packaging

To create and receive new additive or packaging inventory, click on the blue **+Receive Additive** or **+Receive Packaging** button in the top right corner of the Dry Goods Explorer.

This will open a slideover to receive a product via a "Receive Dry Goods" action. Both Receive Dry Goods actions (for additives and packaging) operate in the same way, but additives and packaging products have slightly different fields.

You can also access this slideover by clicking on the blue "Receive more" text in the top right corner of existing [product and batch details pages](https://innovint-6865708.hs-sites.com/hc/en-us/articles/360034896692-navigating-the-additive-explorer?hsLang=en).

### Step 1: Select or Create a New Product

Select an existing additive or packaging item from the dropdown list, or, to create a new additive product, click the blue **+Add new product** button.

#### Creating a new product

![Create and Receive Dry Goods_Creating a new product](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Create%20and%20Receive%20Dry%20Goods_Creating%20a%20new%20product.webp?width=632&height=913&name=Create%20and%20Receive%20Dry%20Goods_Creating%20a%20new%20product.webp)

#### General properties

- **Product type**: Select an option from the dropdown. If you do not see a type listed, select "Custom/Other".
- **Product name**: Use the text field to enter a unique product name, something descriptive that will allow you to identify it.

> **Custom Crush Clients:** Each Additive added to an account requires a unique Product name. This system rule will prevent multiple clients at a facility from entering the same product name for an additive. (i.e. Two owners cannot both have an additive named Go-Ferm in the same facility account.) Contact an Admin at your facility for instructions on how to name your additives in InnoVint.

- (Optional) **Manufacturer**: Enter the manufacturer name as simple text. This field is optional.
- (Optional) **Default Vendor**: Select your preferred vendor for this product from the dropdown list. This field is optional and will pre-populate in your batch details when you receive the dry good. *Note: If your vendor does not appear in the dropdown list, email [support@innovint.us](mailto:support@innovint.us)and we can add it for you.*
- (Optional) **Indicators**: Select indicators for specific additive designations
  **![Create and Receive Dry Goods_indicators dropdown](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Create%20and%20Receive%20Dry%20Goods_indicators%20dropdown.webp?width=378&height=234&name=Create%20and%20Receive%20Dry%20Goods_indicators%20dropdown.webp)**
- **For additives, if your product type is Sulfur Dioxide you have additional fields to select**
  **SO2 form**:
  - Select if your product is in dry, liquid, or tablet form.
  - For Dry and Liquid SO2, enter the concentration.

#### Units

- **Inventory Unit (for additives and packaging)**: how you want to track this product in inventory.  Typically, this is the unit in which you receive a product from the manufacturer - i.e. cases or boxes for packaging, or kilograms or gallons for additives. *Please note that you cannot change this unit after creating the product.*
- **Addition Unit (additives only)**: the unit you use to add this additive to your lots and vessels . *Please note that you cannot change this unit after creating the product*.
- **Individual Item (packaging only):** the individual item that is used when packaging is applied - i.e. bottles. *Please note that you cannot change this unit after creating the product.*
- **Default number of items per inventory unit (packaging only):** default number of items per unit - i.e. 12 bottles per case

**Custom Units**: If the unit in which you measure your additive is not included in the dropdown list, select *Custom*. This automatically sets your Addition Unit to the same. Using Custom units will deactivate the Addition Calculator.

**WARNING:** The Inventory Unit, Addition Unit and Individual Item cannot be edited once a product is created. To change the Inventory Unit and/or Addition Unit, a new product will need to be created in the account.

#### Default addition rate

- (Optional) Enter a default addition rate, if applicable. This can be edited or disregarded later within the Addition action if needed. You can also leave this blank if a rate is not applicable to your additive. **Without a default addition rate, the additive calculator is deactivated.**

- (Optional) Instructions. Enter any default instructions. These will automatically display in Addition tasks and actions when working with this additive. This can be edited each time this additive is loaded into an action or task.
  ![Create and Receive Dry Goods_instructions](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Create%20and%20Receive%20Dry%20Goods_instructions.webp?width=655&height=37&name=Create%20and%20Receive%20Dry%20Goods_instructions.webp)

- (Optional) Enter any Notes that you would like to populate the Product details page. *After* the product is created, you can attach files to notes. This is a great location for SDS or Technical Data Sheets.

### Step 2: Select or create a new batch

After the product is created, you need to create batches to receive the product quantity. Multiple batches can be tracked within a single additive or packaging product.

- Select an existing batch from the dropdown list, or
- To create a new batch, click the blue **+Add new batch** button

![Create and Receive Dry Goods_Select or Create Batch](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Create%20and%20Receive%20Dry%20Goods_Select%20or%20Create%20Batch.webp?width=688&height=631&name=Create%20and%20Receive%20Dry%20Goods_Select%20or%20Create%20Batch.webp)

#### Creating a new batch

![Create and Receive Dry Goods_creating a new batch](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Create%20and%20Receive%20Dry%20Goods_creating%20a%20new%20batch.webp?width=688&height=540&name=Create%20and%20Receive%20Dry%20Goods_creating%20a%20new%20batch.webp)

- (Optional) **Default Vendor:** Select your preferred vendor for this particular batch. It automatically populates the default vendor related to the Product, but can be updated as required. *Note: If your vendor does not appear in the drop-down list, email [support@innovint.us](mailto:support@innovint.us)and we will add it for you.*
- **Mfg. batch ID:**Enter the batch ID as provided from the manufacturer.
- **Batch name:** This field will auto-fill with the Mfg. batch ID, but can be edited to a proprietary batch name. The batch name is displayed as the primary identifier in the Dry Goods Explorer and batch selection dropdown within Addition actions and work orders.
- (Optional) **Expiration date (additives only)**: The expiration date will display next to the batch name on the batch selector in Addition actions and work orders.
  ![Create and Receive Dry Goods_expiration dates](https://support.innovint.us/hs-fs/hubfs/Knowledge%20Base%20Import/Create%20and%20Receive%20Dry%20Goods_expiration%20dates.webp?width=330&height=109&name=Create%20and%20Receive%20Dry%20Goods_expiration%20dates.webp)
- (Optional) **Tags:** Enter any tags you would like to save with this batch. Tags will display beneath the on-hand quantity on the batch selector in Addition actions and work orders.
- (Optional) **Notes:** Enter any notes you would like to save to the Batch details page. *After* the batch is created, you can attach files to notes. This is a great location for SDS or Technical Data Sheets.

### Step 3: Enter amount received and costs

- **Amount or Units received:** Enter the total amount received to add to inventory. If receiving into an existing batch, this value will be added to your current on hand amount. *The amount received unit is determined by the product inventory unit.*
- (Optional) **Total cost:** Enter the cost of the inventory received (if applicable).  If this value is not known at the time that this action is submitted, it can be edited at a later time by opening and editing the Receive Dry Good action. So if you're still waiting on that invoice, you can leave this blank for now. When the action is edited, all related additions will update to reflect the change of costs. Find out more about Dry Goods and COGS Tracking [here](https://support.innovint.us/hc/en-us/cogs-and-dry-goods-tracking?hsLang=en).
- **Number of items per inventory unit (packaging only):** This defaults from the product details, but can be updated for a particular receive action .

### Step 4: Enter purchase and delivery details

- (Optional) **Vendor, PO number, Carrier**, **Driver**, and **Purchase****date**
- (Optional) **Notes**: Enter any notes you would like to save on the Receive Dry Good action.
- **Backdate action:** Backdate the action if applicable. *Note: We recommend backdating your Receive Dry Good action to before any Addition or Bottle actions will be recorded on this batch.*

### Frequently Asked Questions

**Q. What is the difference between vendors on Products, Batches, and Receive Dry Goods actions?**

*A. A default vendor can be set on both Products and Batches. The default vendor of a Batch does not need to match the default vendor of a Product. A special use case may include a scenario in which Vendor A is your primary vendor for KMBS, but on occasion you also purchase small amounts of KMBS from Vendor B. Vendor A and Vendor B might sell different batches of KMBS (ie different manufacturers and/or batch IDs), therefore you can set different default vendors for each batch separately.*

*A vendor can also be selected within a Receive Additive action. The vendor in a Receive Additive action does not need to match the vendor of the Product or Batch. A special use case may include a scenario in which you make a one-off purchase of KMBS from Vendor X. Vendor X sells the same batch of KMBS as Vendor B, which you already have in inventory. In this case, the default vendor of the batch is still Vendor B, but the vendor of this particular purchase is Vendor X.*

*Remember that vendor selections are always optional, and can be edited at any time. InnoVint provides this flexibility to allow you to better track you additive purchase history and easily reference your preferred vendors when it's time to make another order!*

*If your vendor is missing from the list, email [support@innovint.us](mailto:support@innovint.us) to get it added.*

**Q. Can I change the add rate on a product?**

*A. Yes, the default add rate can be changed on the product. Edit the addition rate via the Product details page. This will change the default add rates that populate on addition actions/tasks. You can also edit the add rate on individual addition action or tasks.*
