# Winery Search & Top Navigation Bar

The top navigation bar is always available at the top of your screen when you are logged into the InnoVint web platform. The top bar gives you quick access to search for an inventory item or create a work order, along with some other helpful options. Keep reading to learn more!

This article covers:

- [Winery dropdown](#Winery_dropdown)
- [Search winery](#Search_winery)
- [Record action](#Record_action)
- [Create work order](#Create_work_order)
- [Notifications](#Notifications)
- [Support Center and Announcements](#Support_center_and_announcements)
- [User details](#User_details)
- [and something just for fun!](#Just_for_fun)

#### Winery dropdown

If you have access to multiple wineries in InnoVint, click on the winery name to open a dropdown. Click on the winery name you want to view to switch accounts.

![Winery Search & Top Navigation Bar-dropdown](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-dropdown.webp?width=448&height=257&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-dropdown.webp)

#### Search winery

Use the winery search to quickly navigate to an inventory item. As you start typing, a dropdown will show the top 5 results that match your entered text. The winery search displays inventory that matches the following:

- Lot code or name
- Vessel code
- Vineyard name
- Block name
- Work order title
- Notes

Click on the item you want to view to navigate to the details page.

![Winery Search & Top Navigation Bar-search bar](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-search%20bar.webp?width=688&height=317&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-search%20bar.webp)

#### Record action

Click on the **Record Action** button to access the action list dropdown.
This feature is not available for winery members with Read-only winery access.

![Winery Search & Top Navigation Bar-record](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-record.gif?width=688&height=311&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-record.gif)

#### Create Work Order

Click on the **Create Work Order** button to create a new work order, or select a template.
This feature is not available for winery members with Read-only winery access.

![Winery Search & Top Navigation Bar-create wo](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-create%20wo.gif?width=688&height=228&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-create%20wo.gif)

#### Notifications

Click on the bell to view your notifications or send a [custom notification](https://support.innovint.us/hc/en-us/articles/214943863-what-are-custom-notifications-?hsLang=en) to a winery member. Learn more about receiving notifications [here](https://support.innovint.us/hc/en-us/articles/214943823-when-and-where-do-i-receive-notifications-?hsLang=en).

![Winery Search & Top Navigation Bar-notification](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-notification.webp?width=376&height=109&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-notification.webp)

#### Support Center and InnoVint Announcements

Click on the question mark to access InnoVint's online support center and a list of recent announcements from InnoVint. If you see a gift box, that means InnoVint has posted something new!

![Winery Search & Top Navigation Bar-question](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-question.webp?width=376&height=109&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-question.webp)

#### User details

Click on the user icon to view your login details, change your password, or log out of InnoVint.

![Winery Search & Top Navigation Bar-user details](https://support.innovint.us/hs-fs/hubfs/Winery%20Search%20%26%20Top%20Navigation%20Bar-user%20details.webp?width=376&height=109&name=Winery%20Search%20%26%20Top%20Navigation%20Bar-user%20details.webp)

#### Just for fun!

Customize the color of the title bar by clicking on the InnoVint icon on the far left!
